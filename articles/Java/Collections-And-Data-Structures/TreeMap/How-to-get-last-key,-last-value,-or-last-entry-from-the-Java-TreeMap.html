How to Get Last Key, Last Value, or Last Entry From the Java TreeMap

Using KeySet()

The 

          
           
KeySet()
method in Java is used for fetching the set of keys present in the map and can be used for finding the last key in the TreeMap. The code example below illustrates how [this](http://www.tutorialspoint.com/java/util/treemap_keyset.htm) can be done: 

          
           
Map<String,String> treemap = new TreeMap<String,String>();

//Adding elements to TreeMap
treemap.put("C2", "Green");
treemap.put("C1", "Red");
treemap.put("C4", "White");
treemap.put("C3", "Black");

//Printing the TreeMap (The order is based on the ascending order of the keys)
//Output : {C1=Red, C2=Green, C3=Black, C4=White}
System.out.println(treemap);

//Fetching the last key from the TreeMap
String lastKey = treemap.keySet().toArray()[treemap.size()-1].toString();

//Printing the last key
//Output : C4
System.out.println(lastKey);

Using NavigableMap.lastEntry()

The 

          
           
lastEntry()
method in Java can be used to return a key-value mapping associated with the greatest key strictly less than the given key, or null if there is no such key. The code example below illustrates how [this](http://www.tutorialspoint.com/java/util/treemap_lastentry.htm) can be done: 

          
           
Map<String,String> treemap = new TreeMap<String,String>();

//Adding elements to TreeMap
treemap.put("C2", "Green");
treemap.put("C1", "Red");
treemap.put("C4", "White");
treemap.put("C3", "Black");

//Printing the TreeMap (The order is based on the ascending order of the keys)
//Output : {C1=Red, C2=Green, C3=Black, C4=White}
System.out.println(treemap);

//Fetching the last entry from the TreeMap
Map.Entry lastEnt = ((TreeMap)treemap).lastEntry();

//Printing the last entry in the TreeMap
//Output : C4=White
System.out.println(lastEnt.getKey()+"="+lastEnt.getValue());
