How to Replace a Value for the Given Key in the TreeMap (TreeMap replace Method)

The 

          
           
TreeMap
class from the 

          
           
java.util
package provides a method, 

          
           
replace()
, which can be used to substitute a new value for the specified key in a map. This article will discuss how to use the 

          
           
TreeMap
class's 

          
           
replace()
method to achieve this goal.

Definition and Syntax of the TreeMap replace Methods

The 

          
           
TreeMap
class has two variations of the 

          
           
replace()
method: one that takes two arguments and another that takes three arguments. The definition and syntax of both versions are as follows: 
  • 
            
             
replace(K key, V value)
    : Replaces the entry for the specified key only if it is currently mapped to some value.
  • 
            
             
replace(K key, V oldValue, V newValue)
    : Replaces the entry for the specified key only if currently mapped to the specified value.

Examples of Using the TreeMap replace Method

Here are a few examples to demonstrate the usage of the 

          
           
TreeMap

          

          
           
replace()
method.

Example 1: Replace a Value for a Key in a TreeMap with the Two-argument Version

In this example, we use the two-argument version of the 

          
           
replace()
method to replace the value associated with the key "2" in the 

          
           
TreeMap
instance 

          
           
map
. 

          
           
TreeMap<String, String> map = new TreeMap<>(); 
map.put("1", "A"); 
map.put("2", "B"); 

// Replace the value associated with the key "2" in map 
System.out.println(map.replace("2", "C")); // Prints "B"
System.out.println(map.get("2")); // Prints "C"

In this example, we use the three-argument version of the 

          
           
replace()

TreeMap
instance 

          
           
map
. 

          
           
TreeMap<String, String> map = new TreeMap<>();

// Replace the value associated with the key "2" in map
