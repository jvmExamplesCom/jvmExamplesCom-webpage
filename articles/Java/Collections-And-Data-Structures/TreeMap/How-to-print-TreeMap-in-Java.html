How to Print TreeMap in Java

TreeMap is a data structure that is used to store key-value pairs of data, where the keys are always sorted and cannot contain duplicates. In this article, we will learn how to print out the elements of a TreeMap in Java.

Using the forEach() Method

The simplest way to iterate through a TreeMap is to use the forEach() method, which takes a consumer as an argument. A consumer is a functional interface that takes an argument of the same type as the TreeMap and performs some operation on it. For example, if we have a TreeMap of integers, we can use the consumer to print out each element of the TreeMap in a loop: 

          
           
TreeMap<integer,integer> treeMap = new TreeMap<>();
treeMap.put(1, 10);
treeMap.put(2, 15);
treeMap.put(3, 11);

treeMap.forEach((k,v) -> {
    System.out.println(k + " : " + v);
});</integer,integer>
