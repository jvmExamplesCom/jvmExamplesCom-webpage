How to Get TreeMap Key or TreeMap Value Using Index

Java's TreeMap offers a convenient way to store and access data, but it can sometimes be difficult to access a TreeMap key or value by index. This article will explain how to do this using the TreeMap class and the TreeMap.entrySet() method.

Using TreeMap Entry Set

The TreeMap Entry Set is a collection of all the entries in the map, where each entry is a key-value pair. It can be used to traverse the map in order to get a particular key or value from it. The following code example shows how to use the entrySet() method to get a particular key or value from the map:


          
           
TreeMap<String,String> map = new TreeMap<>();
map.put("key1", "value1");
map.put("key2", "value2");

//Get the entry from the particular index
Entry<String, String> e = map.entrySet().stream().skip(1).findFirst().get();

//Get key
String key  = e.getKey();

//Get value
String value = e.getValue();

In this example, we first create a TreeMap and add two entries to it. Then, we get the entry at the particular index (in this case, 1) using the entrySet() method. Finally, we get the key and value from the entry object.

Using the Iterator

The iterator interface of the TreeMap can also be used to retrieve a particular key or value from the map. The following code example shows how to do this:


          
           
TreeMap<String,String> map = new TreeMap<>();
map.put("key1", "value1");
map.put("key2", "value2");

//Create an iterator
Iterator<Entry<String,String>> iter = map.entrySet().iterator();

//Move the iterator to the desired index
iter.next();

//Get the entry
Entry<String,String> entry = iter.next();

//Get the key
String key = entry.getKey();

//Get the value
String value = entry.getValue();

In this example, we first create a TreeMap and add two entries to it. Then, we create an iterator for the map. We move the iterator to the desired index. Finally, we get the key and value from the entry object.

Conclusion

In this article, we have seen how to get a TreeMap key or value using index. We have seen two different ways of doing this: using the TreeMap Entry Set and the Iterator. While both methods are equally effective, the Entry Set method allows us to skip directly to the index we are interested in, while the Iterator approach requires iterating through the map until the correct index is reached.
