Java TreeMap

TreeMap is a Red-Black tree based NavigableMap implementation. It is used to store key-value pairs in a sorted order. It provides an efficient means of storing key/value pairs in sorted order, and allows rapid retrieval. It also provides guaranteed log(n) time cost for the containsKey, get, put and remove operations.

Example

Let's look at a simple example of how to use TreeMap in Java:


          
           
// Create a TreeMap 
TreeMap<String, Integer> treeMap = new TreeMap<String, Integer>(); 

// put values 
treeMap.put("A", 1); 
treeMap.put("B", 2); 
treeMap.put("C", 3); 

// Print the size of the treeMap 
System.out.println("Size of TreeMap = " + treeMap.size()); 

// Get head Map from treeMap 
SortedMap<String, Integer> headMap = treeMap.headMap("B"); 

// Print elements of headMap 
System.out.print("Elements of headMap : "); 
for (Map.Entry<String, Integer> entry : headMap.entrySet()) { 
	System.out.print(entry.getKey() + " " + entry.getValue() + "  "); 
}
The above code will produce the following output: 

          
           
Size of TreeMap = 3
Elements of headMap : A 1
