Tail Map from Java TreeMap Example

The TreeMap class extends AbstractMap and implements NavigableMap interfaces in java. The TreeMap class is a binary tree structure that stores the elements in sorted order. This article covers the concept of getting the tail map from Java TreeMap with a simple example.

Example

Let's understand this concept by an example. Consider a TreeMap collection that has 5 elements as given below:


          
           
TreeMap<Integer,String> treemap = 
        new TreeMap<Integer,String>(); 
  
// Use put() method to add elements 
// into the TreeMap 
treemap.put(2, "Two"); 
treemap.put(1, "One"); 
treemap.put(3, "Three"); 
treemap.put(6, "Six"); 
treemap.put(5, "Five");

Now, we want to get the tail map from the TreeMap collection. To do this, we will use the tailMap() method available in the TreeMap class. The syntax of the tailMap() method is given below:


          
           
SortedMap tailMap(K fromKey)

This method returns a view of the portion of the treemap whose keys are greater than or equal to fromKey. Thus, we will pass the fromKey parameter to the tailMap() method which acts as a starting point for the tail map. We can also specify the boolean inclusive parameter to the tailMap() method that indicates whether the fromKey should be included in the tail map or not.

Now, letâ€™s use the above-mentioned syntax to get the tail map from the TreeMap collection that we created above using the following statement:


          
           
SortedMap<Integer, String> tailMap = treemap.tailMap(3, true);

When we execute the above statement, we get the tail map that starts from the key 3 including it as well, as output.


          
           
{3=Three, 6=Six, 5=Five}

Therefore, the tailMap() method returned a view of the TreeMap collection where all the keys are greater than or equal to 3.
