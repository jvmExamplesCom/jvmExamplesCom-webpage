Check if a Particular Value Exists in Java Hashtable Example

Java Hashtable is an implementation of the Hashtable class which is a keyed collection. It stores key-value pairs and provides constant time complexity for lookups, insertions, and deletions. It is an ideal way to store values that need to be retrieved quickly, as it provides very efficient retrieval of associated values. This article will discuss how to check if a particular value exists in a Java Hashtable example.

Creating the Hash Table and Adding Key-Value Pairs

To begin, we will create the Hashtable and add some key-value pairs. We can do so by using the following code: 

          
           
Map<string, integer> ht = new Hashtable<>();
ht.put("One", 1);
ht.put("Two", 2);
ht.put("Three", 3);
</string,>
Here, we have created a Hashtable named “ht” and added three key-value pairs. The key is a string and the value is an integer.

Checking if a Particular Value Exists

Now that we have created the Hashtable and added our key-value pairs, we can check if a particular value exists in the Hashtable. To do this, we can use the containsValue() method on the Hashtable. This method takes a single argument - the value that we would like to check for. 

          
           
boolean exists = ht.containsValue(2);
In this example, we have checked to see if the value “2” exists in the Hashtable. The method returns true if the value exists and false if it does not.

Printing the Result

Finally, we can print out the result of our check for the existence of the value “2” in the Hashtable. 

          
           
System.out.println("Does the value 2 exist in the Hashtable? " + exists);
This will print out either “Does the value 2 exist in the Hashtable? true” or “Does the value 2 exist in the Hashtable? false” depending on whether or not the value exists in the Hashtable.

Conclusion

In this article, we discussed how to check if a particular value exists in a Java Hashtable example. We created the Hashtable, added some key-value pairs, used the containsValue() method to check if a value existed, and printed out the result. This is an important concept that can be applied to many different situations.
