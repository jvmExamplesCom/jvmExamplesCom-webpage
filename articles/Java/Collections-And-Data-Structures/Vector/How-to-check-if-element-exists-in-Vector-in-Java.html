How to Check if Element Exists in Vector in Java

Java is a platform-independent, object-oriented programming language that is used to develop a range of applications. Many developers find it an efficient and easy-to-use language to create sophisticated applications. One of the most essential tools in Java is a Vector, which is an ordered collection of data. It stores elements in contiguous memory locations, so you can quickly search for elements by their position index. In this article, we’ll discuss how to check if a particular element exists in a Vector in Java.

Using contains() Method

The simplest and most efficient way to check if an element exists in a Vector is to use the contains() method from Java. This method checks whether the specified element is present in the Vector or not. It returns true if the element exists, otherwise it returns false. 

          
           
public static boolean checkIfElementExists(Vector v1, int elem) {
    return v1.contains(elem);
}

// Create a new Vector
Vector<Integer> v1 = new Vector<>();

// Insert some elements into it
v1.add(1);
v1.add(2);
v1.add(3);

// Check if the element 2 exists in Vector
boolean result = checkIfElementExists(v1, 2);

System.out.println(result); // Output: true
In the above code, we’ve created a new Vector and added some elements to it. We then pass this Vector along with the element we want to search for to the checkIfElementExists() function. This function uses the contains method to check if the element is present in the Vector or not. If it is found, it returns true, otherwise it returns false.

Using indexOf() Method

Another way to check if an element exists in a Vector is to use the indexOf() method. It searches the Vector for the specified element and returns the index of the first occurrence of the element. If the element is not found in the Vector, it returns -1. 

          
           
public static boolean checkIfElementExists(Vector v1, int elem) {
    return v1.indexOf(elem) >= 0;
}

// Create a new Vector
Vector<Integer> v1 = new Vector<>();

// Insert some elements into it
v1.add(1);
v1.add(2);
v1.add(3);

// Check if the element 4 exists in Vector
boolean result = checkIfElementExists(v1, 4);

System.out.println(result); // Output: false
In the above code, we’ve again created a new Vector and added some elements to it. We then pass this Vector along with the element we want to search for to the checkIfElementExists() function. This function uses the indexOf() method to check if the element is present in the Vector or not. If the index of the element is greater than or equal to 0, it means the element exists, so it returns true. Otherwise, it returns false.

Using Streams API

The last way to check if an element exists in a Vector is to use the Java Streams API. This API enables you to filter, map, reduce and execute many other operations on collections. Using Streams, you can quickly check if a particular element exists in a Vector. 

          
           
public static boolean checkIfElementExists(Vector v1, int elem) {
    return v1.stream().anyMatch(x -> x == elem);
}

// Create a new Vector
Vector<Integer> v1 = new Vector<>();

// Insert some elements into it
v1.add(1);
v1.add(2);
v1.add(3);

// Check if the element 3 exists in Vector
boolean result = checkIfElementExists(v1, 3);

System.out.println(result); // Output: true
In the above code, we’ve again created a new Vector and added some elements to it. We then pass this Vector along with the element we want to search for to the checkIfElementExists() function. This function uses the Streams API to check if the element is present in the Vector or not. If the element is found, it returns true, otherwise it returns false. To summarize, if you want to check if an element exists in a Vector, you can use the contains() method, the indexOf() method or the Streams API. Each of these methods has its own advantages and disadvantages, but they are all fairly simple and straightforward ways to check if a particular element exists in a Vector.
