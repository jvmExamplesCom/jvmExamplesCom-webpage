Converting Vector to ArrayList and ArrayList to Vector in Java

Converting Vector to ArrayList

In Java, we can use the built-in function 

          
           
Collections.list()
to easily convert a vector to an ArrayList. The collection framework provides the method 

          
           
Collections.list(vector)
which returns a list object containing the elements of the given vector object in the same order. For example, if we have the code below: 

          
           
Vector<Integer> vect = new Vector<>();  
vect.add(4);  
vect.add(7);  
vect.add(2);  
vect.add(9);
We can convert it to an ArrayList using the following code: 

          
           
ArrayList<Integer> arrayList = Collections.list(vect);
The resulting output will be an ArrayList with the elements 4, 7, 2, and 9, in that order.

Converting ArrayList to Vector

To convert an ArrayList to a Vector, we can use the built-in method 

          
           
Collections.enumeration()
. The 

          
           
Collections.enumeration()
method returns an enumeration of all the elements in the specified collection. For example, if we have the following code: 

          
           
ArrayList<Integer> arrayList = new ArrayList<>();  
arrayList.add(4);  
arrayList.add(7);  
arrayList.add(2);  
arrayList.add(9);
We can convert it to a Vector using the following code: 

          
           
Vector<Integer> vect = Collections.enumeration(arrayList);
The resulting output will be a Vector with the elements 4, 7, 2, and 9, in that order.
