Add an Element at Specified Index of Java Vector Example

Java Vector class can be used to store a group of objects. It is similar to ArrayList, but the difference between them is that Vector is synchronized while ArrayList is not. Elements in a Vector object can be accessed randomly using an index number. In this article, we will look at how to add an element at a specified index of a Vector object.

Method Syntax


          
           
public void add(int index, Object element)

Example

Let's take an example to understand how this method works. We will create a Vector object and then add elements at specified indexes.


          
           
import java.util.Vector;

public class VectorExample {
  public static void main(String[] args) {
    Vector<String> vec = new Vector<String>();

    // adding elements at specified index
    vec.add(2, "apple");
    vec.add(0, "mango");
    vec.add(1, "orange");

    System.out.println("Elements are: " + vec);
  }
}
The output of the above code will be: 

          
           
Elements are: [mango, orange, apple]

As you can see, the elements have been added at the specified indexes.

Conclusion

In this article, we looked at how to add an element at specified index of a Vector object. This method is useful when you need to insert elements in a specific order in a Vector object.
