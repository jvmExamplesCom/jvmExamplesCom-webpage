How To Convert an Array to Vector In Java

Java provides a powerful data structure, the vector, which is like an array but can change size dynamically. This article will explain how to convert an array to a vector in Java.

Using the Copy Constructor

One way to convert an array to a vector is to use the copy constructor. To use this method, create a new Vector object and pass in the array as an argument to the Vector constructor. You can either pass the array directly or use the toArray() method first. The following example shows how to convert an array of type String to a Vector: 

          
           
String[] array = {"Hello", "World"};
Vector<String> vector = new Vector<>(Arrays.asList(array));
The output will be a Vector of Strings.

Using Java Streams

Java also provides a stream API that can be used to convert an array to a vector. This can be done by using the Stream.of() method and then calling the collect() method with the Collectors.toCollection() method as an argument. The Collectors.toCollection() method creates a new collection of the same type as the array. The following example shows how to use Java streams to convert an array of type Integer to a Vector: 

          
           
Integer[] array = {1, 2, 3};
Vector<Integer> vector = Arrays.stream(array)
        .collect(Collectors.toCollection(Vector::new));
The output will be a Vector of Integers.

Using the addAll() Method

Another way to convert an array to a vector is to use the addAll() method. This method takes a collection as an argument, so you can pass in an array as the argument. The following example shows how to use the addAll() method to convert an array of type Double to a Vector: 

          
           
Double[] array = {4.5, 5.6, 6.7};
Vector<Double> vector = new Vector<>();
vector.addAll(Arrays.asList(array));
The output will be a Vector of Doubles.
