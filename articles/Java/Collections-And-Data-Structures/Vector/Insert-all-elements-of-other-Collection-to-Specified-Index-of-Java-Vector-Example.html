Insert all elements of other Collection to Specified Index of Java Vector Example

Java Vector class allows us to insert all the elements of the other Collection to the specified index in Vector. We can achieve this by using addAll (int index, Collection c) method which is defined in java.util.Vector class. This method adds all the elements of the specified collection c to the vector at the specified position ‘index’. Let's have a look at an example that illustrates this concept. 

          
           
import java.util.*;
public class Demo {
    public static void main(String[] args) {
        // Create Vector object
        Vector v = new Vector(); 
        
        // Add elements to Vector
        v.add("Adil");
        v.add("Ayesha");
        
        // Print the Vector
        System.out.println("Vector: " + v);

        // Create a HashSet object
        HashSet hs = new HashSet();

        // Add elements to HashSet
        hs.add("Ismail");
        hs.add("Fawad");

        // Insert all elements of HashSet to Vector
        v.addAll(1, hs);

        // Print the Vector
        System.out.println("Vector: " + v);
    }
}
Output: 

          
           
Vector: [Adil, Ayesha]
Vector: [Adil, Ismail, Fawad, Ayesha]
In the above example, we have created an empty Vector. Then, we have added two elements to it. Before invoking the addAll() method, the Vector contains two elements. After invoking the addAll() method, the Vector contains all the elements of Vector and Collection. The syntax of addAll () method is as follows: 

          
           
public boolean addAll(int index, Collection c)
This method takes two arguments. The first argument specifies the index at which the elements of this collection are to be inserted and the second argument is the Collection from which the elements need to be added. This method returns a boolean value. If the elements are successfully added to the Vector, it returns true and if the elements are not added to the Vector, it returns false. import java.util.*;
public class Demo {
    public static void main(String[] args) {
        // Create Vector object
        Vector v = new Vector(); 
        
        // Add elements to Vector
        v.add("Adil");

        // Print the Vector

        // Create a LinkedList object
        LinkedList ll = new LinkedList();

        // Add elements to LinkedList
        ll.add("Noor");
        ll.add("Arif");

        // Insert all elements of LinkedList to Vector

        // Print the Vector
        System.out.println("Vector: " + v);
    }
}
Output: 

          
           
Vector: [Adil, Ayesha]
Vector: [Adil, Ayesha, Noor, Arif]
In this example, we have created a Vector and added two elements to it. Then, we have created a LinkedList which contains two elements. After invoking addAll () method, the Vector contains all the elements of Vector and LinkedList. Hence, we can conclude that addAll () method helps us to insert all the elements of the other Collection to the specified index in Vector.
