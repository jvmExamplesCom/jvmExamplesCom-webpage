Replacing an Element in a Vector

Vectors are a type of data structure used to store objects in Java. They are similar to arrays, but offer more flexibility in how they are used. For example, they allow you to easily add and remove elements from the beginning and end of the vector without having to shift all of the other elements. They also provide methods to get, set, and replace elements.

In this tutorial, we will learn how to replace an element at a specific index of the Vector. We will cover how to do this with both the set() and replaceAll() methods.

Using the set() Method

The set() method can be used to replace an element at a specific index of the Vector. This method takes two parameters: the index of the element to be replaced and the new element to be set. The following example shows how to use the set() method to replace the second element of an integer Vector with the value 10:


          
           
Vector<Integer> vec = new Vector<Integer>();
vec.add(5);
vec.add(7);
vec.set(1, 10);

After this code is executed, the vector will now contain the elements 5 and 10. The element at index 1 (7) has been replaced by 10.

Using the replaceAll() Method

The replaceAll() method can also be used to replace an element at a specific index of the Vector. This method takes two parameters: a Predicate to identify the elements to be replaced and a Function to specify what they should be replaced with. The following example shows how to use the replaceAll() method to replace all int values greater than 5 with the value 10:


          
           
Vector<Integer> vec = new Vector<Integer>();
vec.add(3);
vec.add(5);
vec.add(7);

vec.replaceAll(x -> x > 5, x -> 10);

After this code is executed, the vector will now contain the elements 3, 5, and 10. All of the elements greater than 5 (7) have been replaced by 10.

Conclusion

In this tutorial, we learned how to replace an element at a specific index of the Vector using the set() and replaceAll() methods. With these methods, you can easily modify your Vector when necessary.
