How to Check If Vector Is Empty in Java

In this article, we will discuss how to use the 
isEmpty()
method to determine if a Vector is empty in Java.

Vector isEmpty() Method

The 
isEmpty()
method of the Vector class determines whether the Vector is empty. The method returns 
true
if the Vector is empty and 
false
if it contains elements.

Syntax:


          
           
public boolean isEmpty()

It takes no arguments and returns a boolean value.

Example 1:


          
           
import java.util.*;

public class Main {
    public static void main(String[] args) {

        // Create an empty Vector
        Vector<integer> vec = new Vector<integer>();

        // Check if the Vector is empty
        System.out.println(vec.isEmpty());
    }
}
</integer></integer>
