Iterate through Elements Java Vector Using Iterator Example

The Java Vector class provides an iterator() method that returns an Iterator over the elements in the Vector. We can use this Iterator to loop through the Vector and perform operations on its elements. In this tutorial, we'll learn how to use the Java Vector iterator() method to loop through the Vector's elements.

Java Vector Iterator Example

Let's start by creating a simple Vector of Strings:


          
           
Vector<String> vector = new Vector<>();
vector.add("One");
vector.add("Two");
vector.add("Three");

Now, let's iterate over the Vector using the iterator() method:


          
           
Iterator<String> iterator = vector.iterator();
while (iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}
The output of the above code is: 

          
           
One
Two
Three

As we can see, we were able to loop over the Vector's elements using the iterator() method.

Iterate Java Vector in Reverse Order Using Iterator

We can also iterate over the Vector's elements in reverse order. To do this, we need to create a ListIterator from the Vector's iterator() method:


          
           
ListIterator<String> listIterator = vector.listIterator(vector.size());
while (listIterator.hasPrevious()) {
    String element = listIterator.previous();
    System.out.println(element);
}
The output of the above code is: 

          
           
Three
Two
One

As we can see, we were able to loop over the Vector's elements in reverse order using the ListIterator obtained from the Vector's iterator() method.
