Converting LinkedHashMap to Two Arrays

In Java, it is possible to convert a LinkedHashMap into two separate arrays easily. By converting the LinkedHashMap into two separate arrays, you can easily access each element of the map quickly and efficiently. In this article, we will walk you through the steps to convert a LinkedHashMap into two separate arrays using various examples.

Example 1: Using the Java Stream API

In this example, we will use the Java Stream API to convert a LinkedHashMap into two separate arrays. To do this, we will have to first create a LinkedHashMap with the data we want to convert.


          
           
LinkedHashMap<String, Integer> lMap = new LinkedHashMap<>();
lMap.put("Apple", 10);
lMap.put("Banana", 20);
lMap.put("Cherry", 30);
lMap.put("Durian", 40);

Once we have created the LinkedHashMap, we can now use the Stream API to convert it into two separate arrays. To do this, we will use the 

Collectors.toMap()
method to create two separate arrays: one containing the keys, and the other containing the values. 

          
           
Object[] keys = lMap.keySet().stream()
				.toArray(Object[]::new);

Object[] values = lMap.values().stream()
				.toArray(Object[]::new);
The resulting arrays look like this: 

          
           
keys: { Apple, Banana, Cherry, Durian }
values: { 10, 20, 30, 40 }

And that's it! We have successfully converted our LinkedHashMap into two separate arrays. This method is very efficient as it allows us to quickly and easily access each element of the map.

Example 2: Using Iteration

In this example, we will use iteration to convert a LinkedHashMap into two separate arrays. This example is a bit more involved than the previous one, but it is still relatively simple.

First, we will create a LinkedHashMap with the data we want to convert:


          
           
LinkedHashMap<String, Integer> lMap = new LinkedHashMap<>();
lMap.put("Apple", 10);
lMap.put("Banana", 20);
lMap.put("Cherry", 30);
lMap.put("Durian", 40);

Next, we will create two empty array lists to store the results of the conversion: one for the keys and one for the values.


          
           
List<Object> keysList = new ArrayList<>();
List<Object> valuesList = new ArrayList<>();

Now, we will iterate over the LinkedHashMap and add each element to its respective array list.


          
           
for (Map.Entry<String,Integer> entry : lMap.entrySet()) {
    keysList.add(entry.getKey());
    valuesList.add(entry.getValue());
}

Finally, we will convert the array lists into arrays:


          
           
Object[] keys = keysList.toArray(new Object[keysList.size()]);
Object[] values = valuesList.toArray(new Object[valuesList.size()]);
The resulting arrays look like this: 

          
           
keys: { Apple, Banana, Cherry, Durian }
values: { 10, 20, 30, 40 }

And that's it! We have successfully converted our LinkedHashMap into two separate arrays. This method is slightly more involved than the first, but it still provides us with the same result.

Conclusion

In this article, we walked you through the steps to convert a LinkedHashMap into two separate arrays in Java. We saw how to use the Stream API, as well as how to iterate over the LinkedHashMap to achieve this result. Both methods provide us with a fast and efficient way to access each element of the map.
