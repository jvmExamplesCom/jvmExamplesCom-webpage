How to Get LinkedHashMap Key by Index

Java is a popular, versatile programming language that can be used in a wide range of applications. One of the most useful features of Java is its wide range of data structures, which can be used to store and manipulate data efficiently. One such data structure is the LinkedHashMap, which stores key-value pairs and maintains an order for them. In this article, we'll take a look at how to get the key of a particular item in a LinkedHashMap using its index.

Method 1: Using forEach()

The forEach() method of LinkedHashMap allows us to iterate over the elements of the map and perform some action on each element. We can use this method to find a particular key based on its index. Consider the following example where we've stored some sample data in a LinkedHashMap: 

          
           
LinkedHashMap<Integer, String> numbers = new LinkedHashMap<>();
numbers.put(1, "one");
numbers.put(2, "two");
numbers.put(3, "three");
numbers.put(4, "four");
numbers.put(5, "five");
Now, we can use this forEach() method to iterate over the elements and find the key for a given index. For example, to find the key for the third element (i.e., the one with the index 2), we can do the following: 

          
           
int key = 0;
int count = 0;
numbers.forEach((k, v) -> {
if (count == 2) {
    key = k;
}
count++;
});

System.out.println("The key for the third element is " + key);
When we run this code, we get the following output: 

          
           
The key for the third element is 3

Method 2: Using entrySet()

Another way to get the key for a particular index in a LinkedHashMap is to use the entrySet() method. This method returns a set of all the entries in the map, and we can use this to iterate over the elements and find the desired key. Let's say we want to find the key for the fourth element (with the index 3). We can do this as follows: 

          
           
int key = 0;
int count = 0;
for (Map.Entry<Integer, String> entry : numbers.entrySet()) {
if (count == 3) {
    key = entry.getKey();
}
count++;
}

System.out.println("The key for the fourth element is " + key);
When we run this code, we get the following output: 

          
           
The key for the fourth element is 4

Conclusion

In this article, we took a look at how to get the key of a particular item in a LinkedHashMap using its index. We saw two different ways of doing this: using the forEach() method and using the entrySet() method. Try out these methods in your own projects and see how they work for you!
