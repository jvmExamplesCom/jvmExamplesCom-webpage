Print LinkedHashMap in Java

Printing a LinkedHashMap in Java is an essential task when coding, as it allows you to see the contents of the data structure at any time. This article will explain how to print a LinkedHashMap in both formats: console and HTML.

Print LinkedHashMap in Console

The following is an example of how to print a LinkedHashMap in the console using the entrySet() method. We'll assume that the LinkedHashMap contains String keys and values. 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<String, String>(); 
map.put("key1", "value1");
map.put("key2", "value2");


for (Map.Entry<String, String> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}
Output: 

          
           
key1 = value1
key2 = value2

Print LinkedHashMap in HTML

If you need to display the contents of a LinkedHashMap in HTML format, the following code snippet shows how to do it using the entrySet() method. 

          
           
LinkedHashMap<String, String> map = new LinkedHashMap<String, String>(); 
map.put("key1", "value1");
map.put("key2", "value2");

System.out.println("<ul>");
for (Map.Entry<String, String> entry : map.entrySet()) {
    System.out.println("<li>" + entry.getKey() + " = " + entry.getValue() + "</li>");
}
System.out.println("</ul>");
Output: 

          
           
<ul>
<li>key1 = value1</li>
<li>key2 = value2</li>
</ul>
This will create an unordered list of the LinkedHashMap's key-value pairs.

Conclusion

Printing a LinkedHashMap in Java is a simple task that can be accomplished by using the entrySet() method. Whether printing in console format or HTML format, the code snippets above should be enough to get you started.
