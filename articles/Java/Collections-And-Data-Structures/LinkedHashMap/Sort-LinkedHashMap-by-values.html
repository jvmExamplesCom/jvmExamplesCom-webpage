Sorting LinkedHashMap by Values in Java

LinkedHashMap is one of the most frequently used implementations of the Java Map interface. It stores key-value mappings in a way that preserves the insertion order of the elements. A LinkedHashMap object also supports sorting of values if desired. In this article, we will look at how to sort a LinkedHashMap object by values in Java. We will discuss two different approaches for achieving this.

Using Comparator and TreeMap

The comparator interface provides the compare(Object o1, Object o2) method for comparing two objects. This method returns an integer value which can be negative, zero or positive depending on whether o1 is less than, equal to, or greater than o2 respectively. We can use this compare() method to define a custom comparator for our LinkedHashMap object. This comparator can then be used to create a TreeMap which sorts the LinkedHashMap by its values. The following example illustrates this approach: 

          
           
// Create a LinkedHashMap
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();
map.put("Alpha", 1);
map.put("Beta", 3);
map.put("Gamma", 2);

// Define a comparator
Comparator<String> comparator = new Comparator<String>(){
  @Override
  public int compare(String s1, String s2){
    return map.get(s2).compareTo(map.get(s1));
  }
};

// Use the comparator to create a TreeMap
TreeMap<String, Integer> treeMap = new TreeMap<>(comparator);
treeMap.putAll(map);

// Print the sorted map
System.out.println(treeMap);
The above code will print the following output: 

          
           
{Beta=3, Gamma=2, Alpha=1}
As we can see, the LinkedHashMap has been successfully sorted by its values.

Using Stream API

The Stream API was introduced in Java 8 and it can be used to process collections of data in a declarative manner. We can use the Stream API to sort a LinkedHashMap object by its values as shown in the following example: 

          
           
// Create a LinkedHashMap
LinkedHashMap<String, Integer> map = new LinkedHashMap<>();
map.put("Alpha", 1);
map.put("Beta", 3);
map.put("Gamma", 2);

// Sort the LinkedHashMap by its values
map = map.entrySet().stream()
            .sorted(Map.Entry.comparingByValue())
            .collect(Collectors.toMap(
                Map.Entry::getKey,
                Map.Entry::getValue, 
                (e1, e2) -> e1,
                LinkedHashMap::new
            ));

// Print the sorted map
System.out.println(map);
The above code will produce the same output as the previous approach: 

          
           
{Alpha=1, Gamma=2, Beta=3}
Conclusion In this article, we looked at how to sort a LinkedHashMap object by its values in Java. We discussed two different approaches for achieving this: one using Comparator and TreeMap and the other using Stream API.
