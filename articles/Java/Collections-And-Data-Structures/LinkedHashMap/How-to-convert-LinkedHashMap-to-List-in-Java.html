How to Convert LinkedHashMap to List in Java

Converting a LinkedHashMap object to a List object can be useful in many situations. LinkedHashMap objects contain the same underlying data as List objects, but LinkedHashMap objects are able to access and manipulate the data much faster. The following section will give a detailed description of how to convert LinkedHashMap to List in Java.

Using Java 8 Stream API

The easiest way to convert a LinkedHashMap to List is to use Java 8 Stream API. This approach allows us to convert the LinkedHashMap entries into a List in a single line of code. Here’s an example: 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(1, “one”);
map.put(2, “two”);
map.put(3, “three”);

List<String> list = map.entrySet().stream()
    .map(Map.Entry::getValue)
    .collect(Collectors.toList());
In the above example, we first create a LinkedHashMap with some key-value pairs. Then we use the stream API to map the values from the LinkedHashMap entries to a new List. Finally, we use the collect() method to collect the mapped values into a List.

Using For-Each Loop

Another approach for converting a LinkedHashMap to List is to use a for-each loop. This type of loop iterates over each entry in the LinkedHashMap and adds the values to a List. Here’s an example: 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(1, “one”);
map.put(2, “two”);
map.put(3, “three”);

List<String> list = new ArrayList<>();
for (Map.Entry<Integer, String> entry : map.entrySet()) {
    list.add(entry.getValue());
}
In the above example, we first create a LinkedHashMap with some key-value pairs. Then we iterate over each entry in the map and add the value associated with the entry to the List.

Using Apache Commons Collections

If you have Apache Commons Collections library installed, you can use its CollectionUtils.removeDuplicates() method to convert a LinkedHashMap to List. Here’s an example: 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(1, “one”);
map.put(2, “two”);
map.put(3, “three”);

List<String> list = new ArrayList<>(CollectionUtils.removeDuplicates(map.values()));
In the above example, we first create a LinkedHashMap with some key-value pairs. Then we pass the collection of values from the LinkedHashMap to the removeDuplicates() method. This method will return a collection of unique values which we can then store in a List.

Conclusion

In this tutorial, we discussed three different approaches for converting a LinkedHashMap to List in Java.
