Convert Set to Array (Convert HashSet to Array)

In Java, a Set is an interface used to represent a collection of unique elements. The HashSet class implements the Set interface, and it is useful in a variety of scenarios. There are scenarios where one may need to convert a HashSet to an array. In this article we will look at a few different approaches to accomplish this task.

Using the toArray() Method

The Set interface has a method called 

          
           
toArray()
which can be used to convert a set to an array. This method takes an array as an argument and returns an array containing the elements of the Set. 

          
           
HashSet<Integer> hashSet = new HashSet<>();
hashSet.add(1);
hashSet.add(2);
hashSet.add(3);

Integer[] array = hashSet.toArray(new Integer[hashSet.size()]);

for (int i = 0; i < array.length; i++) {
    System.out.println(array[i]);
}
In the code above, we create a HashSet and add three elements to it. We then use the 

          
           
toArray()
method to convert the HashSet to an array. The argument we pass to the 

          
           
toArray()
method is an array of type 

          
           
Integer
with the same size as the set. After the conversion is complete, we iterate through the array and print its contents. The output of this code is: 

          
           
1
2
3

Using the for-each Loop

Another way of converting a HashSet to an array is using the 

          
           
forEach()
method of the Set interface. This method takes a Consumer object as an argument and iterates over the set, executing the specified code for each element. 

          
           
HashSet<Integer> hashSet = new HashSet<>();
hashSet.add(1);
hashSet.add(2);
hashSet.add(3);

Integer[] array = new Integer[hashSet.size()];
int index = 0;

hashSet.forEach(element -> {
    array[index] = element;
    index++;
});

for (int i = 0; i < array.length; i++) {
    System.out.println(array[i]);
}
In the code above, we create a HashSet and add three elements to it. We then use the 

          
           
forEach()
method to iterate over the set and add each element to the array. After the loop is complete, we iterate through the array and print its contents. The output of this code is: 

          
           
1
2
3

Using the Stream API

The Java 8 Stream API offers another way to convert a HashSet to an array. The 

          
           
stream()
method of the Set interface can be used to create an IntStream of the set elements, and the 

          
           
toArray()
method of the IntStream can be used to convert the stream to an array. 

          
           
HashSet<Integer> hashSet = new HashSet<>();
hashSet.add(1);
hashSet.add(2);
hashSet.add(3);

int[] array = hashSet.stream().mapToInt(Integer::intValue).toArray();

for (int i = 0; i < array.length; i++) {
    System.out.println(array[i]);
}
In the code above, we create a HashSet and add three elements to it. We then use the 

          
           
stream()
method to create a stream of the set elements and the 

          
           
mapToInt()
and 

          
           
toArray()
methods to convert the stream to an array. After the conversion is complete, we iterate through the array and print its contents. The output of this code is: 

          
           
1
2
3
In this article, we looked at a few different ways to convert a HashSet to an array. The 

          
           
toArray()
method of the Set interface, the 

          
           
forEach()
method and the Java 8 Stream API can all be used to accomplish this task.
