Getting Random Elements from a LinkedHashSet in Java

In this article, we will explore the process for retrieving random elements from a LinkedHashSet in Java. We will be looking at two core methods of achieving this; one utilising the “Random” class and another that takes advantage of the LinkedHashSet’s iterator.

Retrieving Random Element Using the “Random” Class

The most straightforward way of obtaining a random element from a LinkedHashSet object is to use the Random class. The Random class provides a range of methods for producing random values, including generating double and integer data. The sequence of random numbers produced by the Random class is determined by the seed value supplied when creating the Random object. Once the Random object has been created, we can then use it to provide random indices for our LinkedHashSet: 

          
           
LinkedHashSet<Object> mySet = new LinkedHashSet<>();
// Add elements to the set

Random rand = new Random();
int randomIndex = rand.nextInt(mySet.size());
Object randomElement = (Object) mySet.toArray()[randomIndex];
In the above code example, the Random instance is used to generate an integer between 0 (inclusive) and the size of the mySet (exclusive). This number is then used to select an element at random from the array returned by the LinkedHashSet’s “toArray()” method.

Retrieving a Random Element Using the Iterator

Alternatively, we can make use of the LinkedHashSet’s iterator to retrieve a random element from the collection. To do this, we first create an iterator over the LinkedHashSet, and then generate a random index, using the same approach as before: 

          
           
LinkedHashSet<Object> mySet = new LinkedHashSet<>();
// Add elements to the set

Random rand = new Random();
int randomIndex = rand.nextInt(mySet.size());

Iterator<Object> iter = mySet.iterator();
Object randomElement = null;

for (int i = 0; i < randomIndex; i++) {
    randomElement = iter.next();
}
In this example, the iterator is used to traverse the set until it reaches the random index generated beforehand, returning the element at this index.

Conclusion

In this article, we have explored two methods for obtaining random elements from a LinkedHashSet in Java. The first approach involves using the Random class to generate a random index, which is then used to select an element from the array returned by the “toArray()” method. The second approach makes use of the LinkedHashSet’s iterator to traverse the collection until it reaches the target index.
