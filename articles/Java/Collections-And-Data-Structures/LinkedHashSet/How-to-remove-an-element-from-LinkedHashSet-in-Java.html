How to Remove an Element from LinkedHashSet in Java

LinkedHashSet is a part of the Java Collections Framework which implements the Set interface. It is a data structure that provides constant-time insertion and removal of elements while preserving the order of the elements in the set. The following tutorial goes over how to remove an element from a LinkedHashSet in Java.

Removing Elements With the remove() Method

The simplest way to remove an element from a LinkedHashSet is to use the remove() method. This method accepts an element as an argument and removes it from the set if it is present. 

          
           
LinkedHashSet<String> mySet = new LinkedHashSet<String>();
mySet.add("Hello");
mySet.add("World");

if (mySet.remove("Hello")) {
    System.out.println("Element removed");
}
In this example, we create a LinkedHashSet and add two elements to it. Then, we use the remove() method to try to remove the element “Hello”. If the element is present in the set, it will be removed and we will print out “Element removed”.

Iterating Over the Set to Find the Element

If you don’t know the exact element that you want to remove, you can iterate over the entire set to find it and then remove it. This approach is more expensive than using the remove() method, since it involves more operations. 

          
           
LinkedHashSet<String> mySet = new LinkedHashSet<String>();
mySet.add("Hello");
mySet.add("World");

String elementToRemove = "Hello";

Iterator<String> iterator = mySet.iterator();
while (iterator.hasNext()) {
    String element = iterator.next();
    if (element.equals(elementToRemove)) {
        iterator.remove();
    }
}
In this example, we create a LinkedHashSet and add two elements to it. Then, we create an iterator for the set and loop over all the elements. If the current element is equal to the element that we want to remove, we call the iterator’s remove() method to remove it from the set.

Conclusion

In this tutorial, we went over two different ways to remove an element from a LinkedHashSet in Java. We saw that the simplest way is to use the remove() method, which accepts an element as an argument and removes it from the set if it is present. We also saw how to iterate over the entire set and remove the desired element, although this approach is more expensive.
