How to Iterate LinkedHashSet Elements in Java

LinkedHashSet is a type of HashSet which maintains the order of elements it stores. LinkedHashSet can be iterated in multiple ways. In this article we will explore different ways of iterating through its elements.

Using For Loop

The simplest way of iterating over the elements of a LinkedHashSet is by using a for loop. The following code demonstrates this. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();
linkedHashSet.add("Hello");
linkedHashSet.add("World");

for (String s : linkedHashSet) {
    System.out.println(s);
}
The output of this code is: 

          
           
Hello
World

Using Iterator

Another way to iterate over LinkedHashSet elements is using Iterator. The code for this is given below. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();
linkedHashSet.add("Hello");
linkedHashSet.add("World");

Iterator<String> iterator = linkedHashSet.iterator();
while (iterator.hasNext()) {
    String s = iterator.next();
    System.out.println(s);
}
For the above code the output will be same as before, 

          
           
Hello
World

Using Lambda Expressions

Java 8 also provides a way to iterate over LinkedHashSet elements using Lambda Expressions. The code is given below. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();
linkedHashSet.add("Hello");
linkedHashSet.add("World");

linkedHashSet.forEach(s -> System.out.println(s));
The output of this code will also be same as before, 

          
           
Hello
World

Using forEachRemaining() Method

LinkedHashSet implements Iterable interface, so it also has a forEachRemaining() method. Here is a code snippet that shows how to use this method. 

          
           
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<String>();
linkedHashSet.add("Hello");
linkedHashSet.add("World");

Iterator<String> iterator = linkedHashSet.iterator();
iterator.forEachRemaining(s -> System.out.println(s));
The output of this code is: 

          
           
Hello
World
