Get Size of Java LinkedHashSet Example

In this tutorial, we will learn how to determine the size of a Java LinkedHashSet. We will also look at various code examples to illustrate the size of a LinkedHashSet.

LinkedHashSet Class in Java

The LinkedHashSet class is an implementation of the Set interface which uses a doubly-linked list to store elements in a particular order. This means that when iterating through a LinkedHashSet, the elements will be in the same order they were inserted in. This class offers constant time performance for the basic operations like add, remove, contains and size.

Size of a LinkedHashSet in Java

The size of a LinkedHashSet can be found using the method size(). This method returns the number of elements currently in the LinkedHashSet.

Example 1: Get Size of a LinkedHashSet

In this example, we will create a LinkedHashSet and then use the size() method to determine its size: 

          
           
// Create a LinkedHashSet
LinkedHashSet<String> set = new LinkedHashSet<>();
 
// Add elements to the LinkedHashSet
set.add("Java");
set.add("Python");
set.add("C++");
 
// Get the size of the LinkedHashSet
int size = set.size();
 
System.out.println("Size of LinkedHashSet: " + size);
Output: 

          
           
Size of LinkedHashSet: 3

Example 2: Empty LinkedHashSet

In this example, we will create an empty LinkedHashSet and then use the size() method to determine its size: 

          
           
// Create an empty LinkedHashSet
LinkedHashSet<String> set = new LinkedHashSet<>();
 
// Get the size of the LinkedHashSet
int size = set.size();
 
System.out.println("Size of LinkedHashSet: " + size);
Output: 

          
           
Size of LinkedHashSet: 0

Example 3: Check LinkedHashSet Is Empty

We can also use the isEmpty() method to check if a LinkedHashSet is empty. This method returns true if the LinkedHashSet is empty and false otherwise: 

          
           
// Create an empty LinkedHashSet
LinkedHashSet<String> set = new LinkedHashSet<>();
 
// Check if the LinkedHashSet is empty
boolean isEmpty = set.isEmpty();
 
System.out.println("Is LinkedHashSet empty? " + isEmpty);
Output: 

          
           
Is LinkedHashSet empty? true

Conclusion

In this tutorial, we looked at how to determine the size of a Java LinkedHashSet. We used the size() and isEmpty() methods to get the size of a LinkedHashSet and to check whether a LinkedHashSet is empty.
