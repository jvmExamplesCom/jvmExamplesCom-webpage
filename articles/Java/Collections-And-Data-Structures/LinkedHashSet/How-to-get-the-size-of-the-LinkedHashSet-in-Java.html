How to Get the Size of a LinkedHashSet in Java

Java’s LinkedHashSet data structure is a hybrid of the HashSet and the LinkedList. A LinkedHashSet is an ordered version of a HashSet which maintains a doubly-linked list running through all of its entries. This makes it useful for tasks such as retrieving elements in the insertion-order. In this article, we’ll discuss how to get the size of a LinkedHashSet in Java. We’ll also review various ways of achieving this.

Getting the Size of a LinkedHashSet

The first way to get the size of a LinkedHashSet is to use the size() method of the LinkedHashSet class. This method returns an integer value representing the number of elements in the set. Here’s an example: 

          
           
LinkedHashSet<Integer> numbers = new LinkedHashSet<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);

int numOfElements = numbers.size();
System.out.println("Size of the set is: " + numOfElements);
In the example above, we’ve created a LinkedHashSet called ‘numbers’. Using the size() method, we’re able to retrieve the number of elements in the set. The output of the code will be: 

          
           
Size of the set is: 3
The second way to get the size of a LinkedHashSet is to use the isEmpty() method of the LinkedHashSet class. This method returns a boolean value (true or false) indicating whether or not the LinkedHashSet is empty or not. It’s useful when we want to know if the LinkedHashSet is empty before attempting to make a size comparison. Here’s an example: 

          
           
LinkedHashSet<String> names = new LinkedHashSet<>();
boolean isEmpty = names.isEmpty();
if (isEmpty) {
    System.out.println("Set is empty!");
} else {
    System.out.println("Set is not empty!");
}
In the example above, we’ve created a LinkedHashSet called ‘names’. We’ve then used the isEmpty() method to check whether the set is empty or not. The output of the code will be: 

          
           
Set is empty!
The third way to get the size of a LinkedHashSet is to use the iterator() method of the LinkedHashSet class. This method returns an Iterator object which can be used to iterate through the elements of the LinkedHashSet. By counting the number of elements visited during the iteration, we can determine the size of the LinkedHashSet. Here’s an example: 

          
           
LinkedHashSet<Character> characters = new LinkedHashSet<>();
characters.add('a');
characters.add('b');
characters.add('c');

Iterator iterator = characters.iterator();
int count = 0;
while(iterator.hasNext()) {
    count++;
    iterator.next();
}
System.out.println("Size of the set is: " + count);
In the example above, we’ve created a LinkedHashSet called ‘characters’. Using the iterator() method, we’ve retrieved an Iterator object. We’ve then used the Iterator to iterate through the elements of the set and count them. The output of the code will be: 

          
           
Size of the set is: 3
