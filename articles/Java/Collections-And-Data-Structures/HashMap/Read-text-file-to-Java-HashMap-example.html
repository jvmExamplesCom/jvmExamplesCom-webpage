Read Text File to Java HashMap Example

Reading a text file into a 
HashMap
in Java is a convenient and efficient way to store data. In this example, we will demonstrate how to read a text file into a 
HashMap
using the 
BufferedReader
class. We will also illustrate how to display the keys and values of the 
HashMap
.

Example Code


          
           
// Create a Map object to store entries from the text file
HashMap<String, String> myHashMap = new HashMap<String, String>();

// BufferedReader to read the contents of the text file
BufferedReader br = new BufferedReader(new FileReader("example.txt"));

// line is used to store each line read from the text file
String line;
while((line = br.readLine()) != null) {
    // Use the split() method of String to divide each line into two strings
    String[] entry = line.split("\\s+");
    // Store key and value in the map
    myHashMap.put(entry[0], entry[1]);
}

// Get the entries of the HashMap
Set<Map.Entry<String,String>> entries = myHashMap.entrySet();

// Iterate over the entries
for (Map.Entry<String, String> entry : entries) {
    System.out.println("Key = " + entry.getKey() + ", Value = " + entry.getValue());
}

Expected Output


          
           
Key = key1, Value = value1
Key = key2, Value = value2
Key = key3, Value = value3

Conclusion

In this example, we demonstrated how to read a text file into a 

HashMap
in Java and display the keys and values of the 
HashMap
. We used the 
BufferedReader
class to read the contents of the text file and the 
split()
method of 
String
to create two strings. Finally, we used the 
put()
method of 
HashMap
to store the key and value.
