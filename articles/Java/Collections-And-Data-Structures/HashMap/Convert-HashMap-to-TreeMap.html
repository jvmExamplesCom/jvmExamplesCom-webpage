Converting HashMap to TreeMap in Java

Java provides the Java Collections Framework, which contains classes that store and manipulate groups of objects. Two of the most commonly used collections are the 

          
           
HashMap
and 

          
           
TreeMap
. A 

          
           
HashMap
is implemented based on hashing and provides fast random access to elements in its collection. On the other hand, a 

          
           
TreeMap
is an implementation of a Red-Black Tree, and provides an ordered view of its data, allowing for fast retrieval and sorting of elements in the collection. It is possible to convert an existing 

          
           
HashMap
into a 

          
           
TreeMap
in Java. This article explores the process of doing so.

Creating a HashMap

Let's start by creating a 

          
           
HashMap
called 

          
           
hashmap
, and adding some elements to it. 

          
           
// Create a HashMap 
Map<String, Integer> hashmap = new HashMap<>();  

// Adding elements to the HashMap  
hashmap.put("one", 1); 
hashmap.put("two", 2); 
hashmap.put("three", 3);  
hashmap.put("four", 4);  
hashmap.put("five", 5);

Creating a TreeMap

We can now create a 

          
           
TreeMap
using the 

          
           
hashmap
we have created: 

          
           
// Create a TreeMap from HashMap 
Map<String, Integer> treemap = new TreeMap<String, Integer>(hashmap);

Sorting the TreeMap

The elements in the 

          
           
TreeMap
will be sorted in natural order. To be able to sort them in a different order, we need to provide a 

          
           
Comparator
object to the constructor at the time of creation. Let's see how this is done by creating a 

          
           
TreeMap
that is sorted by the values instead of the keys. To do so, we will define a 

          
           
Comparator
object that compares two 

          
           
Map.Entry
objects based on their values: 

          
           
// Define a Comparator that sorts Map.Entry<K,V> objects by value
Comparator<Map.Entry<String, Integer>> comparator = Comparator.comparing(Map.Entry::getValue);
We can then use the 

          
           
Comparator
to create a 

          
           
TreeMap
: 

          
           
// Use the Comparator to create a TreeMap
Map<String, Integer> treemap = new TreeMap<String, Integer>(comparator); 

// Put all elements from the HashMap to the TreeMap
treemap.putAll(hashmap);

Conclusion

In this article, we looked at how to convert an existing 

          
           
HashMap
into a 

          
           
TreeMap
in Java. We explored how to create a 

          
           
TreeMap
using either a natural ordering of the keys or a custom ordering of the values. We also saw how to provide a 

          
           
Comparator
object to the 

          
           
TreeMap
constructor to sort the elements by values.
