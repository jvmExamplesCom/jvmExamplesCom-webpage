Removing Values from Java HashMap

Java HashMap is a data structure used to store and retrieve data based on a key/value pair. It provides an efficient way to access and modify data, making it a popular choice for many applications.

In this article, we will explore how to remove values from a Java HashMap using various methods.

Removing All Key/Value Pairs From A HashMap

The easiest way to remove all the key/value pairs from a HashMap is to use the clear() method.


          
           
HashMap<String, Integer> numbers = new HashMap<>();

numbers.put("one", 1);
numbers.put("two", 2);
numbers.put("three", 3);

// Clear all key/value pairs
numbers.clear();

Removing A Single Key/Value Pair From A HashMap

To remove a single key/value pair from a HashMap, you can use the remove() method.


          
           
HashMap<String, Integer> numbers = new HashMap<>();

numbers.put("one", 1);
numbers.put("two", 2);

// Remove "two" key/value pair
numbers.remove("two");

Removing Key/Value Pairs From A HashMap By Value

If you need to remove one or more key/value pairs from a HashMap based on their values, you can use the entrySet() method and the removeIf() method.


          
           
HashMap<String, Integer> numbers = new HashMap<>();

numbers.put("one", 1);
numbers.put("two", 2);
numbers.put("three", 3);

// Remove entries whose value is greater than 2
numbers.entrySet().removeIf(entry -> entry.getValue() > 2);

Conclusion

In this article, we explored how to remove values from a Java HashMap using various methods. We looked at how to remove all the key/value pairs from a HashMap, how to remove a single key/value pair from a HashMap, and how to remove key/value pairs from a HashMap by value.
