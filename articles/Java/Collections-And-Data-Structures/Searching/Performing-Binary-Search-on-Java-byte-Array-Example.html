Performing Binary Search on Java byte Array Example

Binary search is an efficient algorithm that searches for a specified element in a sorted array. It works by comparing the middle element of the array with the target value, and either eliminating the lower or upper half of the array, depending on whether the target element is greater or lesser than the middle element. This process is repeated until the element is found or the search space is exhausted. The following example shows how to perform binary search on a Java byte array.

Example

We will define a byte array, and create a method to perform binary search on the array. The method takes three arguments: a byte array, the target element, and the start index. The method returns the index of the target element if it is found; otherwise, it returns -1.


          
           
public static int binarySearch(byte[] arr, byte b, int start) {
    int low = start;
    int high = arr.length - 1;
    
    while (low <= high) {
        int mid = (low + high) / 2;
        byte midVal = arr[mid];
        
        if (b == midVal) {
            return mid; //target element found
        } else if (b < midVal) {
            high = mid - 1;
        } else { 
            low = mid + 1;
        }
    }
    return -1; //element not found
}

The following code instantiates a byte array and calls the binary search method:


          
           
byte[] arr = { 1, 3, 5, 7, 9 };
int idx = binarySearch(arr, 5, 0);
System.out.println(idx); // prints 2
The output of the program is: 

          
           
2

In this example, the binary search finds the target element at index 2.
