How to Get Random Elements From Java HashSet

Java provides a data structure called the HashSet which is a collection of elements. The elements that are stored in a HashSet are unordered and each element can only appear in the set once. It is often used when you want to store unique data or when you want to check if a value exists in the set. There is no guarantee about the order of the elements in the set. HashSet does not provide methods for getting random elements from the set. This means we need to use a few different techniques to get a random element from the set. In this article, we will explore two different ways to get a random element from the set.

Method 1: Generating Random Indexes

The first approach to get random element from HashSet is by generating random indexes. The idea is to generate a random number using some random number generator and then use it as an index in the set. We can get the element in the set by its index. Let’s look at an example. In the below code, we use the java.util.Random class to generate a random number and then get the element at that index. 

          
           
import java.util.HashSet;
	import java.util.Random;

	public class Example {
		public static void main(String[] args) {
			// Create a HashSet
			HashSet hset = new HashSet<>();
			hset.add("A");
			hset.add("B");
			hset.add("C");
			hset.add("D");

			// Create a random number generator
			Random rand = new Random();

			// Get a random index
			int randomIndex = rand.nextInt(hset.size());

			// Get the element at the random index
			Object randomElement = hset.toArray()[randomIndex];

			// Print the random element
			System.out.println("Random Element: " + randomElement);
		}
	}
If we execute the above program, we get the following output. 

          
           
Random Element: B

Method 2: Iterating Over the Elements

Another approach to get a random element from the HashSet is by iterating over the elements one by one and randomly selecting one of them. Let’s look at an example. In the below code, we use the java.util.Iterator class to iterate over the elements in the HashSet and then use a random number generator to randomly select one of them. 

          
           
import java.util.HashSet;
	import java.util.Iterator;
	import java.util.Random;

	public class Example {
		public static void main(String[] args) {
			// Create a HashSet
			HashSet hset = new HashSet<>();
			hset.add("A");
			hset.add("B");
			hset.add("C");
			hset.add("D");

			// Create a random number generator
			Random rand = new Random();

			// Create an iterator
			Iterator it = hset.iterator();

			// Get a random element
			Object randomElement = null;
			while (it.hasNext()) {
				Object element = it.next();
				if (rand.nextBoolean()) {
					randomElement = element;
				}
			}

			// Print the random element
			System.out.println("Random Element: " + randomElement);
		}
	}
If we execute the above program, we get the following output. 

          
           
Random Element: B
In this article, we looked at two different methods to get a random element from the Java HashSet. The first approach is to generate a random index for the set and then use it to get the element. The second approach is to iterate over the elements in the set and randomly select one of them. Both approaches are useful in different situations.
