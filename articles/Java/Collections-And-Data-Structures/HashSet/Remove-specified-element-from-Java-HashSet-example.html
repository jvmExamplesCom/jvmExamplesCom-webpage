Java HashSet Remove Element

A HashSet is a type of Collection used to store unique elements. It is an unordered and unsorted data structure in Java. The remove() method is used to remove a particular element from the HashSet.

Remove specified element from Java HashSet example

The remove() method takes an argument of the specified element to be removed from the HashSet. If the element is present, then the element is removed. The following example shows how to remove a specified element from a Java HashSet.


          
           
import java.util.HashSet;

public class Example {
   public static void main(String[] args) {
      HashSet<String> hSet = new HashSet<String>();
      //add elements to HashSet
      hSet.add("Hello");
      hSet.add("World");
      hSet.add("This");
      hSet.add("is");
      hSet.add("a");
      hSet.add("test");
      
      System.out.println("HashSet before remove: " + hSet);
      //remove element
      hSet.remove("World");

      System.out.println("HashSet after remove: " + hSet);
   }
}
The output of the above program will be: 

          
           
HashSet before remove: [This, a, test, Hello, World, is]
HashSet after remove: [This, a, test, Hello, is]
