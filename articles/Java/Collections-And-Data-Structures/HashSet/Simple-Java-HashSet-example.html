Java HashSet example

The Java HashSet is an implementation of the Set interface. It uses a hash table to store the elements. The elements are not ordered. A HashSet does not contain any duplicate elements. The following example shows how to use the HashSet in Java.

Creating a HashSet

To create a HashSet in Java, we can use the following syntax:


          
           
HashSet<String> myHashSet = new HashSet<String>();

Adding Elements to the HashSet

To add elements to the HashSet, we can use the 
add()
method, as follows: 

          
           
myHashSet.add("Apple");
myHashSet.add("Banana");
myHashSet.add("Cherry");
We can also use the 
addAll()
method to add multiple elements to the HashSet: 

          
           
ArrayList<String> fruits = new ArrayList<String>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Cherry");

myHashSet.addAll(fruits);

Removing Elements from the HashSet

To remove an element from the HashSet, we can use the 

remove()
method, as follows: 

          
           
myHashSet.remove("Apple");
We can also use the 
removeAll()
method to remove multiple elements from the HashSet: 

          
           
ArrayList<String> fruits = new ArrayList<String>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Cherry");

myHashSet.removeAll(fruits);

Iterating Over the Elements of a HashSet

We can iterate over the elements of a HashSet using an 

Iterator
: 

          
           
Iterator<String> iterator = myHashSet.iterator();
while(iterator.hasNext()) {
    System.out.println(iterator.next());
}

Checking if a HashSet Contains an Element

To check if a HashSet contains an element, we can use the 

contains()
method, as follows: 

          
           
boolean containsFruit = myHashSet.contains("Cherry");

Clearing a HashSet

To clear a HashSet, we can use the 
clear()
method, as follows: 

          
           
myHashSet.clear();

Checking the Size of a HashSet

To check the size of a HashSet, we can use the 
size()
method, as follows: 

          
           
int size = myHashSet.size();
