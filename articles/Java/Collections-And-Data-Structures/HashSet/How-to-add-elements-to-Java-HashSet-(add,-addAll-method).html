Adding elements to Java HashSet

Java HashSet is an implementation of the Set interface that stores elements in a hash table. It provides fast insertion, removing, and searching operations and keeps track of the insertion order. In this article, we will discuss two ways to add elements to a Java HashSet- the add() and addAll() methods.

Add Method

The add() method inserts a given object into a HashSet. It returns a boolean value â€“ true if the element was successfully added to the HashSet, false if it was already present in the set. 

          
           
HashSet<String> hSet = new HashSet<>();

boolean isAdded1 = hSet.add("Apple");  //true
boolean isAdded2 = hSet.add("Apple");  //false
System.out.println(isAdded1);   //true
System.out.println(isAdded2);   //false
As shown above, the add() method returns true when a new element is added to the HashSet, and false when an existing element is added to the HashSet.

addAll Method

The addAll() method is used to add all of the elements of a collection to the HashSet. It takes a collection as its argument, such as List, Set, Map, etc., and inserts all of the elements of the specified collection into the HashSet. The following example shows how to use the addAll() method to add elements to a HashSet: 

          
           
HashSet<String> hSet = new HashSet<>();
List<String> list = Arrays.asList("Apple", "Banana", "Orange");

System.out.println(hSet);   //[]
hSet.addAll(list);
System.out.println(hSet);   //[Apple, Banana, Orange]
As you can see, the elements of the list have been successfully added to the HashSet.
