Preserving Insertion Order of Java HashSet Elements

The insertion order of elements in a HashSet is not preserved by default. However, it is possible to preserve the insertion order of this structure. This article will explain how to do this by providing code examples and detailing their outputs.

Using a LinkedHashSet

The easiest way to preserve insertion order in a HashSet is to use a 

          
           
LinkedHashSet
. This type of set maintains its elements in insertion order, meaning that the order in which elements are added is the same as the order in which they are output. Example 1: 

          
           
import java.util.*;

public class Test {
    public static void main(String[] args) {
        Set<Integer> linkedHashSet = new LinkedHashSet<>();

        linkedHashSet.add(10);
        linkedHashSet.add(15);
        linkedHashSet.add(20);
        linkedHashSet.add(25);

        System.out.println("LinkedHashSet elements:");
        for (Integer element : linkedHashSet)
            System.out.println(element);
    }
}
Output: 

          
           
LinkedHashSet elements:
10
15
20
25

Using a TreeSet

Another way to preserve the order in which elements are added to a HashSet is to use a 

          
           
TreeSet
. A 

          
           
TreeSet
is sorted according to the natural ordering of its elements, which means that the order in which they are added will be preserved. Example 1: 

          
           
import java.util.*;

public class Test {
    public static void main(String[] args) {
        Set<Integer> treeSet = new TreeSet<>();

        treeSet.add(10);
        treeSet.add(15);
        treeSet.add(20);
        treeSet.add(25);

        System.out.println("TreeSet elements:");
        for (Integer element : treeSet)
            System.out.println(element);
    }
}
Output: 

          
           
TreeSet elements:
10
15
20
25

Conclusion

Preserving the order in which elements are added to a HashSet is possible using either a 

          
           
LinkedHashSet
or a 

          
           
TreeSet
. For most cases, a 

          
           
LinkedHashSet
is preferable because it is simpler to implement and produces results that are easier to understand.
