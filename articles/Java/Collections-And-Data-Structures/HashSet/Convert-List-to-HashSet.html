Convert List to HashSet in Java

Java is a popular programming language and it provides a range of useful methods for manipulating collections. One of the most useful methods for working with collections is converting a List to a HashSet. In this article, we'll explore how to do this using the Java API and offer some code examples.

What is a HashSet?

Before we dive into the details of how to convert a List to a HashSet, let's take a quick look at what a HashSet is. A HashSet is a type of Set, which is a collection that does not allow duplicate elements. It uses a hash table for storage and provides fast lookup and insertion times. The elements in a HashSet are not ordered and cannot be sorted.

How to Convert List to HashSet in Java

Now that we have an understanding of what a HashSet is, let's look at how to convert a List to a HashSet in Java. Java provides a few different ways to perform this conversion. The first is to use the Set.addAll(Collection c) method. This method takes a Collection argument and adds all of its elements to the Set. The syntax looks like this: 

          
           
HashSet<String> hashSet = new HashSet<>();
List<String> list = new ArrayList<>();

hashSet.addAll(list);
The second way to convert a List to a HashSet is to use the Collections.addAll method. This method takes a Collection and adds all of its elements to the specified collection. The syntax looks like this: 

          
           
HashSet<String> hashSet = new HashSet<>();
List<String> list = new ArrayList<>();

Collections.addAll(hashSet, list);
These two methods are the simplest and most efficient way to convert a List to a HashSet in Java.

Example: Converting a List to a HashSet in Java

Let's take a look at an example of how to convert a List to a HashSet in Java. In this example, we have a List of Strings and we want to convert it to a HashSet. We will use the Set.addAll() method to do this. 

          
           
List<String> list = Arrays.asList("Apple", "Banana", "Orange", "Pear");
HashSet<String> hashSet = new HashSet<>();

hashSet.addAll(list);

System.out.println(hashSet);
