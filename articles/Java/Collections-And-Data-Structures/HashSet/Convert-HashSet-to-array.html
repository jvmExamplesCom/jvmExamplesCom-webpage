Converting HashSet to Array in Java

HashSet is an unordered collection of elements, meaning that elements can be stored in any order. To convert a HashSet to an array in Java, one must first create an object of the appropriate type and specify the size of the array, then iterate through the HashSet and assign each element to the array.

Example 1: Using toArray() Method

The simplest way to convert a HashSet to an array is by using the 

          
           
toArray()
method. This method takes an array as the argument and returns an array of the same type. 

          
           
import java.util.*; 
  
public class HashSetToArray { 
  
    public static void main(String[] args) 
    { 
  
        // Creating a HashSet 
        Set<String> set = new HashSet<String>(); 
  
        // Adding elements to the HashSet 
        set.add("Apple"); 
        set.add("Banana"); 
        set.add("Orange"); 
  
        // Converting HashSet to an array 
        String[] array = set.toArray(new String[set.size()]); 
  
        System.out.println("HashSet elements: "); 
        for (String temp : array) { 
            System.out.println(temp); 
        } 
    } 
}
Output: 

          
           
HashSet elements:
Apple
Banana
Orange

Example 2: Iterator and For-Each Loop

Another way to convert a HashSet to an array is by using the iterator and a for-each loop. This method iterates through all the elements of the HashSet and assigns them to an array. 

          
           
import java.util.*; 
  
public class HashSetToArray { 
  
    public static void main(String[] args) 
    { 
        // Creating a HashSet 
        Set<String> set = new HashSet<String>( 
            Arrays.asList("Apple", "Banana", "Orange")); 
  
        // Converting HashSet to an array 
        String[] array = new String[set.size()]; 
        Iterator<String> itr = set.iterator(); 
        int i = 0; 
        while (itr.hasNext()) 
            array[i++] = itr.next(); 
  
        System.out.println("HashSet elements: "); 
        for (String temp : array) { 
            System.out.println(temp); 
        } 
    } 
}
Output: 

          
           
HashSet elements:
Apple
Banana
Orange
