How to get Elements by Index from HashSet in Java

HashSet is a data structure available in the Java programming language. It is an unordered collection of elements that cannot contain duplicate values. It is used for storing and manipulating sets of data. This tutorial will explain how to get elements by index from HashSet in Java.

Example 1

The following code example shows how to get elements by index from HashSet:


          
           
import java.util.*;
public class Test {
 public static void main(String args[]) {
  HashSet<Integer> hash = new HashSet<Integer>();
  hash.add(1);
  hash.add(2);
  hash.add(3);
  // get element at index 0
  int element = (int)hash.toArray()[0];
  System.out.println(element);
 }
}

In the above example, we create a HashSet of integers and add three elements to it. Then, we use the toArray() method to get the elements present in the HashSet as an array, and finally, we get the element at the specified index (0 in this case). The output of this program is:


          
           
1

 }
