Retrieving Elements by Index from a Java TreeSet

A 
TreeSet
is an implementation of the Set interface which stores its elements in a tree. The elements are sorted according to their natural ordering, or based on a custom comparator that you can provide at construction time. This makes it easy to retrieve elements from the set using an index.

Accessing Elements Using Natural Ordering

If you are using the TreeSet with its natural ordering, you can access the elements by their index using the 

treeSet.toArray()
method. This will return an array containing all the elements of the TreeSet in their natural order, and you can then access the element at the desired index. 

          
           
// Create a TreeSet
TreeSet<String> treeSet = new TreeSet<>();

// Add elements to the set
treeSet.add("element1");
treeSet.add("element2");
treeSet.add("element3");

// Retrieve an element at the given index
Object[] elements = treeSet.toArray();
String elementAtIndex2 = (String) elements[2];
System.out.println(elementAtIndex2);
// Output: element3
