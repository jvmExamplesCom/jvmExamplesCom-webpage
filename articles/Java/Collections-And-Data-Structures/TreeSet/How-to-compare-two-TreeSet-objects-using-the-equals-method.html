Comparing Two TreeSet Objects Using The Equals Method

TreeSet is a collection that stores elements in a sorted order. It extends the AbstractSet class and implements the NavigableSet interface. The equals() method is used to compare two TreeSet objects and check if they are equal. In this tutorial, we will learn how to use the equals() method for comparing two TreeSet objects.

Syntax

The syntax of the equals() method is as follows: 

          
           
public boolean equals(Object o)

Parameters

The equals() method takes one parameter:
  • o: This is the object to be compared with the current TreeSet object.

Return Value

The equals() method returns a boolean value:
  • True: If the two TreeSet objects are equal.
  • False: If the two TreeSet objects are not equal.

Example 1: Comparing Two Empty TreeSet Objects

In the following example, we will create two empty TreeSet objects, t1 and t2. Then, we will use the equals() method to compare them. We will print the result to the console.


          
           
import java.util.TreeSet;

public class Main {
    public static void main(String[] args) {
        TreeSet<Integer> t1 = new TreeSet<>();
        TreeSet<Integer> t2 = new TreeSet<>();
    
        //Compare t1 and t2
        boolean result = t1.equals(t2);
    
        //Print the result
        System.out.println(result);
    }
}

When we run the above program, the output will be:


          
           
true
