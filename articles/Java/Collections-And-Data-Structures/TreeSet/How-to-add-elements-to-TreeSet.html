Creating a TreeSet Collection

To use TreeSet in a Java program, you must first initialize a new instance of the collection with the new keyword. You may also need to specify the type of elements the collection will contain, as in the following example: 

          
           
TreeSet<Integer> intSet = new TreeSet<>();
This code creates a new TreeSet of Integer type called intSet. You can also use this general template to create other types of TreeSets, such as String, Float, or Double.

Adding Elements to a TreeSet

add()
method, which adds a single element to the collection. For example, to add 10 to the above intSet, use the following code: 

          
           
intSet.add(10)
You can also use the 

          
           
addAll()

List<Integer> list = new ArrayList<>();
list.add(11);

intSet.addAll(list);

remove()

intSet.remove(10)
