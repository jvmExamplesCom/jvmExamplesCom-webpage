Getting Tail Set from Java TreeSet Example

The tailSet() is a method of the TreeSet class in Java that returns a view of the portion of this set whose elements are greater than or equal to a given element. This method can be used to obtain elements from a specified index, or to perform a sub-range retrieval operation on a sorted set or a navigable set. This article demonstrates how the 

tailSet()
method works with a code example.

Java TreeSet tailSet() Method Syntax


          
           
public SortedSet<E> tailSet(E fromElement)
//Returns a view of the portion of this set whose elements are greater than or equal to fromElement.

Java TreeSet tailSet() Method Example

To better understand how the 
tailSet()
method works, let’s look at a code example below: 

          
           
import java.util.TreeSet;

public class TreeSetTailSet {
   public static void main(String[] args) {
      // Create a TreeSet and add some elements
      TreeSet<String> ts1 = new TreeSet<String>();
      ts1.add("A");
      ts1.add("B");
      ts1.add("C");
      ts1.add("D");
      ts1.add("E");

      // Print out the TreeSet
      System.out.println("TreeSet values: " + ts1);

      // Get the TailSet using tailSet() method
      System.out.println("The TailSet: " + ts1.tailSet("C"));
   }
}
The output of the program is: 

          
           
TreeSet values: [A, B, C, D, E]
The TailSet: [C, D, E]

In the above example, the TreeSet class is used to create a sorted set of Strings. The 

tailSet()
method is used to obtain a view of the portion of the set whose elements are greater than or equal to the argument passed in. In this case, it is “C”, so all elements from “C” onward (including “C”) are returned as a TreeSet.
