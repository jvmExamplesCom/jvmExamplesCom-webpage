How to Convert ArrayList to TreeSet

Java provides an easy way to convert an ArrayList to a TreeSet. The TreeSet is an implementation of the Set interface, while the ArrayList is an implementation of the List interface. A TreeSet stores its elements in a sorted and ascending order, while an ArrayList stores its elements in the order in which they are added. This article will provide a step-by-step guide to converting an ArrayList to a TreeSet.

Creating an ArrayList

Before you can convert an ArrayList to a TreeSet, you must first create an ArrayList. The following example shows how to create an ArrayList with five elements: 

          
           
ArrayList<Integer> arrayList = new ArrayList<>();
arrayList.add(1);
arrayList.add(2);
arrayList.add(3);
arrayList.add(4);
arrayList.add(5);
The output of this code is the following ArrayList: [1, 2, 3, 4, 5].

Converting the ArrayList to a TreeSet

To convert an ArrayList to a TreeSet, we use the constructor of the TreeSet class. This constructor takes a collection as an argument and creates a TreeSet with the elements of the collection. In this case, the collection will be our ArrayList. The following example shows how to convert the ArrayList to a TreeSet: 

          
           
TreeSet<Integer> treeSet = new TreeSet<>(arrayList);
The output of this code is the following TreeSet: [1, 2, 3, 4, 5]. Note that the elements of the TreeSet are sorted in ascending order.

Printing the TreeSet to the Console

If you want to print the TreeSet to the console, then you can use the for-each loop. The following example shows how to print the TreeSet to the console: 

          
           
for (Integer element : treeSet) {
    System.out.println(element);
}
The output of this code is the following:

2
3
4
5
