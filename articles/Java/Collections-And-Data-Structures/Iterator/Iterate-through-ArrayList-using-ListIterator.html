Iterate through ArrayList using ListIterator

Java provides various ways to iterate through ArrayList. One of the most common ways is to use ListIterator. ListIterator provides a bidirectional iterating functionality and offers several methods that allow you to traverse a list in both directions. The below example demonstrates how to iterate through an ArrayList using ListIterator. 

          
           
ArrayList<String> arrList = new ArrayList<>();
arrList.add("Apple");
arrList.add("Banana");
arrList.add("Orange");

ListIterator<String> iterator = arrList.listIterator();

while(iterator.hasNext()) {
    System.out.println(iterator.next());
}
The output of the above code: 

          
           
Apple
Banana
Orange

What is ListIterator?

ListIterator is an interface that is used to traverse a list. It provides more methods compared to its Iterator counterpart and can be used to traverse the list in both directions (forward & backward). The following methods are commonly used when working with ListIterator: 

          
           
next() - returns the next element in the list

previous() - returns the previous element in the list

hasNext() - checks if there is a next element

hasPrevious() - checks if there is a previous element

add() - adds an element

remove() - removes an element

set() - replaces the last returned element by a call to next() or previous()

Adding Elements To ArrayList

We can add elements to an ArrayList while iterating over it using the add() method of ListIterator. Below is an example: 

          
           
ArrayList<String> arrList = new ArrayList<>();
arrList.add("Apple");
arrList.add("Banana");
arrList.add("Orange");

ListIterator<String> iterator = arrList.listIterator();

while(iterator.hasNext()) {
    String element = iterator.next();
    if(element.equals("Banana")) {
        iterator.add("Pineapple");
    }
    System.out.println(iterator.next());
}
The output of the above code: 

          
           
Apple
Banana
Pineapple
Orange
