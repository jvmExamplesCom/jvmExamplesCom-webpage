Iterate through a LinkedList in Java

Linked lists are a popular data structure used in programming languages like Java. A linked list is a collection of nodes connected by pointers. Each node contains a value or data and a pointer pointing to the next node in the list. In this tutorial, we are going to learn how to iterate through a LinkedList in Java. Iterating through a LinkedList means visiting each node in the list and performing an action.

Iteration using while loop

The conventional way of iterating through a LinkedList is to use a while loop. This involves creating a node pointer which points to the first node of the list and then looping until that node is null. At each iteration, the pointer is moved to the next node in the list. Here is an example of how to iterate through a LinkedList using while loop: 

          
           
Node current = head;
while (current != null) {
    // process current node
    current = current.next;
}
The above code snippet will loop through each node in the list starting from the first node (head). It will print out each value in the list until it reaches the end.

Iteration using forEach loop

Using a forEach loop is another efficient way of iterating through a LinkedList. This involves calling a consumer lambda within the forEach method, passing the current node as an argument. The consumer lambda will perform an action on each node in the list. Here is an example of how to iterate through a LinkedList using a forEach loop: 

          
           
list.forEach(node -> {
    // process current node
});
The above code snippet will perform an action on each node in the list. It will print out each value in the list until it reaches the end.

Iteration using an Iterator

Using an iterator is another way of iterating through a LinkedList. This involves first obtaining an iterator from the LinkedList, and then looping until the iterator hasNext() method returns false.
