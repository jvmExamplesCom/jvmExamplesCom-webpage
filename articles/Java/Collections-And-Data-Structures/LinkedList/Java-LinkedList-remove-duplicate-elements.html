Java LinkedList Remove Duplicate Elements

Java's LinkedList data structure provides a useful, but often overlooked, mechanism for removing duplicate elements. With a few steps, you can easily remove duplicates from a LinkedList and streamline the data stored within.

Creating a LinkedList

First, let's create a LinkedList with some duplicate elements in it. We'll use Java's generics syntax to explicitly type the elements as Strings: 

          
           
LinkedList<String> myLinkedList = new LinkedList<String>();
myLinkedList.add("cat");
myLinkedList.add("dog");
myLinkedList.add("cat");

Removing Duplicate Elements

Now that we have our LinkedList set up, we can begin removing the duplicates. We'll start by converting the elements into a Set, which only stores unique elements. 

          
           
Set<String> mySet = new HashSet<String>(myLinkedList);
Now that our set of unique elements is established, we can use it to create a new LinkedList that contains only the unique elements. 

          
           
LinkedList<String> noDupes = new LinkedList<String>(mySet);

Verifying the Results

Finally, we can verify that our operation was successful by printing out the elements of the new LinkedList. 

          
           
for(String s : noDupes) {
    System.out.println(s);
}
The above code should output: 

          
           
cat
dog
As you can see, the duplicate element of "cat" has been removed from the LinkedList.

Conclusion

Removing duplicate elements from a LinkedList doesn't need to be a tedious process. With a few lines of code, you can transform an overflowing LinkedList into an optimized set of unique data.
