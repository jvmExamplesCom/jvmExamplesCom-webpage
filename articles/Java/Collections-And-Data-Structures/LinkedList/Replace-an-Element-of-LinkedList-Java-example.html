Replacing an Element of a Linked List Java Example

Linked Lists are one of the most commonly used data structures in Java and many other popular programming languages. An element within a linked list can be replaced quite easily though there are a few steps that need to be taken in order for it to succeed. In this article, we will provide an example of how to replace an element of a Linked List in Java.

Creating a LinkedList

Before any elements can be replaced, a linked list must first be created. To do this, we will create a simple linked list with 5 elements using the following code:


          
           
LinkedList<String> list = new LinkedList<>();
list.add("Element 1");
list.add("Element 2");
list.add("Element 3");
list.add("Element 4");
list.add("Element 5");

Replacing an Element

Once the linked list has been created, we are now ready to replace one of the elements. To do this, we will use the set() method which takes two parameters - the index of the element to be replaced and the new element. For example, if we want to replace “Element 3” with “New Element”, we could use the following code:


          
           
list.set(2, "New Element");

Once the code is executed, the list will look like this:


          
           
"Element 1", "Element 2", "New Element", "Element 4", "Element 5"

Conclusion

In this article, we provided an example of how to replace an element of a Linked List in Java. As you can see, the process is quite simple and straightforward and just requires the use of the set() method. We hope this article has helped you gain a better understanding of Linked Lists in Java.
