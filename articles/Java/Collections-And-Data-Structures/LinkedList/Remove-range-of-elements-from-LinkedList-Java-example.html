Remove Range of Elements from LinkedList Java Example

In this article, we'll discuss how to remove a range of elements from a LinkedList in Java. We will also provide code examples and output for each operation. LinkedList is an implementation of the List interface which allows us to store data in a linear order while preserving constant time insertion and deletion. It has two pointers - head and tail - which point to the first and last elements of the list. We can use the removeRange() method to remove a given range of elements from the list. This method takes two arguments: the index of the first element to be removed, and the index of the last element to be removed (inclusive).

Example 1

Let's consider the following example where we remove a range of elements from our LinkedList: 

          
           
LinkedList<Integer> list = new LinkedList<Integer>();
list.add(1);
list.add(2);
list.add(3);
list.add(4);
list.add(5);
list.add(6);

list.removeRange(2, 4);

System.out.println(list);
The output of this program will be: 

          
           
[1, 2, 5, 6]
As can be seen from the output, the elements at indices 2, 3, and 4 have been successfully removed from our list.

Example 2

In the following example, we will remove the first and last elements of the list using the removeRange() method: 

          
           
LinkedList<Integer> list = new LinkedList<Integer>();
list.add(1);
list.add(2);
list.add(3);
list.add(4);
list.add(5);
list.add(6);

list.removeRange(0, list.size() - 1);

System.out.println(list);
The output of this program will be: 

          
           
[]
As expected, all elements in the list have been successfully removed. Therefore, the list is now empty.

Example 3

In the following example, we will attempt to remove a range of elements from an empty list: 

          
           
LinkedList<Integer> list = new LinkedList<Integer>();

list.removeRange(2, 4);

System.out.println(list);
The output of this program will be: 

          
           
[]
As expected, the attempt to remove a range of elements from an empty list fails, and the list remains unchanged since there are no elements to remove.

Conclusion

In this article, we discussed how to remove a range of elements from a LinkedList in Java. We provided code examples and output for each operation. We demonstrated how to use the removeRange() method to remove a given range of elements from the list. We also discussed how the attempt to remove a range of elements from an empty list fails.
