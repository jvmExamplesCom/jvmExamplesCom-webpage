Remove Specified Element from LinkedList Java Example

LinkedList is one of the most popular data structures in Java, and can be used to store a sequence of elements. However, sometimes it may be necessary to remove a particular element from the list. This can be done using the remove() method, which takes a parameter specifying the element to be removed. In this article, we will discuss how to remove a specified element from a LinkedList in Java.

Example 1: Using remove() Method

The simplest way to remove a specified element from a LinkedList is to use the remove() method. This method takes a parameter representing the element to be removed. For example, if we have a LinkedList of Strings, we can remove an element from it by passing the String value to the remove() method: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("element1");
list.add("element2");
list.add("element3");

String elementToBeRemoved = "element2";
list.remove(elementToBeRemoved);
After executing this code, the list no longer contains “element2”.

Example 2: Using Iterator

Another way to remove a specified element from a LinkedList is to use an Iterator. The Iterator provides a way to traverse the list, and allows us to remove elements while specifying them by their position. Here is an example of how to use an Iterator to remove an element from a LinkedList: 

          
           
LinkedList<String> list = new LinkedList<>();
list.add("element1");
list.add("element2");
list.add("element3");

String elementToBeRemoved = "element2";
Iterator<String> iterator = list.iterator();

while (iterator.hasNext()) {
    String element = iterator.next();
    if (element.equals(elementToBeRemoved)) {
        iterator.remove();
    }
}
After executing this code, the list no longer contains “element2”.

Conclusion

In this article, we discussed two ways of removing a specified element from a LinkedList in Java. The first way is using the remove() method, which takes a parameter specifying the element to be removed. The second way is using an Iterator, which allows us to specify the element to be removed by its position. Whichever method we choose, the result is that the specified element will no longer be present in the list after the operation is complete.
