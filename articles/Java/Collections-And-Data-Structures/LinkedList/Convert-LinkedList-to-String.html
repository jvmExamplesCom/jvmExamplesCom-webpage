Converting a LinkedList to String in Java

In this tutorial, we will learn how to convert a 
LinkedList
to a 
String
in Java. The basic approach for this conversion involves looping through the current 
LinkedList
, appending the elements together into a single string, and then displaying the resulting string.

Using a For Loop with the toString() Method

The most basic approach to converting a 
LinkedList
to a 
String
in Java is to use a 
for
loop. This method uses the 
toString()
method of each element in the list to return a 
String
representation of the list. Here is an example of how this approach can be used: 

          
           
public static String linkedListToString(LinkedList list) {
    StringBuilder builder = new StringBuilder();
    for(Object obj : list){
        builder.append(obj.toString());
    }
    return builder.toString();
}

Using Streams

Another way to achieve our goal is to use Java 8's Stream API. This can be done with the following code:


          
           
public static String linkedListToString(LinkedList list) {
    return list.stream()
               .map(Object::toString)
               .collect(Collectors.joining());
}

Using the StringJoiner Class

The 
StringJoiner
class can also be used to convert a 
LinkedList
to a 
String
. This class can be used by passing in the desired characters or strings to be included between each element in the list. Here is an example of how this can be used: 

          
           
public static String linkedListToString(LinkedList list, String delimiter) {
    StringJoiner joiner = new StringJoiner(delimiter);
    list.forEach(joiner::add);
    return joiner.toString();
}
