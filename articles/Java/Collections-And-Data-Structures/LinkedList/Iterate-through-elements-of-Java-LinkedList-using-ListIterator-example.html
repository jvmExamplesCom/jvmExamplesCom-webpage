Iterate through Java LinkedList using ListIterator

LinkedList is one of the most commonly used data structures in Java. It is an implementation of the List interface. The list provides several methods to iterate through its elements. One such method is the ListIterator. It allows you to traverse the list in either direction, and you can add, remove or replace existing elements in the list. In this article, we will be covering how to iterate through a Java LinkedList using the ListIterator.

A Simple Example

First, let's create a LinkedList object: 

          
           
LinkedList<String> list = new LinkedList<String>();
Now we can add some elements to it: 

          
           
list.add("Apple");
list.add("Banana");
list.add("Cherry");
Finally, we can obtain the ListIterator object: 

          
           
ListIterator<String> iterator = list.listIterator();

Using the ListIterator

The ListIterator provides two methods for traversing the list: 

          
           
next()
and 

          
           
previous()
. Each of these methods returns the next or previous element in the list, respectively. They also return a boolean value to indicate whether there are more elements available to be traversed. Let's start by iterating through the list from the beginning: 

          
           
while(iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}
This will produce the following output: 

          
           
Apple
Banana
Cherry
We can also iterate through the list in reverse using the 

          
           
previous()
method: 

          
           
while(iterator.hasPrevious()) {
    String element = iterator.previous();
    System.out.println(element);
}
This will produce the following output: 

          
           
Cherry
Banana
Apple

Adding and Removing Elements

The ListIterator also provides methods for adding and removing elements from the list. The 

          
           
add(E e)
method adds the specified element to the list and returns 

          
           
true
if the element was successfully added. For example, let's add a new element: 

          
           
iterator.add("Durian");
We can then use the iterator to traverse the list and verify that the new element has been added: 

          
           
while(iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}
This will produce the following output: 

          
           
Apple
Banana
Cherry
Durian
The 

          
           
remove()
method can be used to remove the last element returned by 

          
           
next()
or 

          
           
previous()
. For example, let's remove the element we just added: 

          
           
iterator.remove();
When we traverse the list again, we can see that the element has been removed: 

          
           
while(iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}
This will produce the following output: 

          
           
Apple
Banana
Cherry

Replacing Elements

The ListIterator also provides a 

          
           
set(E e)
method, which can be used to replace the last element returned by 

          
           
next()
or 

          
           
previous()
. For example, let's replace the element "Cherry" with "Strawberry": 

          
           
iterator.previous();
iterator.set("Strawberry");
Now, when we traverse the list, we can see that the element has been replaced: 

          
           
while(iterator.hasNext()) {
    String element = iterator.next();
    System.out.println(element);
}
This will produce the following output: 

          
           
Apple
Banana
Strawberry

Conclusion

In this article, we have covered how to iterate through a Java LinkedList using the ListIterator. We have also seen how we can use the iterator to add, remove, and replace elements in the list.
