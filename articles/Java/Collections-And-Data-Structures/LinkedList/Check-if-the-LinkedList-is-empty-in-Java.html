Check if the LinkedList is empty in Java

A LinkedList in Java is a linear data structure where each element is a separate object that is connected to another element, or node, using references.

Checking if a LinkedList is empty is an important task while working with this type of data structure. This can be easily done with the help of two methods which are provided by the LinkedList class. The first one is the isEmpty() method which returns true if the list is empty else it returns false. The second one is the size() method which returns the total number of elements present in the list.

Example 1: Using isEmpty() Method


          
           
import java.util.LinkedList;

public class Main {

 public static void main(String[] args) {

   // Create a LinkedList
   LinkedList<Integer> l_list = new LinkedList<Integer>();

   // Check if the list is empty using the isEmpty() method
   System.out.println("Is the LinkedList empty? " + l_list.isEmpty());
 }
}
Output: 

          
           
Is the LinkedList empty? true

Example 2: Using size() Method


          
           
import java.util.LinkedList;

public class Main {

 public static void main(String[] args) {

   // Create a LinkedList
   LinkedList<Integer> l_list = new LinkedList<Integer>();

   // Check if the list is empty using the size() method
   if(l_list.size() == 0)
    System.out.println("The LinkedList is empty!");
  else
    System.out.println("The LinkedList is not empty!");
 }
}
Output: 

          
           
The LinkedList is empty!

These two methods can be used to check if a LinkedList is empty and depending on the requirement either of them can be used.
