Finding index of LinkedList Elements using indexOf & lastIndexOf Methods

Java's LinkedList class provides two methods for searching elements and finding their indices: the indexOf(Object o) and lastIndexOf(Object o). Both methods are used to locate the position of an element in a LinkedList, but they differ in the way they search through the list.

indexOf Method

The method 

          
           
indexOf(Object o)
locates the position of an element in a LinkedList by searching sequentially, starting at element 0 and progressing to the end of the list. The method returns the index of the first occurrence of the element being searched, and if the element is not found in the LinkedList then the returned value is -1. For example, consider the 

          
           
LinkedList
below: 

          
           
LinkedList<String> list = new LinkedList<>(); 
list.add("one"); 
list.add("two"); 
list.add("three"); 
list.add("four"); 
list.add("five");
list.add("two");
If we want to locate the index of the element "two", the indexOf method can be used as follows: 

          
           
int index = list.indexOf("two"); 
System.out.println(index);
This will print out the index of the first occurrence of "two" which is 1.

lastIndexOf Method

The method 

          
           
lastIndexOf(Object o)
is similar to the indexOf method, except it begins the search at the end of the LinkedList and returns the index of the last occurrence of the element being searched. If the element is not found in the LinkedList, then the returned value is -1. For example, using the same LinkedList from before, if we want to locate the index of the last occurrence of "two", the lastIndexOf method can be used as follows: 

          
           
int index = list.lastIndexOf("two"); 
System.out.println(index);
This will print out the index of the last occurrence of "two" which is 5.

Conclusion

The indexOf and lastIndexOf methods are useful for finding the position of an element in a LinkedList. The indexOf method searches from the beginning of the list and returns the index of the first occurrence of the element, while the lastIndexOf method searches from the end of the list and returns the index of the last occurrence of the element.
