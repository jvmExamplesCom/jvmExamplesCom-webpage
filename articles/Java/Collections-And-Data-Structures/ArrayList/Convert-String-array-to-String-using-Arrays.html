Convert String Array to String using Arrays

The goal of the following tutorial is to demonstrate how to convert a String array to a String using the Arrays class. In java, a String array is an array of Strings. This article provides code examples for each step in the process.

Create a String Array

To get started, we will first create a String array: 

          
           
String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
Now that we have our array, we can move on to the next step.

Use Arrays.toString() Method

Once we have created a String array, we can use the 

          
           
Arrays.toString()
method to convert it to a String. This method takes any type of array and returns a String representation of the array. Here is an example of using 

          
           
Arrays.toString()
to convert the String array to a String: 

          
           
String carString = Arrays.toString(cars);
The output of this code will be: 

          
           
["Volvo", "BMW", "Ford", "Mazda"]

Remove Brackets from Output

If you do not want the brackets to be included in the output, you can use the 

          
           
replaceAll()
method to remove them. This method takes two parameters: a regular expression and a replacement string. To remove the brackets, we can use the regular expression 

          
           
"[\[\]]"
and replace it with an empty string. Here is an example of using the 

          
           
replaceAll()
method to remove the brackets from the output: 

          
           
carString = carString.replaceAll("[\\[\\]]", "");
The output of this code will be: 

          
           
Volvo, BMW, Ford, Mazda

Conclusion

In conclusion, the Arrays class provides us with the 

          
           
toString()
method which can be used to convert a String array to a String. Additionally, the 

          
           
replaceAll()
method can be used to remove any unwanted characters from the output.
