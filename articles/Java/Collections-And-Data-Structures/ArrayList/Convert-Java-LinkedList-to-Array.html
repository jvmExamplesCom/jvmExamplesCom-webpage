Convert Java LinkedList to Array

Converting a LinkedList to an array can be useful for many different tasks, such as processing data in the array format, or passing the array as a parameter to a method. In Java, LinkedLists are objects that store multiple, ordered elements in a single object. To convert a LinkedList object to an array, Java offers a simple method.

Using the toArray() Method in Java

To convert a Java LinkedList to an array, you can use the toArray() method. The toArray() method converts a LinkedList into an array by taking each element of the LinkedList, one at a time, and putting it into an array. When using the toArray() method, you must pass the size of the array that the LinkedList will be converted to.

The syntax for the toArray() method is as follows:


          
           
LinkedList<String> myLinkedList = new LinkedList<String>();
    String[] myArray = myLinkedList.toArray(new String[myLinkedList.size()]);

The example above shows how to convert a LinkedList of type String to an array of type String. As you can see, the size of the array is specified as the argument to the toArray() method.

Example Program

The following example program will demonstrate how to convert a Java LinkedList to an array:


          
           
import java.util.LinkedList; 

public class ConvertLinkedList { 

    public static void main(String[] args) 
    { 

        // Creating object of class linked list 
        LinkedList<String> myLinkedList =  
                      new LinkedList<String>(); 

        // Adding elements to the linked list 
        myLinkedList.add("Apple"); 
        myLinkedList.add("Mango"); 
        myLinkedList.add("Orange"); 
        myLinkedList.add("Strawberry"); 

        // Printing the linked list 
        System.out.println("Linked List: " + myLinkedList); 

        // Creating an array of strings 
        String[] myArray = myLinkedList.toArray(new String[myLinkedList.size()]); 

        // Printing the array 
        System.out.println("Array elements: "); 
        for (String element : myArray) { 
            System.out.print(element + " "); 
        } 
    } 
}

When the above code is compiled and executed, it produces the following result:


          
           
Linked List: [Apple, Mango, Orange, Strawberry]
Array elements: Apple Mango Orange Strawberry
