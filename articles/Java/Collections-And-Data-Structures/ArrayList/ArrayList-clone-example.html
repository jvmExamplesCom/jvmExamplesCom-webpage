Java: ArrayList Clone Example

Cloning an arraylist in Java is a useful capability that allows you to make a copy of an array and store it in a separate object. This can be helpful in many scenarios where you need to keep track of the original array while making modifications to the clone. In this article, we'll take a look at how to clone an arraylist in Java and explore some examples of this in action.

Cloning an ArrayList in Java

The easiest way to clone an arraylist in Java is to use the clone() method. This method will create a shallow clone of the array, meaning that it will create a new arraylist object but will not clone each element that exists within the array. If you need a deep clone, then you will need to use other methods to create a new arraylist from scratch with the elements from the old list. Here's an example of using the clone() method to create a clone of an arraylist: 

          
           
// Create an ArrayList 
  ArrayList<string> list = new ArrayList<>();
  list.add("a");
  list.add("b");
  list.add("c");
  
  // Clone the ArrayList 
  ArrayList<string> clonedList = (ArrayList<string>) list.clone();

The cloned arraylist will contain the same elements as the original arraylist, though the objects stored in the list will not have been deep cloned. This may be sufficient for some use cases, but it is important to remember that references to the same objects will be kept in both the original and cloned arraylist.

Deep Cloning an ArrayList

The clone() method should only be used when a shallow clone is sufficient for your use case. If you need to create a deep clone, then you will need to use other methods to create the array. Here's an example of creating a deep clone of an arraylist by looping through the elements of the original array and adding them to a new arraylist: 

          
           
// Create an ArrayList 
  ArrayList<string> list = new ArrayList<>();
  list.add("a");
  list.add("b");
  list.add("c");
  
  // Deep clone the ArrayList 
  ArrayList<string> deepClonedList = new ArrayList<>();
  for(String s : list) {
    deepClonedList.add(new String(s));

Cloning an ArrayList of Objects

If you are working with an arraylist that contains objects, then you may need to perform additional steps to ensure that you have created a deep clone. You can do this by looping through the elements of the original arraylist and creating new instances of each object and adding them to the cloned arraylist. Here's an example of cloning an arraylist of objects: 

          
           
// Create an Object
  class Person {
    
    private String name;
    private int age;
    
    public Person(String name, int age) {
      this.name = name;
      this.age = age;
    } 
    
    public String getName() {
      return name;
    }
    
    public int getAge() {
      return age;
    }
  }
  
  // Create an ArrayList 
  ArrayList<person> list = new ArrayList<>();
  list.add(new Person("John", 30));
  list.add(new Person("Jane", 25));
  
  // Deep clone the ArrayList
  ArrayList<person> deepClonedList = new ArrayList<>();
  for(Person p : list) {
    Person clonePerson = new Person(p.getName(), p.getAge());
    deepClonedList.add(clonePerson);
  }

We have then used a for loop to iterate through the elements in the arraylist and create clones of each person before adding them to the cloned arraylist. This approach ensures that each instance of the Person class is deep cloned.
