Adding All Elements of an Array to an ArrayList using Arrays

The Arrays class provides a static method 

          
           
asList()
which can be used to convert a given array into a list. This method is available in the 

          
           
java.util
package and it was added to Java 1.4. It can be used for adding all elements of an array to an ArrayList. The following example shows how to add all elements of an array, passed as argument to the List constructor, to an ArrayList. 

          
           
String[] myArray = new String[]{"Apple", "Banana", "Grape"};

List<String> myList = new ArrayList<>(Arrays.asList(myArray));
The above code will create an 

          
           
ArrayList
object with all the elements of 

          
           
myArray
.

Using addAll() Method of ArrayList

We can also use the 

          
           
addAll()
method of the 

          
           
ArrayList
class to add all elements of an array to the desired 

          
           
ArrayList
. 

          
           
String[] myArray = new String[]{"Apple", "Banana", "Grape"};

List<String> myList = new ArrayList<>();
myList.addAll(Arrays.asList(myArray));

ArrayList

myArray
.

Using foreach Loop

We can also use the 

          
           
forEach
loop to add all the elements of the array to the 

          
           
ArrayList
. 

          
           
String[] myArray = new String[]{"Apple", "Banana", "Grape"};

List<String> myList = new ArrayList<>();

for (String element : myArray) {
    myList.add(element);
}
This code will also create an 

          
           
ArrayList
object with all the elements of 

          
           
myArray
.

The new Streams API introduced in Java 8 can also be used for adding all elements of an array to an 

          
           
ArrayList
. 

          
           
String[] myArray = new String[]{"Apple", "Banana", "Grape"};

List<String> myList = 
    Arrays.stream(myArray).collect(Collectors.toCollection(ArrayList::new));
