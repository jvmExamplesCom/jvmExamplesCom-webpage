Get Random Elements from ArrayList in Java

The ArrayList class in Java provides a convenient way of storing and accessing objects. One useful feature of the ArrayList is being able to retrieve a random element from it without having to loop through the entire list. In this article, we will look at how we can retrieve random elements from an ArrayList using Java.

Using Random Class

The most common way to get a random element from an ArrayList is by using the java.util.Random class. The Random class provides a variety of methods to generate random numbers, including one for generating a random integer between zero and the size of the list. Here is an example of how to use the Random class to get a random element from an ArrayList: 

          
           
import java.util.ArrayList;
import java.util.Random;
 
public class RandomElement {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<String>();
        list.add("Apple");
        list.add("Orange");
        list.add("Grapes");
        list.add("Pineapple");
 
        Random random = new Random();
        int index = random.nextInt(list.size());
        String randomElement = list.get(index);
        System.out.println(randomElement);
    }
}
In the above example, we are creating an ArrayList that contains four elements. We then create an instance of the Random class to generate a random integer between 0 and the size of the list (4). We then use the get() method to retrieve the element with the corresponding index.

Using Collections Class

Another way to get a random element from an ArrayList is by using the java.util.Collections class. The Collections class provides a variety of utility methods for working with collections such as lists, maps, and sets. Among these utility methods is the shuffle() method, which shuffles the elements of the supplied collection. The shuffled elements are then returned in a List, which we can then use to get a random element. Here is an example of how to use the shuffle() method to get a random element from an ArrayList: 

          
           
import java.util.ArrayList;
import java.util.Collections;
import java.util.List;
 
public class RandomElement {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<String>();
        list.add("Apple");
        list.add("Orange");
        list.add("Grapes");
        list.add("Pineapple");
 
        Collections.shuffle(list);
        List<String> shuffledList = list.subList(0, 1);
        String randomElement = shuffledList.get(0);
        System.out.println(randomElement);
    }
}
In the above example, we are first shuffling the elements of the list using the shuffle() method. We then use the subList() method to get a sublist that contains only one element. We then use the get() method to retrieve the element from the sublist.

Conclusion

In this article, we looked at how to get a random element from an ArrayList in Java. We saw two different approaches to do this: using the java.util.Random class and using the java.util.Collections class. Both approaches provide a convenient way to get a random element from a list.
