Adding Elements to an ArrayList in Java

Adding elements to an 

          
           
ArrayList
in Java is a simple task that can be done with a few basic methods. Depending on your needs, the elements you add to the 

          
           
ArrayList
can range from simple primitive types to complex objects. In this article we'll go over some examples of how to add elements to an 

          
           
ArrayList
.

Adding Primitives

Adding primitives to an 

          
           
ArrayList
is done using the 

          
           
add()
method. The following example shows how to add an int to an 

          
           
ArrayList
: 

          
           
ArrayList<Integer> myList = new ArrayList<>();
myList.add(5);
We can also add boolean values to an 

          
           
ArrayList
: 

          
           
ArrayList<Boolean> myList = new ArrayList<>();
myList.add(true);

Adding Objects

Adding objects to an 

          
           
ArrayList
is done in a similar way as primitives. The following example shows how to add a 

          
           
String
object to an 

          
           
ArrayList
: 

          
           
ArrayList<String> myList = new ArrayList<>();
myList.add("Hello World!");
We can also add custom objects to an 

          
           
ArrayList
. The following example shows how to add a 

          
           
Person
object to an 

          
           
ArrayList
: 

          
           
public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // getters and setters
}

ArrayList<Person> myList = new ArrayList<>();
Person person = new Person("John", 34);
myList.add(person);

Adding Collection of Objects

We can also add a collection of objects to an 

          
           
ArrayList
. This can be done using the 

          
           
addAll()
method. The following example shows how to add a 

          
           
List
of 

          
           
String
objects to an 

          
           
ArrayList
: 

          
           
List<String> list = Arrays.asList("Apple", "Banana", "Orange");
ArrayList<String> myList = new ArrayList<>();
myList.addAll(list);

Conclusion

Adding elements to an 

          
           
ArrayList
in Java is a very simple process. Depending on your needs, the elements can range from simple primitive types to complex objects. We've gone over some examples of how to add primitive types and objects to an 

          
           
ArrayList
, as well as how to add a collection of objects.
