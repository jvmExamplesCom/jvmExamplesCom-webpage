Append All Elements of Other Collection to Java ArrayList Example

Java ArrayList is used to store a collection of elements. It is dynamically re-sizable and allows duplicate elements. The ArrayList provides useful methods to work with its elements, such as append() which append all elements of other Collection to this ArrayList instance.

Example 1: Appending one Collections' elements into another

In this example, we will create two different ArrayList instances and append one Collection's elements into the other. We will use addAll() method of ArrayList class for this purpose.


          
           
import java.util.*;
 
public class Main {
  public static void main(String[] args) {
    // Creating two ArrayLists
    List list1 = new ArrayList();
    List list2 = new ArrayList();
 
    // Populating list1
    list1.add("A");
    list1.add("B");
    list1.add("C");
 
    // Populating list2
    list2.add("D");
    list2.add("E");
    list2.add("F");
 
    // Appending list2 elements into list1
    list1.addAll(list2);
    System.out.println("Final list: " + list1);
  }
}
Output: 

          
           
Final list: [A, B, C, D, E, F]

Example 2: Appending one element of a Collection into another

In this example, we will use addAll() method of ArrayList class to append one element of a Collection into another. Here, we will create two different ArrayList instances and append one element from the one ArrayList instance into the other.


          
           
import java.util.*;
 
public class Main {
  public static void main(String[] args) {
    // Creating two ArrayLists
    List list1 = new ArrayList();
    List list2 = new ArrayList();
 
    // Populating list1
    list1.add("A");
    list1.add("B");
    list1.add("C");
 
    // Populating list2
    list2.add("D");
    list2.add("E");
    list2.add("F");
 
    // Appending one element from list2 into list1
    list1.addAll(list2.subList(0,1));
 
    System.out.println("Final list: " + list1);
  }
}
Output: 

          
           
Final list: [A, B, C, D]
