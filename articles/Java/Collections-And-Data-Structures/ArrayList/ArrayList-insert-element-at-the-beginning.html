Java: Inserting an Element at the Beginning of an ArrayList

It is possible to insert an element at the beginning of an 

ArrayList
in Java. The recommended approach to accomplish this task is to use the 

          
           
add(int index, E element)

ArrayList

add(0, E element)
syntax.

String

ArrayList<String> al = new ArrayList<String>();
