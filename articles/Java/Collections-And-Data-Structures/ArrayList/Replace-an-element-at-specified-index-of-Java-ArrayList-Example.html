Replace an Element at Specified Index of Java ArrayList Example

Introduction

Java ArrayList is used to store dynamically sized collection of elements and is backed by an array. We can use set() method to specify a particular element in the list and replace it with another element. In this article, we will be discussing how to replace an element at specific index of ArrayList with an example.

Example

Let’s understand how to replace an element at a specified index with an example. 

          
           
// Java Program to replace an element 
// at specified index 
import java.util.*; 

public class ReplaceElement 
{ 
	public static void main(String[] args) 
	{ 
		// Creating a list 
		List<string> my_list 
			= new ArrayList<string>( 
				Arrays.asList("Java", "C++", 
							"Python", "Go", 
							"Scala")); 

		// Displaying initial list 
		System.out.println("Initial List: "
							+ my_list); 

		// Replacing an element using 
		// set() method 
		my_list.set(2, "Ruby"); 

		// Displaying final list 
		System.out.println("Final List: "
							+ my_list); 
	} 
}

Final List: [Java, C++, Ruby, Go, Scala]
In the above example, we have created a list named

method to replace the element at index 2 with the string “Ruby”. Finally, we have displayed the updated list.
