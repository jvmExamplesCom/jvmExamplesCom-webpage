Copy All Elements of the Java ArrayList to an Object Array Example

This article will demonstrate how to copy all elements from a Java ArrayList to an Object Array using several code examples.

Example 1: With Casting

In this example, we will cast the ArrayList object to create an Object array. 

          
           
ArrayList<String> arrList = new ArrayList<String>();

arrList.add("String 1");
arrList.add("String 2");
arrList.add("String 3");

Object[] objArray = arrList.toArray();
In this example, all elements from the ArrayList were successfully copied to the Object array.

Example 2: Without Casting

In this example, we will create an Object array and pass the size of the ArrayList to avoid casting. 

          
           
ArrayList<String> arrList = new ArrayList<String>();

arrList.add("String 1");
arrList.add("String 2");
arrList.add("String 3");

Object[] objArray = new Object[arrList.size()];
arrList.toArray(objArray);
In this example, all elements from the ArrayList were successfully copied to the Object array.

Example 3: With Collection Framework

In this example, we will use the Collection Framework to copy all the elements from the ArrayList to the Object Array. 

          
           
ArrayList<String> arrList = new ArrayList<String>();

arrList.add("String 1");
arrList.add("String 2");
arrList.add("String 3");

Object[] objArray = arrList.toArray(new Object[arrList.size()]);
In this example, all elements from the ArrayList were successfully copied to the Object array.

Conclusion

In this article, we demonstrated three different ways to copy all elements from a Java ArrayList to an Object Array. We used casting, creation of objects, and the Collection Framework in order to achieve our goal. With any of these techniques, you should be able to easily copy all elements from a Java ArrayList to an Object Array.
