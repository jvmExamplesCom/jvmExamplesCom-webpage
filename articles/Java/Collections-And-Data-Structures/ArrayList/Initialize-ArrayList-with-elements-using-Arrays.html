Initializing ArrayList with Elements Using Arrays

The ArrayList class provides an easy way to initialize lists with elements from an array. It can be used to quickly and efficiently create a list of items that are initialized with a set of elements. In this article, we will explore how to use the ArrayList class to initialize a list with elements from an array.

Creating an ArrayList

The ArrayList class is part of the Java Collections Framework, and it provides an efficient way of creating lists from arrays. It is easy to create an ArrayList from an array using the constructor that takes an array as a parameter. Here is an example of how to use this constructor to create an ArrayList of String objects: 

          
           
String[] words = {"hello", "world", "!", "goodbye"};
ArrayList<String> list = new ArrayList<>(Arrays.asList(words));
In this example, the constructor takes an array of strings and creates an ArrayList object containing the same strings. Now that we have an ArrayList, we can use the add method to add new elements to it, or use the remove method to remove elements from it.

Adding Elements to an ArrayList

Adding elements to an ArrayList is straightforward. We can simply use the add method to add new elements to the end of the list. Here is an example of how to add a new element to an ArrayList of Strings: 

          
           
list.add("new word");
We can also add multiple elements at once using the addAll method. This method takes an array or Collection as a parameter and adds all of the elements to the ArrayList: 

          
           
String[] newWords = {"foo", "bar"};
list.addAll(Arrays.asList(newWords));
Now the ArrayList contains all of the words from the original array plus the two new words.

Removing Elements from an ArrayList

Removing elements from an ArrayList is just as easy as adding them. We can use the remove method to remove a single element from the list. Here is an example of how to remove a word from an ArrayList of Strings: 

          
           
list.remove("world");
We can also use the removeAll method to remove multiple elements at once. This method takes an array or Collection as a parameter and removes all of the elements from the list that are contained in the array or Collection: 

          
           
String[] wordsToRemove = {"!","goodbye"};
list.removeAll(Arrays.asList(wordsToRemove));
Now the ArrayList contains only the words from the original array minus the two words that were removed.

Conclusion

In this article, we explored how to use the ArrayList class to initialize a list with elements from an array. We saw how to use the constructor to create an ArrayList from an array, how to use the add and addAll methods to add elements to the list, and how to use the remove and removeAll methods to remove elements from the list.
