Sorting LinkedHashSet elements using Comparable

LinkedHashSet is a subclass of the HashSet class. It is an ordered collection of objects, stored as key-value pairs. This article will discuss how to sort elements in a LinkedHashSet using the Comparable interface.

What is Comparable?

Comparable is an interface that provides a natural ordering for the objects in a collection. Objects can be compared to each other using their natural ordering, which can be defined by implementing the compareTo() method in the Comparable interface.

Example: String Sorting Using Comparable Interface


          
           
import java.util.LinkedHashSet;
import java.util.Set;

public class ComparableExample {

    public static void main(String[] args) {
        Set<string> set = new LinkedHashSet<>();

        set.add("Apple");
        set.add("Banana");
        set.add("Pineapple");
        set.add("Grapes");

        // Sort the elements in the LinkedHashSet using Comparable
        set.stream().sorted((s1, s2) -> s1.compareTo(s2)).forEach(System.out::println);
    }
}</string>

The above code snippet shows how to sort the elements of a LinkedHashSet using the Comparable interface. We have created a LinkedHashSet and added four strings. Then we have applied the stream() method to the set and sorted the elements by their natural order. The output is as follows:


          
           
Apple
Banana
Grapes
Pineapple
