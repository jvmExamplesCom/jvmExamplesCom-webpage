Sort LinkedList Using Comparator in Java

LinkedList is a linear data structure that allows efficient insertion and deletion of elements, and allows random access unlike arrays. It can be used to store data that needs to be accessed in an efficient way. Sorting a LinkedList can be done using the Comparator interface. In this article, we will learn how to sort a LinkedList using Comparator in Java.

Example 1: Sorting LinkedList using Comparator by String Length

In this example, we will create a LinkedList of Strings and then sort them by the length of the String. We will use Comparator.comparingInt(String::length) to get the length of each String and compare them. 

          
           
import java.util.*;

public class SortLinkedListByLength {

    public static void main(String[] args) {
        // Create a list of strings 
        List<String> list = new LinkedList<>();
        list.add("Java");
        list.add("C++");
        list.add("Python");
        list.add("Scala");
        list.add("Kotlin");
        
        // Sort the list by length
        list.sort(Comparator.comparingInt(String::length));
        
        // Print the sorted list
        System.out.println(list);
    }
}
Output: 

          
           
[Java, C++, Scala, Kotlin, Python]

Example 2: Sorting LinkedList using Comparator by Numeric Value

In this example, we will create a LinkedList of numbers and sort them in ascending order. We will use the Comparator.comparingInt() method to compare the numeric values. 

          
           
import java.util.*;

public class SortLinkedListByNumber {

    public static void main(String[] args) {
        // Create a list of numbers 
        List<Integer> list = new LinkedList<>();
        list.add(10);
        list.add(5);
        list.add(20);
        list.add(1);
        list.add(15);
        
        // Sort the list in ascending order
        list.sort(Comparator.comparingInt(Integer::intValue));
        
        // Print the sorted list
        System.out.println(list);
    }
}
Output: 

          
           
[1, 5, 10, 15, 20]

Example 3: Sorting LinkedList using Comparator by Custom Logic

In this example, we will create a LinkedList of objects and sort them using a custom comparator. We will use the Comparator.comparing() method to compare the field values of each object according to our custom logic. 

          
           
import java.util.*;

public class SortLinkedListByCustomLogic {

    public static void main(String[] args) {
        // Create a list of Person objects 
        List<Person> list = new LinkedList<>();
        list.add(new Person("John", 25));
        list.add(new Person("David", 30));
        list.add(new Person("Alice", 20));
        list.add(new Person("Bob", 24));
        
        // Sort the list by age
        list.sort(Comparator.comparing(Person::getAge));
        
        // Print the sorted list
        System.out.println(list);
    }
    
    public static class Person {
        private String name;
        private int age;
        
        public Person(String name, int age) {
            this.name = name;
            this.age = age;
        }
        
        public String getName() {
            return name;
        }
        
        public int getAge() {
            return age;
        }
        
        @Override
        public String toString() {
            return String.format("Name: %s, Age: %d", name, age);
        }
    }
}
Output: 

          
           
[Name: Alice, Age: 20, Name: Bob, Age: 24, Name: John, Age: 25, Name: David, Age: 30]
