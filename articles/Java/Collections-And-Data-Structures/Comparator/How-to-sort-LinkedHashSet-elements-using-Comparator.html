How to Sort LinkedHashSet Elements Using Comparator

LinkedHashSet is a great class for storing elements in the form of a linked list. It provides fast and efficient access to the elements stored in the set. It also maintains insertion order, which is useful for applications that require a predefined order of elements. However, there are times when you may need to sort the elements in a LinkedHashSet according to some criteria. This is where Comparator comes in handy. In this article, we'll look at how to sort LinkedHashSet elements using a Comparator.

Using the sort() Method

The easiest way to sort LinkedHashSet elements is to use the sort() method. This method takes a Comparator as an argument and returns a new sorted LinkedHashSet. Here's a simple example: 

          
           
Set<String> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add("One");
linkedHashSet.add("Two");
linkedHashSet.add("Three");

// Create a comparator
Comparator<String> comparator = Comparator.comparing(String::length);

// Sort the LinkedHashSet elements using the comparator
Set<String> sortedSet = linkedHashSet.sort(comparator);

// Print the sorted LinkedHashSet
System.out.println(sortedSet); // Prints [One, Two, Three]
In the above example, we create a Comparator that compares strings by their length. We then pass this Comparator to the sort() method, which returns a new sorted LinkedHashSet.

Using the Stream API

Another way to sort LinkedHashSet elements is to use the Stream API. The Stream API allows us to transform collections of data into streams and perform various operations on the stream. Here's an example: 

          
           
Set<String> linkedHashSet = new LinkedHashSet<>();
linkedHashSet.add("One");
linkedHashSet.add("Two");
linkedHashSet.add("Three");

// Create a comparator
Comparator<String> comparator = Comparator.comparing(String::length);

// Create a stream from the LinkedHashSet
Stream<String> stream = linkedHashSet.stream();

// Sort the stream using the comparator
Stream<String> sortedStream = stream.sorted(comparator);

// Collect the sorted stream elements into a LinkedHashSet
Set<String> sortedSet = sortedStream.collect(Collectors.toCollection(LinkedHashSet::new));

// Print the sorted LinkedHashSet
System.out.println(sortedSet); // Prints [One, Two, Three]
In the above example, we create a Comparator that compares strings by their length. We then create a stream from the LinkedHashSet and sort it using the comparator. Finally, we collect the sorted stream elements into a LinkedHashSet.

Conclusion

In this article, we looked at how to sort LinkedHashSet elements using a Comparator. We looked at two ways of sorting the elements: using the sort() method and using the Stream API. Both methods provide easy and efficient ways to sort LinkedHashSet elements.
