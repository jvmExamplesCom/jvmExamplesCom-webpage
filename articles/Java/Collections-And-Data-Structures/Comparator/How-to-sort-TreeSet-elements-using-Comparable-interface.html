How to sort TreeSet elements using Comparable interface

TreeSet is a collection in Java which stores its elements in a sorted and ascending order. Its elements are stored in a tree structure and it implements the SortedSet interface. It is widely used for storing data which needs to be sorted in an ascending order. The ordering of the elements is done using the Comparable interface.

Implementing Comparable Interface

We can use the Comparable interface to sort the elements of a TreeSet according to a certain criteria. We need to implement the Comparable interface into our class and define the method compareTo() which compares two objects of the same type.


          
           
public class Person implements Comparable<Person> {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public int compareTo(Person p) {
        if (this.age > p.age) return 1;
        else if (this.age < p.age) return -1;
        else return 0;
    }
}

Creating TreeSet

Once we have implemented the Comparable interface, we can create a TreeSet of Person objects in the following manner:


          
           
TreeSet<Person> persons = new TreeSet<Person>();
persons.add(new Person("John", 20));
persons.add(new Person("Dave", 30));
persons.add(new Person("Alice", 25));

Sorting the TreeSet Elements

Once the TreeSet has been created, we can then sort the elements in the set according to their age. This can be done in the following manner:


          
           
List<Person> sortedList = new ArrayList<Person>(persons);
Collections.sort(sortedList, new Comparator<Person> () {
    public int compare(Person p1, Person p2) {
        return p1.compareTo(p2);
    }
});

for (Person person : sortedList) {
    System.out.println(person.getName() + ": " + person.getAge());
}
The output of the above code will be: 

          
           
John: 20
Alice: 25
Dave: 30
