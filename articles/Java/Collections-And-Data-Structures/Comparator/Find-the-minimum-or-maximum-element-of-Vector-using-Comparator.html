Find the Minimum or Maximum Element of Vector using Comparator in Java

The Vector class in Java provides a collection for a group of objects. A Comparator is an interface used to define the ordering of each element in the Vector. This tutorial will show you how to find the minimum or maximum element of a Vector by implementing a Comparator in Java.

Creating the Vector and Comparator Implementation

First, let's create a Vector object and add some elements to it: 

          
           
Vector<Integer> v = new Vector<Integer>();
v.add(1);
v.add(5);
v.add(3);
v.add(4);
v.add(2);
Then, we will create a Comparator class to define the ordering of each element: 

          
           
class MyComparator implements Comparator<Integer> {
    public int compare(Integer o1, Integer o2) {
        if(o1 < o2) return -1;
        if(o1 > o2) return 1;
        return 0;
    }
}

Finding the Minimum Element

To find the minimum element of the Vector, we will use the Collections.min() method. This method takes two parameters: the Vector and the comparator created above. The snippet below shows how to use this method: 

          
           
Integer minElement = Collections.min(v, new MyComparator());
After running the above code, the value of minElement will be 1.

Finding the Maximum Element

To find the maximum element of the Vector, we will use the Collections.max() method. This method takes two parameters: the Vector and the comparator created above. The snippet below shows how to use this method: 

          
           
Integer maxElement = Collections.max(v, new MyComparator());
After running the above code, the value of maxElement will be 5.

Conclusion

In this tutorial, you have learned how to find the minimum or maximum element of a Vector by implementing a Comparator in Java. You have also seen examples of how to use the Collections.min() and Collections.max() methods.
