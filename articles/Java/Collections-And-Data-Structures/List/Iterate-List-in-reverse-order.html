Iterate List in Reverse Order Using Java

Java provides a great set of tools and libraries for working with Lists. One of the most commonly used methods is iterating a List in reverse order, which can be achieved by using various techniques. In this article, we will explore the different ways to iterate a List in reverse order using Java.

Using for Loop

The first way of iterating a List in reverse order is by using a for loop. The idea is to start an index from the end of the list and keep decrementing it until it reaches the beginning of the list. Here is an example of how this can be done: 

          
           
List<Integer> list = Arrays.aslist(10, 20, 30, 40, 50);
for (int i = list.size() - 1; i >= 0; i--) {
    System.out.println(list.get(i));
}
The output of the above code snippet would be: 

          
           
50
40
30
20
10

Using Lambda Expression

Another way of iterating a List in reverse order is by using the Lambda expression. This approach involves creating a Stream from the List and then using the reverse method to reverse the order of the elements. Here is an example: 

          
           
List<Integer> list = Arrays.aslist(10, 20, 30, 40, 50);
list.stream().sorted().reverse().forEach(System.out::println);
The output of the above code snippet would be: 

          
           
50
40
30
20
10

Using Iterator Interface

It is also possible to iterate a List in reverse order using the Iterator interface. This approach involves creating a ListIterator object from the List and then using the previous() method to access the previous element in the List. Here is an example of how this can be done: 

          
           
List<Integer> list = Arrays.aslist(10, 20, 30, 40, 50);
ListIterator<Integer> iter = list.listIterator(list.size());
while (iter.hasPrevious()) {
    System.out.println(iter.previous());
}
The output of the above code snippet would be: 

          
           
50
40
30
20
10

Conclusion

In this article, we explored different ways to iterate a List in reverse order using Java. We saw how a for loop, Lambda expression, and the Iterator interface can be used to achieve this.
