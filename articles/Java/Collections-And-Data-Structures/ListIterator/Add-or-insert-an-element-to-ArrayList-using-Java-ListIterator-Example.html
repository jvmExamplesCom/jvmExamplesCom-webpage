Adding an Element to ArrayList using Java ListIterator

Adding elements to an ArrayList during the development process is often necessary. Java provides several ways of adding elements to an ArrayList, one of which is the ListIterator interface. In this article, we will discuss how to add an element to an ArrayList using the ListIterator interface by way of code examples.

How to Use the ListIterator Interface

The ListIterator interface provides a method called add() that can be used to add elements to an ArrayList. The add() method takes the element to be added as an argument and adds it to the end of the list. This method throws an UnsupportedOperationException if the underlying collection does not support the add() operation.

The following example demonstrates how to use the ListIterator interface to add an element to an ArrayList:


          
           
//Create a new ArrayList
ArrayList<String> list = new ArrayList<String>();

//Add some elements to the list
list.add("Java");
list.add("Python");
list.add("C++");

//Create a ListIterator
ListIterator<String> iter = list.listIterator();

//Add an element to the list
iter.add("C#");

//Print the list
for(String s : list)
{
 System.out.println(s);
}
Output: 

          
           
Java
Python
C++
C#

In the example above, we have created an ArrayList containing three elements and then created a ListIterator. We then used the add() method to add the element "C#" to the end of the list. Finally, we used a for-each loop to print out the list.

Conclusion

In this article, we discussed how to add elements to an ArrayList using the ListIterator interface. We saw how to use the add() method to add elements to a list and also how to print out the list after adding the element. Adding elements to a list during development can be a useful tool and understanding how to use the ListIterator interface is essential in order to do so.
