Remove an Element from ArrayList using Java ListIterator Example

Removing an element from an 
ArrayList
using 
Java ListIterator
is a simple process. The ListIterator interface provides a bidirectional iterator that allows objects to be removed from a List, replaced, or added. In this tutorial, we will learn how to use the ListIterator class to remove an element from an ArrayList.

Create and populate an ArrayList:

We will create an instance of an 
ArrayList
and populate it with some strings for testing purposes: 

          
           
ArrayList<String> list = new ArrayList<>();
list.add("One");
list.add("Two");
list.add("Three");
list.add("Four");
list.add("Five");

Using ListIterator to Remove an Element from an ArrayList:

To remove an element from an ArrayList we can first use the iterator() method to get a 

ListIterator
instance. We can then use the 
remove()
method of the 
ListIterator
interface to remove a specific element. The following example shows how to remove the element "Three" from the 
ArrayList
: 

          
           
// Get ListIterator instance
ListIterator<String> itr = list.listIterator();

// Iterate over the list
while (itr.hasNext()) { 

   // Get the next element 
   String element = itr.next(); 

   // Check if the element equals to the one to be removed 
   if (element.equals("Three")) { 

      // Remove the element if found 
      itr.remove(); 
   } 
}

Verify the Result:

We can now verify that the element "Three" has been successfully removed from the 

ArrayList
:
