Java: ArrayList ListIterator Example

ArrayList ListIterator is a Java utility class which provides the facility to traverse a list in either direction, i.e., to go forward or backward. It also provides the facility to add, remove and replace elements while traversing the list.

Example 1: Traversing a List in Forward Direction

Let's look at a simple example of how to traverse an ArrayList in forward direction using ListIterator in Java: 

          
           
import java.util.ArrayList;
import java.util.ListIterator;

public class ArrayListExample {

	public static void main(String[] args) {
		 
        //creating an ArrayList object
        ArrayList<String> colors = new ArrayList<String>();
 
        //adding elements to the list
        colors.add("Red");
        colors.add("Blue");
        colors.add("Green");
        colors.add("Orange");
 
        //getting ListIterator
        ListIterator <String> iterator = colors.listIterator();
 
        //traversing the list in forward direction
        System.out.println("Traversing the list in forward direction:");
        while(iterator.hasNext()){
           System.out.println(iterator.next());
        }
	}

}
Output: 

          
           
Traversing the list in forward direction:
Red
Blue
Green
Orange

Example 2: Traversing a List in Backward Direction

We can also traverse an ArrayList in backward direction using ListIterator in Java. Let's look at the following example: 

          
           
import java.util.ArrayList;
import java.util.ListIterator;

public class ArrayListExample {

	public static void main(String[] args) {
		 
        //creating an ArrayList object
        ArrayList<String> colors = new ArrayList<String>();
 
        //adding elements to the list
        colors.add("Red");
        colors.add("Blue");
        colors.add("Green");
        colors.add("Orange");
 
        //getting ListIterator
        ListIterator<String> iterator = colors.listIterator(colors.size());
 
        //traversing the list in backward direction
        System.out.println("Traversing the list in backward direction:");
        while(iterator.hasPrevious()){
           System.out.println(iterator.previous());
        }
	}

}
Output: 

          
           
Traversing the list in backward direction:
Orange
Green
Blue
Red
