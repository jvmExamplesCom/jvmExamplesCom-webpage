How to Iterate ArrayList in Reverse Direction using ListIterator

Java provides multiple ways of iterating over the elements of an ArrayList or any other collection for that matter. ArrayList has a 

listIterator()
method that returns a list iterator object of the same type as the list. Using this list iterator, we can traverse the collection in reverse order.

Example: Iterating ArrayList in Reverse Direction

Consider an ArrayList with some elements. In this example, we are iterating the ArrayList in reverse direction by using ListIterator.


          
           
import java.util.ArrayList;
import java.util.ListIterator;

public class ReverseIterationExample {
    public static void main(String[] args) {
        ArrayList<Integer> numbers = new ArrayList<>();

        // Adding elements to the ArrayList
        numbers.add(1);
        numbers.add(2);
        numbers.add(3);
        numbers.add(4);
        numbers.add(5);

        System.out.println("Original list : " + numbers);

        // Getting list iterator object in forward direction
        ListIterator<Integer> itr = numbers.listIterator(numbers.size());

        // Printing list elements in reverse order
        System.out.println("Elements in reverse order:");
        while (itr.hasPrevious()) {
            System.out.print(itr.previous() + " ");
        }
    }
}
When you run the program, the output will be: 

          
           
Original list : [1, 2, 3, 4, 5]
Elements in reverse order: 5 4 3 2 1

Explanation

To iterate the ArrayList in reverse direction, first, we get the ListIterator object of the list by calling the 

listIterator()
method. We can pass the index of the element to start with while obtaining the 
ListIterator
object. Since we are traversing in the reverse direction, we pass the size of the list to the 
listIterator()
method.

Once we have the ListIterator object, then we can call the 

hasPrevious()
and 
previous()
methods to access the elements in reverse order. The 
hasPrevious()
method returns true if there are elements in the ListIterator object before the current element and 
previous()
method returns the previous element from the ListIterator object.

Conclusion

In this tutorial, we have seen how to iterate over an ArrayList in reverse direction using ListIterator. ListIterator has several useful methods to traverse a list in both directions. We can also use 

forEachRemaining()
method to iterate the list elements in reverse order.
