Copy Elements of One Java Vector to Another Java Vector Example

Java provides an API for dealing with vectors and arrays. In this article, we will look at an example of how to copy elements of one Java Vector to another.

Creating the Vector Objects

We will create two Vector objects, one for the source and one for the destination. To do this, we use the following code: 

          
           
Vector sourceVec = new Vector();
Vector destVec = new Vector();

Populating the Source Vector

Next, we need to populate the source vector with some data. We can do this by adding elements to the vector using the add() method: 

          
           
sourceVec.add("Element 1");
sourceVec.add("Element 2");
sourceVec.add("Element 3");

Copying the Elements

Now that the source Vector has been populated, we can copy the elements over to the destination Vector. To do this, we use the addAll() method from the Collections class. This method takes two parameters: the source collection and the destination collection. 

          
           
Collections.addAll(destVec, sourceVec);

Checking the Results

To check that the elements have been copied correctly, we can iterate over the destination Vector and print out the elements. 

          
           
for (String element : destVec) {
    System.out.println(element);
}
This should print out the following output: 

          
           
Element 1
Element 2
Element 3

Conclusion

In this article, we looked at how to copy elements of one Java Vector to another. We created two Vector objects and then used the addAll() method from the Collections class to copy the elements. Finally, we iterated over the destination Vector and printed out the elements to confirm that they had been copied correctly.
