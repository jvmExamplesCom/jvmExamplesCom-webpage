Converting Lists to Maps

The 

          
           
java.util.List
and 

          
           
java.util.Map
are two of the core data structures in Java programming. A list is an ordered collection of elements, whereas a map is composed of key-value pairs. Fortunately, Java 8 provides a powerful utility method to convert lists into maps - the 

          
           
toMap()
method of the 

          
           
Collectors
class. In this article, we'll discuss how to use this method to convert a list into a HashMap. Before we dive into the examples, let's have a quick look at some of the relevant class and methods associated with list-to-map conversions.

Relevant Classes & Methods

The 

          
           
Collectors
class is part of the 

          
           
java.util.stream
package, which provides specialized stream operations for performing reduce, gather, and collect operations on streams. The 

          
           
Collectors

toMap()

An Example List-to-Map Conversion

toMap()

books.add(new Book(2, "Harry Potter and the Philosopher's Stone"));

We used the third parameter to handle collisions (i.e. if two elements in the list have the same id, the resulting map only contains the entry with the higher index in the list). The fourth parameter is used to specify the type of map to be used (
