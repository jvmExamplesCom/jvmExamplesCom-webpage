Enumeration over Java HashSet Example

The HashSet class in Java is one of the most commonly used collections in Java for storing group of elements. It allows fast access and retrieval of elements, and is based on the hashing technique. It is an unordered collection that contains unique elements. Generally, enumerating the elements of a collection is done using the iterator API or using the enhanced for-each loop, if the collection implements Iterable. But it is also possible to enumerate the HashSet using the legacy Enumeration API.

Example

Let us see a simple example of how to use the Enumeration API with a HashSet.


          
           
import java.util.Enumeration;
import java.util.HashSet;

public class EnumerationHashSetExample {
 
  public static void main(String args[]) {
    // create object of HashSet
    HashSet<String> set = new HashSet<String>();
 
    // add elements to HashSet object
    set.add("Java");
    set.add("JDK");
    set.add("JVM");
    set.add("JRE");
 
    // get Enumeration
    Enumeration<String> e = Collections.enumeration(set);
 
    System.out.println("Enumerating elements of HashSet: ");
 
    // enumerate elements of Hashtable
    while(e.hasMoreElements())
      System.out.println(e.nextElement());
  }
}

In this example, we create a HashSet object and add some elements to it. We then create an Enumeration object by calling the enumeration(HashSet) method of the Collections class. Finally, we use the hasMoreElements() and nextElement() methods of the Enumeration class to loop over and print out the elements of the HashSet.

Output


          
           
Enumerating elements of HashSet:
JDK
JRE
JVM
Java
