Java Vector Find Maximum Element Example

In this article, we will look at the example of finding the maximum element in a Java Vector. We will discuss the algorithm used, provide code examples and output our results as HTML.

Algorithm Overview

The algorithm for finding the maximum element in a Java vector is fairly straightforward: 1. Starting from the beginning, compare each element in the vector to the one before it. 2. Whichever one is greater, set that as the maximum value. 3. Once you reach the last element, you will have the maximum value stored in a variable.

Code Examples and Outputs

Let's dive into the code examples and how they can be outputted in HTML format. First, let's create a vector and fill it with some data: 

          
           
Vector<Integer> vec = new Vector<>();  
vec.add(10);  
vec.add(25);  
vec.add(6);  
vec.add(15); 
vec.add(45);
Now that we have our data, let's create a method to find the maximum element within the vector: 

          
           
public static int findMax(Vector<Integer> vec) {  
    int max = vec.get(0);  
    for (int i = 0; i < vec.size(); i++) {  
        if (vec.get(i) > max) {  
            max = vec.get(i);  
        }  
    }  
    return max;  
}
Finally, let's call the method and output the result in HTML format: 

          
           
int max = findMax(vec);   
System.out.println("The maximum element is " + max); 
// Output: The maximum element is 45
