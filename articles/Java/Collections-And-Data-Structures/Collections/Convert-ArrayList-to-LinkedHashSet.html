Converting ArrayList to LinkedHashSet

Converting a 
ArrayList
to a 
LinkedHashSet

LinkedHashSet

LinkedHashSet

Using Constructor

A quick way to convert an 
ArrayList
to a 
LinkedHashSet

LinkedHashSet

Collection
type, such as an 
ArrayList
. The constructor will automatically add the elements from the provided Collection type into the 
LinkedHashSet
. 

          
           
ArrayList<String> list = new ArrayList<String>();
list.add("a");
list.add("b");
list.add("c");

LinkedHashSet<String> set = new LinkedHashSet<String>(list);
