Converting HashMap to TreeMap in Java

HashMap and TreeMap are two of the most popular and commonly used classes from Java's Collection Framework. The main difference between the two lies in their implementations; HashMap uses a hashing algorithm, whereas TreeMap implements the Red-Black tree structure. In this article, we'll discuss how to convert a HashMap to a TreeMap.

Creating a HashMap

We'll begin by creating a HashMap that stores the Integer key-value pair (1,”One”). 

          
           
HashMap<Integer,String> hashMap = new HashMap<>();
hashMap.put(1, "One");

Converting HashMap to TreeMap Using Constructor

The easiest way to convert a HashMap to a TreeMap is by using the TreeMap constructor. The constructor takes a Map object as an argument, which means it can be used with any Map implementation such as HashMap, LinkedHashMap, or even TreeMap. 

          
           
// Creating a TreeMap from a HashMap
TreeMap<Integer,String> treeMap = new TreeMap<>(hashMap);
The TreeMap constructor will also take a Comparator as an argument, which can be used to define the sorting order of the keys. If no Comparator is specified, the TreeMap will be sorted according to the natural ordering of its keys.

Converting HashMap to TreeMap Using putAll() Method

Another way to convert a HashMap to TreeMap is by using the putAll() method. This method takes a Map object as an argument, which means it can be used with any Map implementation such as HashMap, LinkedHashMap, or even TreeMap. 

          
           
// Creating a TreeMap
TreeMap<Integer,String> treeMap = new TreeMap<>();

// Adding all entries from HashMap to TreeMap
treeMap.putAll(hashMap);
As with the constructor, the putAll() method also allows you to specify a Comparator. If no Comparator is specified, the TreeMap will be sorted according to the natural ordering of its keys.

Conclusion

In this article, we discussed how to convert a HashMap to a TreeMap in Java. We looked at two different approaches - using the TreeMap constructor and using the putAll() method. Both of these methods allow you to specify a Comparator for ordering the keys in the TreeMap.
