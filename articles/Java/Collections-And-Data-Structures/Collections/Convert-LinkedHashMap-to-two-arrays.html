Convert LinkedHashMap to Two Arrays

The LinkedHashMap class in Java is an ordered version of the HashMap that allows us to store our key-value pairs in a predefined order. However, if we want to convert this mapping into two separate arrays, one containing the keys and the other containing the values, it can be a bit tricky. This article will examine the methods by which we can accomplish this conversion.

Using a Loop

One way to convert our LinkedHashMap into two separate arrays is to use a loop. We can iterate through the entries in the map and extract the keys and values into separate arrays. 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(1, "A");
map.put(2, "B");

int[] keys = new int[map.size()];
String[] values = new String[map.size()];

int i = 0;
for (Map.Entry<Integer, String> entry : map.entrySet()) {
    keys[i] = entry.getKey();
    values[i] = entry.getValue();
    i++;
}
In the above example, we create a LinkedHashMap containing integer keys and string values, then use a loop to iterate through the entries and extract the keys into an integer array and the values into a string array.

Using Streams and Collectors

Java 8 introduced the Stream API, which allows us to process collections of data in a more functional-style manner. We can use streams to convert our LinkedHashMap into two separate arrays by mapping the entries to their respective keys and values, then collecting the results. 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(1, "A");
map.put(2, "B");

int[] keys = map.entrySet().stream()
        .mapToInt(Map.Entry::getKey)
        .toArray();

String[] values = map.entrySet().stream()
        .map(Map.Entry::getValue)
        .toArray(String[]::new);
In this example, we first use the stream() method to convert our map into a Stream, then use the mapToInt() and map() methods to map each entry to its respective key or value. We then call the toArray() method to collect the mapped values into two separate arrays.

Using Apache Commons Library

For those who are using the Apache Commons library, there is an easy way to convert our LinkedHashMap into two separate arrays. The org.apache.commons.collections4.MapUtils class contains the toArray() method, which takes a Map and two array types as parameters and returns two arrays, one containing the keys and the other containing the values. 

          
           
LinkedHashMap<Integer, String> map = new LinkedHashMap<>();
map.put(1, "A");
map.put(2, "B");

int[] keys = MapUtils.toArray(map, int.class);
String[] values = MapUtils.toArray(map, String.class);
In this example, we simply pass the map and the desired array type to the toArray() method and the the keys and values are returned in two separate arrays.

Conclusion

In this article, we examined the various ways we can convert a LinkedHashMap into two separate arrays, one containing the keys and the other containing the values. We looked at using a loop, using streams and collectors, and using the Apache Commons library. No matter which method you choose, converting a LinkedHashMap into two separate arrays is a useful operation and can be used in a variety of situations.
