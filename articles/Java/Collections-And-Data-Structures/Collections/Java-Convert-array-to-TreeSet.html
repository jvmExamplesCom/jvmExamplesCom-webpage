Java Convert array to TreeSet

The java.util.TreeSet is an implementation of the Set interface. It provides an ordered collection of elements, sorted according to their natural order, or by a Comparator provided at set creation time.

The TreeSet supports the addition, removal, and retrieval of elements in constant time. It also provides a range view of the elements in the set. A range view allows you to retrieve a subset of elements from the set with a given range of values. This article will demonstrate how to convert a Java array to a TreeSet.

Convert Array to TreeSet

The first step is to create a new TreeSet: 

          
           
TreeSet<String> treeSet = new TreeSet<String>();

Next, we need to loop through the array and add each element to the TreeSet:


          
           
String[] myArray = {"Apple", "Banana", "Cherry"};
for (String element : myArray) {
    treeSet.add(element);
}

Finally, we can print out the contents of the TreeSet: 


          
           
System.out.println(treeSet);
This will output the following: 

          
           
[Apple, Banana, Cherry]

Conclusion

In this article, we demonstrated how to convert a Java array to a TreeSet. We discussed the importance of using a Set, instead of a List, to store a collection of unique elements. Finally, we saw how to loop through an array, adding each element to the TreeSet, and printing out the contents of the TreeSet.
