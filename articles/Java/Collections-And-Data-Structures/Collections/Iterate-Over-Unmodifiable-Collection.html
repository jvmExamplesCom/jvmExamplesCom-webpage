Iterate Over Unmodifiable Collection

Java provides a variety of ways for developers to iterate over unmodifiable collections. These are helpful when dealing with collections that are not intended to be modified. This article will explore the different techniques and provide code examples for each.

Using forEach Method

The simplest way to iterate over an unmodifiable collection is to use the forEach method. This method takes a Consumer object as a parameter, which is used to process each element in the collection. The following example demonstrates how to use forEach to print out the elements in a List:


          
           
List<String> unmodifiableList = List.of("apple","banana","orange");
unmodifiableList.forEach(System.out::println);
The output of this code will be: 

          
           
apple
banana
orange

Using Stream API

Another way to iterate over an unmodifiable collection is to use the Stream API. This API provides several methods for processing collections in a functional style. The following example shows how to use filter and forEach to print out only the elements in a List that start with the letter “a”:


          
           
List<String> unmodifiableList = List.of("apple", "banana", "orange");
unmodifiableList.stream().filter(s -> s.charAt(0) == 'a').forEach(System.out::println);
The output of this code will be: 

          
           
apple

Using Iterator

Another technique for iterating over an unmodifiable collection is to use the Iterator interface. This interface provides the hasNext and next methods that allow you to process each element in the collection. The following example shows how to use Iterator to print out the elements in a Set:


          
           
Set<String> unmodifiableSet = Set.of("apple", "banana", "orange");
Iterator<String> iterator = unmodifiableSet.iterator();
while(iterator.hasNext()) {
    System.out.println(iterator.next());
}
The output of this code will be: 

          
           
apple
banana
orange

Conclusion

Iterating over an unmodifiable collection can be done in a variety of ways. The forEach, Stream, and Iterator APIs all provide different methods that can be used to process the elements in a collection. It is important to understand the differences between these APIs and when it is appropriate to use each one.
