Java Break Statement with Label Example

The break statement in Java is used to alter the control flow of the program by breaking out of the loop. The flow of the program branches out from its normal order of execution and terminates the loop. It is also possible to specify a label to a loop for the break statement to be used for exiting an outer loop. In this article, we will discuss the break statement with a label example in Java.

Java Break Statement with Label Syntax

The syntax for the break statement with a label in Java is as follows:


          
           
label:
while (boolean-expression) {
   // code that needs to be executed
   if (some-condition) {
      break label;
   }
}

In the above syntax, the label keyword is used to assign a name to the statement, and whenever a break statement is encountered, the code exits out from the loop with the label name.

Break Statement with Label Example

Let's look at a program to understand the usage of the break statement with a label example:


          
           
class BreakStatementWithLabelExample {
    public static void main(String[] args) {

        label:
        for (int i = 1; i <= 5; i++) {
            for (int j = 1; j <= 3; j++) {
                if (i == 3 &amp;&amp; j == 2) {
                    break label;
                }
                System.out.println("Running i:" + i + " j:" + j);
            }
        }
    }
}

In the above program, we have used the break statement with a label to exit out of the loop when the condition is met. We have labeled the outer for loop with the label name label. When the condition i==3 and j==2 is met, the break label; statement is executed, which branches the program out of the loop with the label name label. Running i:2 j:1
