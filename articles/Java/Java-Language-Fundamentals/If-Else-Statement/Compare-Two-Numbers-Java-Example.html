Compare Two Numbers Java Example

In this Java example, we will learn how to compare two numbers using the if-else statement. We will be comparing two numbers, x and y, and deciding if x is larger, smaller, or equal to y.

Example 1: Simple comparison strings

We will start with a simple example that uses comparison strings for our comparison. 

          
           
int x = 4;
int y = 6;

if (x == y) {
  System.out.println("x is equal to y");
} else if (x > y) {
  System.out.println("x is greater than y");
} else {
  System.out.println("x is less than y");
}
In the above example, x and y are compared using the comparison strings "==" (equal) , ">" (greater than), and "<" (less than). The output of the code is: 

          
           
x is less than y

Example 2: Comparison operators

In this example we will use comparison operators instead of comparison strings: 

          
           
int x = 4;
int y = 6;

if (x == y) {
  System.out.println("x is equal to y");
} else if (x > y) {
  System.out.println("x is greater than y");
} else {
  System.out.println("x is less than y");
}
In the above example, x and y are compared using comparison operators. The output of the code is: 

          
           
x is less than y

Example 3: Multiple comparison operators

In this example, we will use multiple comparison operators in our comparison: 

          
           
int x = 4;
int y = 6;

if (x == y || x > y) {
  System.out.println("x is equal to or greater than y");
} else {
  System.out.println("x is less than y");
}
In the above example, x and y are compared using the operator "||" (or). The output of the code is: 

          
           
x is less than y

Conclusion

In this article, we learned how to compare two numbers in Java using the if-else statement. We compared two numbers, x and y, and determined if x was larger, smaller, or equal to y. We used both comparison strings and comparison operators to make our comparisons.
