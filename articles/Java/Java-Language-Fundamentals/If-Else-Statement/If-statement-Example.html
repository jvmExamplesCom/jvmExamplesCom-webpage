If statement example in Java

If Statements are a powerful tool in any programming language, and Java is no exception. The if statement allows you to control the flow of your program by executing code based on conditions set by you. By using an if statement, you can check for certain conditions and then execute code accordingly.

Syntax

The syntax for an if statement in Java is as follows:


          
           
if (condition) {
    //code to execute
}

In this example, the condition is checked and if it evaluates to true, the code inside the braces is executed. If the condition evaluates to false, the code is not executed.

Example

Let’s take a look at a simple example that uses an if statement to check for a certain condition and prints a message based on the result:


          
           
int x = 5;
if (x == 5) {
    System.out.println("x is equal to 5");
}

In this example, we declare an int variable x and assign it the value of 5. We then use an if statement to check if x is equal to 5, and if it is, we print out a message stating that x equals 5.

Else Statement

You can also use an else statement with if statements to provide an alternative block of code to be executed if the condition is false. The syntax for an else statement is as follows:


          
           
if (condition) {
    //code to execute if condition is true
} else {
    //code to execute if condition is false
}

Let’s take a look at an example that uses an else statement to print a different message based on the result of the condition:


          
           
int x = 5;
if (x == 5) {
    System.out.println("x is equal to 5");
} else {
    System.out.println("x is not equal to 5");
}

In this example, we again declare an int variable x and assign it the value of 5. We then use an if statement to check if x is equal to 5, and if it is, we print out a message stating that x equals 5. If the condition evaluates to false, the else statement is executed and the alternate message is printed.

Else If Statements

You can also use multiple conditions with if statements by using else if statements. The syntax for an else if statement is as follows:


          
           
if (condition1) {
    //code to execute if condition1 is true
} else if (condition2) {
    //code to execute if condition1 is false and condition2 is true
}

Let’s take a look at an example that uses an else if statement to check for several conditions and print out different messages based on the result:


          
           
int x = 5;
if (x == 5) {
    System.out.println("x is equal to 5");
} else if (x > 5) {
    System.out.println("x is greater than 5");
} else {
    System.out.println("x is less than 5");
}

In this example, we again declare an int variable x and assign it the value of 5. We then use an if statement to check if x is equal to 5. If it is, a message stating that x is equal to 5 is printed. If the condition evaluates to false, the else if statement is executed and the program checks for the condition if x is greater than 5. If so, the alternate message is printed. If not, the else statement is executed and the last alternate message is printed.
