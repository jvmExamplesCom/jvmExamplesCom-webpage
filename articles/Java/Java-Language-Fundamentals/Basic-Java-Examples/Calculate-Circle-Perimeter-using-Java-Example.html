Calculate Circle Perimeter using Java Example

Calculating the perimeter of a circle is a useful skill for many different applications. Knowing how to do this in Java can help you create programs that require this knowledge. In this article, we'll discuss how to calculate the perimeter of a circle using Java and provide some examples of code.

Java Code to Calculate Circle Perimeter

The formula for calculating the perimeter of a circle is 2πr, where r is the radius of the circle. The code below shows how to calculate the perimeter of a circle in Java: 

          
           
public class CirclePerimeter {
    public static void main(String[] args) {
        double radius = 10; // Radius of circle
        double perimeter = 2 * Math.PI * radius; 
 
        System.out.println("Perimeter of Circle = " + perimeter);
    }
}
In this example, we've set the radius of the circle to 10. We then calculate the perimeter with the formula 2πr and print the result to the console.

Example Output of Circle Perimeter Java Code

When the above code is compiled and run, it will produce the following output: 

          
           
Perimeter of Circle = 62.83185307179586
As you can see, the result of 62.83185307179586 is calculated correctly.
