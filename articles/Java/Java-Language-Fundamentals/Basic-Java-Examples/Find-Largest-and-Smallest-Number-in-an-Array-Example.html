Find Largest and Smallest Number in an Array Example

Java is a powerful programming language that can be used to create programs for desktop, mobile, and web applications. It is widely used for a variety of tasks, from creating simple scripts to web applications. This tutorial will show you how to find the largest and smallest numbers in an array using the Java language.

Example 1: Finding the Largest Number in an Array

In this example, we will use the following array: 

          
           
[2, 4, 6, 8, 10]
We will also use a loop to go through the array, and a variable to store the largest number. The code is as follows: 

          
           
int[] numbers = {2, 4, 6, 8, 10};
int largestNumber = numbers[0];

for(int i = 0; i < numbers.length; i++){
    if(numbers[i] > largestNumber){
        largestNumber = numbers[i];
    }
}

System.out.println("The largest number is " + largestNumber);
In the code above, we set the initial value of the largestNumber variable to the first item in the array. Then, we use a for loop to go through each item in the array, and check if it is larger than the current largest number. If so, then we set the largestNumber variable to the current value from the array. At the end of the loop, the largestNumber variable should contain the largest value. We can then print it out. The output of this code is: 

          
           
The largest number is 10

Example 2: Finding the Smallest Number in an Array

In this example, we will use the same array as before: 

          
           
[2, 4, 6, 8, 10]
We will use a similar approach to the previous example, but instead of looking for the largest number, we will look for the smallest number. The code is as follows: 

          
           
int[] numbers = {2, 4, 6, 8, 10};
int smallestNumber = numbers[0];

for(int i = 0; i < numbers.length; i++){
    if(numbers[i] < smallestNumber){
        smallestNumber = numbers[i];
    }
}

System.out.println("The smallest number is " + smallestNumber);
In this code, we set the initial value of the smallestNumber variable to the first item in the array. Then, we use a for loop to go through each item in the array, and check if it is smaller than the current smallest number. If so, then we set the smallestNumber variable to the current value from the array. At the end of the loop, the smallestNumber variable should contain the smallest value. We can then print it out. The output of this code is: 

          
           
The smallest number is 2

Conclusion

In this tutorial, we've seen how to use Java to find the largest and smallest numbers in an array. We've also seen two examples of how to do this. To recap, we use a loop to go through each item in the array, and compare it with our variable. If the current array item is larger (or smaller, depending on our goal) than the one we have stored, then we set our variable to the new value. At the end of the loop, we should have the desired result.
