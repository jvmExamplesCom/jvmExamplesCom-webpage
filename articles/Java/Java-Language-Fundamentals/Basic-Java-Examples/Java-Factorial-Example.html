Java Factorial Example

A factorial is a mathematical operation that computes the product of an integer with all the integers below it. In Java, the factorial of a number is computed using a for loop and the multiplicative operator (the asterisk). The output of the code will be an integer that is the product of the input values.

Code Examples


          
           
int factorial(int n) {
  int result = 1;
  for (int i = 1; i <= n; i++) {
    result *= i;
  }
  return result;
}
The above example uses a for loop to iterate through the integers from one and up to the given number. For each number, the result is multiplied by the current value of i. Finally, the result is returned. 

          
           
System.out.println("Factorial of 4 is " + factorial(4));
This code prints out the factorial of a number, in this case four. It calls the factorial function and passes in the number four as an argument. 

          
           
int num = 5;
System.out.println("Factorial of " + num + " is " + factorial(num));
In this example, a variable is used to store the number five. This value is then passed into the factorial function and the result is printed out.

Output Examples

The code examples above will print out the following: 

          
           
Factorial of 4 is 24
Factorial of 5 is 120
