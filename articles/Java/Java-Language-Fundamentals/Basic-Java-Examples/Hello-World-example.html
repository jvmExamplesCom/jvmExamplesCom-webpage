Java Hello World Example

The traditional "Hello World" program is one of the most commonly used initial examples to teach programming in general and the Java language specifically. It serves as a great introduction to the world of programming and Java.

In this article we will look at a basic Java "Hello World" example and explain the code line by line. We will also look at how to compile and run the program.

Creating the Hello World Program

The first step is to create a text file called "HelloWorld.java" with the following code:


          
           
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
Let's look at each of these lines in detail.
  • public class HelloWorld: This is the definition of the class which contains the main method.
  • public static void main(String[] args): The main method where the program execution begins. All Java programs have a main method with the exact same signature, even the Hello World program.
  • System.out.println("Hello World!"): This line prints out the string "Hello World!".

Compiling, Linking and Running the Program

Once you have written the program, you need to compile it so that it can be executed. To do this, open a terminal window and navigate to the directory containing the Java program. Then use the 

javac
command to compile the program: 

          
           
$ javac HelloWorld.java

This will create a new file called "HelloWorld.class". This file contains the bytecode version of the program which can be executed by the JVM (Java Virtual Machine). To run the program, use the 

java
command: 

          
           
$ java HelloWorld

The program will then print out "Hello World!". Congratulations, you have just written and run your first Java program!
