Java Static Method Example

Java allows programmers to define static methods within a class. Static methods are shared by all instances of the class and are often used to group related utility methods together. This article will explain how they work and provide some examples of their usage.

Declaring a Static Method

A Java static method is declared using the 
static
keyword. The following is an example of a static method that returns the sum of two integers: 

          
           
public static int add(int x, int y) {
  return x + y;  
}

Invoking a Static Method

Static methods can be invoked without creating an instance of the class. Instead, they are invoked using the class name followed by the dot operator. For example, the following code invokes the 

add
method defined above: 

          
           
int result = MyClass.add(1, 2);

Accessing a Static Variable

Static variables can be accessed in the same way as static methods, using the class name followed by the dot operator. For example, the following code accesses the 

PI
constant in the 
Math
class: 

          
           
double radius = 10;
double circumference = radius * Math.PI;

Example: Implementing Factorials with Recursion

Recursive functions are often difficult to implement using object-oriented programming languages, but can be easily done using static methods. The following example implements the factorial function using recursion:


          
           
public static long factorial(int n) {
  if (n == 0) {
    return 1;  
  } 
  return n * factorial(n - 1);
}
The following code invokes the 
factorial
method: 

          
           
long result = MyClass.factorial(5);

Conclusion

In this article we have seen how to declare and invoke static methods, and how to use them to implement recursive functions. Static methods are a useful way to group utility functions together and make them accessible from anywhere in your program.
