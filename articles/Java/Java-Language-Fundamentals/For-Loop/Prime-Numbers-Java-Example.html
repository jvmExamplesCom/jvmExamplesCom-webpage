Java: Prime Numbers Java Example

Prime numbers are numbers that can only be divided evenly by 1 and itself. In this tutorial, we will discuss prime numbers in Java and how to determine whether a number is prime or not. We will provide code examples along the way to help explain the concepts.

Prime Number Program in Java

To check if a number is prime or not, we can use a 

          
           
for
loop to iterate from 2 up to the number itself and check if it is divisible by any of the numbers in the loop. A number is prime if none of the numbers between 2 and itself divides it evenly. If we find a number that divides the given number then it is not prime. Here is a sample program in Java to check if a number is prime or not: 

          
           
public class PrimeNumber { 
  
    // Function to check if a number is prime or not 
    public static boolean isPrime(int n) 
    { 
        // Check if n is less than 2, if yes then 
        // it is not prime 
        if (n < 2) { 
            return false; 
        } 
  
        // Iterate from 2 to n-1 
        for (int i = 2; i < n; i++) { 
            // If n is divisible by any number between 
            // 2 and n-1, it is not prime 
            if (n % i == 0) { 
                return false; 
            } 
        } 
  
        // If there is no number between 2 and n-1 that 
        // can divide n, it is prime 
        return true; 
    } 
  
    // Driver method 
    public static void main(String[] args) 
    { 
        System.out.println(isPrime(47)); 
    } 
}
When you run the above program, it should output 

          
           
true
since 47 is a prime number.

Finding Prime Numbers in an Array in Java

Now that we know how to check if a number is prime or not, we can start looking at how to find all the prime numbers in an array. In this example, we will use a simple algorithm to find the prime numbers in an array. The idea is to iterate over the given array and for each element check if it is a prime number or not. If it is a prime number, add it to another array. Here is the code for this example: 

          
           
import java.util.ArrayList; 
  
public class PrimeNumber { 
  
    // Function to check if a number is prime or not 
    public static boolean isPrime(int n) 
    { 
        // Check if n is less than 2, if yes then 
        // it is not prime 
        if (n < 2) { 
            return false; 
        } 
  
        // Iterate from 2 to n-1 
        for (int i = 2; i < n; i++) { 
            // If n is divisible by any number between 
            // 2 and n-1, it is not prime 
            if (n % i == 0) { 
                return false; 
            } 
        } 
  
        // If there is no number between 2 and n-1 that 
        // can divide n, it is prime 
        return true; 
    } 
  
    // Function to find all prime numbers in an array 
    public static int[] findPrimesInArray(int[] arr) 
    { 
        ArrayList<integer> primes = new ArrayList<>(); 
  
        for (int i = 0; i < arr.length; i++) { 
            if (isPrime(arr[i])) { 
                primes.add(arr[i]); 
            } 
        } 
  
        // Convert ArrayList to array 
        int[] res = new int[primes.size()]; 
        for (int i = 0; i < primes.size(); i++) { 
            res[i] = primes.get(i); 
        } 
        return res; 
    } 
  
    // Driver method 
    public static void main(String[] args) 
    { 
        int[] arr = { 1, 2, 3, 4, 5, 6, 7, 8, 9 }; 
        int[] res = findPrimesInArray(arr); 
        for (int i = 0; i < res.length; i++) { 
            System.out.print(res[i] + " "); 
        } 
    } 
}

2 3 5 7
, since those are the prime numbers in the array.
