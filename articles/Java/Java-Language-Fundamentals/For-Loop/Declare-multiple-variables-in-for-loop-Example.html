Declare Multiple Variables in For Loop Example

The Java 
for
loop is a control statement that makes it possible to execute a block of code multiple times based on a condition. It is often used to perform a certain task for a certain number of times. The for loop can also be used to declare multiple variables, making them available for use inside the for loop. This article will explain how to declare multiple variables in a for loop with code examples.

Declare Multiple Variables in a For Loop

To declare multiple variables in a for loop, use the following syntax:


          
           
for ( datatype variable1, datatype variable2, ... ; condition ; increment/decrement ) {
        // code block
}
In the above syntax, 
datatype
, 
variable1
, 
variable2
, 
condition
and 
increment/decrement
are all placeholder variables. Replace these placeholders with the appropriate values. For example, to declare two integer variables 
i
and 
j
with values 1 and 10 respectively and print their sum, use the following code: 

          
           
for (int i=1, j=10; i<=j; i++) {
	int sum = i + j;
	System.out.println("Sum is " + sum);
}
The output of the above code is:

Sum is 20
Similarly, to declare two double variables 
x
and 
y
with values 0.5 and 10.5 respectively, and print the product, use the following code: 

          
           
for (double x=0.5, y=10.5; x<=y; x++) {
	double product = x * y;
	System.out.println("Product is " + product);
}
