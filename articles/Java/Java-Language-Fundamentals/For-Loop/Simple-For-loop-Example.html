Java For Loop Example

For loops are a common programming language structure which allows code to repeat until a certain condition is met. Java is one language which makes use of for loops, and in this article we will cover an example of a for loop and provide a few examples.

For Loop Structure

In general, a for loop consists of three components: An initialization statement, a loop-continuation condition, and an increment statement. In Java the for loop has the following basic structure: 

          
           
for (initialization; loop-continuation-condition; increment) {
   statement(s)
}
Here is an example of a for loop in Java which prints out the numbers from 1 to 10: 

          
           
for (int i=1; i<=10; i++) {
   System.out.println(i);
}
The output of this loop will be: 

          
           
1
2

10

For Loop With an Array

It is also possible to use a for loop with an array, as seen in the following example: 

          
           
int[] numbers = { 1, 2, 3, 4, 5 };

for (int i : numbers) {
   System.out.println(i);
}
The output of this loop will be: 

          
           
1
2
3
4
5

Nested For Loops

Nested for loops are for loops which are nested within other for loops. They can be used to iterate over two or more collections at the same time. Here is an example of a nested for loop in Java: 

          
           
for (int i=0; i<10; i++) {
   for (int j=0; j<10; j++) {
      System.out.println("i is "+i+" and j is "+j);
   }
}
