Convert String to Short in Java

In Java, you can convert a String to a short value by using the parseShort() method of the Short class. You can also use other methods such as the getShort() method of the Byte class if you need to do more complex conversions.

Using the parseShort() Method

The simplest way to convert a String to a short is to use the parseShort() method of the Short class. This method takes a String as an argument and returns a short that is parsed from the String. Here is an example:


          
           
String str = "35";
short s = Short.parseShort(str);
System.out.println(s); // prints 35

This code snippet parses the String "35" into a short value and prints it to the console. As you can see, the output is the same as what the String was represented as.

Using the getShort() Method

The getShort() method of the Byte class is another way to convert a String to a short. This method takes a String as an argument and returns a short that is parsed from the String. Here is an example:


          
           
String str = "35";
short s = Byte.getShort(str);
System.out.println(s); // prints 35

This code snippet parses the String "35" into a short value and prints it to the console. As you can see, the output is the same as what the String was represented as.

Converting Other Types of Strings to Shorts

You can also use the parseShort() and the getShort() methods to convert other types of Strings into shorts. For example, if you want to convert a hexadecimal String to a short, you can use the following code:


          
           
String str = "0xFF";
short s = Short.parseShort(str, 16);
System.out.println(s); // prints 255

In this example, we pass the radix 16 (hexadecimal) as a second argument to the parseShort() method, so it knows how to interpret the String correctly.

We can also use the getShort() method for this conversion. Here is an example:


          
           
String str = "0xFF";
short s = Byte.getShort(str, 16);
System.out.println(s); // prints 255

As you can see, both methods are equally effective for converting hexadecimal Strings to shorts.

Conclusion

In this article, we looked at how to convert a String to a short in Java. We explored two methods, the parseShort() method of the Short class and the getShort() method of the Byte class. Both of these methods can be used to convert any type of String to a short, including hexadecimal Strings.
