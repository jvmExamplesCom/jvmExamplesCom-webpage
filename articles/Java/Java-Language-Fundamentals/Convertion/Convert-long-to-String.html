Converting Long to String in Java

Java provides a number of built-in ways to convert a Long value to a String. In this article, we'll take a look at some of the different and most popular options.

Using String.valueOf()

The simplest way is to use the 

          
           
String.valueOf()
method. This method takes a long as an argument and returns its string representation: 

          
           
String str = String.valueOf(123456L); // "123456"

Using Long.toString()

The 

          
           
Long.toString()
method is another way to convert a long value to a String. This method takes one argument, a long, and returns its String representation: 

          
           
String str = Long.toString(123456L); // "123456"

Using Long.toString(radix)

If you need to convert a long to a string with a given radix, the 

          
           
Long.toString(radix)
method is what you're looking for. This method takes two arguments, the long and a radix. It returns the string representation of the long in the given radix. For example, to convert the long 123456 to a string in base 16 (hexadecimal): 

          
           
String hexStr = Long.toString(123456L, 16); // "1e240"

Using Long.toUnsignedString()

The 

          
           
Long.toUnsignedString()
method is like the 

          
           
Long.toString()
method but it returns the unsigned representation of the long. This means that a negative long will be converted to its positive counterpart. For example: 

          
           
String unsignedStr = Long.toUnsignedString(-123456L); // "18446744073709551440"

Using Long.toHexString()

The 

          
           
Long.toHexString()
method is a convenience method for converting a long to a hexadecimal string. It takes one argument, a long, and returns its hexadecimal string representation. For example: 

          
           
String hexStr = Long.toHexString(123456L); // "1e240"
