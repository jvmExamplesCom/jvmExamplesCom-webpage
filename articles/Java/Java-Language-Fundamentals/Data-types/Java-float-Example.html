Java Float Example

A float is a data type that holds a numerical value that is a single precision 32-bit IEEE 754 floating point. It is used to represent decimal values, such as 3.14. In Java, there are two types of floats: float and double. Float is a single-precision 32-bit IEEE 754 computer number format. Double is a double-precision 64-bit IEEE 754 computer number format. Float takes four bytes of memory while Double takes 8 bytes.

Declaring a float in Java

To declare a float in Java, use the following syntax: 

          
           
float myFloat = 3.14;
In this example, "myFloat" is the name of the float and the value of the float is 3.14.

Operators for float

The following operators can be used with float values: * Addition ( + ) * Subtraction ( - ) * Multiplication ( * ) * Division ( / ) * Modulus ( % )

Example of using Operators with floats


          
           
float x = 2.5;
float y = 3.3;

float result = x + y; // result is 5.8
In this example, we have declared two floats, x and y, and then added them together to get the result 5.8.

Comparing floats in Java

It is possible to compare two floats in Java using the following comparison operators: * Greater than ( > ) * Less than ( < ) * Greater than or equal to ( >= ) * Less than or equal to ( <= )

Example of comparing floats


          
           
float x = 2.5;
float y = 3.3;

boolean isXGreaterThanY = x > y; // isXGreaterThanY is false
In this example, we have declared two floats, x and y, and then compared them using the greater than operator. The result is false, since x is not greater than y.

Rounding floats

It is possible to round a float in Java using the Math.round() method. This method takes a double value as an argument and returns the closest long value.

Example of rounding a float


          
           
float x = 2.5;
long roundedX = Math.round(x); // roundedX is 3
In this example, we have declared a float, x, and then rounded it using the Math.round() method. The result is 3, since 2.5 is closer to 3 than to 2.
