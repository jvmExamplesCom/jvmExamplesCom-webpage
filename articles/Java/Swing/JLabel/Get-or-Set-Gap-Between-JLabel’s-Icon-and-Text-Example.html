Get or Set Gap Between JLabel’s Icon and Text Example

Java Swing is an API used for developing Graphical User Interfaces (GUI) with Java. Swing Components are components like buttons, labels, text fields which provide the building blocks for graphical user interfaces. A JLabel is one such component which is used to display a short text string or an image, or both. In this tutorial we look at how to set and get the gap between a JLabel's icon and text.

Setting the gap Between the Icon and Text

The amount of gap between a JLabel's icon and text can be set with the setIconTextGap method of the JLabel class. This method takes an int argument which specifies the gap in pixels. Here is an example: 

          
           
JLabel label = new JLabel("Label Text");
label.setIcon(new ImageIcon("icon.png"));
label.setIconTextGap(10);

Getting the Gap Between the Icon and Text

To get the gap between a JLabel's icon and text we use the getIconTextGap method. This returns an int value which represents the gap in pixels. Here is an example: 

          
           
int gap = label.getIconTextGap();

Putting it All Together

Here is an example of setting and getting the gap between the JLabel's icon and text: 

          
           
public class LabelExample {
    public static void main(String[] args) {
        JLabel label = new JLabel("Label Text");
        label.setIcon(new ImageIcon("icon.png"));
        label.setIconTextGap(10);
        int gap = label.getIconTextGap();
        System.out.println("Gap between the icon and text is: " + gap);
    }
}

Gap between the icon and text is: 10
