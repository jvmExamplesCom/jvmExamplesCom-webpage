JLabel Horizontal Text Position Example

A JLabel is a display area for a short text, an image, or both. It can display a text or an image or both. The JLabel class provides several constructors used to create labels. This tutorial will guide you through the steps of setting the horizontal position of a text in a JLabel component.

Example Program: Setting Horizontal Position of Text in JLabel Component

The following program demonstrates how to set the horizontal position of a text in a JLabel component: 

          
           
import javax.swing.JFrame;
import javax.swing.JLabel;

public class JLabelHorizontalTextPositionExample {
    public static void main(String[] args) {
        // Create a frame
        JFrame frame = new JFrame("JLabel Horizontal Text Position Example");

        // Create a label
        JLabel label = new JLabel("Hello World!");

        // Set the horizontal position of the text to RIGHT
        label.setHorizontalTextPosition(JLabel.RIGHT);

        // Add the label to the frame
        frame.add(label);

        // Set the size of the frame
        frame.setSize(200, 100);

        // Set the frame visible
        frame.setVisible(true);
    }
}
By default, the horizontal position of the text in a JLabel component is set to LEFT.In this example, we have set it to RIGHT by calling the setHorizontalTextPosition() method of the JLabel class.
