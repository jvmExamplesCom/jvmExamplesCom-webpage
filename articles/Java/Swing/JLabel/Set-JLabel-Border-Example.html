Java Set JLabel Border Example

Java's Swing library provides many powerful components for creating user interfaces. One such component is the 

          
           
JLabel
object. In this tutorial, we will discuss how to set the border of a 

          
           
JLabel
component in Java. Creating a 

          
           
JLabel
object is simple. You can do it either by creating a new 

          
           
JLabel
instance and passing it a text string as an argument: 

          
           
JLabel label = new JLabel("Text");
Or by using a constructor that allows you to specify the text string and an icon: 

          
           
Icon icon = new ImageIcon("image.jpg");
JLabel label = new JLabel("Text", icon, JLabel.LEFT);
Now that you have created your 

          
           
JLabel
component, you can set its border as follows: 

          
           
Border lineBorder = BorderFactory.createLineBorder(Color.BLACK);
label.setBorder(lineBorder);
The 

          
           
createLineBorder()
method takes in a 

          
           
Color
argument and creates a line border with the specified color. Other than a line border, there are other types of borders you can create as well. For example, you can create an empty border with the following code: 

          
           
Border emptyBorder = BorderFactory.createEmptyBorder();
label.setBorder(emptyBorder);
Similarly, you can also create an etched border as follows: 

          
           
Border etchedBorder = BorderFactory.createEtchedBorder();
label.setBorder(etchedBorder);
You can also create a raised or lowered bevel border, which has a 3D effect: 

          
           
Border raisedBevelBorder = BorderFactory.createRaisedBevelBorder();
label.setBorder(raisedBevelBorder);

Border loweredBevelBorder = BorderFactory.createLoweredBevelBorder();
label.setBorder(loweredBevelBorder);
Finally, you can also create a compound border that combines two or more borders together: 

          
           
Border lineBorder = BorderFactory.createLineBorder(Color.BLACK);
Border emptyBorder = BorderFactory.createEmptyBorder(5,5,5,5);
CompoundBorder compoundBorder = BorderFactory.createCompoundBorder(lineBorder, emptyBorder);
label.setBorder(compoundBorder);
In the code above, we have created a compound border that combines a line border and an empty border. This is just one of the many possibilities when creating a compound border. As you can see, setting the border of a 

          
           
JLabel
component in Java is quite easy. If you want to learn more about Java's Swing library and how to use it to build user interfaces, check out our comprehensive tutorials.
