Create JLabel With Text Example

This article will provide you with an example of how to create a 

          
           
JLabel

JLabel

JLabel

Create JLabel Object

To create a new 

          
           
JLabel
object, use the following syntax: 

          
           
JLabel label = new JLabel("My Label");
This will create a 

          
           
JLabel
object named 

          
           
label
with the text 

          
           
"My Label"
.

Set Text for JLabel

You can change the text for your 

          
           
JLabel
object by using the 

          
           
setText()

label.setText("My New Label");

label

"My New Label"
.

JLabel
with both text and an icon. To do this, use the following syntax: 

          
           
ImageIcon icon = new ImageIcon("myicon.png");
JLabel label = new JLabel("My Label", icon, JLabel.LEFT);
