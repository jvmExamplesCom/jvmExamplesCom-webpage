JLabel

Create JLabel Object

To create a new 

          
           
JLabel
object, use the following syntax: 

          
           
JLabel label = new JLabel("My Label");
This will create a 

          
           
JLabel
object named 

          
           
label
with the text 

          
           
"My Label"
.

Set Text for JLabel

You can change the text for your 

          
           
JLabel

setText()

"My New Label"
.

JLabel

ImageIcon icon = new ImageIcon("myicon.png");
