Convert java Double to Double object Example

Code Examples

To convert a double primitive type to a Double object, you can use the 


          
           
Double.valueOf(double d)
method. 

          
           
double d = 12.5;
Double obj = Double.valueOf(d);
The 
Double.valueOf()
method is a static factory method that returns an instance of the Double class. It is equivalent to invoking the constructor 
Double(double)
.

Example

In this example, we'll use the 
Double.valueOf()
method to convert a double primitive type to a Double object. 

          
           
public class ConvertDoubleToDoubleObject {
    public static void main(String[] args) {
        double d = 12.5;
        Double obj = Double.valueOf(d);
        System.out.println("Double object value  : " + obj);
        System.out.println("Type of returned obj: " + obj.getClass().getName());
    }
}
// Outputs
// Double object value  : 12.5
// Type of returned obj: java.lang.Double
As you can see, the 
Double.valueOf()
method returns an instance of the Double class and the primitive double value has been successfully converted to the Double object.

Performance

As a static factory method, the 
Double.valueOf()
method is more performant than the constructor 
Double(double)
, since it doesn't create a new object each time it's invoked. The 
Double.valueOf()
method caches instances of objects that represent a range of values from -128 to 127. When an application invokes this method repeatedly with a given value, it will receive the same instance of a Double object each time.

Conclusion

Double.valueOf()
method.
