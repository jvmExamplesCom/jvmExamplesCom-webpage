Converting Long Object to Bytes, Shorts, Ints, Longs, Floats, and Doubles

Java provides multiple helper methods to convert Long objects to other data types. It's important to remember that all of these conversions involve losing precision, meaning that some information will be lost when going from a larger data type to a smaller data type. This article explores the various conversion options and provides code examples.

Long to Byte

The simplest way to convert a Long object to a (signed) byte is to use the 

          
           
longValue()
method, which returns the value of the Long as a 

          
           
long
. The returned value can then be cast to a 

          
           
byte
: 

          
           
Long l = Long.valueOf(123);
byte b = (byte) l.longValue();
System.out.println(b); // prints 123

Long to Short

The Long type has the convenient 

          
           
shortValue()
method which returns the value of the Long as a 

          
           
short
: 

          
           
Long l = Long.valueOf(123);
short s = l.shortValue();
System.out.println(s); // prints 123

Long to Integer

Similar to the approach used for bytes, you can use the 

          
           
longValue()
method to return the value of the Long as a 

          
           
long
, then cast it to an 

          
           
int
: 

          
           
Long l = Long.valueOf(123);
int i = (int) l.longValue();
System.out.println(i); // prints 123

Long to Long

We can use the same approach used for converting to integers, but this time the cast is not necessary since the value of the Long is already of type 

          
           
long
: 

          
           
Long l = Long.valueOf(123);
long l2 = l.longValue();
System.out.println(l2); // prints 123

Long to Float

The Float class provides the static 

          
           
floatToIntBits()
method, which we can use to convert a Long to a Float: 

          
           
Long l = Long.valueOf(123);
float f = Float.intBitsToFloat(l.longValue());
System.out.println(f); // prints 123.0

Long to Double

Lastly, we can use the static 

          
           
doubleToLongBits()
method of the Double class to convert a Long to a Double: 

          
           
Long l = Long.valueOf(123);
double d = Double.longBitsToDouble(l.longValue());
System.out.println(d); // prints 123.0
