Converting an Integer Object to byte, short, int, long, float and double

In this article, we will discuss how to convert an Integer object into common data types such as byte, short, int, long, float and double. Java provides several ways to convert an Integer Object into these primitive data types and we will discuss the most common ones here.

Converting to a byte

The following code demonstrates how to convert an Integer Object to a byte: 

          
           
Integer num = new Integer(25);
byte b = num.byteValue();
System.out.println("Converted to byte: " + b);
Output 

          
           
Converted to byte: 25

Converting to a short

To convert an Integer Object to a short, use the shortValue() method: 

          
           
Integer num = new Integer(25);
short s = num.shortValue();
System.out.println("Converted to short: " + s);
Output 

          
           
Converted to short: 25

Converting to an int

To convert an Integer Object to an int, use the intValue() method: 

          
           
Integer num = new Integer(25);
int i = num.intValue();
System.out.println("Converted to int: " + i);
Output 

          
           
Converted to int: 25

Converting to a long

To convert an Integer Object to a long, use the longValue() method: 

          
           
Integer num = new Integer(25);
long l = num.longValue();
System.out.println("Converted to long: " + l);
Output 

          
           
Converted to long: 25

Converting to a float

To convert an Integer Object to a float, use the floatValue() method: 

          
           
Integer num = new Integer(25);
float f = num.floatValue();
System.out.println("Converted to float: " + f);
Output 

          
           
Converted to float: 25.0

Converting to a double

To convert an Integer Object to a double, use the doubleValue() method: 

          
           
Integer num = new Integer(25);
double d = num.doubleValue();
System.out.println("Converted to double: " + d);
Output 

          
           
Converted to double: 25.0
