Converting Primitive to Wrapper Objects

Java provides two classes in java.lang package to deal with primitive data type values: 
  • java.lang.Integer
    — deals with integer values (int)
  • java.lang.Float
    — deals with floating-point values (float)
These classes are called wrapper classes because they “wrap” the primitive data type inside an object. Wrapper objects are useful for certain operations, such as sorting, that can't be done using a primitive type directly.

Autoboxing and Unboxing

In Java 5, it was necessary to write extra code to convert from a primitive data type to its corresponding wrapper class. However, with the introduction of auto-boxing and unboxing, this task is now automated by the Java compiler, so you don't have to write the extra code manually.

Auto-boxing

Auto-boxing is the process of automatically converting a primitive data type to its corresponding wrapper class when it is required. For example, 

          
           
double d = 1.0;
Double dObj = d; // auto-boxing
Here, the 

          
           
double
primitive value 

          
           
d
is automatically converted by the compiler to its corresponding wrapper class 

          
           
Double
.

Unboxing

Unboxing is the process of automatically converting a wrapper class to its corresponding primitive data type when it is required. For example, 

          
           
Integer iObj = new Integer(1);
int i = iObj; // unboxing
Here, the 

          
           
Integer
wrapper class 

          
           
iObj
is automatically converted by the compiler to its corresponding primitive data type 

          
           
int
.

Manual Conversion

Although auto-boxing and unboxing make the conversion process simpler and automated, there may still be some cases where you need to perform a manual conversion. For example, if you have an 

          
           
int
value and you need to convert it to an 

          
           
Integer
wrapper object, you can use the 

          
           
valueOf()
method of the 

          
           
Integer
class: 

          
           
int i = 1;
Integer iObj = Integer.valueOf(i);
Similarly, if you have an 

          
           
Integer
wrapper object and you need to convert it to an 

          
           
int
primitive value, you can use the 

          
           
intValue()
method of the 

          
           
Integer
class: 

          
           
Integer iObj = new Integer(1);
int i = iObj.intValue();

Conclusion

To sum up, auto-boxing and unboxing allow you to convert a primitive data type to its corresponding wrapper class, and vice versa, automatically. However, if required, you can also manually convert a primitive data type to its corresponding wrapper class, or vice versa, using the 

          
           
valueOf()
and 

          
           
intValue()
methods, respectively.
