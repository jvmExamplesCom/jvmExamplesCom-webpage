Convert Java Float to Numeric Primitive Data Types Example

The Java programming language has several primitive data types including float that allow developers to work with decimal numbers. Converting a float to one of the other primitive data types is a common programming task. In this article, we'll discuss the principles of conversion and walk through a few code examples.

Exploring the float Primitive Type

Float is represented in Java as a 32-bit value that can hold values from ---3.4E+38 to +3.4E+38. It can also store NaN, which stands for Not a Number. Additionally, float values can be expressed in scientific notation.

Converting Float to Other Primitive Types

A float can be converted to an integer through the use of an explicit cast. The example below shows how to cast a float to an integer: 

          
           
float f = 5.5;
int x = (int)f; // cast the float to an int

System.out.println("Float value: " + f);
System.out.println("Integer value: " + x);
In this code example, the output will be: 

          
           
Float value: 5.5
Integer value: 5
Note that when casting a float to an integer, any digits after the decimal point will be truncated. To convert a float to a double, you can simply assign it to a double variable. There is no need to use an explicit cast. The example below shows how this can be done: 

          
           
float f = 5.5;
double d = f; // assign float to a double

System.out.println("Float value: " + f);
System.out.println("Double value: " + d);
In this code example, the output will be: 

          
           
Float value: 5.5
Double value: 5.5
If a float value needs to be converted to a long, then again an explicit cast must be used. The example below shows how this can be done: 

          
           
float f = 5.5;
long l = (long)f; // cast the float to a long

System.out.println("Float value: " + f);
System.out.println("Long value: " + l);
In this code example, the output will be: 

          
           
Float value: 5.5
Long value: 5
Just like when converting to an integer, any digits after the decimal point will be truncated when converting to a long.

Conclusion

In this article, we explored how to convert a float to other primitive data types in Java. We discussed the principles of conversion and walked through a few code examples. Be sure to keep these principles in mind when writing your own programs that involve converting floats to other primitive types.
