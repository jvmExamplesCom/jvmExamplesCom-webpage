Convert Java Float Object to String Object Example

Java's Float class provides us the float data type value. By using the Float class, we can convert float primitive type to Float object. In order to convert Float Object to String Object, we can use the toString() method of Float class. Let's take a look at the example of converting Float Object to String Object with the help of toString() method

Example 1: Using toString() Method


          
           
public class FloatToString {
  public static void main(String[] args) {
    Float floatObj = new Float(45.5f);
    String strObj = floatObj.toString();
    System.out.println("Float Object: "+floatObj);
    System.out.println("String Object: "+strObj);
  }
}
Output: 

          
           
Float Object: 45.5
String Object: 45.5
In the above example, we have created a Float object floatObj with value 45.5f. Then we have used toString() method of the Float class to convert our Float object to a String object. Finally, we have printed the Float and String object values.

Example 2: Using String.valueOf() Method


          
           
public class FloatToString {
  public static void main(String[] args) {
    Float floatObj = new Float(45.5f);
    String strObj = String.valueOf(floatObj);
    System.out.println("Float Object: "+floatObj);
    System.out.println("String Object: "+strObj);
  }
}
Output: 

          
           
Float Object: 45.5
String Object: 45.5
In this example, we have used the String.valueOf() method to convert our Float object to String object. The String.valueOf() method takes the argument as an Object and returns the equivalent String representation of the argument. We have passed our Float object as an argument and rate the String equivalent of it.

Example 3: Using StringBuilder Class


          
           
public class FloatToString {
  public static void main(String[] args) {
    Float floatObj = new Float(45.5f);
    String strObj = new StringBuilder().append(floatObj).toString();
    System.out.println("Float Object: "+floatObj);
    System.out.println("String Object: "+strObj);
  }
}
Output: 

          
           
Float Object: 45.5
String Object: 45.5
In this example, we have used the StringBuilder class to convert our Float object to String object. The StringBuilder class provides us the append() method that takes the argument as an Object and appends the String equivalent of the argument to the String. We have used the StringBuilder class to append the String equivalent of our Float object and returned the String object. Example 4: Using String.format() Method


          
           
public class FloatToString {
  public static void main(String[] args) {
    Float floatObj = new Float(45.5f);
    String strObj = String.format("%f", floatObj);
    System.out.println("Float Object: "+floatObj);
    System.out.println("String Object: "+strObj);
  }
}
Output: 

          
           
Float Object: 45.5
String Object: 45.5000000
In this example, we have used the String.format() method to convert our Float object to String object. The format() method takes the argument as a format specifier and returns the formatted string. We have passed our Float object as a format specifier to the format() method and returned its string equivalent.
