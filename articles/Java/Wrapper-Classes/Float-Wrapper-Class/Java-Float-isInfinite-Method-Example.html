Java Float isInfinite Method Example

The Float class provides the isInfinite() method, which determines whether a given float value is infinite or not. It returns true if the given float value is an infinite, otherwise it returns false. This method belongs to the Float class which falls under the java.lang package.

Syntax


          
           
public static boolean isInfinite(float f)

Parameter

The isInfinite() method accepts a single parameter of float type. This parameter represents the float value which should be checked for infinite value.

Return Value

This method returns a boolean value true if the parameter is an infinite, otherwise it returns false.

Example 1: Determine whether a float value is infinite or not


          
           
public class IsInfiniteExample {
 
  public static void main(String[] args) {
 
    float a = Float.NEGATIVE_INFINITY;
    System.out.println("a == Float.NEGATIVE_INFINITY : " + Float.isInfinite(a));
 
    float b = (float) 1.0 / 0.0;
    System.out.println("b == (float)1.0/0.0 : " + Float.isInfinite(b));
 
    float c = (float) 0.0 / 0.0;
    System.out.println("c == (float)0.0/0.0 : " + Float.isInfinite(c));
  }
}

          

          
           
Output:
a == Float.NEGATIVE_INFINITY : true
b == (float)1.0/0.0 : true
c == (float)0.0/0.0 : false
In the above example, we have declared three float variables a, b, and c. Then we have checked each of the values using the isInfinite() method. We can see from the output that the first two values are infinity and the last value is not an infinite.
