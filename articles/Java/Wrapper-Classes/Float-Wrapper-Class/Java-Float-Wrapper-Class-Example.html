Java Float Wrapper Class Example

A wrapper class is a class that encapsulates a primitive data type. The Java language provides wrapper classes for each of the primitive data types. In this article, we'll take a look at the Float wrapper class in Java and provide some examples of how to use it.

Creating a Float Object

We can create a Float object from a float or double value like this: 

          
           
Float f1 = new Float(3.14);
Float f2 = new Float(2.718);
Or we can create a Float object using a String representation of the value: 

          
           
Float f3 = new Float("3.14");
Float f4 = new Float("2.718");

Converting float to Float

We can convert a float to a Float like this: 

          
           
float f1 = 3.14;
Float f2 = Float.valueOf(f1);

Comparing Floats

We can compare two Float objects like this: 

          
           
Float f1 = new Float(3.14);
Float f2 = new Float(2.718);

if (f1.equals(f2)) {
    System.out.println("f1 and f2 are equal");
} else {
    System.out.println("f1 and f2 are not equal");
}
To compare two Floats using their primitive values, we can use the compareTo() method: 

          
           
Float f1 = new Float(3.14);
Float f2 = new Float(2.718);

if (f1.compareTo(f2) == 0) {
    System.out.println("f1 and f2 are equal");
} else if (f1.compareTo(f2) < 0) {
    System.out.println("f1 is less than f2");
} else {
    System.out.println("f1 is greater than f2");
}

Formatting Floats

We can format floating-point numbers with the DecimalFormat class. For example: 

          
           
Float f1 = new Float(3.14);
DecimalFormat df = new DecimalFormat("#0.00");
System.out.println(df.format(f1)); // prints 3.14

Rounding Floats

We can round a Float to the nearest integer using the round() method: 

          
           
Float f1 = new Float(3.14);
long rounded = Math.round(f1);
System.out.println(rounded); // prints 3
We can also use the round() method to round a Float to a specified number of decimal places: 

          
           
Float f1 = new Float(3.14159);
double rounded = Math.round(f1 * 100.0) / 100.0;
System.out.println(rounded); // prints 3.14

Conclusion

In this article, we looked at the Float wrapper class in Java and provided examples of how to use it. We saw how to create Float objects, convert float values to Floats, compare Floats, format Floats, and round Floats.
