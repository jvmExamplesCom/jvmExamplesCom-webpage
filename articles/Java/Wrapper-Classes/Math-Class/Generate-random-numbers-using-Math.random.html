Generating Random Numbers with Java's Math.random() Method

Java's Math.random() method can be used to generate random numbers. It is a Pseudo-Random Number Generator (PRNG) that returns a floating-point value between 0 (inclusive) and 1 (exclusive). This floating-point number can then be scaled to meet the desired range and rounded off to the desired precision.

Syntax for the Math.random() Method

The Math.random() method takes no arguments and returns a floating-point value between 0 (inclusive) and 1 (exclusive): 

          
           
double randomNumber = Math.random();

Examples of Generating Random Numbers with Math.random()

Generating a Random Integer Between Two Values (inclusive)

The following code example demonstrates how to generate a random integer between two values, x and y (inclusive): 

          
           
// Generate a random integer between x and y (inclusive)
int x = 10;
int y = 20;
int range = y-x+1; 
int randomNumber = (int)(Math.random()*range)+x;
