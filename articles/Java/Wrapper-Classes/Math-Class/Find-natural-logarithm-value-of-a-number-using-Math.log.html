Find Natural Logarithm Value Of A Number Using Math.log

The natural logarithm of a number is the power to which e (2.71828...) must be raised to equal that number. The natural logarithm of x is typically written as ln x and pronounced "loge x".

In Java, you can use the 
Math.log()
static method to find the natural logarithm (base e) of a number. This method takes a single argument which must be a double value, and returns its natural logarithm.

Example - Finding The Natural Logarithm Of 10

To find the natural logarithm of 10 using the 
Math.log()
method, we can write the following code: 

          
           
double result = Math.log(10);
System.out.println(result);
This will output the following: 

          
           
2.302585092994046
