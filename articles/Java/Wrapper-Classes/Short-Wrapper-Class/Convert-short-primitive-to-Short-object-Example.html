Converting a Short Primitive to Short Object Example in Java

A Short object in Java is an object that wraps a short primitive value and provides numerous utility methods. Converting a short primitive value to a Short object can be done using the 

valueOf()
static method of the 
Short
class.

Example 1: Converting a Short Primitive To Short Object

In the code example below, we are converting a short primitive value to a Short object. This is done by passing the short value to the 

valueOf()
static method of the 
Short
class. 

          
           
short shortVal = 7; 
Short shortObjectVal = Short.valueOf(shortVal); 
System.out.println(shortObjectVal);
The output of this program will be: 

          
           
7

Example 2: An Alternate Way to Convert a Short Primitive to a Short Object

An alternate way to convert a short primitive to a Short object is to use the constructor 

Short(short val)
. This constructor creates a Short object with the given short value. 

          
           
short shortVal = 7; 
Short shortObjectVal = new Short(shortVal); 
System.out.println(shortObjectVal);
The output of this program will be: 

          
           
7
