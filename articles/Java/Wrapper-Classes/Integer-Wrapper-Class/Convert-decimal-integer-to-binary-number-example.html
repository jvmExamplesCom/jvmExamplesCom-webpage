Decimal Integer to Binary Number Conversion Example with Java

Java provides many tools for developers to convert an integer from its decimal form to its binary representation. The following are a few examples of how to use these tools to convert a decimal integer to a binary number.

Example 1: Using Integer.toBinaryString() Method

The 

          
           
Integer.toBinaryString()
method is the simplest way to convert a decimal integer to a binary number. This example shows how to use this method to convert the decimal integer 
42
to its binary representation: 

          
           
int decimal = 42;
String binary = Integer.toBinaryString(decimal);
System.out.println("Binary representation of " + decimal + " is " + binary);
// Outputs: "Binary representation of 42 is 101010"

Example 2: Using Bitwise Operators

If you want more control over the conversion process, you can use bitwise operators to manually convert a decimal integer to a binary number. This example converts the decimal integer 

42
to its binary representation: 

          
           
int decimal = 42;
String binary = "";

while (decimal > 0) {
    int remainder = decimal % 2;
    binary = remainder + binary;
    decimal /= 2;
}

System.out.println("Binary representation of " + decimal + " is " + binary);
// Outputs: "Binary representation of 42 is 101010"

Example 3: Using BigInteger Class

If you need to convert a large decimal integer to its binary representation, you can use the 

BigInteger
class. This example converts the decimal integer 
123456789
to its binary representation: 

          
           
BigInteger decimal = BigInteger.valueOf(123456789);
String binary = decimal.toString(2);
System.out.println("Binary representation of " + decimal + " is " + binary);
// Outputs: "Binary representation of 123456789 is 111010110111100110100010101"
