Converting Java Integer Object to Numeric Primitive Types

Java Integer objects are references of type java.lang.Integer, used to store integer values. They are useful when you want to store an int value as an object, or pass an int value as a parameter. However, in some cases you may need to convert the Integer object to one of the primitive numerical types. This article will explain how to convert an Integer object to a numeric primitive type in Java.

Using the intValue() method

The simplest way to convert an Integer object to a numeric primitive type is to use the intValue() method. This method will return a primitive int value from an Integer object. The following example shows how to use this method: 

          
           
Integer myIntObj = new Integer(5);
int myIntPrimitive = myIntObj.intValue();
System.out.println(myIntPrimitive); // Outputs 5

Using the parseInt() method

Another way to convert an Integer object to a primitive int is to use the static parseInt() method of the Integer class. The syntax for this method takes a String argument and returns its integer value as an int primitive. The following example shows how to use this method: 

          
           
String myIntStr = "5";
int myIntPrimitive = Integer.parseInt(myIntStr);
System.out.println(myIntPrimitive); // Outputs 5

Using Typecasting

You can also convert an Integer object to a numeric primitive type by typecasting it. The syntax for this is as follows: 

          
           
Integer myIntObj = new Integer(5);
int myIntPrimitive = (int)myIntObj;
System.out.println(myIntPrimitive); // Outputs 5

Conclusion

In this article, we looked at how to convert an Integer object to a numeric primitive type in Java. We discussed three different ways to do this: using the intValue() method, using the parseInt() method, and using typecasting. All three methods produce the same result, so choose the one that fits best with your code.
