Converting Strings to Long Example

Java provides many ways to convert strings to longs, as long as the string contains a valid representation of a number. As such, it is important to understand how to do this conversion correctly, and how to handle the exceptions that may arise during the process.

Using the Long.parseLong() Method

The most straightforward way to convert a string to a long is to use the 

Long.parseLong()
method. This method takes a single string argument and will return the corresponding long value. 

          
           
String numString = "123456";
long numLong = Long.parseLong(numString);
System.out.println(numLong); // prints 123456

Using the Long.valueOf() Method

Another option for converting a string to a long is to use the 

Long.valueOf()
method. Like its name implies, this method will return the 
Long
object corresponding to the argument. It also has several overloaded versions that accept different types of data. 

          
           
String numString = "123456";
Long numLong  = Long.valueOf(numString);
System.out.println(numLong); // prints 123456

Using the DecimalFormat Class

The 
DecimalFormat
class can also be used to convert strings to longs. This class provides various methods to convert strings to different types of numbers, including longs. 

          
           
String numString = "123456";
DecimalFormat format = new DecimalFormat("#");
long numLong = format.parse(numString).longValue();
System.out.println(numLong); // prints 123456

Handling Exceptions

When using any of the methods outlined above, it is important to remember that they may throw a 


          
           
NumberFormatException
. This type of exception is thrown when a string is not a valid representation of a number. Therefore, it is important to catch this exception in order to properly handle it. 

          
           
String numString = "invalid";
long numLong;
try {
  numLong = Long.parseLong(numString);
} catch (NumberFormatException ex) {
  System.err.println("Cannot convert string to number");
  numLong = 0;
}
System.out.println(numLong); // prints 0
