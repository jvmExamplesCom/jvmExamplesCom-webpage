Convert Java String to Long Example

In the Java programming language, you may often need to convert a string to a Long. Converting a String to a Long requires that the string is a valid representation of a number in a base 10 numeral system. Fortunately, Java makes this process simple with the parseLong() method of the Long class.

Method Syntax

The basic syntax for using the parseLong() method is as follows:


          
           
Long.parseLong(String s)

The parseLong() method takes a single argument, which is the value of the String that you want to convert. Note that the argument must be a valid representation of a number in base 10.

Example 1 - Convert String to Long

In this example, we use the parseLong() method to convert a String to a Long. We pass in the String "12345" as an argument, and the result is the Long 12345.


          
           
public class StrToLongExample {
    public static void main(String args[]){
         String str_num = "12345"; 
         long long_num = Long.parseLong(str_num); 
         System.out.println("The Long value is " + long_num); 
     } 
}
When we run this code, the output is: 

          
           
The Long value is 12345

Example 2 - Convert Hexadeximal String to Long

In this example, we use the parseLong() method to convert a hexadecimal String to a Long. We pass in the String "0xFFFF" as an argument, and the result is the Long 65535.


          
           
public class HexStrToLongExample {
    public static void main(String args[]){
         String str_hex = "0xFFFF"; 
         long long_num = Long.parseLong(str_hex, 16); 
         System.out.println("The Long value is " + long_num); 
     } 
}
When we run this code, the output is: 

          
           
The Long value is 65535

Example 3 - Convert Octal String to Long

In this example, we use the parseLong() method to convert an octal String to a Long. We pass in the String "0777" as an argument, and the result is the Long 511.


          
           
public class OctStrToLongExample {
    public static void main(String args[]){
         String str_hex = "0777"; 
         long long_num = Long.parseLong(str_hex, 8); 
         System.out.println("The Long value is " + long_num); 
     } 
}
When we run this code, the output is: 

          
           
The Long value is 511

Conclusion

As you can see, the Java programming language makes it easy to convert a String to a Long. Just remember to ensure that the String is a valid representation of a number in the appropriate base before attempting to parse it.
