Converting Long to Numeric Primitive Data Types in Java

In this article, we will learn how to convert a long value to different primitive data types in Java.

Long to int Conversion

We can convert a long type to an integer type by using the 

intValue()
method of the Long wrapper class. The following example demonstrates how to achieve this conversion: 

          
           
long l = 87654321L; 
int i = l.intValue();
System.out.println("The int value is: " + i);
The output of the above code is: 

          
           
The int value is: 87654321
