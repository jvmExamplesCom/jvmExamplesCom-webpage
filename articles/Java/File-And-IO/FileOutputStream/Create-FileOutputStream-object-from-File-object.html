Create FileOutputStream object from File Object

Java supports two different approaches for creating a 

          
           
FileOutputStream
object. The first approach involves using the 

          
           
File
class to create the desired object. The second approach uses the 

          
           
FileOutputStream
class directly to create the desired object.

Using the File Class to Create a FileOutputStream Object

The following code example creates a 

          
           
FileOutputStream
object from an existing 

          
           
File
object: 

          
           
File myFile = new File("myfile.txt");
FileOutputStream fos = new FileOutputStream(myFile);
In this example, we are creating a 

          
           
FileOutputStream
object from an existing 

          
           
File
object. Notice that we do not need to explicitly specify the path to the file.

Using FileOutputStream Class to Create a FileOutputStream Object

We can also use the 

          
           
FileOutputStream
class directly to create the desired object. The following code example creates a 

          
           
FileOutputStream
object from a specified path: 

          
           
FileOutputStream fos = new FileOutputStream("myfile.txt");
In this example, we are creating a 

          
           
FileOutputStream
object from a specified path. Notice that we need to explicitly specify the path to the file.

Writing the Contents of a File

Once a 

          
           
FileOutputStream
object has been created, it can be used to write contents to a file. The following code example writes a string of text to a file using a 

          
           
FileOutputStream
object: 

          
           
FileOutputStream fos = new FileOutputStream("myfile.txt");
String text = "Hello, World!";
byte[] bytes = text.getBytes();
fos.write(bytes);
fos.close();
In this example, we are writing a string of text to a file. Notice that we need to convert the string to a byte array before writing it to the file. We also need to close the 

          
           
FileOutputStream
object once we have finished writing the contents of the file.

Conclusion

In this article, we discussed two different approaches for creating a 

          
           
FileOutputStream
object from a 

          
           
File
object. We also discussed how to use a 

          
           
FileOutputStream
object to write contents to a file.
