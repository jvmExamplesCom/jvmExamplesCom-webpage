Read byte array using ByteArrayInputStream Example

Java provides the ByteArrayInputStream class, which allows users to read bytes from an array of primitive data types. This is useful for situations where you need to process or store the contents of a large array, but do not want to use extra memory to copy the contents of the array. In this example, we will use a ByteArrayInputStream object to print out a byte array. The program will read each byte in the array and print out its value. We will also demonstrate how to read from the ByteArrayInputStream object and write it to a file.

Creating a ByteArrayInputStream

To create a ByteArrayInputStream object, we must first define an array of bytes. The array can contain any type of primitive data type; in this example, we’ll use an array of 8-bit integers (bytes). 

          
           
byte[] data = {0x12, 0x34, 0x56, 0x78, 
                  0x90, 0xAB, 0xCD, 0xEF};
We then create a ByteArrayInputStream object, passing the byte array as a parameter to the constructor: 

          
           
ByteArrayInputStream bis = new ByteArrayInputStream(data);

Reading Bytes From a ByteArrayInputStream

To read a byte from the ByteArrayInputStream, we use the read() method. This method returns the next available byte from the stream. It also advances the stream’s “cursor” so that it points to the next byte in the array. 

          
           
int b = bis.read();
If the stream has reached the end of the array, the read() method will return -1.

Printing Each Byte of the Array

We can use a while loop and the read() method to print out each byte in the array. We keep calling the read() method until it returns -1, indicating that the end of the array has been reached. 

          
           
while (bis.available() > 0) {
   int b = bis.read();
   System.out.println("byte = " + b);
}

Writing the Byte Array to a File

We can use the write() method to write the contents of the ByteArrayInputStream to a file. To do this, we need to create a FileOutputStream object that points to the file we want to write to. 

          
           
String fileName = "data.dat";
FileOutputStream fos = new FileOutputStream(fileName);
Then we use a loop to read each byte from the ByteArrayInputStream and write it to the file. 

          
           
while (bis.available() > 0) {
   int b = bis.read();
   fos.write(b);
}
Finally, we close the streams. 

          
           
bis.close();
fos.close();

Summary

In this article, we looked at how to use a ByteArrayInputStream to read a byte array. We also saw how we can print out the values of each byte in the array, and how to write the array to a file.
