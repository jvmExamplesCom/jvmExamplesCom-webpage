Reading Files in Strings Using Java BufferedInputStream Example

In this article, we will show you how to read a file in String using Java BufferedInputStream Example. There are various methods available for reading a file in String, however, in this article, we will cover reading a file line-by-line using Java BufferedInputStream.

Using BufferedInputStream

Java provides the 
BufferedInputStream
class which is used to read data from a stream. The 
BufferedInputStream
class has a method 
readLine()
which reads the next line of data from the InputStream. The following example shows how to use the 
BufferedInputStream
class to read a file in String. 

          
           
import java.io.BufferedInputStream;
import java.io.FileInputStream;
import java.io.IOException;
 
public class ReadFileInString {
    public static void main(String[] args) {
        try {
            BufferedInputStream bis = new BufferedInputStream(new FileInputStream("data.txt"));
 
            StringBuilder sb = new StringBuilder();
            while (bis.available() > 0) {
                sb.append((char) bis.read());
            }
            System.out.println(sb.toString());
            bis.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

The above example reads the contents of a data.txt file into a String variable by using a 

BufferedInputStream
. We open the 
FileInputStream
and pass it to the constructor of 
BufferedInputStream
. Then, we create a 
StringBuilder
object and loop until there is data available for reading. Inside the loop, we append the data into the 
StringBuilder
and finally, print the String.
