Writing String as bytes to a file using DataOutputStream in Java

DataOutputStream is one of the classes in Java that helps in writing primitive data types and strings to an OutputStream. It can be used to convert any Java string or other primitive type data into bytes and write them to a file.

Steps to write String as bytes to a file Using DataOutputStream

The following are the steps to write a String as bytes to a file using DataOutputStream:

  1. Create a FileOutputStream object by providing a location for the output file.
  2. Create a DataOutputStream object by passing the FileOutputStream object to its constructor.
  3. Write the desired string as bytes in the DataOutputStream object using the writeBytes() method.
  4. Close the DataOutputStream object by calling the close() method.

Example

Let's take a look at a code example which demonstrates how to write a String as bytes to a file using DataOutputStream class:


          
           
import java.io.DataOutputStream;
import java.io.FileOutputStream;

public class App {
    public static void main(String[] args) {
        try { 
            // Create a FileOutputStream object
            FileOutputStream out = new FileOutputStream("file.dat"); 

            // Create a DataOutputStream object by passing the FileOutputStream object to its constructor
            DataOutputStream dos = new DataOutputStream(out);
            
            // Writing the String "Hello World!" as bytes in the DataOutputStream object
            dos.writeBytes("Hello World!");
            
            // Closing the DataOutputStream object
            dos.close();
        } catch (Exception e) { 
            System.out.println("Exception: " + e); 
        }
    }
}

Here, we have created a FileOutputStream object with a location for the output file. And then, a DataOutputStream object has been created by passing the FileOutputStream object to its constructor. After that, we wrote the desired string "Hello World!" into the DataOutputStream object using the writeBytes() method. Finally, we closed the DataOutputStream object by calling the close() method.

Now, let's verify if the code has worked as expected. To do this, open the file "file.dat" with a text editor and check for the content. It will show the same string "Hello World!" which we have written in the DataOutputStream.
