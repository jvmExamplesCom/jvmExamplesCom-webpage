Determining the number of bytes written to DataOutputStream in Java

DataOutputStream is a type of stream used by programs in the Java programming language to write primitive data types to an output stream. When writing primitive data types to a DataOutputStream, the number of bytes written can be determined based on the type of data being written. This article will demonstrate how to determine the number of bytes written to a DataOutputStream.

Using the DataOutputStream Write Methods

The DataOutputStream class has a number of write methods for writing primitive data types to an output stream. These methods all return the number of bytes written to the stream. For example, the following code writes a single byte to the DataOutputStream and prints out the number of bytes written:


          
           
DataOutputStream dos = new DataOutputStream(...);
int bytesWritten = dos.writeByte(1);
System.out.println("Bytes written: " + bytesWritten);

The above code will print out "Bytes written: 1" since a single byte was written to the DataOutputStream.

Working With Strings

When writing strings to a DataOutputStream, the number of bytes written will depend on the character encoding being used, as well as the length of the string. For example, using UTF-8, the following code will print out the number of bytes written when writing the string "Hello":


          
           
DataOutputStream dos = new DataOutputStream(...);
int bytesWritten = dos.writeUTF("Hello");
System.out.println("Bytes written: " + bytesWritten);

This will print out "Bytes written: 7" since the string "Hello" takes up 7 bytes when encoded with UTF-8.

Using the DataOutputStream Flush Method

Calling the DataOutputStream flush method will write any pending data to the output stream and return the number of bytes written. For example, the following code will write a single byte to the DataOutputStream and flush the stream, printing out the number of bytes written:


          
           
DataOutputStream dos = new DataOutputStream(...);
int bytesWritten = dos.writeByte(1);
bytesWritten += dos.flush();
System.out.println("Bytes written: " + bytesWritten);

The above code will print out "Bytes written: 1" since a single byte was written to the DataOutputStream.

Conclusion

In this article, we demonstrated how to determine the number of bytes written to a DataOutputStream in Java. We showed how to use the write methods and flush method to determine the number of bytes written as well as how the number of bytes written changes depending on the data type and character encoding being used.
