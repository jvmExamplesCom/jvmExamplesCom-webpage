Flush Output Stream in Java

The flush operation in Java is a method of the OutputStream class that is used for output buffering to control the flow of data from an output device, such as a file, network connection or printer. Flushing the output stream clears all previously written data and sends a signal to the device that data needs to be written.

Types of Output Streams

In Java, there are two types of output streams: buffered and unbuffered. Buffered output streams store data in an internal buffer, while unbuffered output streams send data directly to the device. Both types of output streams use the flush operation, which is defined in the java.io.OutputStream interface.

How to Use the Flush Operation

The flush operation can be used to flush the written data from the output stream before it is sent to the device. To use it, first create an OutputStream object by passing a FileOutputStream or other OutputStream to its constructor:
