Free Memory of Java Virtual Machine (JVM) Example

Java Virtual Machine (JVM) offers a way to access free memory for its programs. This can be useful in situations where the application requires additional memory or when a program is running out of memory due to an unexpected circumstance. In this article, we will demonstrate how to get the free memory from JVM using two code examples.

Example 1 – Getting Free Memory from JVM

In this example, we will take a look at the 

          
           
Runtime
class, which contains some methods for interacting with the JVM. We will use the 

          
           
freeMemory()
method to get the total amount of free memory available to the JVM: 

          
           
long freeMemory = Runtime.getRuntime().freeMemory();
System.out.println("Free Memory: " + freeMemory);
The output of the above code will look like this: 

          
           
Free Memory: 134217728

Example 2 – Calculating Free Memory Percentage

In this example, we will demonstrate how to calculate the percentage of free memory as compared to the total memory available to the JVM. To do this, we will use the 

          
           
totalMemory()
and 

          
           
maxMemory()
methods of the 

          
           
Runtime
class: 

          
           
long totalMemory = Runtime.getRuntime().totalMemory();
long maxMemory = Runtime.getRuntime().maxMemory();
double freeMemoryPercentage = (double)totalMemory / maxMemory * 100;
System.out.println("Free Memory % : " + freeMemoryPercentage);
The output of the above code will look like this: 

          
           
Free Memory % : 50.71682929717643
