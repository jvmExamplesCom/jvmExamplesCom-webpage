Getting the Java Classpath Using System Class

The 

          
           
System

System

Getting the Classpath

The first step to getting the classpath is to create a new instance of 

          
           
System
. This can be done by using the following code: 

          
           
System system = new System();

System
class, you can use the 

          
           
getProperty()
method to retrieve the classpath. This is done by passing the argument 

          
           
"java.class.path"

getProperty()
method. The following code snippet shows an example of how to do this: 

          
           
String classPath = system.getProperty("java.class.path");

ClassLoader classLoader = Thread.currentThread().getContextClassLoader();
