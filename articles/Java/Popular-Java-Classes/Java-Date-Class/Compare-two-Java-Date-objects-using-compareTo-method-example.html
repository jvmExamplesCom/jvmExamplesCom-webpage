Comparing Java Date Objects using compareTo Method

Java's 
java.util.Date
class provides comparison methods to compare two date objects. We can use the 

          
           
compareTo(Date anotherDate)
method to compare two dates and it will return a negative, 0 or positive integer depending on whether the current date is earlier, equal to or later than the supplied date.

Code Examples


          
           
public static void main(String[] args)
{
    // creating two Date objects
    Date date1 = new Date(2019, 11, 15);
    Date date2 = new Date(2018, 12, 23);

    // comparing two dates using compareTo() method
    int result = date1.compareTo(date2);

    // result will be 1 if date1 is greater than date2
    System.out.println(result);
}
The above code example will output 
1
because the first date, 
date1
, is greater than the second date, 
date2
.
