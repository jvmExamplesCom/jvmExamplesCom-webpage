Java String endsWith Example

The Java String endsWith() method is used to check whether the given string ends with a specified suffix. The endsWith() method is case sensitive, which means that if the suffix is in different case than the string, it will not return true.

Basic Example

The following example shows how to use the endsWith() method to check if a string ends with a specific character.


          
           
String str = "Hello World!";
boolean result = str.endsWith("!");
System.out.println(result); // true
In the above example, we have a string called 
str
, which holds the value 
"Hello World!"
. We then call the endsWith() method, passing in the character 
'!'
as an argument. This returns a boolean, which we store in the variable 
result
. We then print out the result, which is 
true
.

Multiple Characters

The endsWith() method can also be used to check if a string ends with multiple characters. Here is an example:


          
           
String str = "Hello World!";
boolean result = str.endsWith("ld!");
System.out.println(result); // true

In this example, we are checking to see if the string 

str
ends with the characters 
"ld!"
. Again, this returns a boolean, which is stored in the variable 
result
. When we print out this result, we get 
true
.

Ignoring Case

The endsWith() method can also be used to ignore case when checking for the last characters of a string. Here is an example:


          
           
String str = "Hello World!";
boolean result = str.endsWith("LD!");
System.out.println(result); // true

In this example, we are checking to see if the string 

str
ends with the characters 
"LD!"
. As the method is case-insensitive, this returns 
true
, even though the last characters of the string are 
"d!"
and not 
"LD!"
.

Negative Example

Finally, here is an example of the endsWith() method returning 

false
: 

          
           
String str = "Hello World!";
boolean result = str.endsWith("X");
System.out.println(result); // false

In this example, we are checking to see if the string 

str
ends with the character 
"X"
. As the string does not end with this character, the endsWith() method returns 
false
.

Conclusion

In this article, we looked at the Java String endsWith() method, which is used to check if a string ends with a specified suffix. We looked at examples of how to use the endsWith() method and examined a few scenarios in which it might return either 

true
or 
false
.
