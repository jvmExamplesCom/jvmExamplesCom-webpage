Java Date to String Example

Working with dates and times in Java requires an understanding of the Java Date class and how to convert dates from one format to another. In this article, we will take a look at converting a Java Date to a String using some basic formatting options and then examine some common patterns to use when working with Dates.

Java Date to String Conversion with SimpleDateFormat

The easiest way to convert a Java Date to a String is to use the SimpleDateFormat class. This class has a variety of constructors and methods for manipulating date and time values. To convert a date to a string, pass a Date object to the SimpleDateFormat.format() method. The following example shows how to convert a Date to a String containing the text “Monday, October 8, 2018”: 

          
           
SimpleDateFormat formatter = new SimpleDateFormat("EEEE, MMMM d,yyyy");
Date date = new Date();
String dateString = formatter.format(date);
System.out.println(dateString);
Output: 

          
           
Monday, October 8, 2018
In this example, the “EEEE, MMMM d, yyyy” pattern is used to specify the format of the output string. The “EEEE” part of the pattern specifies that the full name of the day should be used (e.g. Monday). The “MMMM” part of the pattern specifies that the full month name should be used (e.g. October). The “d” part of the pattern specifies that the day of the month should be used (e.g. 8). Finally, the “yyyy” part of the pattern specifies that the year should be used (e.g. 2018).

Java Date to String Conversion Patterns

The Java Date to String conversion can be used with a variety of different patterns. Here is a list of the most commonly used patterns: 
  • EEE, MMM d, yyyy
    - Produces a string of the form “Wed, Oct 8, 2018”
  • MM/dd/yyyy
    - Produces a string of the form “10/08/2018”
  • h:mm a z
    - Produces a string of the form “1:34 pm EDT”
  • H:mm:ss Z
    - Produces a string of the form “13:34:45 -0500”
  • 
            
             
yyyy-MM-dd'T'HH:mm:ss.SSSZ
    - Produces a string of the form “2018-10-08T13:34:45.123+0000”
For a more comprehensive list of available patterns, please see the SimpleDateFormat documentation.
