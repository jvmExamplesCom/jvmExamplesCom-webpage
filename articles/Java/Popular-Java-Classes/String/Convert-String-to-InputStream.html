Converting a String to an InputStream in Java

One of the advantages of using Java is its versatile set of tools and features. One of these features is the ability to convert a Java 

          
           
String
into an 

          
           
InputStream

String to InputStream Basics

In order to convert a 

          
           
String
to an 

          
           
InputStream
, we must first understand what each of these terms mean. A 

          
           
String
is a sequence of characters stored within the Java language. Theses characters can contain any combination of numbers, letters, and symbols. An 

          
           
InputStream
is a stream of bytes which is used as input for a program.

Converting a String to InputStream

In order to convert a 

          
           
String
to an 

          
           
InputStream
, we will use the 

          
           
java.io.ByteArrayInputStream
class. This class provides a constructor which takes an array of bytes and creates an 

          
           
InputStream
. In order to create this array of bytes, we will use the 

          
           
String.getBytes()
method, which will return the string parsed into an array of bytes. The code example below shows a method which converts a 

          
           
String
to an 

          
           
InputStream
: 

          
           
public static InputStream stringToInputStream(String str) {
    byte[] bytes = str.getBytes();
    return new ByteArrayInputStream(bytes);
}
This method will take a 

          
           
String
as an argument, use the 

          
           
String.getBytes()
method to parse the 

          
           
String
into an array of bytes, and then use the constructor of the 

          
           
ByteArrayInputStream
class to create the 

          
           
InputStream
.
