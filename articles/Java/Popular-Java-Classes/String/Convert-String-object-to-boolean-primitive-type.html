Converting String Object to Boolean Primitive Type in Java

Java provides a facility to convert a String object to a boolean primitive type. This conversion can be done using the Boolean.parseBoolean(String) method. This method returns a boolean data type, which can then be used in various programming tasks, such as making decisions and other operations. In this article, we will look at how to convert a String object to a boolean primitive type in Java with detailed code examples and output.

Syntax of Boolean.parseBoolean() Method

The syntax of the 

          
           
Boolean.parseBoolean()
method is given below: 

          
           
Boolean.parseBoolean(String s);
The 

          
           
Boolean.parseBoolean()
method takes a single argument of type 

          
           
String
, and it returns a 

          
           
boolean
value.

Code Examples to Convert a String to a Boolean Primitive Type in Java

Let us now look at some code examples to convert a 

          
           
String
object to a 

          
           
boolean
primitive type in Java.

Example 1: Converting a String to boolean Using Boolean.parseBoolean() Method

In this example, we are converting a 

          
           
String
object ‘true’ to a 

          
           
boolean
primitive type using the 

          
           
Boolean.parseBoolean()
method. 

          
           
public class Main {

    public static void main(String[] args) {

        String str = "true";
         
        // Converting String to boolean
        boolean flag = Boolean.parseBoolean(str);
         
        // print boolean
        System.out.println("Boolean Value: " + flag);
    }
}
Output: 

          
           
Boolean Value: true

Example 2: Converting a String to boolean Using Boolean.valueOf() Method

In this example, we are converting a 

          
           
String
object ‘false’ to a 

          
           
boolean

Boolean.valueOf()

public class Main {
