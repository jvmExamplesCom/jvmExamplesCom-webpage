Convert int Array to String in Java

Java provides multiple methods to convert an integer array to string. The simplest way to do this is by using the 

          
           
toString()
method in Arrays class of java.util package. A few other options to this conversion are discussed below.

Using the 
Arrays.toString()
Method

The Arrays class in Java provides a static method 

          
           
toString()
that can receive any type of array as a parameter and return its content as a String. For example: 

          
           
int[] array = { 1, 2, 3 };
String str = Arrays.toString(array);
System.out.println(str);
This will output the following: 

          
           
[1, 2, 3]
.

Using a Loop

If you want to customize the output, then you can use any type of loop to iterate through the array and convert it into a string. For example: 

          
           
int[] array = { 1, 2, 3 };
String str = "";
for (int i = 0; i < array.length; i++) {
    str += array[i];
}
System.out.println(str);
This will output the following: 

          
           
123
. You can also use the 

          
           
StringBuilder
class in the same way to create a String from an array. For example: 

          
           
int[] array = { 1, 2, 3 };
StringBuilder sb = new StringBuilder();
for (int i = 0; i < array.length; i++) {
    sb.append(array[i]);
}
String str = sb.toString();
System.out.println(str);
This will output the same as the previous example.

Using the 
Arrays.stream()
Method

Another way to convert an array to a string is to use the 

          
           
Arrays.stream()
method. This will convert the array into a Stream object, which can then be converted into a string using the 

          
           
collect()
method. For example: 

          
           
int[] array = { 1, 2, 3 };
String str = Arrays.stream(array)
                     .mapToObj(String::valueOf)
                     .collect(Collectors.joining());
System.out.println(str);
This will output the same as the previous example.

Using Java 8 Streams and the 
collect()
Method

This is a more concise example using Java 8 Streams and the 

          
           
collect()
method. For example: 

          
           
int[] array = { 1, 2, 3 };
String str = Arrays.stream(array)
                   .map(String::valueOf)
                   .collect(Collectors.joining());
System.out.println(str);
This will output the same as the previous examples.
