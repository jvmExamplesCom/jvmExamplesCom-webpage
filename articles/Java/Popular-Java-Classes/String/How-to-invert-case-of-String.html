Inverting the Case of a String in Java

Java offers an easy way to convert the case of a string. It is often used to modify user input, such as converting an input field of mixed case characters to all uppercase or all lowercase. This article provides examples of how to use the Java 

          
           
toUpperCase()
and 

          
           
toLowerCase()
methods to do a case conversion.

Using toUpperCase()

The 

          
           
toUpperCase()
method converts a string from lowercase to uppercase characters. Here's an example of code that outputs the uppercase version of 'hello world': 

          
           
String str = "hello world";
String upperStr = str.toUpperCase();

System.out.println(upperStr); 
// Outputs: HELLO WORLD
The 

          
           
toUpperCase()
method is case-insensitive and will convert any character with an uppercase equivalent. For example, it will convert the accented character Ä to the uppercase equivalent Ä.

Using toLowerCase()

The 

          
           
toLowerCase()
method converts a string from uppercase to lowercase characters. Here's an example of code that outputs the lowercase version of 'HELLO WORLD': 

          
           
String str = "HELLO WORLD";
String lowerStr = str.toLowerCase();

System.out.println(lowerStr); 
// Outputs: hello world
As with the 

          
           
toUpperCase()
method, the 

          
           
toLowerCase()
method is also case-insensitive and will convert any character with a lowercase equivalent.

Conclusion

Using the 

          
           
toUpperCase()
and 

          
           
toLowerCase()
methods is a convenient way to convert strings from one case to another. Always remember that the methods are case-insensitive and will convert any character with an uppercase or lowercase equivalent.
