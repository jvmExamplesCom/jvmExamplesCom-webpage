How to Split String by New Line Characters in Java

The task of splitting a string based on new line characters can be quite tricky. There are several solutions available, depending on your needs and the desired output.

Using String.split()

The most common approach is to use the 
String.split()
method. This is a simple and straightforward way to split a string based on a single character or set of characters. 

          
           
String str = "This\nis\na\ntest";
String[] parts = str.split("\\n");
for (String part : parts) {
  System.out.println(part);
}
The output of the code above would be: 

          
           
This
is
a
test

Using StringTokeneizer

Another option is to use the 
StringTokenizer
class. This class is more flexible than 
String.split()
, as it allows you to specify multiple delimiters. For example: 

          
           
String str = "This;is;;a;;;test";
StringTokenizer st = new StringTokenizer(str, ";");
while (st.hasMoreTokens()) {
  System.out.println(st.nextToken());
}
The output of the code above would be: 

          
           
This
is
a
test

Using Regular Expressions

Finally, if you need more control over how the string is split, you can use regular expressions. The 

Pattern
and 
Matcher
classes provide a powerful and flexible way to perform complex string manipulation. In this example, we will use a regular expression to split the string based on any number of new line characters: 

          
           
String str = "This\nis\na\ntest";
Pattern p = Pattern.compile("\\R");
Matcher m = p.matcher(str);
while (m.find()) {
  System.out.println(m.group());
}
The output of the code above would be: 

          
           
This
is
a
test

As you can see, each of these approaches has its own advantages and disadvantages. Depending on your requirements, one solution may be better suited for your particular problem.
