Java Convert int Array To String Example

Converting int arrays to strings can be tricky since they are different types of objects. In this article, we will look at several ways to convert an int array to a string in Java.

Using Arrays.toString() Method

The 

          
           
Arrays.toString()
method is the simplest way to convert an int array to a string in Java. The 

          
           
Arrays.toString()
method takes an array as an argument and returns a string representation of the array. Here is an example of using the 

          
           
Arrays.toString()
method to convert an int array to a string: 

          
           
int[] myArray = {1, 2, 3, 4}; 
String str = Arrays.toString(myArray); 
System.out.println(str); 
// Output: [1, 2, 3, 4]
In the above example, we create an int array with four elements and then use the 

          
           
Arrays.toString()
method to convert it to a string. The output of the code is 

          
           
[1, 2, 3, 4]
.

Using StringBuffer And Append() Method

Another way to convert an int array to a string in Java is using the 

          
           
StringBuffer
class and its 

          
           
append()
method. The 

          
           
StringBuffer
class is a mutable sequence of characters and the 

          
           
append()
method appends the string representation of any type of data to the end of the buffer. Here is an example of using the 

          
           
StringBuffer
class and its 

          
           
append()
method to convert an int array to a string: 

          
           
int[] myArray = {1, 2, 3, 4}; 
StringBuffer sb = new StringBuffer(); 
for (int i : myArray) { 
    sb.append(i).append(","); 
} 
String str = sb.deleteCharAt(sb.length() - 1).toString(); 
System.out.println(str); 
// Output: 1,2,3,4
In the above example, we create an int array with four elements and then use a for loop to iterate through the array and append each element to a 

          
           
StringBuffer
object. We also add a comma after each element. Finally, we delete the last character which is the extra comma and then convert the 

          
           
StringBuffer
object to a string using the 

          
           
toString()
method. The output of the code is 

          
           
1,2,3,4
.

Using Java Streams

Java 8 introduced the concept of streams which made it easier to process collections of data. We can use streams to convert an int array to a string in Java. Here is an example of using Java Streams to convert an int array to a string: 

          
           
int[] myArray = {1, 2, 3, 4}; 
String str = Arrays.stream(myArray) 
                   .mapToObj(String::valueOf) 
                   .collect(Collectors.joining(",")); 
System.out.println(str); 
// Output: 1,2,3,4
In the above example, we use the 

          
           
Arrays.stream()
method to convert our int array to a stream. We use the 

          
           
mapToObj()
method to map each element of the stream to a 

          
           
String
object and then use the 

          
           
collect()
method to collect the elements and join them together with a comma. The output of the code is 

          
           
1,2,3,4
.

Conclusion

In this article, we looked at several ways to convert an int array to a string in Java. We looked at using the 

          
           
Arrays.toString()
method, the 

          
           
StringBuffer
class and its 

          
           
append()
method and finally using Java Streams.
