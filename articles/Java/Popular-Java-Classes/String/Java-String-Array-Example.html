Java String Array Example

Java supports several types of array, with each type allowing for the creation of arrays of a specific type. The most commonly used type is the string array. A string array is an array of strings, or text, which can be manipulated and used as needed. This article will discuss how to create, access, and manipulate string arrays in Java.

Creating a String Array

Java provides several ways to create a string array. The easiest and simplest way to create a string array is to use the built-in class, 

          
           
String[]
. You may also pass a length argument in order to specify the size of the array: 

          
           
String[] myArray = new String[10];
Alternatively, you may also use the 

          
           
Array.newInstance()
method to create an array of any type: 

          
           
String[] myArray = (String[]) Array.newInstance(String.class, 10);
Finally, you may also use the 

          
           
Array.of()
method to create a string array from the passed elements: 

          
           
String[] myArray = Array.of("Foo", "Bar", "Baz");

Accessing Elements of a String Array

To access and manipulate the elements within a string array, you must use the array index. Each element within an array has a unique index that can be used to access and modify it's content. You may use either the 

          
           
indexOf()
or 

          
           
get()
methods to access a particular element in the array: 

          
           
String element = myArray[myArray.indexOf("Foo")];
// returns "Foo"

String element = myArray.get(1);
// returns "Bar"

Manipulating Elements of a String Array

Once an element has been accessed from the array, you may manipulate it however you see fit. You may change the value of a string element using the 

          
           
set()
method: 

          
           
myArray.set(1, "Bang");
You may add a new element to the array using the 

          
           
add()
method: 

          
           
myArray.add("Quux");
You may also delete an item from the array using the 

          
           
remove()
method: 

          
           
myArray.remove(2);
In addition, you may also perform various operations on the array itself, such as sorting it using the 

          
           
sort()
method: 

          
           
Arrays.sort(myArray);
Lastly, you may also convert the array into a comma-separated string using the 

          
           
join()
method: 

          
           
String result = String.join(", ", myArray);
// returns "Foo, Bar, Baz"

Conclusion

This article explored how to create, access, and manipulate a string array in Java. By using the built-in methods on the array, you can easily access and manipulate the data within it.
