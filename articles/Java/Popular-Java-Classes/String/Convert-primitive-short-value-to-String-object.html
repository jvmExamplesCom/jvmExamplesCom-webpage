Using the Integer.toString() Method

The simplest way to convert a primitive short value to a String object is by using the Integer.toString() method: 

          
           
short val = 25;
String strVal = Integer.toString(val);
System.out.println(strVal);
// Outputs: 25

// Outputs: 25
