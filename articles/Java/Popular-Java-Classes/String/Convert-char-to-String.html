Convert char to String in Java

Java provides multiple ways to convert char to String. In this article we will discuss the different approaches and the differences between them.

Using + Operator

The simplest way of converting a char to String is by using the + operator. This method concatenates the character with an empty string to give us a new string. 

          
           
char c = 'x';
String s = "" + c;
System.out.println(s); // Output: x

Using Character.toString() Method

The Character class provides a static method toString() which can be used to convert a primitive char type to a String object. 

          
           
char c = 'x';
String s = Character.toString(c);
System.out.println(s); // Output: x

Using String.valueOf() Method

Another way of converting a char to String is by using the static method valueOf() of the String class. 

          
           
char c = 'x';
String s = String.valueOf(c);
System.out.println(s); // Output: x

Using Integer.toString() Method

The Integer class provides a static method toString() which can be used to convert a primitive char type to a String object. 

          
           
char c = 'x';
int i = (int)c;
String s = Integer.toString(i);
System.out.println(s); // Output: x
As we can see, there are multiple ways of converting a char to a String in Java. However, the most commonly used methods are the + operator and the Character.toString() method.
