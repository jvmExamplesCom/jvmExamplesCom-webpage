Java Substring Example

Using substring() Method

The 

          
           
substring()
method is used to extract a part of the String for further processing. It takes below two parameters:
  • startIndex
  • endIndex (optional)

Example 1

In the following Java program, we will extract a part of the String by using the substring() method. 

          
           
String str = "Hello World";
System.out.println(str.substring(3));
Output: 

          
           
lo World

Example 2

In this example, we will extract a part of the String using the specified startIndex and endIndex. 

          
           
String str = "Hello World";
System.out.println(str.substring(3, 7));
Output: 

          
           
lo W

Example 3

Let's see another example where we will use the specified endIndex but not the startIndex. 

          
           
String str = "Hello World";
System.out.println(str.substring(7));
Output: 

          
           
World

Example 4

In the final example, we will try to call the substring() method with a negative start index. 

          
           
String str = "Hello World";
System.out.println(str.substring(-2));
Output: 

          
           
This will throw an exception.
