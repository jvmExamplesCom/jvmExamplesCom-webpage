Java String to java.sql.Date Example

In this tutorial we will be learning how to convert a String to an instance of java.sql.Date using the SimpleDateFormat class. Creating a SimpleDateFormat Instance

The date format pattern must match the string we wish to convert in order for the parse() method to work correctly. In this example, we will set the date format pattern to ‘yyyy-MM-dd’. 

          
           
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");

Parsing the String

java.util.Date date = sdf.parse(str);
