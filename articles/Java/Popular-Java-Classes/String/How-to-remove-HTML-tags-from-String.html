How to remove HTML tags from String in Java

Java provides several different ways of removing HTML tags from a string. In this article, we will look at some of the most commonly used methods to remove HTML tags from a string in Java.

Using Pattern matcher and replaceAll()

The simplest way to remove HTML tags from a string is to use the Pattern Matcher class and the replaceAll() method. The following example shows how to remove all HTML tags from a string in Java: 

          
           
String str = "<p>This is a sample string</p>"

Pattern pattern = Pattern.compile("<[^>]*>");
Matcher matcher = pattern.matcher(str);
String strWithoutHtmlTags = matcher.replaceAll("");
The output of the above code would be “This is a sample string”.

Using Builder class

Another way to remove HTML tags from a string in Java is to use the StringBuilder class. The following example shows how to use the StringBuilder class to remove HTML tags from a string: 

          
           
String str = "<p>This is a sample string</p>"

StringBuilder sb = new StringBuilder();
for (char c : str.toCharArray()) {
    if (c != '<' &amp;&amp; c != '>')
        sb.append(c);
}
String strWithoutHtmlTags = sb.toString();
The output of the above code would be “This is a sample string”.

Using replaceAll() method

Another method to remove HTML tags from a string in Java is to use the replaceAll() method. The following example shows how to use the replaceAll() method to remove HTML tags from a string: 

          
           
String str = "<p>This is a sample string</p>"
String strWithoutHtmlTags = str.replaceAll("<[^>]*>","");
The output of the above code would be “This is a sample string”.

Using Regular Expressions

Regular expressions (regex) can also be used to remove HTML tags from a string in Java. The following example shows how to use regular expressions to remove HTML tags from a string: 

          
           
String str = "<p>This is a sample string</p>"
String strWithoutHtmlTags = str.replaceAll("<[\\w\\W]+?>","");
The output of the above code would be “This is a sample string”.
