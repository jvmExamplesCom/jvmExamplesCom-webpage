Converting Primitive int Values to String Object in Java

Java is an object-oriented programming language and uses a wrapper class to convert primitive values to their respective object types. In this article, we will discuss how to convert a primitive int value to a String object in Java.

Using the Integer.toString() Method

The Integer.toString() method accepts a primitive int and returns a String object version of that integer. The following code snippet demonstrates how to use the Integer.toString() method: 

          
           
int num = 123;
String str = Integer.toString(num);
System.out.println(str); // prints "123"
In the code example above, we declare an int variable named num, set it to 123 and then use the Integer.toString() method to convert it to a String. After the conversion, we print the resulting String to the console.

Using the String.valueOf() Method

The String.valueOf() method also accepts a primitive int and returns a String object version of that integer. The following code snippet demonstrates how to use the String.valueOf() method: 

          
           
int num = 123;
String str = String.valueOf(num);
System.out.println(str); // prints "123"
In the code example above, we declare an int variable named num, set it to 123 and then use the String.valueOf() method to convert it to a String. After the conversion, we print the resulting String to the console.

Using the String Constructor

The String class also has a constructor which accepts a primitive int and converts it to a String object. The following code snippet demonstrates how to use the String constructor: 

          
           
int num = 123;
String str = new String(num);
System.out.println(str); // prints "123"
In the code example above, we declare an int variable named num, set it to 123 and then use the String constructor to convert it to a String. After the conversion, we print the resulting String to the console.

Conclusion

In this article, we discussed how to convert a primitive int value to a String object in Java. We looked at using the Integer.toString(), String.valueOf(), and String constructor methods to do the conversion.
