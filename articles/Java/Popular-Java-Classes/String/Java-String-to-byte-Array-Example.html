Java String to byte Array Example

The following examples demonstrate how to convert a Java String to a byte array using the 

          
           
getBytes()
method. This method is used to allocate a new byte array object and populates it with the data from the supplied String.

Example 1: Basic Conversion

The following example shows how to use the 

          
           
getBytes()
method to convert a Java String to a byte array 

          
           
String testString = "Test String"; 
byte[] bytesArray = testString.getBytes(); 

for (int i=0; i < bytesArray.length; i++) {
   System.out.println(bytesArray[i]);
}
In this example, 

          
           
testString
is initialized with a String literal "Test String". The 

          
           
getBytes()
method is then used to convert the String to a byte array. A loop is then used to iterate over the elements of the byte array and print each element out on a new line.

Example 2: Encoding a String in a Specific Charset

The 

          
           
getBytes(String charsetName)
method can be used to convert a String to a byte array while specifying the encoding scheme to use when converting the String. In the following example, the String "Test String" is encoded in UTF-8. 

          
           
String testString = "Test String"; 
byte[] bytesArray = testString.getBytes("UTF-8"); 

for (int i=0; i < bytesArray.length; i++) {
   System.out.println(bytesArray[i]);
}
The output from this example is the same as the output from the first example. However, in this example, the bytes of the String are encoded using UTF-8 when the 

          
           
getBytes()
method is invoked.

Example 3: Using a Byte Array Output Stream

The 

          
           
ByteArrayOutputStream
class can also be used to convert a String to a byte array. The 

          
           
write(String string)
method can be used to write the contents of a String to a 

          
           
ByteArrayOutputStream
. The 

          
           
toByteArray()
method can then be used to retrieve the contents of the 

          
           
ByteArrayOutputStream
as a byte array. 

          
           
String testString = "Test String"; 
ByteArrayOutputStream stream = new ByteArrayOutputStream(); 
stream.write(testString.getBytes()); 
byte[] bytesArray = stream.toByteArray(); 

for (int i=0; i < bytesArray.length; i++) {
   System.out.println(bytesArray[i]);
}
In this example, the 

          
           
ByteArrayOutputStream
is used to write the contents of 

          
           
testString
to the stream. The 

          
           
toByteArray()
method is then used to convert the 

          
           
ByteArrayOutputStream
to a byte array. A loop is used to iterate over the elements in the byte array and print each element out on a new line.

Example 4: Copying the Contents of an Existing Byte Array

The 

          
           
getBytes(byte[] byteArray)
method can be used to copy the contents of an existing byte array to another byte array. In the following example, the contents of a byte array are copied to another byte array. 

          
           
byte[] sourceArray = {1, 2, 3, 4}; 
byte[] targetArray = new byte[4]; 
targetArray = sourceArray.getBytes(); 

for (int i=0; i < targetArray.length; i++) {
   System.out.println(targetArray[i]);
}
In this example, 

          
           
sourceArray
is initialized with a byte array containing four elements. A new byte array, 

          
           
targetArray
, is allocated with the same size as 

          
           
sourceArray
. The 

          
           
getBytes()
method is then used to copy the contents of 

          
           
sourceArray
to 

          
           
targetArray
. Finally, a loop is used to iterate over the elements of 

          
           
targetArray
and print each element to the console.

Example 5: Copying the Contents of a String to an Array

The 

          
           
getBytes(byte[] byteArray, int offset, int length)
method can be used to copy the contents of a String to an array. In the following example, the contents of a String are copied to a byte array. 

          
           
String testString = "Test String"; 
byte[] targetArray = new byte[testString.length()]; 
testString.getBytes(targetArray, 0, testString.length()); 

for (int i=0; i < targetArray.length; i++) {
   System.out.println(targetArray[i]);
}
In this example, the 

          
           
getBytes()
method is used to copy the contents of 

          
           
testString
to 

          
           
targetArray
. The 

          
           
offset
and 

          
           
length
parameters specify the position of the first element in the source array and the number of elements to copy respectively. A loop is then used to iterate over the elements in the 

          
           
targetArray
and print each element on a new line. You can use the 

          
           
getBytes()
methods to quickly convert a Java String to a byte array. There are various overloaded versions of the 

          
           
getBytes()
method which can be used to specify the encoding scheme used when converting the String to a byte array. The 

          
           
ByteArrayOutputStream
class can also be used to convert a String to a byte array.
