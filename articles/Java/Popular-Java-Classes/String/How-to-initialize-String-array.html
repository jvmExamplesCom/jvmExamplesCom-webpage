Java: How to Initialize String Array

A String array is a data structure used by Java applications to store a set of Strings. This tutorial will demonstrate how to initialize a String array in Java.

Creating a String Array With Strings Already Specified

The following example shows how to create a String array with strings already specified:


          
           
String[] stringArray = {"String1", "String2", "String3"};

Creating an Empty String Array With Specified Length

The following example shows how to create an empty String array with a specified length:


          
           
String[] stringArray = new String [9];

Adding Elements to a String Array

Once you've initialized the array, you can add elements to it using the 


          
           
array[index] = value;
syntax: 

          
           
stringArray[0] = "String1";
stringArray[1] = "String2";
stringArray[2] = "String3";

Iterating Over a String Array

The following example shows how to iterate over a String array and print out its elements:


          
           
for (String element : stringArray) {
    System.out.println(element);
}

Using The Arrays Class for Common Array Operations

The Java Arrays class provides several methods that can be used to quickly sort, search, fill, compare, and copy arrays. The following example shows how to use the Arrays.sort() method to sort an array:


          
           
Arrays.sort(stringArray);
