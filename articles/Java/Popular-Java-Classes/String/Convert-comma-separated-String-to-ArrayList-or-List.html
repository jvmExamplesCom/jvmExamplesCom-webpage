Convert comma separated String to ArrayList or List in Java

Java offers a number of ways to split a string that contains a delimiter such as a comma. The most common way to convert a String to an ArrayList or List is by using the split() method of the String class. This method takes a regular expression and splits the string around matches of this regular expression. In the following example, we will show you how to use the split() method to convert a comma-separated String into an ArrayList. We will also cover how to convert the same comma-separated String into a List.

Example 1: Convert comma-separated String to ArrayList


          
           
String str = "jdk1.8, jdk1.7, jdk1.6";
 
//split the string by coma
String[] items = str.split(",");
 
//create an ArrayList object
ArrayList<String> list = new ArrayList<String>();
 
//add items to the Arraylist
for (int i = 0; i < items.length; i++) {
   list.add(items[i]);
}
 
//print out the list
System.out.println(list);
 
//output : [jdk1.8, jdk1.7, jdk1.6]
In this example, we have declared a String str that contains a comma-separated list. We used the split() method to split the String into an array of items and then looped through the array to add each item to the ArrayList. Finally, we printed out the ArrayList to confirm it contains the items from the String.

Example 2: Convert comma-separated String to List


          
           
String str = "jdk1.8, jdk1.7, jdk1.6";
 
//split the string by coma
String[] items = str.split(",");
 
//create a List object
List<String> list = new ArrayList<String>();
 
//add items to the List
for (int i = 0; i < items.length; i++) {
   list.add(items[i]);
}
 
//print out the list
System.out.println(list);
 
//output : [jdk1.8, jdk1.7, jdk1.6]
This example is similar to the previous one but instead of creating an ArrayList we created a List. The code works the same way and produces the same result.

Conclusion

Split() is the most convenient way to convert a comma-separated String into an ArrayList or List. We have provided two examples of how to use split() to achieve this.
