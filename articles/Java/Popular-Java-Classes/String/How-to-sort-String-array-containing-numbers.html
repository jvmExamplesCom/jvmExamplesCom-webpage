Sorting a String Array Containing Numbers in Java

Java provides a number of methods for sorting arrays of objects. One such method is the Arrays.sort() method which takes an array as an argument and sorts it in place according to its natural order (i.e., alphabetical/numeric). In this tutorial, we'll go through how to sort a String array containing numbers using the Arrays.sort() method.

Creating an Array of Strings

Let's start by creating an array of strings containing numbers that we want to sort: 

          
           
String[] numbersArray = {"2", "10", "13", "15", "20", "9"};
Now that we have our array of Strings containing numbers, we can move on to sorting it.

Using the Arrays.sort() Method

The Arrays.sort() method takes an array as an argument and sorts it in place according to its natural order. This means that the array is sorted in ascending order by default. This method will work for numerical data stored as Strings since the sorting will be based on the lexicographical order of the characters. To use the Arrays.sort() method, we simply need to pass in our array as an argument: 

          
           
Arrays.sort(numbersArray);
Once the array has been sorted, we can print it out to check that it has been sorted correctly: 

          
           
System.out.println(Arrays.toString(numbersArray));
This will print out the following: 

          
           
[2, 9, 10, 13, 15, 20]
As you can see, our array has been successfully sorted in ascending numerical order.

Conclusion

In this tutorial, we went through how to sort a String array containing numbers using the Arrays.sort() method. With the Arrays.sort() method, we are able to sort our array in place according to its natural order. This method is useful for sorting numerical data stored as Strings since it takes into consideration the lexicographical order of the characters.
