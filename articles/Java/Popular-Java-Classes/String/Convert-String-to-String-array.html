Convert String to String Array in Java

Converting a String to a String array is a common task in Java. It can be done using either the 

          
           
split()
method of the String class or by creating a StringTokenizer object. These two methods offer different levels of flexibility, depending on the structure of the String you are trying to convert. This article will guide you through the process of converting a String to a String array in Java.

Using the split() Method

The 

          
           
split()
method of the String class can be used to convert a String to a String array. This method takes a String as a parameter, which is used as a delimiter for splitting the String into an array. The following example shows how to use the 

          
           
split()
method to convert a String to a String array in Java: 

          
           
String str = "This is a test string";
String[] arr = str.split(" ");

// Output:
// arr[0] = "This"
// arr[1] = "is"
// arr[2] = "a"
// arr[3] = "test"
// arr[4] = "string"
In this example, the 

          
           
split()
method is used to split the String on the space character ( 

          
           
" "
). This will result in a String array containing five elements, each element containing one word from the original String.

Using the StringTokenizer Object

The 

          
           
StringTokenizer
class can be used to convert a String to a String array. This class takes a String as a parameter and uses it to tokenize the String. The tokens are then returned as an array of Strings. The following example shows how to use the 

          
           
StringTokenizer
class to convert a String to a String array in Java: 

          
           
String str = "This is a test string";
StringTokenizer st = new StringTokenizer(str);
String[] arr = new String[st.countTokens()];

for (int i = 0; st.hasMoreTokens(); i++) {
    arr[i] = st.nextToken();
}

// Output:
// arr[0] = "This"
// arr[1] = "is"
// arr[2] = "a"
// arr[3] = "test"
// arr[4] = "string"
In this example, a 

          
           
StringTokenizer
object is created with the original String. The 

          
           
countTokens()
method is then used to get the number of tokens in the String, which is used to create an array of the correct size. The 

          
           
hasMoreTokens()
and 

          
           
nextToken()
methods of the 

          
           
StringTokenizer
object are then used to iterate over the tokens, adding each token to the array.

Conclusion

Converting a String to a String array is a common task in Java. This article has shown how it can be accomplished using either the split() method of the String class or by creating a StringTokenizer object. Each method offers its own level of flexibility, depending on the structure of the String.
