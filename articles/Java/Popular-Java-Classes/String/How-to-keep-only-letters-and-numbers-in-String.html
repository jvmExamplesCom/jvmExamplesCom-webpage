How to Keep Only Letters and Numbers in a String Using Java

Java is a powerful programming language with many functions and tools. One powerful feature of Java allows developers and programmers to manipulate strings by keeping or removing characters. This article will cover how to keep only letters and numbers in a String using Java.

Using the String.replaceAll() Method

One of the easiest and most efficient ways to remove all non-letters and non-numbers from a String is to use the 

String.replaceAll()
method. This method takes a regular expression pattern as an argument, and it replaces all the matched characters with an empty string. To keep only letters and numbers, you can use the following pattern: 

          
           
String str = "Hello World!";
// Replace all non-letters and non-numbers.
str = str.replaceAll("[^A-Za-z0-9]", "");
// Outputs "HelloWorld"
System.out.println(str);
The above code snippet will take the given String 
Hello World!
and output it as 
HelloWorld
. Notice that all the non-alphanumeric characters have been removed.

Using the Character.isLetterOrDigit() Method

Another way to keep only letters and numbers in a String is to use the 


          
           
Character.isLetterOrDigit()
method. This method returns a boolean value, 
true
or 
false
, depending on whether the character is a letter or digit. You can loop through the characters of a String and check each character using this method, and then concatenate the resulting character values into a new String. For example: 

          
           
String str = "Hello World!";
StringBuilder sb = new StringBuilder();
for (int i = 0; i < str.length(); i++) {
  char c = str.charAt(i);
  if (Character.isLetterOrDigit(c)) {
    sb.append(c);
  }
}
// Outputs "HelloWorld"
System.out.println(sb.toString());

Here we are looping through each character of the String and checking if it's a letter or digit. If it is, then we add it to a 

StringBuilder
, which is used to build a new String. The resulting String will be 
HelloWorld
, as expected.

Conclusion

In this article we covered two methods for keeping only letters and numbers in a String using Java. The first was using the 

String.replaceAll()
method, and the second was using the 

          
           
Character.isLetterOrDigit()
method. Both of these methods are fairly simple to use, and they can be used in various combinations to manipulate Strings in many different ways.
