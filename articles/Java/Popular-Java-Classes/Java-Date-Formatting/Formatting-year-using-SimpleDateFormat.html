Formatting Year Using SimpleDateFormat in Java

Java is a powerful programming language used for a wide range of development tasks, from web apps to embedded systems. One common task is dealing with dates and times, which requires the use of the 

SimpleDateFormat
class. This tutorial explains how to use the 
SimpleDateFormat
class to format a year given in milliseconds.

Getting Started

Before you can use 
SimpleDateFormat
, you need to import it into your Java program with the following commands: 

          
           
import java.text.SimpleDateFormat;
import java.util.Date;

Using SimpleDateFormat

To format a year given in milliseconds, create an instance of 

SimpleDateFormat
and set the format with the following code: 

          
           
SimpleDateFormat formatter = new SimpleDateFormat("yyyy");

The “yyyy” part sets the date format to 4-digit years. To format the year given in milliseconds, you will use the 

format()
method as follows: 

          
           
long yearInMillis = System.currentTimeMillis();
String formattedYear = formatter.format(new Date(yearInMillis));
The 
formattedYear
variable now contains a string with the year formatted according to your format.

Example Program

To demonstrate how to format a year using SimpleDateFormat, here is a complete Java program that prints the current year in 4-digit format: 


          
           
import java.text.SimpleDateFormat;
import java.util.Date;

public class YearFormatter {
    public static void main(String[] args) {
        // Create a SimpleDateFormat object
        SimpleDateFormat formatter = new SimpleDateFormat("yyyy");

        // Get the current time in milliseconds
        long millis = System.currentTimeMillis();

        // Format the year
        String formattedYear = formatter.format(new Date(millis));

        // Print the formatted year
        System.out.println("The year is: " + formattedYear);
    }
}

When you run this program, it prints out the current year in 4-digit format: 


          
           
The year is: 2021
