Java: StringBuffer Trim Java Example

The 

          
           
StringBuffer
object in Java is an efficient way to manipulate strings and perform various operations on them. One of the common operations is trimming the leading and trailing whitespace from a 

          
           
StringBuffer
. This tutorial will demonstrate how to do this using the trim() method of the 

          
           
StringBuffer
class.

Trimming a StringBuffer in Java

The 

          
           
StringBuffer
class includes the 

          
           
trim()
method, which can be used to remove leading and trailing whitespace from a 

          
           
StringBuffer
. The following code example shows how to use the 

          
           
trim()
method to trim a 

          
           
StringBuffer
. 

          
           
StringBuffer sb = new StringBuffer("    Hello World!    ");
sb.trim();
System.out.println(sb);
The output of the above code will be: 

          
           
Hello World!
The 

          
           
StringBuffer
is trimmed before being printed out. Note that the 

          
           
trim()
method does not modify the original 

          
           
StringBuffer
, but rather creates a new 

          
           
StringBuffer
with the trimmed contents.

Trimming with a Custom Character Set

The 

          
           
StringBuffer
class also provides a 

          
           
trim(CharSequence)
method, which allows you to specify a custom character set to be removed from the beginning and end of the 

          
           
StringBuffer
. The following code example shows how to use the 

          
           
trim(CharSequence)
method to trim a 

          
           
StringBuffer
with a custom character set. 

          
           
StringBuffer sb = new StringBuffer("##_Hello World_#_#");
sb.trim("#_");
System.out.println(sb);
The output of the above code will be: 

          
           
Hello World
In this example, the custom character set " # _" is specified for the 

          
           
trim(CharSequence)
method. The characters from the custom character set are removed from the beginning and end of the 

          
           
StringBuffer
.

Trimming a Substring of a StringBuffer

The 

          
           
StringBuffer
class also provides a 

          
           
trimSubstring(start, end)
method, which allows you to specify a substring of the 

          
           
StringBuffer
to be trimmed. The following code example shows how to use the 

          
           
trimSubstring(start, end)
method to trim a substring of a 

          
           
StringBuffer
. 

          
           
StringBuffer sb = new StringBuffer("Hello   World!");
sb.trimSubstring(0, 5);
System.out.println(sb);
The output of the above code will be: 

          
           
HelloWorld!
In this example, the substring from index 0 to index 5 is specified for the 

          
           
trimSubstring(start, end)
method. The whitespace from the specified substring is removed from the 

          
           
StringBuffer
.

Trimming a Subsequence of a StringBuffer

The 

          
           
StringBuffer
class also provides a 

          
           
trimSubsequence(start, end)
method, which allows you to specify a subsequence of the 

          
           
StringBuffer
to be trimmed. The following code example shows how to use the 

          
           
trimSubsequence(start, end)
method to trim a subsequence of a 

          
           
StringBuffer
. 

          
           
StringBuffer sb = new StringBuffer("    Hello World");
sb.trimSubsequence(2, 9);
System.out.println(sb);
The output of the above code will be: 

          
           
HeWorld
In this example, the subsequence from index 2 to index 9 is specified for the 

          
           
trimSubsequence(start, end)
method. The whitespace from the specified subsequence is removed from the 

          
           
StringBuffer
.

Conclusion

In this tutorial, we have demonstrated how to trim a 

          
           
StringBuffer
using the 

          
           
trim()
, 

          
           
trim(CharSequence)
, 

          
           
trimSubstring(start, end)
, and 

          
           
trimSubsequence(start, end)
methods. All of these methods are useful for trimming whitespace from a 

          
           
StringBuffer
, and can be used in a variety of scenarios.
