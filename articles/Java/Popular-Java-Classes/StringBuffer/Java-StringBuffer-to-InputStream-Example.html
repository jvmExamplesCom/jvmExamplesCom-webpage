Java StringBuffer to InputStream Example

The Java StringBuffer is a utility class that allows strings to be manipulated. It is often used when dealing with large amounts of text or data, as it provides the flexibility to change parts of the string without having to re-create the entire string. This can be especially useful when dealing with streams of data coming from a file or a database connection. In some cases, it may be necessary to convert a StringBuffer to an InputStream object in order to manipulate the data as desired. In this tutorial, we'll explore how to convert a StringBuffer to an InputStream using Java. We'll look at examples for each of the major Java versions, including Java 8. Additionally, we'll cover how to work with different kinds of data within the stream.

Convert StringBuffer to InputStream (Java 8)

The following code example shows how to convert a StringBuffer to an InputStream in Java 8. The method begins by creating a new StringReader object, which allows us to read characters from the StringBuffer. Then, we create a BufferedReader, which is used to read chunks of characters from the StringReader. Finally, we create an InputStreamReader, which will read the contents of the buffered reader and create an InputStream object that can be used in our application. 

          
           
StringBuffer sb = new StringBuffer("This is a test"); 

StringReader sr = new StringReader(sb.toString());
BufferedReader br = new BufferedReader(sr);
InputStreamReader isr = new InputStreamReader(br); 
InputStream is = new BufferedInputStream(isr);

Convert StringBuffer to InputStream (Java 7 and earlier)

The same task can be accomplished in Java 7 and earlier. This approach uses the ByteArrayInputStream class, which represents an input stream of bytes. In the code example below, we first convert the StringBuffer to a byte array using the getBytes() method. Then, we use the ByteArrayInputStream constructor to create an InputStream object from the byte array. 

          
           
StringBuffer sb = new StringBuffer("This is a test"); 

byte[] bytes = sb.toString().getBytes(); 
InputStream is = new ByteArrayInputStream(bytes);

Working With Different Types of Data

When converting a StringBuffer to an InputStream, you may need to work with different types of data.
