Adding or Subtracting Seconds to Current Time Using Java Calendar

There are many ways to add or subtract seconds from the current date and time in Java. One way to do this is by using the Java Calendar class, which contains methods for performing various calendar-related calculations. In this article, we will discuss how to add or subtract seconds from the current time using the Calendar class.

Create Calendar Object

Before we can use the Calendar class to calculate time, we must first create an instance of the Calendar class. This can be done using the following code: 

          
           
Calendar cal = Calendar.getInstance();
After the instance of the Calendar class is created, we can add or subtract seconds from the current time.

Adding Seconds to Current Time

To add seconds to the current time, we can use the 

          
           
add()
method. The 

          
           
add()
method takes two parameters: the field to modify (in this case, the value for 

          
           
Calendar.SECOND
) and the number of seconds to add. For example, the following code adds 5 seconds to the current time: 

          
           
cal.add(Calendar.SECOND, 5);
After executing the above code, we can then get the updated time using the 

          
           
getTime()
method: 

          
           
Date newTime = cal.getTime();

Subtracting Seconds from Current Time

Similarly, we can subtract seconds from the current time using the same 

          
           
add()
method, but we must set the second parameter to a negative number. For example, the following code subtracts 5 seconds from the current time: 

          
           
cal.add(Calendar.SECOND, -5);
We can then get the updated time using the 

          
           
getTime()
method as before: 

          
           
Date newTime = cal.getTime();

Example

Let's look at a full example. The following code adds 10 seconds to the current time and prints out the updated time: 

          
           
// Create Calendar object
Calendar cal = Calendar.getInstance();

// Add 10 seconds to the current time
cal.add(Calendar.SECOND, 10);

// Get the updated time
Date newTime = cal.getTime();

// Print out the updated time
System.out.println("Updated time: " + newTime);
If we run the above code, we should get an output similar to this: 

          
           
Updated time: Thu May 23 15:37:39 PDT 2019
As you can see, the above code successfully adds 10 seconds to the current time and prints out the updated time.

Conclusion

In this article, we discussed how to add or subtract seconds from the current time using the Java Calendar class. We discussed how to create an instance of the Calendar class, how to add or subtract seconds using the 

          
           
add()
method, and we looked at a full example.
