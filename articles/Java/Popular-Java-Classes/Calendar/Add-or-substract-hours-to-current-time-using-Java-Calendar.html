Add or Substract Hours to Current Time using Java Calendar

Using the Java Calendar class, you can easily add or substract hours from the current time. The Calendar class provides a set of methods that allow you to adjust the current time. There are a few different methods available for adding or subtracting hours, and each has its own use case.

Adding Hours

The simplest way to add hours to the current time is to use the add() method. This method takes a single integer parameter which specifies the number of hours to add. Here's an example of how to use this method:


          
           
Calendar cal = Calendar.getInstance();
cal.add(Calendar.HOUR, 2);
System.out.println(cal.getTime());

In this example, we first get an instance of the Calendar class and then add two hours to the current time. Finally, we print out the new time. The output of this code will be two hours ahead of the current time.

Subtracting Hours

If you need to subtract hours from the current time, you can use the roll() method. This method takes two parameters, the first is the field you want to adjust (in this case, the HOUR field) and the second is the amount you want to adjust it by. Here's an example of how to use this method:


          
           
Calendar cal = Calendar.getInstance();
cal.roll(Calendar.HOUR, -2);
System.out.println(cal.getTime());

In this example, we first get an instance of the Calendar class and then subtract two hours from the current time. Finally, we print out the new time. The output of this code will be two hours behind the current time.

Conclusion

Using the methods provided by the Calendar class, you can easily add or subtract hours from the current time. The add() method is the simplest way to add hours, while the roll() method is the simplest way to subtract hours. With these methods, you can easily adjust the current time according to your needs.
