Introduction to org.osgi.framework.FrameworkUtil in Java

The 

          
           
org.osgi.framework.FrameworkUtil
class provides utility methods to access the current Framework or BundleContext instance and to create filter objects. It is a utility class that enables easy access to both the framework and bundle context. This class is typically used by bundles that need access to the framework or bundle context.

Methods in org.osgi.framework.FrameworkUtil

  • getBundle(Class)
    - Returns the bundle associated with a given class, or null if the class was loaded by the boot class loader.
  • 
            
             
getBundleContext(Bundle)
    - Returns the BundleContext for the specified bundle, or null if the specified bundle is not in the ACTIVE state.
  • getFramework()
    - Returns the Framework instance, or null if the framework has not been started.

getServiceReference(Class)
    - Returns the service reference for the specified service class, or null if no service is registered for the specified class.

    - Returns the service object for the specified service reference, or null if the specified service reference is invalid.
  •
