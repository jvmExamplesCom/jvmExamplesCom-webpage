Introduction to org.springframework.data.domain.AuditorAware

org.springframework.data.domain.AuditorAware is an interface within the Spring Data project that provides a convenient way to capture and persist audit information, such as date and the user who created or modified a record. This audit information is typically stored in separate tables from the primary data in the database.

Usage

AuditorAware should be used when dealing with persistent entities.
