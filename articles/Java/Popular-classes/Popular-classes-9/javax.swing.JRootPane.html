javax.swing.JRootPane

JRootPane is a Swing container that provides an application's root pane and gives the application window basic internal frame features, such as a layered pane, content pane, and glass pane. It can also add control buttons for minimizing, maximizing, and restoring frames. It also provides support for setting the look and feel for all windows in the application.

Example 1 - Creating a JRootPane

The following example shows how to create a JRootPane:


          
           
JRootPane rootPane = new JRootPane();

Example 2 - Setting the Layout of a JRootPane

The following example shows how to set the layout of a JRootPane:


          
           
rootPane.setLayout(new BorderLayout());

Example 3 - Adding Components to a JRootPane
