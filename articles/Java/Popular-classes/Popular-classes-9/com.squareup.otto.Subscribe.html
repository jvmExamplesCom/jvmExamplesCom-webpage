Introduction to com.squareup.otto.Subscribe

The 

          
           
com.squareup.otto.Subscribe
annotation is a way to register methods for event-based messaging on Android, enabling communication between components without explicit registration or knowledge of each other. It does this by allowing subscribers to register with an EventBus and receive events from publishers. The main advantage of using the 

          
           
com.squareup.otto.Subscribe
annotation is its simplicity. By just adding the annotation to a method, you can make it a subscriber to an event. It also makes it easier to see which classes are a part of the event system since they are all annotated.
