org.jfree.data.xy.XYSeries

Overview

org.jfree.data.xy.XYSeries is a class in the JFreeChart library that provides an implementation of the XYDataset interface. It stores data as an xy-series, with each item consisting of a unique x and y value.

Properties and Attributes

The class contains several properties related to its data set, such as the title and maximum number of items it can store. It also supports notification events, allowing other objects to be notified when changes occur in the data set.

Constructors

Three constructors are available for creating an XYSeries instance. The most commonly used constructor takes two arguments: 

          
           
XYSeries(String title, boolean autoSort)
The title argument specifies the name of the series, while the autoSort argument specifies whether the data should be automatically sorted by the x-values.

Methods

There are several methods in this class used for manipulating its data.

add()


          
           
add(double x, double y)
The add() method adds a new item with the given x and y values to the series.

update()


          
           
update(int index, double y)
The update() method updates the y-value of an item at the specified index.

remove()


          
           
remove(int index)
The remove() method removes an item at the specified index from the series.

Example

Here is a simple example showing how to create and use an XYSeries: 

          
           
XYSeries series = new XYSeries("My Series", true);
series.add(1.0, 1.0);
series.add(2.0, 4.0);
series.add(3.0, 9.0);
double sum = 0.0;
for (int i=0; i<series.getItemCount(); i++) {
    sum += series.getY(i).doubleValue();
}
System.out.println("Sum = " + sum);
The output of the code above is: 

          
           
Sum = 14.0
