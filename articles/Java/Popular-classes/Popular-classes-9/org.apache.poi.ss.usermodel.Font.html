Java org.apache.poi.ss.usermodel.Font

The org.apache.poi.ss.usermodel.Font class is a representation of a font in a workbook. Fonts are used to render the text for various objects such as rich text, header and footers, charts, titles, and others.

Creating Font Objects

To create a Font object, use the following code: 

          
           
Workbook workbook = new XSSFWorkbook();
Font font = workbook.createFont();

Font Properties

The Font object has several properties that can be set or read:

  • Font name - set with setFontName() or getFontName()
  • Font size - set with setFontHeightInPoints() or getFontHeightInPoints()
  • Bold attribute - set with setBold() or getBold()
  • Italic attribute - set with setItalic() or getItalic()
  • Underline attribute - set with setUnderline() or getUnderline()
  • Strike-through attribute - set with setStrikeThrough() or getStrikeThrough()
  • Color - set with setColor() or getColor()

Example Usage

The code below shows an example of setting some of the above properties on a Font object: 


          
           
Font font = workbook.createFont();
font.setFontName("Arial");
font.setFontHeightInPoints((short) 12);
font.setBold(true);
font.setItalic(true);
font.setStrikeThrough(true);
font.setUnderline(Font.U_SINGLE);
font.setColor(IndexedColors.BLUE.getIndex());

Using the Font

Once the Font object is created and configured, it can be associated with various objects in the workbook. To associate the Font with a cell, use the following code: 


          
           
CellStyle style = workbook.createCellStyle();
style.setFont(font);
cell.setCellStyle(style);

To associate the Font with a RichText object, use the following code: 


          
           
RichTextString rtString = new XSSFRichTextString("Hello World!");
rtString.applyFont(font);
