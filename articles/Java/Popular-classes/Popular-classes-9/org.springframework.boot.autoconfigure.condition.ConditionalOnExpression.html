org.springframework.boot.autoconfigure.condition.ConditionalOnExpression
class enables developers to configure a bean with the annotation 

          
           
@ConditionalOnExpression

Example:

import org.springframework.stereotype.Component;

@Component
@ConditionalOnExpression("#{systemProperties['environment'] == 'dev'}")

    // code here
}

@ConditionalOnExpression
is used. The annotation is defined on the class and indicates that the bean will only be created if the SpEL expression evaluates to true. In this case, we are evaluating a system property called 'environment'. If the value is equal to 'dev', then the bean will be created.
