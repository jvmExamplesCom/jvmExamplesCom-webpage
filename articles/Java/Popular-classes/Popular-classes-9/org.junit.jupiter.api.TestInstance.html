JUnit Jupiter TestInstance

JUnit Jupiter TestInstance is an annotation that is used to configure the test instances lifecycle for a given test class. By using this annotation, one can configure the lifecycle mode and per instance or per class algorithm.

Lifecycle Modes

The @TestInstance annotation provides two lifecycle modes, 

PER_CLASS
and 
PER_METHOD
. 
  • PER_CLASS
    :     The lifecycle will be applied once per test class.
  • PER_METHOD
    :     The lifecycle will be applied before each test method.

Example

The following example defines a test class that uses the 

PER_CLASS
lifecycle mode. 

          
           
@TestInstance(TestInstance.Lifecycle.PER_CLASS)
public class MyTestClass {
   // Test methods
}

The following example defines a test class that uses the 

PER_METHOD
lifecycle mode. 

          
           
@TestInstance(TestInstance.Lifecycle.PER_METHOD)
public class MyTestClass {
   // Test methods
}

Algorithm

The @TestInstance annotation also provides an algorithm attribute that is used to determine the run order of the test methods within a given test class.

  • DEFAULT
    :     The test methods will be executed in an undefined order.
  • SORTED
    :     The test methods will be sorted based on their names.
  • RANDOMIZED
    :     The test methods will be randomized using the standard Java random number generator.

Example

The following example defines a test class that uses the 

SORTED
algorithm. 

          
           
@TestInstance(TestInstance.Lifecycle.PER_CLASS,
                TestInstance.Algorithm.SORTED)
public class MyTestClass {
   // Test methods
}

The following example defines a test class that uses the 

RANDOMIZED
algorithm. 

          
           
@TestInstance(TestInstance.Lifecycle.PER_CLASS,
                TestInstance.Algorithm.RANDOMIZED)
public class MyTestClass {
   // Test methods
}

Conclusion

JUnit Jupiter's 
@TestInstance
annotation is a powerful and versatile annotation that allows you to configure the test instances lifecycle and run order of test methods within a given test class.
