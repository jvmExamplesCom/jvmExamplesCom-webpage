org.elasticsearch.action.delete.DeleteResponse

Overview

The org.elasticsearch.action.delete.DeleteResponse class is a part of the Elasticsearch Java API. It provides methods for deleting documents from an Elasticsearch index. This class helps developers to delete documents from an index in an easy and efficient way.

Methods

The DeleteResponse class provides several methods for deleting documents from an index. The following are some of the commonly used methods:

  • setResult(GetResult result): Sets the result of the delete operation. It takes a GetResult object as a parameter.
  • getResult(): Gets the result of the delete operation.
  • isFound(): Checks if the document was found in the index.
  • getIndex(): Gets the name of the index the document was deleted from.
  • getType(): Gets the type of the document that was deleted.
  • getId(): Gets the id of the document that was deleted.
  • getVersion(): Gets the version of the document that was deleted.
