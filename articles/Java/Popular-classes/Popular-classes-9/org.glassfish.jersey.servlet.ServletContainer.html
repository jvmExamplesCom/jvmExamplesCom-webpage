org.glassfish.jersey.servlet.ServletContainer

The org.glassfish.jersey.servlet.ServletContainer is a Java class from the Jersey library that allows developers to build applications based on the Java Servlet API 2.3 and serve them using any servlet container/application server that supports the Servlet API 2.3 or higher.

Overview

The ServletContainer is a high-level abstraction over the HTTP request/response interaction model and provides the basic functionality for serving up servlets from a web application. It provides a read-only access to the incoming request and a write-only access to the outgoing response. The ServletContainer class is designed to be extensible and customizable, allowing developers to easily create various web application architectures, such as RESTful services with JSON support, XML-RPC applications, and traditional HTML web sites.

The ServletContainer is designed to handle multiple requests concurrently and has built-in support for servlet sessions, giving developers the ability to store user data within a session for the duration of the request cycle.
