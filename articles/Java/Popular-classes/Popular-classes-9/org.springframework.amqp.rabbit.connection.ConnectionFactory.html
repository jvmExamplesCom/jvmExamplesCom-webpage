Introduction to org.springframework.amqp.rabbit.connection.ConnectionFactory

The org.springframework.amqp.rabbit.connection.ConnectionFactory is a Java class used to create and configure connections to an AMQP-compliant messaging system. Create a ConnectionFactory

ConnectionFactory factory = new ConnectionFactory(SSLContext.getInstance());

Open a Connection

Connection connection = factory.openConnection();
