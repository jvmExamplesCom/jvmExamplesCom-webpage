JSONObject jsonObject = new JSONObject();

jsonObject.put("key3", true);

jsonObject.put("key1", "newValue");
Object newValue = jsonObject.get("key1");

method returns an Iterator which can be used to loop through all the keys in the JSONObject. 

          
           
JSONObject jsonObject = new JSONObject();

Iterator<String> iterator = jsonObject.keys();
