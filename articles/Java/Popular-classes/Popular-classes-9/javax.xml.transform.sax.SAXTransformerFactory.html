javax.xml.transform.sax.SAXTransformerFactory

The javax.xml.transform.sax.SAXTransformerFactory class provides a way of constructing a SAX (Simple API for XML) parser and generating SAX events from an XML document. It is an implementation of the javax.xml.transform.TransformerFactory interface. This class can be used to create an instance of a SAXTransformerFactory and then process an XML document using the SAX TransformerFactory produced.

Syntax

The syntax for javax.xml.transform.sax.SAXTransformerFactory is as follows −


          
           
SAXTransformerFactory sFactory = SAXTransformerFactory.newInstance();

Parameters

sFactory − This is an instance of SAXTransformerFactory.
