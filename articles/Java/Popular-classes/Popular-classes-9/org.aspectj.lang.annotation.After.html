Introduction to org.aspectj.lang.annotation.After

The 

          
           
org.aspectj.lang.annotation.After
annotation represents an aspect or a set of aspects that will be executed after a certain set of join points have been executed. Syntax

@After(value = "expression")
void aspectName();
