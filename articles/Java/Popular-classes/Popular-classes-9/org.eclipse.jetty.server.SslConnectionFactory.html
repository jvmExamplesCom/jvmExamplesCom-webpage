SslConnectionFactory in Eclipse Jetty

Overview of SslConnectionFactory

The 

          
           
SslConnectionFactory

When a client requests a secure connection, the SslConnectionFactory creates a socket and begins the SSL/TLS handshake process with the client. Once the handshake is successful, a secure connection is established and the server can send encrypted messages to the client. The SslConnectionFactory also provides an authentication mechanism that can be used to authenticate the remote client.

SslConnectionFactory

SslContextFactory
. Finally, you’ll need to specify an IP address and port to listen on when the server receives client requests. Here’s an example of a
