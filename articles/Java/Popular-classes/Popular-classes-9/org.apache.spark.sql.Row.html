Overview of org.apache.spark.sql.Row

org.apache.spark.sql.Row is a class used in Apache Spark that can represent an entire row of data in a schema-based DataFrame. It is a Java type, which gives the ability to a DataFrame to store data of more than one type. Row objects aid in efficient storage, retrieval and processing of data.

Creating a Row Object

A Row object can be created with the help of org.apache.spark.sql.RowFactory class. This class provides various factory methods to create Row objects. The following example creates a Row object from an array of objects.


          
           
Row row = RowFactory.create(new Object[]{ "John", 25 });

Accessing Values of a Row Object

The values of a Row object can be accessed using its get() method which takes an index as the argument and returns the element present at that index. The example below returns "John" i.e. the element stored at index 0.


          
           
row.get(0)

Updating Values of a Row Object

The values of a Row object can be updated using its set() method which takes two arguments - index and value . The example below sets the second element of the row (25) to 30.


          
           
row.set(1, 30);
