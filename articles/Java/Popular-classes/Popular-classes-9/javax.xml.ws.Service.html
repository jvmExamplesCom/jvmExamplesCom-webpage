This class is the starting point for client programs that need to access a web service. It offers access to several methods for accessing web services, such as creating proxy objects or access the Dispatch interface.

The Service class also provides access to the Dispatch interface. The Dispatch interface allows interaction with web services that don't follow the standard Java API for XML Web Services (JAX-WS).
