Introduction to org.bukkit.enchantments.Enchantment

org.bukkit.enchantments.Enchantment is an API class that is used to enchant items in Minecraft. It provides several methods and constants for manipulating enchantments on items, such as adding, removing, and changing their levels. key

Enchantment

ItemMeta.addEnchant(Enchantment enchantment, int level, boolean ignoreLevelRestriction)
