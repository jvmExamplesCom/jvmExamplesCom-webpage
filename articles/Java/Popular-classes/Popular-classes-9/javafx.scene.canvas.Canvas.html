JavaFX Scene Canvas

JavaFX Scene Canvas is a lightweight JavaFX component for drawing 2D graphics from the scene graph. It allows you to easily draw shapes, images and text onto the canvas. It supports basic features such as painting on canvas, blurring and filling in areas.

Creating a Canvas

A Canvas object can be created from the JavaFX Scene Graph by calling the Canvas constructor:


          
           
Canvas canvas = new Canvas();

The size of the Canvas will be determined by the associated width and height properties. The Canvas may be resized by setting the width and height properties:


          
           
canvas.setWidth(200);
canvas.setHeight(100);

Drawing to the Canvas

Once a Canvas has been created, it may be used to draw shapes, images, and text to its associated context. To obtain the GraphicsContext associated with a Canvas, use the getGraphicsContext2D() method:


          
           
GraphicsContext gc = canvas.getGraphicsContext2D();

You can now call methods of GraphicsContext to draw shapes, images, and text on the canvas. For example, to draw a rectangle at coordinates (10, 10) with a width of 80 and a height of 60, use the fillRect() method like this:


          
           
gc.fillRect(10, 10, 80, 60);

Other available methods include drawImage(), strokeLine(), fillOval(), and more. There is also an extensive set of methods for controlling the style of the shapes, including stroke, fill, line width, and more.

Transforming the Canvas

The Canvas is affected by any transformations applied to the associated Scene object. To rotate the Canvas 45 degrees around its origin point, you can use the rotate() method of Scene:


          
           
scene.rotate(45);

This will rotate the Canvas and all of its contents according to the specified angle. Other transformations may be applied in a similar manner.

Using Canvas with Charts

Canvas can be used in combination with other components, such as charts. To add a chart to a Canvas, use the ChartView class. This class allows you to create a ChartNode and attach it to the Canvas:


          
           
ChartView chart = new ChartView();
chart.attachTo(canvas);

This will create a ChartNode which is linked to the Canvas, so changes to the data or layout of the charts are automatically reflected on the Canvas.
