Android Support v7 App NotificationCompat

Android Support v7 App NotificationCompat is a class of the Android support library which provides a consistent API to display notifications in an app. It is used by developers to generate an explicit intent for launching activities, broadcast receivers, and services in response to user interaction with the notification. This class provides advanced features such as being able to specify custom actions, display text and images, and control sound and vibration.

Features of Android Support v7 App NotificationCompat

  • NotificationCompat.Builder class provides a consistent API for creating notifications.
  • The compatibility library provides many features, such as displaying large icons, expanding notifications, and providing a consistent look and feel.
  • It supports all Android versions from 1.6 onwards.
  • The Builder class allows developers to customize their notifications with additional content such as text, images, sounds, and vibration.
  • NotificationCompat.Builder provides an easy way to integrate custom actions into the notification.

How to Create a Notificaiton Using Android Support v7 App NotificationCompat

The following code example shows how to create a notification using Android Support v7 App NotificationCompat. 

          
           
NotificationCompat.Builder builder =
        new NotificationCompat.Builder(context)
                .setSmallIcon(R.drawable.notification_icon)
                .setContentTitle("My Notification")
                .setContentText("Hello World!");

// Gets an instance of the NotificationManager service
NotificationManager notificationManager =
        (NotificationManager) getSystemService(Context.NOTIFICATION_SERVICE);

// Builds the notification and issues it.
notificationManager.notify(0, builder.build());
The above code creates a notification with the specified title, text, and icon. It also retrieves an instance of the NotificationManager service and uses this to issue the notification.
