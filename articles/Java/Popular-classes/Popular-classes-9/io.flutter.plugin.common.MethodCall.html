The io.flutter.plugin.common.MethodCall class is an abstract base class used to process incoming method calls from Flutter code to the host platform. When using the platform channels feature of Flutter, the host platform (Android or iOS) listens for incoming method calls from the Flutter application and invokes the applicable method in response.

Creating a MethodCall Object

To create a MethodCall object, you must provide both the method name (as a String) and the arguments that are passed along with it. The arguments can be of any type, including null.


          
           
MethodCall call = new MethodCall("my_method", "argument1", 4);

Invoking a MethodCall

Response response = await platform.invokeMethod(call);
