Introduction to javax.crypto.spec.PBEParameterSpec

javax.crypto.spec.PBEParameterSpec

PBEKeySpec

encodePassword

byte[] encodePassword(char[] password) {
   int length = password.length;
   byte[] encodedPassword = new byte[length];
   for (int i = 0; i < length; i++) {
      encodedPassword[i] = (byte)password[i];
