IWorkspaceRoot Java Code Examples

Overview

The org.eclipse.core.resources.IWorkspaceRoot class allows developers to access and manipulate Eclipse projects, files, folders and other resources associated with the workspace. With it, a developer can easily create a new project, copy or delete existing ones, access the contents of a file, modify file permissions, and more. This article will provide several code examples related to the IWorkspaceRoot class that demonstrate some of these features.

Creating a New Project

The following code creates a new project in the current workspace: 

          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
String projectName = "MyProject";
IProject project = root.getProject(projectName);
project.create(null);
project.open(null);

Copying an Existing Project

This code example copies an existing project in the workspace: 

          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
IProject sourceProject = root.getProject("ExistingProject");
IProject targetProject = root.getProject("NewProject");
sourceProject.copy(targetProject.getFullPath(), true, null);

Deleting a Project

This code deletes an existing project in the workspace: 

          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
IProject project = root.getProject("MyProject");
project.delete(true, null);

Accessing a File

The following code example demonstrates how to access the contents of a file within a project in the workspace: 

          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
IProject project = root.getProject("MyProject");
IFile file = project.getFile("myFile.txt");
InputStream contents = file.getContents();
byte[] bytes = IOUtils.toByteArray(contents);
String content = new String(bytes, StandardCharsets.UTF_8);
System.out.println(content);

Modifying File Permissions

This code example shows how to modify the permissions of a file in the workspace: 

          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
IProject project = root.getProject("MyProject");
IFile file = project.getFile("myFile.txt");
file.setReadOnly(true);

Finding All Projects in the Workspace

This code example finds all projects in the current workspace: 

          
           
IWorkspaceRoot root = ResourcesPlugin.getWorkspace().getRoot();
IProject[] projects = root.getProjects();
for (IProject project : projects) {
	System.out.println(project.getName());
}
