org.w3c.dom.ls.LSSerializer

The LSSerializer can be used to turn any DOM document into an XML representation of the document. This can be useful not only for displaying the DOM structure, but also for sending the DOM to a file, or to another application.

Document doc = domImpl.createDocument(null, "html", docType);

    <!-- Content Goes Here -->
