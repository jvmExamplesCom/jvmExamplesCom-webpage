Introduction to org.junit.jupiter.api.extension.ParameterContext

The org.junit.jupiter.api.extension.ParameterContext class is a support infrastructure for the parameter injection mechanism in JUnit Jupiter. It enables discovery of parameters and resolution of their values at runtime.

ParameterContext provides access to the following information:

  • The name of the parameter as declared on the method or constructor;
  • The type of the parameter;
  • An Optional that may contain the resolved parameter value.
