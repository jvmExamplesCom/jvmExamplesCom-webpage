ArchiveEntry in Java

The ArchiveEntry interface is part of the org.apache.commons.compress.archivers package of the Apache Commons Compress library. This library can be used to create, read, and write archives in several different compressed file formats. The ArchiveEntry class provides an abstraction layer from the details of how the archives are stored. It allows access to the meta-data of a single entry within an archive.

Constructors

The following constructors are available when creating an ArchiveEntry:


          
           
ArchiveEntry(String name)
ArchiveEntry(String name, boolean isDirectory)
ArchiveEntry(String name, boolean isDirectory, int size)
ArchiveEntry(String name, boolean isDirectory, int size, long lastModifiedTime)
ArchiveEntry(String name, boolean isDirectory, int size, long lastModifiedTime, String comment)

Methods

The following methods are available for ArchiveEntry objects:


          
           
getName(): Returns the name of the entry.
isDirectory(): Returns true if the entry is a directory.
getSize(): Returns the size of the entry.
getLastModifiedTime(): Returns the last modified time.
getComment(): Returns the comment associated with the entry.
setComment(String comment): Sets the comment associated with the entry.

Examples

The following example creates an ArchiveEntry object and prints its properties.


          
           
ArchiveEntry entry = new ArchiveEntry("example.txt", false, 12345, System.currentTimeMillis());
System.out.println(entry.getName()); // example.txt
System.out.println(entry.isDirectory()); // false
System.out.println(entry.getSize()); // 12345
System.out.println(entry.getLastModifiedTime()); // timestamp
System.out.println(entry.getComment()); // null

The following example sets a comment for the entry.


          
           
ArchiveEntry entry = new ArchiveEntry("example.txt", false, 12345, System.currentTimeMillis());
entry.setComment("comment");
System.out.println(entry.getComment()); // comment
