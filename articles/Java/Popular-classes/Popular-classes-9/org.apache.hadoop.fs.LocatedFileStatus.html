org.apache.hadoop.fs.LocatedFileStatus

org.apache.hadoop.fs.LocatedFileStatus is a Java class in the hadoop-common package of the Apache Hadoop framework that provides information about files in the Hadoop distributed file system. This class extends the FileStatus class, and adds an extra field to identify the location of the file in the Hadoop cluster.

Methods

  • getBlockLocations: Gets the locations of all blocks of the file.
  • getLen: Returns the length of the file.
  • getModificationTime: Gets the timestamp of when the file was last modified.
  • getPath: Returns the file's path.
