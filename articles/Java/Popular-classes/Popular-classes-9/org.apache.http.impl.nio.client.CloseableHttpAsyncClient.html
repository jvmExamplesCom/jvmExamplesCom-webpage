CloseableHttpAsyncClient
is the main entry point for using this library. It is responsible for managing the lifecycle of the HTTP request and response.

CloseableHttpAsyncClient
class is simple. To create an instance, you need to provide a valid implementation of the 

          
           
CloseableHttpAsyncClient
interface. The code below creates an asynchronous client that can be used with any HTTP 1.1 compliant server: 

          
           
CloseableHttpAsyncClient httpclient = HttpAsyncClients.createDefault();
httpclient.start();
