org.opencv.core.Scalar

The Scalar class holds four double values, referred to as x, y, z and w. The Scalar class is used in OpenCV mainly for two application scenarios:

  • 1st scenario, the parameters are used to define a color value in the BGR or grayscale color space using the 
    
            
             
Scalar(Blue, Green, Red, Alpha)
    constructor. Alpha parameter is optional and defaults to 255(completely opaque).
  • 2nd scenario, the parameters are used in a multi-dimensional array to identify each individual element of the matrix or image.

Example 1

In this example, let's say you have a 64x64 pixel binary matrix and you want to identify an element at coordinate (3, 4):


          
           
// Create a binary matrix of size 64x64
Mat binaryMatrix = Mat::zeros(64, 64, CV_8UC1);
// Find the element at coordinate (3,4)
Scalar element = binaryMatrix.at<uchar>(3, 4);
std::cout << "Element at (3, 4): " << element << std::endl;
Output: 

          
           
Element at (3, 4): 0

Example 2

In this example, let's say you want to create a color image with a green background and a red circle in the center:


          
           
// Create a color image with a green background
Mat greenBackground = Mat::zeros(640, 480, CV_8UC3);
greenBackground.setTo(Scalar(0, 255, 0));
// Draw a red circle in the center
circle(greenBackground, Point(320, 240), 50, Scalar(0, 0, 255), -1);
Output: 

          
           
A 640 x 480 color image with a green background and a red circle in the center.
