javax.swing.event.TreeModelEvent

javax.swing.event.TreeModelEvent is an event class used to notify listeners of changes to a tree structure.   • getChildIndices(): This method returns an array of integers that represent the indices of the changed children in their parent node.
  • getChildren(): This method returns an array of TreeNode objects that represent the changed children in their parent node.
