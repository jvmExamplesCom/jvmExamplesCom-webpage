Java Security Spec - ECParameterSpec

java.security.spec
package that provides an immutable class for specifying elliptic curve domain parameters. It can be used for Elliptic Curve Cryptography (ECC) algorithms such as ECDH, ECDSA, and ECIES. The parameters are encoded as an ASN.1 SEQUENCE and the first component specifies the globals fields, the second component is the base point and the third component is the order of the base point.
