Introduction to Java.awt.MouseInfo

Java.awt.MouseInfo is a Java AWT class that provides methods for getting information on the current mouse state like location, number of buttons and so on. This class enables applications to determine the presence of a mouse or obtain the current coordinates.

Methods

  • getNumberOfButtons() : Gets the number of buttons on the mouse.
  • getPointerInfo() : Returns a PointerInfo object which contains the location of the mouse pointer.
  • getX() : Returns the x-coordinate of the mouse pointer.
  • getY() : Returns the y-coordinate of the mouse pointer.

Example

In this example, we will write a program to get the current mouse location. 


          
           
import java.awt.MouseInfo;

public class MouseLocation {
	public static void main(String[] args) {
		int x = MouseInfo.getX();
		int y = MouseInfo.getY();
		System.out.println("The mouse location is: (" + x + ", " + y + ")");
	}
}
Output: 

          
           
The mouse location is: (x, y)
