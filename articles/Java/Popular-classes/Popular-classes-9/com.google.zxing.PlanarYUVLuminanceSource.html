com.google.zxing.PlanarYUVLuminanceSource

Introduction

com.google.zxing.PlanarYUVLuminanceSource is a Java library for reading and decoding 1D/2D barcodes in real-time using the ZXing (Zebra Crossing) library.

Features

The library offers a wide range of features which makes it suitable for a variety of barcode scanning applications. It supports many different types of barcodes, such as Aztec, Data Matrix and PDF417. It can also generate encoded images from barcodes and output them as JPEG or PNG.
