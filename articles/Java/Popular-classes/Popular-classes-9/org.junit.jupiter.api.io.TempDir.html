org.junit.jupiter.api.io.TempDir

The org.junit.jupiter.api.io.TempDir is an abstraction for a temporary directory that provides a standardized way to create a temporary, isolated file system location for test code with all the necessary cleanup. It aims to provide a simple, safe and efficient way to create and access a writable temporary directory.

Creating a TempDir Instance

TempDir tempDir = TempDir.create();

This will create a new instance of the TempDir class containing a newly created, empty temporary directory.

When you are done using the TempDir instance, you should call the release() method to clean up the resources associated with the TempDir instance:
