Introduction to Java.util.Currency

Java's 

          
           
java.util.Currency
is a class that allows developers to access currency-related information and perform currency conversions. It provides an efficient, easy way to handle currency information in a program. This article will provide a detailed overview of the features and capabilities of the 

          
           
java.util.Currency
class, including examples of its usage.

Features of Java.util.Currency

The 

          
           
java.util.Currency
class has several useful features for handling currency information:
  • It provides an easy way to list all currencies available in a given region.
  • It allows developers to access currency codes and names, as well as details such as currency symbol, fraction digits, and rounding mode.
  • It can be used to perform currency conversions, taking into account current exchange rates.
  • It provides methods for formatting monetary values in a given currency.

Example Usage

The following examples illustrate the usage of the 

          
           
java.util.Currency
class.

Getting a list of currencies

The 

          
           
getAvailableCurrencies()
method can be used to get a list of all the currencies available in a given region. The following example shows how to get a list of all the currencies available in the USA: 

          
           
Set<Currency> currencies = Currency.getAvailableCurrencies("US");
for (Currency currency : currencies) {
    System.out.println(currency);
}
The output of this code will be something like this: 

          
           
US Dollar (USD)
Euro (EUR)
British Pound Sterling (GBP)
Japanese Yen (JPY)
Canadian Dollar (CAD)

Accessing currency information

The 

          
           
getCurrencyCode()
, 

          
           
getSymbol()
, 

          
           
getFractionDigits()
, and 

          
           
getRoundingMode()
methods can be used to access basic information about a given currency. For example, to get the currency code and symbol for the US Dollar: 

          
           
Currency usd = Currency.getInstance("USD");
String currencyCode = usd.getCurrencyCode(); // USD
String symbol = usd.getSymbol(); // $

Currency conversion

The 

          
           
convert()
method can be used to perform currency conversions, taking into account current exchange rates. The following example shows how to convert 10 US Dollars to Euros: 

          
           
Currency usd = Currency.getInstance("USD");
Currency eur = Currency.getInstance("EUR");
double amount = 10.0;
double convertedAmount = usd.convert(amount, eur);
System.out.println("10 USD = " + convertedAmount + " EUR");
The output of this code will be something like this: 

          
           
10 USD = 8.69 EUR

Formatting monetary values

The 

          
           
format()
method can be used to format monetary values in a given currency. The following example shows how to format 10.5 US Dollars: 

          
           
Currency usd = Currency.getInstance("USD");
double amount = 10.5;
NumberFormat formatter = NumberFormat.getCurrencyInstance(usd);
String formattedAmount = formatter.format(amount);
System.out.println(formattedAmount);
The output of this code will be something like this: 

          
           
$10.50
