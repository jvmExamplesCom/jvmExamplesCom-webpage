Introduction to org.apache.poi.hssf.usermodel.HSSFWorkbook

org.apache.poi.hssf.usermodel.HSSFWorkbook is a Java API for Microsoft documents. It is part of the Apache POI extended family and provides classes for reading and writing Microsoft Excel workbooks (XLS). This API allows developers to programmatically create, edit, and manipulate Microsoft Excel spreadsheets in an efficient and automated way.

Example Code 1: Creating a Workbook

The following code example creates an empty workbook with three sheets: 

          
           
// Create the Workbook
HSSFWorkbook wb = new HSSFWorkbook();

// Create Sheet1
Sheet sheet1 = wb.createSheet("Sheet1");

// Create Sheet2
Sheet sheet2 = wb.createSheet("Sheet2");

// Create Sheet3
Sheet sheet3 = wb.createSheet("Sheet3");

// create row and cell
Row row = sheet1.createRow(0);
Cell cell = row.createCell(0);

// set the cell value

Example Code 4: Reading Data from a Workbook

We can also read data from a workbook using the org.apache.poi.hssf.usermodel.HSSFWorkbook class. The following example reads data from an .xls file and prints it out. 

          
           
// get the file input stream
FileInputStream fis = new FileInputStream(fileName);

// create the workbook
HSSFWorkbook wb = new HSSFWorkbook(fis);

// loop through the sheets of the workbook
for (int i = 0; i < wb.getNumberOfSheets(); i++) {
    // get the sheet
    Sheet sheet = wb.getSheetAt(i);

    // loop through the rows
    for (int j = 0; j <= sheet.getLastRowNum(); j++) {
        // get the row
        Row row = sheet.getRow(j);

        // loop through the cells    
        for (int k = 0; k < row.getLastCellNum(); k++) {
            // get the cell
            Cell cell = row.getCell(k);

            // print out the cell value
            System.out.println(cell.getStringCellValue());
        }
    }
}

//close the file input stream
fis.close();
