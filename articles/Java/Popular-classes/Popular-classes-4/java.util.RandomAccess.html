java.util.RandomAccess

The java.util.RandomAccess package provides an interface, implementations and utilities that allow the efficient reading and writing of individual elements of a list, vector, array and other commonly used data structures. This is accomplished by providing an API that allows users to access any element of the data structure in constant time.

Overview of RandomAccess Interface

The RandomAccess interface defines the methods for accessing elements in a list, vector, or array. The implementing class must provide support for the following operations:


          
           
randomAccess(int i): returns the ith element of the data structure

hasRandomAccess(): returns true if random access is supported

getRandomAccessSize(): returns the size of the data structure

removeRandomAccessElement(int i): removes the ith element from the data structure

addRandomAccessElement(int i, Object o): adds the object o at the ith index of the data structure

In addition, the implementing class may choose to implement its own custom methods as defined by its own data structure.

Example Usage of RandomAccess Interface

Below is an example of how to use the RandomAccess interface with a SimpleArrayList, which implements the RandomAccess interface. We will be adding three elements to the list and then accessing the second element of the list.


          
           
// Create a new SimpleArrayList
SimpleArrayList list = new SimpleArrayList();

// Add elements to the list
list.addRandomAccessElement(0, "a");
list.addRandomAccessElement(1, "b");
list.addRandomAccessElement(2, "c");

// Get the second element of the list
Object element = list.randomAccess(1); // returns "b"

The java.util.RandomAccess package provides an API for efficiently reading and writing individual elements of a list, vector, array and other commonly used data structures.
