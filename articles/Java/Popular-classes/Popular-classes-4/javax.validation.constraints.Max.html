javax.validation.constraints.Max

javax.validation.constraints.Max is an annotation used to verify that the value of an annotated element is not greater than a specified maximum.

Usage

The Max annotation can be used with String, primitive, or wrapper types.

Example 1 - Verifying Primitive Types

For example, we can verify that an integer value is not greater than 10 with the following code:


          
           
import javax.validation.constraints.Max; 
 
public class MaxValidationExample { 
 
    @Max(10)  
    private int maxValue; 
 
    public int getMaxValue() { 
        return maxValue; 
    } 
 
    public void setMaxValue(int maxValue) { 
        this.maxValue = maxValue; 
    } 
}

Example 2 - Verifying Wrapper Types

For example, we can verify that a Double value is not greater than 10 with the following code:


          
           
import javax.validation.constraints.Max;

    } 
}

Example 3 - Verifying String Types

}

Conclusion

As you can see, the javax.validation.constraints.Max annotation is a powerful tool for ensuring that values conform to a specified maximum. With it, you can make sure that your application data is valid and secure.
