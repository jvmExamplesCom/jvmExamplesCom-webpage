Introduction to javafx.scene.input.MouseEvent

The 

          
           
javafx.scene.input.MouseEvent
is an event class in JavaFX which contains information about mouse events such as mouse clicks and mouse drags. It also has methods related to mouse coordinates, double-clicking, grabbing the mouse focus, etc. In this article, we will explore these methods and demonstrate their implementations with examples.

Mouse Coordinates

The 
getX()
and 
getY()
methods of the 
MouseEvent
class return the x and y coordinates of the mouse cursor respectively at the time the mouse event was triggered. 

          
           
Node node = ...;
node.setOnMousePressed(e -> {
    double x = e.getX();
    double y = e.getY();
});

Here, when the mouse is pressed, the x and y coordinates are printed.

Double Click

The 
isStillSincePress()
method returns true if the time elapsed since the last press of the mouse button is greater than the time defined in the system configurations. This way we can implement double-click behavior. 

          
           
Node node = ...;
node.setOnMouseClicked(e -> {
    if (e.isStillSincePress()) {
        // handle double click
    }
});

Grabbing the Mouse Focus

The 
grabFocus()
method requests the focus for the Node, so that all mouse events now go to the Node until it releases the focus. 

          
           
Node node = ...;
node.setOnMouseMoved(e -> node.grabFocus());

Here, when the mouse is moved, the node will grab the focus.
