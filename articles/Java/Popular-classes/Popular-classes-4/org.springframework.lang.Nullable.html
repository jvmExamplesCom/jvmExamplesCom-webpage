Overview of org.springframework.lang.Nullable

The org.springframework.lang.Nullable annotation is a type annotation targeting the Java type system, allowing declarative null-safety at the language level. Introduced as part of Spring Framework 4.2, this annotation provides a mechanism to detect and prevent errors related tonull values.

Usages of org.springframework.lang.Nullable
