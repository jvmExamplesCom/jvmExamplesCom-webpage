Introduction to JUnit Jupiter ExtensionContext

The org.junit.jupiter.api.extension.ExtensionContext is an interface in the JUnit 5 extension model. ExtensionContext provides the context in which an extension can be used. It provides access to the current test instance and container (if it exists), as well as other useful methods for accessing records and configuration.

Using the ExtensionContext Interface

The ExtensionContext interface allows extensions to obtain information about the current test instance and the container, as well as a map of records that may have been created by other extensions. The following example shows how to access the ExtensionContext from within an extension:


          
           
@Override
public void beforeAll(ExtensionContext context) {
  Object testInstance = context.getTestInstance();

}
