Introduction to Apache Commons HTTPClient

org.apache.commons.httpclient.HttpClient is a Java language library used to allow users to perform HTTP requests and receive responses from web servers over the internet. It provides a simple interface for crafting and sending requests, and parsing and interpreting responses, including support for HTTPS, authentication and cookies.

Creating an HttpClient Object

The most basic way to create an HttpClient object is to simply use the default constructor: 


          
           
HttpClient httpClient = new HttpClient();

Making Requests

Once you have an HttpClient object, you can make requests using the executeMethod() method. This method has two required parameters: an HttpMethod object that contains the request parameters and a HostConfiguration object that supplies the target server host name and port. For example: 


          
           
HostConfiguration hostConfig = new HostConfiguration();
hostConfig.setHost("www.example.com");
GetMethod getMethod = new GetMethod("/index.html");
int statusCode = httpClient.executeMethod(hostConfig, getMethod);
The executeMethod() method returns the status code of the response.

Handling Responses

Once the request is complete, the response will be available in the HttpMethod object. For example, the contents of the response can be retrieved using: 


          
           
InputStream responseStream = getMethod.getResponseBodyAsStream();

More Advanced Features

HttpClient also supports more advanced features such as authentication, cookies and HTTPS. For example, authentication can be used by setting the credentials on the HttpState object: 


          
           
HttpState state = new HttpState();
state.setCredentials(AuthScope.ANY, new UsernamePasswordCredentials("username", "password"));
httpClient.setState(state);
