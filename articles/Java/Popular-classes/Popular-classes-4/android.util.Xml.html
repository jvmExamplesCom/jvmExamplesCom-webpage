Introduction to android.util.Xml

android.util.Xml is an Android class used for basic XML manipulation. It can be used for both reading and writing XML documents. It provides methods for parsing, navigating, transforming, and serializing XML documents. The XML document could be an entire document from a file, or a portion of a document. In this article, we will look at the various classes and methods of Android's android.util.Xml class and create some code examples of how to use it.

android.util.Xml Classes

The main classes of android.util.Xml are XmlPullParser and XmlSerializer. XmlPullParser is a pull parser that supports the XML 1.0 specification. It is used to parse an XML document. The parser iterates through the elements of the document and reports its type, text content and attributes. XmlSerializer is a serializer used for converting XML documents into a stream of bytes. It provides methods for creating a start tag, adding attributes, creating an end tag, and writing text.

Example: Parsing XML Document

In this example, we will be using XmlPullParser for parsing an XML document. We will be using the following XML document: 

          
           
<!--?xml version="1.0" encoding="UTF-8"?-->
<document>
  <item>
    <name>John</name>
    <age>25</age>
  </item>
  <item>
    <name>Alice</name>
    <age>20</age>
  </item>
</document>
We will use the XmlPullParser to loop through the elements of the document and print out the contents. 

          
           
XmlPullParser parser = Xml.newPullParser();
int eventType = parser.getEventType();
while (eventType != XmlPullParser.END_DOCUMENT) {
    switch (eventType) {
        case XmlPullParser.START_TAG:
            if (parser.getName().equals("name")) {
                System.out.println("Name: " + parser.nextText());
            } else if (parser.getName().equals("age")) {
                System.out.println("Age: " + parser.nextText());
            }
            break;
    }
    eventType = parser.next();
}
The output of the code will be: 

          
           
Name: John
Age: 25
Name: Alice
Age: 20

<document>

We will use the XmlSerializer to create an XML document. 

          
           
StringWriter writer = new StringWriter();
XmlSerializer serializer = Xml.newSerializer();

serializer.startTag("", "document");

serializer.endTag("", "age");
serializer.endTag("", "item");

serializer.endTag("", "document");
serializer.endDocument();

String result = writer.toString();
