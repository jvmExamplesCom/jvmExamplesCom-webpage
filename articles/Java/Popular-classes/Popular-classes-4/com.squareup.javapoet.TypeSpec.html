Introduction to com.squareup.javapoet.TypeSpec

JavaPoet is a Java API for generating .java source files. It is maintained by Square and is used in many of their applications. It is used to generate code from annotations. The most basic class in the JavaPoet API is the 

          
           
TypeSpec
class.

Creating a TypeSpec Object

Creating a TypeSpec object requires two parameters: the type name and either a list of members, an anonymous class, or a static nested class. For example, the following code creates a 

          
           
TypeSpec

Foo
: 

          
           
TypeSpec foo = TypeSpec.classBuilder("Foo")
        .build();
The resulting 

          
           
TypeSpec

JavaFile
class. This class provides methods for writing out a 

          
           
.java
file with a given package name and 

          
           
TypeSpec
