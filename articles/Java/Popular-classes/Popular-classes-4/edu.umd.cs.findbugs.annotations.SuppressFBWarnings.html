edu.umd.cs.findbugs.annotations.SuppressFBWarnings

edu.umd.cs.findbugs.annotations.SuppressFBWarnings is an annotation type that is used to suppress warnings reported by code analysis tools such as FindBugs. It is a part of the FindBugs project and is used to suppress specific warnings, either permanently or temporarily.

Syntax


          
           
@SuppressFBWarnings(value = { "warning1", "warning2" })
The 
value
parameter defines the warnings that should be suppressed. The value can be either a single warning code or multiple warning codes. The warning code refers to the specific warning code reported by FindBugs.

Examples


          
           
@SuppressFBWarnings("UPM_UNCALLED_PRIVATE_METHOD")
This will suppress the warning code 

          
           
UPM_UNCALLED_PRIVATE_METHOD
. 

          
           
@SuppressFBWarnings(value = { "NP_ALWAYS_NULL", "NP_NULL_ON_SOME_PATH" })
This will suppress the warning codes 
NP_ALWAYS_NULL
and 

          
           
NP_NULL_ON_SOME_PATH
.
