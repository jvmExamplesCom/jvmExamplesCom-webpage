JavaFX Scene Parent

The 

          
           
javafx.scene.Parent
class is a base class of all nodes that have children in the JavaFX scenegraph. It provides a convenient set of tools for adding, removing and converting children as well as a tree like structure for organizing them.

public ObservableList<Node> getChilren()

public void add(Node node)

public void remove(Node node)
This method removes a child Node from the given Node. 

          
           
public void removeAll()
This method removes all children from the given Node. 

          
           
public void replace(Node oldNode, Node newNode)

public void setVisible(boolean visible)

public void setScaleX(double x)
