Introduction to Java.util.function.IntFunction

Java.util.function.IntFunction is a functional interface in Java 8 and part of the java.util.function package. It belongs to the family of functional interfaces provided with Java 8, which includes Supplier, Consumer, Function, and Predicate. An IntFunction is an interface that takes an int as argument and returns any value.

IntFunction Method Signature

The IntFunction method signature is as follows: 

          
           
public IntFunction<R> apply(int t)

Example of IntFunction

Here is an example of the IntFunction interface: 

          
           
IntFunction<String> intToString = num -> "" + num;

String getStringFromInt = intToString.apply(5); // return "5"
In the example, we declared an IntFunction called intToString that takes an int argument and returns a String. We then applied the intToString function to the 5 integer using the apply() method, and it returned the String “5”.

Using IntFunction with Streams

We can also use the IntFunction with Java Streams. Here is an example of using the IntFunction map() method with Streams: 

          
           
List<Integer> numbers = Arrays.asList(1, 2, 3, 4);

List<String> strings = numbers.stream()
      .map(num -> intToString.apply(num))
      .collect(Collectors.toList());

System.out.println(strings); // [1, 2, 3, 4]
In this example, we first created an arraylist of integers called numbers. We then used the Streams map() method to apply the intToString function to each element of the list. This returned an arraylist of Strings, which we printed to the console producing the result [1, 2, 3, 4].

IntFunction Use Cases

The IntFunction interface is a useful way to transform one type of data into another type. One common use case for IntFunction is transforming a list of integers into a list of strings. As we saw earlier, this can be done by applying the intToString function to each element in the list of integers. Another practical use case is mapping a list of integers to other custom objects. For example, imagine we had a list of Order IDs (integers) and we wanted to get a list of Order objects.
