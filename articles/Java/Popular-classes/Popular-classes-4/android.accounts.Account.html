Introduction to Android Accounts

Android Accounts are objects that represent an account held by a user on a device or cloud service. An account can store information about the user, such as their name and other personal details. It also stores credentials used to access the device or cloud service, such as an email address and password. By using Android Account objects, apps can securely store and manage user data.

Creating an Account

An Android Account can be created with the android.accounts.Account class. The following code example shows how to create a new Account object: 

          
           
Account account = new Account("username", "com.example");
The first argument of the constructor is the account username, and the second argument is the account type. This example uses a custom account type, but the standard types defined by Google are “com.google” and “com.facebook”.

Adding Data to an Account

Once an Account object has been created, data can be added to it. To do this, an Android app must use the AccountManager class. The following code example shows how to add a data element to an Account using the AccountManager class: 

          
           
Bundle userData = new Bundle();
userData.putString("name", "John Smith");
AccountManager.get(context).addAccountExplicitly(account, null, userData);
The code first creates a bundle to store the data elements. In this example, we are adding the user’s name. The addAccountExplicitly method is then called with the account, a password (which is set to null in this example), and the bundle of user data.

Retrieving Data from an Account

Data stored in an Account can be retrieved using the AccountManager class. The following code example shows how to retrieve a user’s name from an Account: 

          
           
String name = AccountManager.get(context).getUserData(account, "name");
The getUserData method is called with the Account and the name of the data element to retrieve. The returned value is the user’s name.

Deleting an Account

An Android Account can be deleted using the AccountManager class. The following code example shows how to delete an Account: 

          
           
AccountManager.get(context).removeAccount(account, null, null);
The removeAccount method is called with the Account, an activity (which is set to null in this example) and a callback (also set to null). Once the Account has been deleted, all associated data is also deleted.
