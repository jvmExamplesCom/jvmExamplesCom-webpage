Introduction to org.springframework.core.MethodParameter

The 

          
           
org.springframework.core.MethodParameter
class provides an introspection mechanism into a method's parameters, including type and annotations. This class is used in Spring's AOP framework and in the context of creating proxies for beans with lookup methods. By understanding the parameter types and their associated annotations, this class can be used to enable various types of advice and interception.

Constructors

The 

          
           
org.springframework.core.MethodParameter
class has two constructors: one which takes a 
Method
instance as a parameter, and one which takes both a 
Method
and an 
int
value representing the parameter index. 

          
           
public MethodParameter (Method method)

          

          
           
public MethodParameter (Method method, int parameterIndex)

Methods and Properties

The 

          
           
org.springframework.core.MethodParameter
class has several methods and properties which provide useful information about a method parameter: 
  • getDeclaringClass()
    returns the declaring class of the parameter.
  • getParameterName()
    returns the name of the parameter, if available.
  • getParameterType()
    returns the type of the parameter.
  • getParameterIndex()
    returns the index of the parameter within its declaring method.
  • 
            
             
getParameterAnnotations()
    returns an array of all annotations present on the parameter.
  • 
            
             
getGenericParameterType()
    returns the generic type of the parameter.
The following example shows how to use 

          
           
org.springframework.core.MethodParameter
to obtain the declaring class, parameter type and parameter index of a method parameter: 

          
           
MethodParameter param = new MethodParameter(method, 0);
Class<?> declaringClass = param.getDeclaringClass();
Class<?> paramType = param.getParameterType();
int paramIndex = param.getParameterIndex();
System.out.println("Declaring class: " + declaringClass);
System.out.println("Parameter type: " + paramType);
System.out.println("Parameter index: " + paramIndex);
