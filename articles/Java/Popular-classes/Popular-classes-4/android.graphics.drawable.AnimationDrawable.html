android.graphics.drawable.AnimationDrawable

The 

          
           
android.graphics.drawable.AnimationDrawable
is an animation drawable class used to animate a series of Drawables stored in an array. It is a part of the Android graphics package and thus provides access to the basic and powerful features of the Android platform.

Creating an AnimationDrawable

An instance of AnimationDrawable is created by passing it a list of Drawable resources as an array. Then, each Drawable can be added to the AnimationDrawable by iterating through the array. 

          
           
int[] animationDrawableArray = {
    R.drawable.image_1,
    R.drawable.image_2,
    R.drawable.image_3
};

AnimationDrawable animationDrawable = new AnimationDrawable();

for(int i = 0; i < animationDrawableArray.length; i++) {
    animationDrawable.addFrame(getResources().getDrawable(animationDrawableArray[i]), 100);
}

Activating the Animation

Once the AnimationDrawable is created, it must be activated for it to run. This is achieved by calling the start() method on the AnimationDrawable instance. 

          
           
animationDrawable.start();

Changing the Interval Between Frames

By default, the interval between frame changes is set to 1000 milliseconds (1 second). The duration of each frames can be individually set by passing the desired duration to the AnimationDrawable addFrame() method. 

          
           
for(int i = 0; i < animationDrawableArray.length; i++) {
    animationDrawable.addFrame(getResources().getDrawable(animationDrawableArray[i]), variableDuration);
}
In this example, variableDuration is assigned a different value for each frame, allowing for individual timing for each frame in the animation.

Stopping the Animation

To stop the animation, simply call the stop() method on the AnimationDrawable instance. 

          
           
animationDrawable.stop();
