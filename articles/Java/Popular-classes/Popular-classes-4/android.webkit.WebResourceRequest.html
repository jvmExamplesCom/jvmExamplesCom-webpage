shouldInterceptRequest

Creating a WebResourceRequest Object

A 

          
           
WebResourceRequest
can be created with the following code: 

          
           
WebResourceRequest request = new WebResourceRequest(String method, Uri url);
The first parameter is the HTTP method being performed, such as GET or POST. The second parameter is the URL of the resource being requested. This method also has an optional third parameter which is a Map of request headers.

WebResourceRequest
to intercept a request for a specific resource. In this example, we are attempting to intercept a request for a .png image file. 

          
           
public WebResourceResponse shouldInterceptRequest (WebView view, WebResourceRequest request) {
