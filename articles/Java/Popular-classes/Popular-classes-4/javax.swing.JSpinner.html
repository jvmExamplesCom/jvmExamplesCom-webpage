A javax.swing.JSpinner is a component that allows users to select one value from a finite list of values. It is a single line input field that offers a set of predefined values in a drop-down list. The user is able to scroll through the list, or type in a custom value.

The JSpinner class uses a SpinnerModel to determine what values to display in the drop-down list. The most commonly used SpinnerModel is the SpinnerListModel. This model takes an array of objects as its argument and displays them in the drop-down list.
