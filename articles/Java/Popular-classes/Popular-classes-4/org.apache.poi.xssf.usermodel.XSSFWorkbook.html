org.apache.poi.xssf.usermodel.XSSFWorkbook

The org.apache.poi.xssf.usermodel.XSSFWorkbook is a class that offers high-level presentations and support for working with Microsoft Excel documents in Java applications. Its features allow users to read, write, and create Excel documents in the XLSX format and perform various operations on them such as calculating formulas, formatting cells, and much more.

Code Example 1

The following example creates an instance of XSSFWorkbook and reads an Excel file:


          
           
XSSFWorkbook workbook = new XSSFWorkbook("fileName.xlsx");

Code Example 2

The following example creates a new sheet in the workbook and sets some properties for it:


          
           
XSSFSheet sheet = workbook.createSheet("Sheet1");
XSSFCellStyle style = workbook.createCellStyle();
style.setFillPattern(FillPatternType.SOLID_FOREGROUND);
sheet.setDefaultColumnStyle(0, style);

Code Example 3

The following example adds content to the sheet and writes the workbook to an output stream:


          
           
XSSFRow row = sheet.createRow(0);
XSSFCell cell = row.createCell(0);
cell.setCellValue("Hello World");
workbook.write(new FileOutputStream("output.xlsx"));

Code Example 4

The following example closes the workbook and releases any memory associated with it:


          
           
workbook.close();
