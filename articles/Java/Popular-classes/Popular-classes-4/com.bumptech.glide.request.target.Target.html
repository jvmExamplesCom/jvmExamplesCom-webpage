com.bumptech.glide.request.target.Target

Java applications using Glide can use the class com.bumptech.glide.request.target.Target as a target when starting a new load to provide information and receive callbacks throughout the life of the load. When an instance of Target is used, its methods like onLoadStarted(), onResourceReady(), and onLoadFailed() will be invoked in response to important events during the loading process.

Methods

In order to indicate its intention to respond to a particular event or action, an instance of Target must implement one of its three main methods: -

  • onLoadStarted: This method will be called when the process of fetching the resource from its source has begun.
  • onResourceReady: This method will be called after the requested resource has been fetched successfully.
  • onLoadFailed: If for some reason the process of fetching the resource fails, this method will be called with a message indicating what went wrong.
