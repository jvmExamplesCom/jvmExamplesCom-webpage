javax.persistence.Embeddable

javax.persistence.Embeddable is an annotation from the Java Persistence API (JPA). It is used to indicate that a class should be treated as an embeddable component. This annotation marks the class as an embeddable and defines the scope of the embeddable instance in the persistence context. Embeddables are typically used to store state within an entity.

Example of Embeddable Class

Below is an example of a simple Embeddable class: 

          
           
@Embeddable 
public class Address {
    private String street;
    private String city;
    private String zip; 
}

How to Embed the Embeddable in an Entity

Once an Embeddable class has been defined, it can be embedded into an entity class like this:


          
           
@Entity
public class Employee {
    ...
    @Embedded
    private Address address;
    ...
}

The above code will result in the Employee entity having a field for an Address object, which can be persisted along with the Employee record.

How to Use @AttributeOverrides Annotation

The @AttributeOverrides annotation can be used to customize how an embeddable is mapped to the database. For instance, if we wanted to specify a custom column name for the street field of the Address embeddable, we could use the @AttributeOverride annotation like this:


          
           
@Entity
public class Employee {
    @Embedded
    @AttributeOverrides({
        @AttributeOverride(name = "street", column = @Column("home_address"))
    })
    private Address address;
    ...
}

In this example, the street field of the Address embeddable will be mapped to a column named home_address in the database.

How to Use @AttributeOverride with Complex Types

The @AttributeOverride annotation can also be used to customize the mapping of a complex type. For instance, consider the following example:


          
           
@Entity
public class Employee {
    @ElementCollection
    @AttributeOverrides({
        @AttributeOverride(name = "street", column = @Column("work_address"))
    })
    private List<Address> addresses;
    ...
}

In this example, the street field of each Address in the addresses list will be mapped to a column named work_address in the database.

Conclusion

In this article we discussed the javax.persistence.Embeddable annotation from the Java Persistence API. We saw how to annotate an embeddable class, how to embed the embeddable in an entity, and how to customize the mapping using the @AttributeOverrides annotation.
