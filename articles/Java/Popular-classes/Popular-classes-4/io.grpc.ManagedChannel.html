io.grpc.ManagedChannel

io.grpc.ManagedChannel is a class in Java designed for managing a connection to a remote host for RPCs. It provides an API surface for initiating, maintaining and monitoring the associated RPC calls. It is designed to be used with clients that want to initiate multiple RPC calls over the same connection.

Creating a ManagedChannel

Creating a ManagedChannel is fairly easy. All you need to do is provide the host string, port and a ServerBuilder.


          
           
ManagedChannel channel = ManagedChannelBuilder.forAddress("localhost", 8080)
        .usePlaintext()
        .build();

Using the ManagedChannel

Once you have a ManagedChannel, you can make RPC calls to the remote host. You may also use it to monitor channels, such as getting state changes.

Making RPC Calls

You can invoke any method of the remote host by using the ManagedChannel's newCall() method. This method takes two arguments: a method descriptor and a ClientCall.Listener.

        // handle response
    }
