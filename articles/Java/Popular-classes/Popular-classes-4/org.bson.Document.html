org.bson.Document

The org.bson.Document class is a part of the MongoDB Java driver. It's a data structure that maps keys to values, allowing developers to store and manipulate JSON data.

Documents are composed of field-value pairs and have the following characteristics:

  • Field names must be strings
  • Field values may be any BSON data type
  • Fields can be nested documents and arrays

Using Documents, developers can easily construct complex data models in MongoDB while taking advantage of its flexible schema. They can perform CRUD operations, query and index documents, aggregate data, and more.
