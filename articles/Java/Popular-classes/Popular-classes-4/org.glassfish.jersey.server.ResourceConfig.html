org.glassfish.jersey.server.ResourceConfig

This class provides access to the underlying Jersey runtime environment and acts as a bridge between the Java world and the web application. It also allows for customizations of the Jersey runtime for different web applications.

ResourceConfig config = ResourceConfig.forEnvironment(env);
