org.springframework.boot.web.servlet.support.SpringBootServletInitializer

The org.springframework.boot.web.servlet.support.SpringBootServletInitializer class is a convenient way to configure a Spring Boot application as a traditional deployment in web servlet containers. Example

Here is an example of the SpringBootServletInitializer class:


          
           
public class MyApplication extends SpringBootServletInitializer {
    @Override
    protected SpringApplicationBuilder configure(SpringApplicationBuilder application) {
        return application.sources(MyApplication.class);
    }
}

The configure() method of the SpringBootServletInitializer class is called by the container when the application is deployed. The method takes a SpringApplicationBuilder object as an argument and returns the configured application.
