Android Text Util Linkify

Android Text Util Linkify is a class in Android that helps recognize and automatically turn various text kinds into clickable links. Linkify helps to detect web URLs, email addresses, phone numbers, map addresses and non-clickable internationalized resource identifiers (IRIs). It works by matching certain textual patterns and creating hyperlinks based on the pattern.

Methods

Linkify provides static methods that can be used to add clickable links to any text view or editable text:

  • 
            
             
addLinks(View view, int mask)
    - Automatically adds links to a View.
  • 
            
             
addLinks(Spannable spannable, int mask)
    - Automatically adds links to a Spannable.
  • 
            
             
addLinks(TextView textView, Pattern pattern, String scheme)
    - Applies a regex to the content of a TextView turning the matches into links.
  • 
            
             
addLinks(TextView textView, Pattern pattern, String scheme, Linkify.MatchFilter matchFilter, Linkify.TransformFilter transformFilter)
    - Applies a regex to the content of a TextView turning the matches into links while taking into account a matchFilter and a transformFilter.

Mask

The mask is a set of flags defining the default link types that should be matched. The params correspond to the various link types that can be recognized, e.g. WEB_URLS, EMAIL_ADDRESSES, PHONE_NUMBERS, IP_ADDRESSES, etc. Linkify also provides additional flags to define custom web schemes or to indicate that all web schemes should be matched.

Example

In this example, we are going to create a TextView with a String containing a web URL, an email address, and a phone number. We use the Linkify.addLinks method to automatically turn these strings into clickable links.


          
           
TextView textView = findViewById(R.id.text_view);
String str = "Visit us at https://www.example.com, or contact us at info@example.com or call us at +1 123 456 7890";
textView.setText(str);
Linkify.addLinks(textView, Linkify.ALL);

This will output a TextView with the following text:


          
           
Visit us at <a href="https://www.example.com">https://www.example.com</a>, or contact us at <a href="mailto:info@example.com">info@example.com</a> or call us at <a href="tel:+1 123 456 7890">+1 123 456 7890</a>
