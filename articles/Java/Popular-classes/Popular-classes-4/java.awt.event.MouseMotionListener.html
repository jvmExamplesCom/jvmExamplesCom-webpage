Java.awt.event.MouseMotionListener

The 

          
           
java.awt.event.MouseMotionListener
interface is for objects that want to be notified when the mouse is being moved or dragged. This interface is part of the java.awt.event package, which contains all the events used by AWT components. To make a class receive mouse motion events, it needs to implements this interface. The methods that are implemented must be able to respond to mouse movement events generated by the user.

Methods in MouseMotionListener Interface

There are two methods: 
mouseMoved()

mouseDragged()
, that can be implemented to handle mouse motion events.

mouseMoved() Method

The 
mouseMoved()
method receives an instance of 
MouseEvent

MouseMotionListener
interface. Here's an example of how to implement this method: 

          
           
@Override
public void mouseMoved(MouseEvent e) {

MouseMotionListener

@Override
public void mouseDragged(MouseEvent e) {
    System.out.println("Mouse dragged");

MouseMotionListener
