android.widget.TimePicker

The android.widget.TimePicker is a user interface control that is used to set the time on mobile devices running the Android operating system. The TimePicker allows users to select an hour, minute, and second from a dropdown list of choices. The TimePicker also supports specifying an am/pm value.

Usage Example

The following code example shows how to create a TimePicker object and set the time values: 

          
           
TimePicker timePicker = new TimePicker(this);
timePicker.setHour(10);
timePicker.setMinute(30);
timePicker.setSecond(45);
timePicker.setIs24HourView(true);

Listeners

It is possible to attach a ChangeListener to the TimePicker in order to be notified when the value of the TimePicker changes. The ChangeListener is triggered whenever one of the time values (hour, minute, second) is changed. 

          
           
timePicker.setChangeListener(new ChangeListener() { 
  @Override 
  public void onChange(TimePicker view, int hour, int minute, int second) {
     // Do something when the TimePicker value changes
  } 
});

Formatting Output

The TimePicker supports setting the format of the output, so the displayed value can be configured to the desired pattern. The following code example tells the TimePicker to display the time in the format HH:mm:ss a: 

          
           
timePicker.setFormat(TimePicker.Format.HH_MM_SS_A);

Getting Time Values

The values of the TimePicker can be retrieved using the following methods: 

          
           
getHour(): returns the current selected hour
getMinute(): returns the current selected minute
getSecond(): returns the current selected second
getIs24HourView(): returns true if the TimePicker is in 24-hour mode, false otherwise

Setting Default Values

The default values of a TimePicker can be set to the current system time by calling the setToCurrentTime() method. 

          
           
timePicker.setToCurrentTime();
