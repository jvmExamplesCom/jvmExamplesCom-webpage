org.springframework.web.util.UriComponentsBuilder

The org.springframework.web.util.UriComponentsBuilder Java class is a utility class used to create and manipulate uniform resource identifiers (URIs). It can be used to easily build URI components in a safe manner and supports many of the same techniques used by the popular Apache HttpClient project. URI uri = new URI("http://example.com/foo?bar=baz");
UriComponentsBuilder builder = UriComponentsBuilder.fromUri(uri);

        .build();
