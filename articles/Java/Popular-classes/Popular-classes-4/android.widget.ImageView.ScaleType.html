android.widget.ImageView.ScaleType

The android.widget.ImageView.ScaleType class is used to specify how an image should be scaled. It can define the scaling type of an image when it is displayed in an ImageView, as well as some other behaviours related to the imageâ€™s display. This class provides the functionality to scale, crop, translate and center the image in its view.

Code Examples

Below are some code examples showing how to use the android.widget.ImageView.ScaleType class.

Setting a Scale Type:


          
           
ImageView image = (ImageView) findViewById(R.id.image);
image.setScaleType(ImageView.ScaleType.CENTER_CROP);

Retrieving a Scale Type:


          
           
ImageView image = (ImageView) findViewById(R.id.image);
Log.i("Scale type: ", image.getScaleType().name());

Possible Values

The possible values for the ScaleType are as follows:

  • CENTER
  • CENTER_CROP
  • CENTER_INSIDE
  • FIT_CENTER
  • FIT_END
  • FIT_START
  • FIT_XY
  • MATRIX

Each of these values have their own effects on how the image is scaled and displayed.
