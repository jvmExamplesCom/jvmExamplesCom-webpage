Introduction to android.graphics.ColorMatrix

The android.graphics.ColorMatrix class is a class used to store and manipulate color matrix transformations in the Android operating system. It is used to create effects such as color tinting, saturation changes, brightness adjustments, contrast adjustments, and more. Defining ColorMatrix Objects

ColorMatrix cm = new ColorMatrix();
or the following constructor with arguments: 

          
           
ColorMatrix cm = new ColorMatrix(float[] matrix);
Where the matrix argument is a 4x4 array of floats representing the matrix's values.
