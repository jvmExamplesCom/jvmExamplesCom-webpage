android.support.test.rule.ActivityTestRule

Android Support Test Rule is an important library in the Android framework for testing activities. It provides a set of rules which govern how activities and their associated states are tested. The purpose of the rule is to ensure reliable, repeatable tests of activities.

    onView(withId(R.id.view_example))

  }
