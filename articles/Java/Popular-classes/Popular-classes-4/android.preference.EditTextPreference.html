android.preference.EditTextPreference

The android.preference.EditTextPreference class is a subclass of DialogPreference that shows a text input dialog to the user. This allows the user to type in a string and store it.

Example 1:

The following code creates an EditTextPreference with the key “my_pref” and the default value of “default_value”.


          
           
EditTextPreference myPref = new EditTextPreference(getActivity());
myPref.setKey("my_pref");
myPref.setDefaultValue("default_value");

Example 2:

The following code sets the preference’s title, summary, and input type.


          
           
myPref.setTitle("My Preference");
myPref.setSummary("Please enter a value");
myPref.setInputType(InputType.TYPE_CLASS_TEXT);

Example 3:

The following code reads the value from the preference and displays it on the screen.


          
           
SharedPreferences sharedPreferences = getPreferenceManager().getSharedPreferences();
String value = sharedPreferences.getString("my_pref", "default_value");
TextView textView = findViewById(R.id.text_view);
textView.setText(value);
