Introduction to org.springframework.cloud.netflix.zuul.EnableZuulProxy

Setting up org.springframework.cloud.netflix.zuul.EnableZuulProxy

The first step to setting up org.springframework.cloud.netflix.zuul.EnableZuulProxy is to add it as a dependency in your project’s pom.xml file.


          
           
<dependencies>
    <dependency>
        <groupId>org.springframework.cloud</groupId>
        <artifactId>spring-cloud-starter-netflix-zuul</artifactId>
        <version>2.0.3.RELEASE</version>
    </dependency>
</dependencies>

The second step is to enable org.springframework.cloud.netflix.zuul.EnableZuulProxy in your SpringBootApplication class:


          
           
@SpringBootApplication

    public static void main(String[] args) {

}
