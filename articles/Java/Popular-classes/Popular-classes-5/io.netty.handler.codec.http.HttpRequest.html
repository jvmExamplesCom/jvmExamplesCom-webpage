About io.netty.handler.codec.http.HttpRequest

io.netty.handler.codec.http.HttpRequest is an API used for constructing and encoding HTTP requests. It supports most of the methods defined in RFC 7231, though some optional methods are not supported.

Creating Requests

Constructing a request is straightforward. To create a simple GET request with no body:


          
           
HttpRequest req = new DefaultHttpRequest(HttpVersion.HTTP_1_1, HttpMethod.GET, "/");
To add headers to the request, call the 
headers()
method: 

          
           
req.headers().add("Host", "www.example.com");

The request can also be customized with a query string or body content:


          
           
req.setUri("/some/path?foo=bar&amp;la=la");

String jsonContent = "{ \"key\": \"value\" }";
DefaultHttpContent content = new DefaultHttpContent(Unpooled.wrappedBuffer(jsonContent.getBytes()));
req.setContent(content);

Encoding Requests

An HttpRequest instance can be encoded into a ByteBuf which can then be sent out over the wire. To do this, use the HttpRequestEncoder class:


          
           
ByteBuf buf = Unpooled.buffer();
HttpRequestEncoder encoder = new HttpRequestEncoder();
encoder.encode(req, buf);

The encoded request can now be sent out over the wire as a byte stream.
