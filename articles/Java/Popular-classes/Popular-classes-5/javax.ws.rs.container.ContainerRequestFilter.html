javax.ws.rs.container.ContainerRequestFilter

The 

          
           
ContainerRequestFilter
class is part of the Java API for RESTful Web Services (JAX-RS) 2.1 and serves as a request filter for container requests. A filter is an object that performs filtering tasks on either the request to a resource (a servlet or static content) or on the response from a resource, or both. Filters do not usually generate a response; they modify or adapt the requests for a resource, or modify or adapt the responses from a resource. The 

          
           
ContainerRequestFilter
class is one such filtered used to filter container requests.

Usage

The 

          
           
ContainerRequestFilter
class is typically used to perform preprocessing on requests that come into a web application before they are processed by the application. It can be used to perform tasks such as authentication, authorization, logging, and other tasks that need to be done before the request is processed. The 

          
           
ContainerRequestFilter
class is an abstract class that needs to be extended by a concrete implementation in order to be used. The concrete implementation must implement the 

          
           
filter()
method, which is called when a request is processed. The 

          
           
filter()
method is passed a 

          
           
ContainerRequestContext
object, which contains all of the information about the incoming request. 

          
           
public abstract class ContainerRequestFilter {
    public abstract void filter(ContainerRequestContext requestContext) throws IOException;

Example

ContainerRequestFilter
class. In this example, the

class extends the
