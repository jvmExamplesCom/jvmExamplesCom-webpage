The com.codahale.metrics.Counter Class

The com.codahale.metrics package provides a powerful set of classes to help developers monitor the performance of various components in their application. This article will focus on one particular class in the package, the Counter class. This class is used for counting the number of times an event has happened and is trivially simple to use.

Creating Counters

Counters can be created by simply instantiating them. There are a few different types available such as LongAdder, AtomicInteger and AtomicLong: 

          
           
Counter counter = new Counter();

Counter atomicIntegerCounter = new AtomicIntegerCounter();
Counter atomicLongCounter = new AtomicLongCounter();
By default, the counters start with a value of 0. However, it is possible to provide a custom starting value when they are created: 

          
           
Counter counterWithStartValue = new Counter(100);

Incrementing Counters

To increment a counter, simply call the 

          
           
inc()
method. This will increase the value of the counter by one. It is also possible to increment the counter with a value other than one. This can be done by calling the 

          
           
add(long amt)
method and providing the desired amount. For example, the following code will result in the counter value being incremented by ten: 

          
           
counter.add(10);

Decrementing Counters

To decrease the value of the counter, the 

          
           
dec()
method can be used. This will decrease the value of the counter by one. As with the 

          
           
add()
method, it is possible to decrement the counter using a custom value. The following code will decrement the counter by 5: 

          
           
counter.dec(5);

Retrieving the Value of Counters

It is also possible to retrieve the current value of the counter by using the 

          
           
getCount()
method: 

          
           
long count = counter.getCount();
