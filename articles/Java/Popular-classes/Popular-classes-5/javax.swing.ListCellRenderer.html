javax.swing.ListCellRenderer

The javax.swing.ListCellRenderer class is used to render a representation of a single value in a list, such as a text string or an image. A cell renderer can be used to customize the way a cell is rendered based on its contents.

Overview

A ListCellRenderer is used to customize the representation of a single value within a list. The renderer provides a representation of an object as a JComponent that can be displayed within a list. A renderer can be used to customize the way a cell is displayed based on its contents. This could include changing the font or color of the cells, displaying images, or adding custom icons.

Creating a List Cell Renderer

Creating a custom ListCellRenderer requires implementing the ListCellRenderer interface and overriding its getListCellRendererComponent method. This method takes two arguments, the JList and the Object to be rendered.


          
           
public class MyListCellRenderer implements ListCellRenderer {

  public Component getListCellRendererComponent(JList list, Object value, int index, boolean isSelected, boolean cellHasFocus) {
    // Do something with the object
  }

}

The returned component should be a representation of the object being rendered. This could be a JLabel for text, or a JPanel for images and icons.

Parameters


          
           
list
– The JList that is asking the renderer to draw. 

          
           
value
– The value returned by the ListModel for the specified list index. 

          
           
index
– The cell index. 

          
           
isSelected
– True if the specified cell was selected. 

          
           
cellHasFocus
– True if the specified cell has the focus.

Example

In this example, we create a custom ListCellRenderer that displays an icon next to the value:


          
           
public class IconListCellRenderer extends JLabel implements ListCellRenderer {

  public IconListCellRenderer() {
    setOpaque(true);
  }

  @Override
  public Component getListCellRendererComponent(JList list, Object value, int index, boolean isSelected, boolean cellHasFocus) {
    String text = value.toString();
    ImageIcon icon = (ImageIcon)value;
    setIcon(icon);
    setText(text);

    if (isSelected) {
      setBackground(list.getSelectionBackground());
      setForeground(list.getSelectionForeground());
    } else {
      setBackground(list.getBackground());
      setForeground(list.getForeground());
    }
    return this;
  }

}

This renderer will display an icon and text for each element in the list. The icon is taken from the value argument, and the text is derived from the toString() method of the value.
