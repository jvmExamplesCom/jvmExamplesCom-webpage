javax.activation.DataSource

The javax.activation.DataSource interface is used to represent an arbitrary data source in JavaBeans Activation Framework (JAF). A data source represents an input stream of data. It is typically used with JAF to retrieve data in many different formats.

Example 1


          
           
URL url = new URL("http://example.com/image.jpg");
DataSource ds = new URLDataSource(url);

This example creates a new DataSource that uses the URL http://example.com/image.jpg as its source. The DataSource is then used to read data from the URL.

Example 2


          
           
DataSource ds = new FileDataSource("file.txt");

This example creates a new DataSource that uses the file.txt file as its source. The DataSource can then be used to read data from the file.

Example 3


          
           
DataSource ds = new ByteArrayDataSource("Hello World".getBytes(), "text/plain");

This example creates a new DataSource that uses a byte array as its source. The byte array consists of the string "Hello World" and the content type is set to text/plain. The DataSource can then be used to read data from the byte array.

Example 4


          
           
ByteArrayOutputStream baos = new ByteArrayOutputStream();
baos.write("Hello World".getBytes());

DataSource ds = new ByteArrayDataSource(baos.toByteArray(), "text/plain");

This example creates a new DataSource that uses a ByteArrayOutputStream as its source. The ByteArrayOutputStream contains the string "Hello World" and the content type is set to text/plain. The DataSource can then be used to read data from the ByteArrayOutputSteam.
