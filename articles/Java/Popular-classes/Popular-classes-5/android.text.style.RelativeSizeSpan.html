android.text.style.RelativeSizeSpan

The android.text.style.RelativeSizeSpan class increases the size of text in a text line relative to the text's original size. It is important to note that this is based on a percentage, so by default, when you create a RelativeSizeSpan, it will double the size of the text. This class is useful when you want to display text in a certain size relative to the rest of the text in the same line.

Creating a RelativeSizeSpan

Creating a RelativeSizeSpan is simple; all you need to do is invoke the constructor with the desired size as a float value. For example: 

          
           
RelativeSizeSpan span = new RelativeSizeSpan(1.5f);
This code creates a RelativeSizeSpan that increases the size of the text by 50%.

Applying a RelativeSizeSpan

To apply the RelativeSizeSpan to a text, you need to call TextUtils.apply() and pass in the Spannable object, the RelativeSizeSpan, and the start and end index for the span. For example: 

          
           
TextUtils.apply(spanable, span, 0, spanable.length(), Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
This code applies the span to the entire spanable text.

Example

Here is a complete example that demonstrates how to create and apply a RelativeSizeSpan. 

          
           
// Create a RelativeSizeSpan
RelativeSizeSpan span = new RelativeSizeSpan(1.5f);

// Get a SpanableStringBuilder
SpannableStringBuilder spanable = new SpannableStringBuilder("This is some text");

// Apply the RelativeSizeSpan
TextUtils.apply(spanable, span, 0, spanable.length(), Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);

System.out.println(spanable);

This is some text
