javax.jms.Session: Overview

The 
javax.jms.Session
interface provides a single-threaded context for producing and consuming messages. It represents a logical transaction with the server. The session object provides an "umbrella" for the methods needed to create, send and receive messages.

Creating a Session Object

To create a session object, use the 

          
           
Connection.createSession()
method of the javax.jms.Connection object: 

          
           
Connection connection = ...;
Session session = connection.createSession(false, Session.AUTO_ACKNOWLEDGE);

Session.createProducer()
method: 

          
           
MessageProducer producer = session.createProducer(topic);

Messages

Session.createTextMessage()
method: 

          
           
TextMessage message = session.createTextMessage("Hello World!");

Sending Messages

Message Consumers

Receiving Messages
