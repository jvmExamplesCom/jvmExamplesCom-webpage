javax.swing.event.TableModelEvent

The javax.swing.event.TableModelEvent is an event object that is created when a TableModel object has changed, and a view of the model needs to be notified of the change. All changes to the data in the TableModel are reflected in this event, including inserting or removing rows and columns, as well as changing the data in a particular cell.

    public void tableChanged(TableModelEvent e) {

        } else if (e.getType() == TableModelEvent.DELETE) {

        }
    }
