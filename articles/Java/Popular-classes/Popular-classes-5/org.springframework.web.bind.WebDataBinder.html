Introduction to org.springframework.web.bind.WebDataBinder

The org.springframework.web.bind.WebDataBinder is a part of the Spring framework for the Java platform. It provides a convenient way to bind request parameter values to an object or a collection of objects, as well as to register custom editors for a given type. The WebDataBinder can be used in both web-based and non web-based applications.

    private String name;
    private Integer age;

}

We can then use the WebDataBinder to bind the request parameters to a new instance of the Person class like so:


          
           
WebDataBinder binder = new WebDataBinder(new Person());
binder.bind(request);
