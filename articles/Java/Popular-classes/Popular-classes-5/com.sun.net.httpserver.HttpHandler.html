HttpHandler on Java

Example

The following example shows the implementation of an HttpHandler object that handles a simple HTTP request. It prints out the request body and prints out a response message. It also prints out the request headers and the response headers:


          
           
public class MyHandler extends HttpHandler {

   public void handle(HttpExchange exchange, String path) {
      // print request body
      InputStream requestBody = exchange.getRequestBody();
      System.out.println(requestBody);

      // print request headers
      Headers requestHeaders = exchange.getRequestHeaders();
      System.out.println(requestHeaders);

      // send response
      String response = "Hello world";
      exchange.sendResponseHeaders(200, response.length());

      // print response headers
      Headers responseHeaders = exchange.getResponseHeaders();
      System.out.println(responseHeaders);

      // write response
      OutputStream responseBody = exchange.getResponseBody();
      responseBody.write(response.getBytes());
      responseBody.close();
   }
}
