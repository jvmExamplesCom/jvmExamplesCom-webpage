android.telephony.PhoneStateListener

The android.telephony.PhoneStateListener class provides a way to monitor changes in phone state, including incoming and outgoing calls, data connection status, and signal strength. It is an abstract class and must be extended to be used.

Methods

PhoneStateListener has several methods for monitoring various aspects of the phone's state. On each change, the appropriate method will be called.

  • onCallStateChanged: Called when there is a change in the call state (e.g., an incoming call or an outgoing call).
  • onDataConnectionStateChanged: Called when the data connection state changes (e.g., when the device switches from Wi-Fi to cellular data).
  • onSignalStrengthsChanged: Called when the signal strength of a network changes.

Example

Here is an example of extending the PhoneStateListener class and overriding the onCallStateChanged() method. This example will simply log a message when there is a change in the call state. 


          
           
public class MyPhoneStateListener extends PhoneStateListener {
    
    @Override
    public void onCallStateChanged(int state, String incomingNumber) {
        switch (state) {
            case TelephonyManager.CALL_STATE_IDLE:
                Log.d("MyPhoneStateListener", "Idle");
                break;
            case TelephonyManager.CALL_STATE_RINGING:
                Log.d("MyPhoneStateListener", "Ringing (" + incomingNumber + ")");
                break;
            case TelephonyManager.CALL_STATE_OFFHOOK:
                Log.d("MyPhoneStateListener", "Off Hook");
                break;
        }
    }
}

Usage

Once the PhoneStateListener class has been extended, it can be registered with the TelephonyManager. The TelephonyManager can then be used to monitor changes in the phone's state. 


          
           
TelephonyManager manager = (TelephonyManager) getSystemService(Context.TELEPHONY_SERVICE);
MyPhoneStateListener listener = new MyPhoneStateListener();
manager.listen(listener, PhoneStateListener.LISTEN_CALL_STATE);
