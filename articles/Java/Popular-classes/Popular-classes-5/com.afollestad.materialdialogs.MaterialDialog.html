MaterialDialog in Java

MaterialDialog is a powerful, feature-rich library for displaying dialogs in the Android app. It is built on top of the Google Material Design specification and it is extremely lightweight and easy to use. It has many features, including customizable dialog boxes, custom fonts, and size settings. In this article, we will explore how to use the MaterialDialog library and how to customize it for your needs.

Getting Started

To get started, you'll need to include the MaterialDialog library in your project. You can do this by adding the following dependency in your build.gradle file:


          
           
implementation 'com.afollestad.materialdialogs:core:X.X.X'
Where 
X.X.X
is the version number of the library.

Once you have added the library to your project, you can create a new MaterialDialog by calling the following method:


          
           
MaterialDialog materialDialog = new MaterialDialog.Builder(context)
    .title(title)
    .message(message)
    .build();

This will create a new MaterialDialog with the given title and message. To display the dialog, simply call 


          
           
materialDialog.show()
.

Customization Options

The MaterialDialog library provides many customization options for creating dialogs. For example, you can set the following properties to customize the look and feel of the dialog box:

  • 
            
             
setBackgroundColor()
    : Sets the background color of the dialog box.
  • setContentColor()
    : Sets the text color of the content in the dialog box.
  • setCancelable()
    : Sets whether or not the dialog box can be canceled by pressing the back button.
  • setAnimations()
    : Sets the animations used when the dialog box appears and disappears.
  • setTitleTypeface()
    : Sets the font used for the title of the dialog box.
  • 
            
             
setMessageTypeface()
    : Sets the font used for the message in the dialog box.

In addition to the above, you can also customize the buttons used in the dialog box by setting their text, color, and listener. For example, the following code will create a dialog box with two buttons:


          
           
MaterialDialog materialDialog = new MaterialDialog.Builder(context)
    .title(title)
    .message(message)
    .positiveText("Yes")
    .negativeText("No")
    .onPositive(new MaterialDialog.SingleButtonCallback() {
        @Override
        public void onClick(@NonNull MaterialDialog dialog, @NonNull DialogAction which) {
            // Do something
        }
    })
    .onNegative(new MaterialDialog.SingleButtonCallback() {
        @Override
        public void onClick(@NonNull MaterialDialog dialog, @NonNull DialogAction which) {
            // Do something
        }
    })
    .build();

This code will create a dialog box with two buttons, labeled "Yes" and "No". When either button is clicked, the respective 

onClick()
method will be called.

Additionally, you can enable automatic dismissal of the dialog when a button is pressed by calling 

autoDismiss()
before 
build()
. This will cause the dialog to be dismissed after the 
onClick()
method is called.

Conclusion

In this article, we explored the MaterialDialog library and how to use it to create and customize dialog boxes in Android apps. We also looked at how to set properties such as the background color, font, and animations, as well as how to create customized buttons with listeners.
