The 

          
           
android.text.Selection
class contains several methods for selecting text within an Editable or Spannable string. These methods fall into two categories; being able to select a range of text and being able to select an entire line of text. To select a range of text the following methods can be used: 

          
           
public static void selectAll(Editable text) 
public static void setSelection(Editable text, int start, int stop) 
public static final void extendSelection(Editable text, int index)

// Select all text
Selection.selectAll(myEditText.getText());

Selection.extendSelection(myEditText.getText(), 20)
