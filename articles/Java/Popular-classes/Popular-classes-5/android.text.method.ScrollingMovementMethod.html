android.text.method.ScrollingMovementMethod

android.text.method.ScrollingMovementMethod is a movement method that allows a TextView to scroll the document it displays. It supports both horizontal and vertical scrolling, as well as smooth scrolling including flinging. This class can be used to create a scrolling textview from an existing piece of text.

Basic Usage

To use the ScrollingMovementMethod, you must first create a TextView for the document and then set the movement method to the ScrollingMovementMethod: 

          
           
TextView textView = new TextView(context);
textView.setMovementMethod(new ScrollingMovementMethod());
You can then set the content of the TextView using setText() or other methods.

Scrolling and Flinging

The ScrollingMovementMethod supports both scrolling (by clicking and dragging the text with the mouse) and flinging (by clicking and dragging the text with the mouse, then releasing quickly). The scrolling behavior can be configured by setting the flags on the view, such as canScrollHorizontally and canScrollVertically. The flinging behavior can be configured by setting the minFlingVelocity and maxFlingVelocity parameters. The default values are 500 and 2000 respectively.

Smooth Scrolling

The ScrollingMovementMethod also supports smooth scrolling, which takes place when the user scrolls slowly using the mouse or scroll wheel. This can be enabled by setting the smoothScrollingEnabled flag on the view to true.

Example

TextView textView = new TextView(context);
textView.setMovementMethod(new ScrollingMovementMethod());
