org.apache.commons.lang.builder.HashCodeBuilder

org.apache.commons.lang.builder.HashCodeBuilder is a utility class for constructing hash codes for Java objects. HashCodeBuilder builder = HashCodeBuilder.reflectionHashCode(myObject);

Generating the Hash Code

int hashCode = builder.generateHashCode();
