javax.xml.namespace.NamespaceContext

The javax.xml.namespace.NamespaceContext provides the mapping of prefixes to namespace URIs for XML documents. It enables lookup of a namespace URI by prefix or vice versa. It is mainly designed to facilitate access to an XML document and provides an interface for applications that need to manipulate the namespace mappings.

Methods in NamespaceContext

The NamespaceContext class has several methods which enable easy manipulation of namespace mappings. These include:

  • getNamespaceURI(): returns the namespace URI associated with the given prefix.
  • getPrefix(): returns the prefix associated with the given namespace URI.
  • getPrefixes(): returns an enumeration of all prefixes associated with the given namespace URI.

Example Code


          
           
import javax.xml.namespace.NamespaceContext;

public class MyContext implements NamespaceContext {

    /**
     * Retrieves a prefix from a namespace URI
     */
    @Override
    public String getPrefix(String namespaceUri) {
        if (namespaceUri.equals("urn:example:com"))
            return "example";
        return null; // namespace URI not recognized
    }

    /**
     * Retrieves a namespace URI from a prefix
     */
    @Override
    public String getNamespaceURI(String prefix) {
        if (prefix.equals("example"))
            return "urn:example:com";
        return null; // prefix not recognized
    }

    /**
     * Returns all prefixes of a namespace URI
     */
    @Override
    public Iterator<?> getPrefixes(String namespaceUri) {
        List<String> list = new ArrayList<String>();
        if (namespaceUri.equals("urn:example:com"))
            list.add("example");
        return list.iterator();
    }
}

Output

The code above can be used to easily retrieve prefixes and namespace URIs. For example, if we passed the prefix 'example' to the getNamespaceURI() method, it would return 'urn:example:com', and if we passed the namespace URI 'urn:example:com' to the getPrefix() method, it would return 'example'. The getPrefixes() method returns an iterator with the prefix 'example' for the namespace URI 'urn:example:com'.
