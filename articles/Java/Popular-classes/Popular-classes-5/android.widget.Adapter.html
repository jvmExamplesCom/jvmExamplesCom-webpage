Android.widget.Adapter

Android.widget.Adapter is a subclass of the Android framework's base Adapter class and is a key component to making the ListView and GridView user interfaces work properly. This class allows a list of objects to be linked to a list of views and allows for recycling of views.

Syntax


          
           
public abstract class Adapter<VH extends ViewHolder>

Example

The following code snippet is an example of how an Adapter can be used with a ListView, where Image is an example of the data type stored in a collection inside the Adapter.


          
           
public class ExampleAdapter extends BaseAdapter {

private ArrayList<Image> data;

public ExampleAdapter(ArrayList<Image> data) {
	this.data = data;
}

@Override
public int getCount() {
	return data.size();
}

@Override
public Object getItem(int position) {
	return data.get(position);
}

@Override
public long getItemId(int position) {
	return position;
}

@Override
public View getView(int position, View convertView, ViewGroup parent) {
	View view = convertView;
	if (view == null) {
		LayoutInflater inflater = LayoutInflater.from(parent.getContext());
		view = inflater.inflate(R.layout.example_item, parent, false);
	}

	Image image = (Image) getItem(position);
	ImageView imageView = (ImageView) view.findViewById(R.id.imageView);
	imageView.setImageBitmap(image.bitmap);

	TextView titleTextView = (TextView) view.findViewById(R.id.titleTextView);
	titleTextView.setText(image.title);

	return view;
}
}

Methods

The Adapter class provides several methods that can be used to manipulate the data displayed in a list or grid view. The key methods are listed below.

  • getCount(): returns the number of items in the data set.
  • getItem(int position): returns the item at the specified position in the data set.
  • getItemId(int position): returns the id of the item at the specified position in the data set.
  • getView(int position, View convertView, ViewGroup parent): returns a view for each item in the data set. This method is used for creating the view for each item in the list or grid view.

Conclusion

The Android.widget.Adapter class is a powerful tool for creating lists and grids in an Android application. This class allows for objects to be linked to views and reused for efficient memory usage. By implementing methods such as getCount(), getItem(), getItemId(), and getView(), it is possible to create dynamic user interfaces with data from a collection.
