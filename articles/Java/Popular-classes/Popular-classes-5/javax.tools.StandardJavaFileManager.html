javax.tools.StandardJavaFileManager

The javax.tools.StandardJavaFileManager interface is a part of the Java Platform, Standard Edition 6 and later. It provides access to the underlying file system and can be used to read and write Java source and class files. It is the central entry point for tools that need to access compiled classes, source files, and other related resources.

It supports different kinds of resources, for example, resource paths, class paths, and files. It also supports operations such as list, copy, delete, and update.

Examples

Example 1: Listing All Files in a Path


          
           
StandardJavaFileManager javaFileManager = 
    ToolProvider.getSystemJavaCompiler().getStandardFileManager(null, null, null);

File[] files = 
    javaFileManager.list(StandardLocation.CLASS_PATH, "", Collections.emptySet(), true);

for (File file : files) {
    System.out.println(file);
}

javaFileManager.close();

Example 2: Copying a File


          
           
StandardJavaFileManager javaFileManager = 
    ToolProvider.getSystemJavaCompiler().getStandardFileManager(null, null, null);

File oldFile = new File("src/Main.java");
File newFile = new File("bin/Main.class");

javaFileManager.copy(oldFile.toPath(), newFile.toPath());

javaFileManager.close();

Example 3: Deleting a File


          
           
StandardJavaFileManager javaFileManager = 
    ToolProvider.getSystemJavaCompiler().getStandardFileManager(null, null, null);

File file = new File("bin/Main.class");

javaFileManager.delete(file.toPath());

javaFileManager.close();

Example 4: Updating a File


          
           
StandardJavaFileManager javaFileManager = 
    ToolProvider.getSystemJavaCompiler().getStandardFileManager(null, null, null);

File oldFile = new File("src/Main.java");
File newFile = new File("bin/Main.class");

javaFileManager.update(oldFile.toPath(), newFile.toPath());

javaFileManager.close();
