Overview of com.google.android.material.tabs.TabLayout

The com.google.android.material.tabs.TabLayout class is an interactive widget that displays a horizontal row of tabs for Android apps. It provides a way to switch between various fragments or activities with just a single click. It is compatible with the latest version of Android and has the same look and feel of the Material Design components. The TabLayout class works with a ViewPager to provide a smooth scrolling effect when switching between tabs. A selected tab is always indicated by a colored indicator line below it. It also supports text labels, icons and custom views as the tab title. This class also allows to set a custom tab selection listener to listen for changes in the currently selected tab.

Getting Started with com.google.android.material.tabs.TabLayout

To use TabLayout, you must first add the following dependency in your app-level build.gradle file: 

          
           
implementation 'com.google.android.material:material:1.1.0'
Next, you need to add the TabLayout view to the layout file of the Activity or Fragment. 

          
           
<com.google.android.material.tabs.TabLayout
android:id="@+id/tabLayout"
android:layout_width="match_parent"
android:layout_height="wrap_content"/>
To initialize the TabLayout view, you need to setup an adapter. The adapter provides the data model for the tab titles. 

          
           
TabLayout tabLayout = findViewById(R.id.tabLayout);
PagerAdapter adapter = new PagerAdapter(getSupportFragmentManager(), tabLayout.getTabCount());
tabLayout.setupWithViewPager(adapter);

Working with Tabs

You can configure the style, content and behavior of a tab using the TabLayout.Tab class. You can add tabs with text labels, icons and custom views using the addTab() method. 

          
           
// Add a tab with an icon
TabLayout.Tab tab = tabLayout.newTab().setIcon(R.drawable.ic_cloud);
tabLayout.addTab(tab);

// Add a tab with a string resource
tab = tabLayout.newTab().setText(R.string.tab_title);
tabLayout.addTab(tab);
