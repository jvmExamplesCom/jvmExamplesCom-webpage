About org.apache.http.client.methods.HttpHead

The org.apache.http.client.methods.HttpHead class is a type of HTTP request that can be used in Java applications. It is similar to the other types of requests as it sends a request to the server and returns a response, but the HttpHead only retrieves the header fields for an HTTP message. This can be useful for checking the header information without receiving the entire response, which can save time and resources. In this article, we will discuss what the HttpHead class is, its methods and properties, and provide some code examples.

HttpHead Methods and Properties

The HttpHead class has several useful methods and properties to help create and perform HTTP requests:

  • getMethod(): Returns the HTTP request method (GET, POST, etc...)
  • getRequestLine(): Returns the request line (e.g., GET /index.html HTTP/1.1)
  • addHeader(): Adds the specified header to the request
  • addRequestProperty(): Adds the specified property to the request

The HttpHead class also contains several protected methods and properties, such as getProtocolVersion(), getHeaders(), and getURI(). These methods and properties cannot be accessed directly but can be useful when extending the class.

Code Examples

Here are some code examples to demonstrate how to use the HttpHead class.


          
           
// Create a new HttpHead object
HttpHead httpHead = new HttpHead("http://www.example.com");

// Add a header to the request
httpHead.addHeader("User-Agent", "MyApplication/1.0");

// Create a CloseableHttpClient object
CloseableHttpClient httpClient = HttpClients.createDefault();

// Execute the HttpHead request and get the response
CloseableHttpResponse response = httpClient.execute(httpHead);

// Get the response code

// Print out the response code
System.out.println("Response Code: " + responseCode);
