org.apache.log4j.PatternLayout

The org.apache.log4j.PatternLayout class is a key component of the Log4j logging framework that enables developers to write their log messages in a customizable and flexible format. Simple Formatting

Each placeholder is represented by the % symbol followed by a number or letter (from 0 to 9 or from A to Z) that denotes the field that should be printed in that position. %d %p %l - %m%n

12/07/2017 12:00:00pm INFO com.example.MyClass - This is a log message

%d{dd/MM/yyyy HH:mm:ssa} %p %l - %m%n
will produce the same result as the simple format above. Notice that the date placeholder %d was changed to include a specific date format. This allows developers to fully customize the format of the log messages.
