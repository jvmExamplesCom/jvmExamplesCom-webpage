org.apache.log4j.PatternLayout

The org.apache.log4j.PatternLayout class is a key component of the Log4j logging framework that enables developers to write their log messages in a customizable and flexible format. Simple Formatting

%d %p %l - %m%n

12/07/2017 12:00:00pm INFO com.example.MyClass - This is a log message
