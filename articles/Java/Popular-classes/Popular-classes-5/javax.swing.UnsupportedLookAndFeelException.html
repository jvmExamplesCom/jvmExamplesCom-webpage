javax.swing.UnsupportedLookAndFeelException

javax.swing.UnsupportedLookAndFeelException is an exception thrown when a program attempts to set the look and feel (L&F) of a Swing component but the L&F is not supported on the current platform. try {
    UIManager.setLookAndFeel("javax.swing.plaf.nimbus.NimbusLookAndFeel");
