Retrofit2.http.FormUrlEncoded

Retrofit2 is an open source library for making application programming interfaces (APIs) from an Android or Java application. It is a type-safe REST client for Android, which simplifies communication with server-side APIs. Retrofit2.http.FormUrlEncoded is an annotation used to define HTTP requests with type as x-www-form-urlencoded.

Example Usage

We can use Retrofit2.http.FormUrlEncoded to define the @POST method in an interface of our retrofit service. The example code given below shows how we can send a form encoded HTTP request using the Retrofit2.http.FormUrlEncoded annotation. Here, we have defined a simple API endpoint for sending POST data.


          
           
@POST("/my-endpoint")
@FormUrlEncoded
Call<ResponseBody> sendPostRequest(@Field("name") String name);

Here, we are using the @Field annotation to define the field names and values that will be sent in the POST request body.

Example Code

To use the Retrofit2.http.FormUrlEncoded annotation, we need to first create a Retrofit instance. We can do this by using the following code. Here, we are creating a retrofit instance with a base URL and an OkHttpClient instance.


          
           
OkHttpClient okHttpClient = new OkHttpClient.Builder()
          .readTimeout(60, TimeUnit.SECONDS)
          .connectTimeout(60, TimeUnit.SECONDS)
          .build();

Retrofit retrofit = new Retrofit.Builder()

        .build();

MyService myService = retrofit.create(MyService.class);
