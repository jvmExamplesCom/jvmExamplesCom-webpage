org.springframework.data.repository.PagingAndSortingRepository

The org.springframework.data.repository.PagingAndSortingRepository is an interface in the Spring Data repository infrastructure. It adds extra methods to an existing repository for the purpose of paging and sorting query results. All repositories based on this interface have access to the following methods.

Paging Methods

There are several methods available to allow pagination of query results.


          
           
Page<T> findAll(Pageable pageable);
List<T> findAll(Sort sort);
Slice<T> findAll(Pageable pageable);

Example Usage

Let's say we have a repository for users which implements the PagingAndSortingRepository interface:


          
           
@Repository
public interface UserRepository extends PagingAndSortingRepository<User, Long> {

}

We can use thefindAll method to get a page of results from our query:


          
           
PageRequest pageRequest = PageRequest.of(0, 10);
Page<User> page = userRepository.findAll(pageRequest);

List<User> users = page.getContent();

This example will return the first page of 10 users from the query results.

We can also use the findAll method with a Sort object to order our query results by a specified field:


          
           
Sort sort = Sort.by("lastName").ascending();
List<User> users = userRepository.findAll(sort);

This example will order the query results by the last name field in ascending order.

Conclusion

The org.springframework.data.repository.PagingAndSortingRepository provides an easy way to add pagination and sorting to existing repositories in the Spring Data infrastructure. By using this interface, you can quickly and easily add these features to your application code.
