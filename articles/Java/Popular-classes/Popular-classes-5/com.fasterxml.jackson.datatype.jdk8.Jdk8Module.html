com.fasterxml.jackson.datatype.jdk8.Jdk8Module

The com.fasterxml.jackson.datatype.jdk8.Jdk8Module is a module used to add support for the Java 8 data types. Adding the Module to an Application

In order to use the Jdk8Module, it must first be added to an application's Jackson ObjectMapper. This is done by calling registerModule on the mapper. 

          
           
ObjectMapper om = new ObjectMapper();
om.registerModule(new Jdk8Module());

The Jdk8Module adds support for the Java 8 LocalDate and LocalTime. This allows these types to be serialized to and from JSON without additional configuration or customization. LocalDate and LocalTime can be written to JSON using the ObjectMapper: 

          
           
LocalDate ld = LocalDate.of(2019, 5, 27);
