org.eclipse.swt.widgets.Display in Java

Java is a powerful language that provides many facilities to create graphical user interfaces and other GUIs. One of the most important items in this toolkit is the org.eclipse.swt.widgets.Display class, which is part of the Standard Widget Toolkit (SWT). This class helps to create displays and provides methods to control them.

What is Display Class?

Display is an abstract class that serves as a window for drawing on the screen or printing on a printer. It provides methods for creating native data structures from primitive types, such as bitmaps and windows, as well as for managing event queues and creating controls.

Methods in Display Class

Display class contains several important methods to help create and control displays and their related resources, including: 
  • getCurrent()
    : Gets the current display.
  • update()
    : Causes the operating system to change the state of the device to reflect the new contents of the Client Area.
  • sleep()
    : Causes the thread which has invoked this method to wait until another thread calls the 
    notify()
    or 
    notifyAll()
    method of this object.
  • 
            
             
syncExec(Runnable runnable)
    : Causes the Runnable to have its run method called in the main thread at some point before the next event is picked up and dispatched.
  • 
            
             
asyncExec(Runnable runnable)
    : Causes the runnable to have its run method called in the user interface thread. It will be run after all pending user interface events are processed.

Example Code

Let's look at the following example to see how we can use the Display class in Java. This example creates a window with a text field, allows you to enter text, and prints out the text when the window is closed. 

          
           
import org.eclipse.swt.widgets.Display;
import org.eclipse.swt.widgets.Shell;
import org.eclipse.swt.widgets.Text;
import org.eclipse.swt.layout.FillLayout;

public class DisplayExample {

    public static void main(String[] args) {
        // Create the display
        Display display = new Display();

        // Create the shell
        Shell shell = new Shell(display);
        shell.setLayout(new FillLayout());

        // Create a text field in the shell
        Text textField = new Text(shell, 0);

        // Open the shell
        shell.open();

        // Event loop to keep the window open
        while(!shell.isDisposed()) {
            if (!display.readAndDispatch())
                display.sleep();
        }

        // Get the text from the text field
        String text = textField.getText();
        System.out.println(text);

        // Dispose the display
        display.dispose();
    }

}
When this code is run, a window opens with a text field. Once the window is closed, the text entered in the field is printed out.
