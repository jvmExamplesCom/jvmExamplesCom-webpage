org.springframework.test.web.servlet.MvcResult

The 
MvcResult
class is part of the Spring Framework, specifically the Spring MVC testing framework. It is used to store the results of invoking a controller method, and allows developers to access request, response, and handler objects from within a test.

Basic Usage Example

The following example shows how to use the 
MvcResult
object in a test. This example invokes a controller method to retrieve data from a service and then asserts that the results are as expected. 

          
           
@Test
public void testController() throws Exception {
  // Invoke the controller
  MvcResult result = this.mockMvc.perform(get("/")
      .param("name", "John Doe"))
      .andReturn();

  // Assert the response
  assertEquals(200, result.getResponse().getStatus());
  assertEquals("John Doe", result.getResponse().getContentAsString());
}
