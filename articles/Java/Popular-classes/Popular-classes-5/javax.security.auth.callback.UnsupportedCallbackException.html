javax.security.auth.callback.UnsupportedCallbackException

The javax.security.auth.callback.UnsupportedCallbackException is an exception thrown by a CallbackHandler to indicate that the requested Callback is not supported. This class is used to signal unsupported Callback classes being passed to a CallbackHandler. In particular, Callback implementations can throw this exception to indicate that they are not supported.

Example 1

Here is an example of using the javax.security.auth.callback.UnsupportedCallbackException.


          
           
try {
    // Perform some authentication
} catch (UnsupportedCallbackException e) {
    System.err.println("Unsupported callback: " + e.getCallback());
}
In the code above, we are catching a 

          
           
UnsupportedCallbackException
and printing out the unsupported callback.
