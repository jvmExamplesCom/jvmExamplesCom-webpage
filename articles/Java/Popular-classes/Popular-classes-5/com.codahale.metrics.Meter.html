Introduction to Java com.codahale.metrics.Meter

The 

          
           
com.codahale.metrics.Meter
class is a powerful tool for tracking the rate at which a set of operations occur. It is useful for monitoring the rate of requests within a system, or the rate of data being read from a file. Creating an instance of the 

          
           
Meter

Meter

Timer

start()
method of the 

          
           
Meter
: 

          
           
Timer timer = new Timer();
meter.start(timer);
Once the 

          
           
Meter

Marking Events

The 

          
           
Meter
class allows events to be marked. This is done by calling the 

          
           
mark()

Each call to

Meter

Stopping the Meter

When the 

          
           
Meter

meter.stop();
This will stop the internal timer, and reset the internal counter of the 

          
           
Meter

. These methods include:

Meter
