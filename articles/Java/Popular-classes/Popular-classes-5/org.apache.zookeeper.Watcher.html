org.apache.zookeeper.Watcher

Java's  

          
           
org.apache.zookeeper.Watcher
is an object that receives notifications of changes to the ZooKeeper data tree.

Creating a Watcher

process(WatchedEvent event)
    — This method is called when the node that is being watched has been changed, deleted or created
  • 
            
             
onConnected(Watcher.Event.KeeperState connectedState)
    — This method is triggered when a connection is established between the client and the server
  • 
            
             
onDisconnected(Watcher.Event.KeeperState disconnectedState)
    − This method is triggered when a connection between the client and the server is lost
  • onExpired()
    − This method is triggered when the session is expired

Once the Watcher has been created, the methods can be used to watch for changes regarding a specific node in the ZooKeeper data tree.
