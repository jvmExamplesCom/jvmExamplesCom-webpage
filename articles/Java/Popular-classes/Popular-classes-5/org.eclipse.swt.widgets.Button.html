org.eclipse.swt.widgets.Button

The org.eclipse.swt.widgets.Button class is used to represent a push button control in a graphical user interface. It is a subclass of org.eclipse.swt.widgets.Control and provides additional methods that allow applications to create a button with text, an image, or both.

Creating a Button

Creating a button consists of constructing an instance of the Button class using one of the available constructors and then setting the desired properties. The following example creates a basic button: 

          
           
Button b = new Button(parent, SWT.PUSH);
where parent is the Composite from which the button will be created. The SWT flag indicates the type of the button (in this case a push button).

Setting Button Properties

The properties of a Button can be set by calling the corresponding methods provided by the Button class. The following examples demonstrate how to set the text, image and alignment of the button: 

          
           
b.setText("Click Me!");
b.setImage(image);
b.setAlignment(SWT.LEFT);
The setText() method sets a String as the text of the button. The setImage() method takes an Image object as a parameter and sets it as the button's image. The setAlignment() method takes one of the constants defined in the SWT class and sets the alignment of the text and image to one of the values: LEFT, CENTER or RIGHT.

Handling Button Events

When a button is clicked, an event is generated. This can be handled by registering a Listener object with the addListener() method of the Button class. The following example registers an event listener with the button: 

          
           
b.addListener(SWT.Selection, new Listener(){
    public void handleEvent(Event e){
        //handle the event here
    }
});
The above code registers a Listener object for the Selection event of the button. The handleEvent() method should contain the code to handle the event (e.g. displaying a message, performing an action, etc.).

Example

The following example demonstrates how to create a button and handle its events: 

          
           
public class ButtonExample {
    public static void main(String[] args){
        Display display = new Display();
        Shell shell = new Shell(display);
        shell.setSize(200, 200);
    
        //Create the button
        Button b = new Button(shell, SWT.PUSH);
        b.setText("Click Me!");
        b.setBounds(100,50,100,30);
    
        //Register a listener
        b.addListener(SWT.Selection, new Listener(){
            public void handleEvent(Event e){
                System.out.println("Button clicked!");
            }
        });
    
        shell.open();
    
        while(!shell.isDisposed()){
            if(!display.readAndDispatch())
                display.sleep();
        }
        display.dispose();
    }
}
In the above example, a button is created and a Listener object is registered to listen for the Selection event of the button. When the button is clicked, the handleEvent() method of the Listener prints out a message.
