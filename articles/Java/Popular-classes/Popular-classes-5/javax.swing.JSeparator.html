javax.swing.JSeparator

The 

          
           
javax.swing.JSeparator
class is a component that can create a horizontal or vertical visual separator between components in a container. It can be used to provide a more cohesive and organized user interface.

Example

In the following example, we will create a 
JFrame
with two components, separated by a 
JSeparator
: 

          
           
JFrame frame = new JFrame("My Frame");
JPanel panel1 = new JPanel();
JPanel panel2 = new JPanel();
JSeparator separator = new JSeparator(SwingConstants.HORIZONTAL);

frame.add(panel1);
frame.add(separator);
frame.add(panel2);

frame.pack();
frame.setVisible(true);

We can customize the appearance of a 
JSeparator
by setting its color, size, fonts, etc. The following code example demonstrates how to customize the color of a separator: 

          
           
JSeparator separator = new JSeparator(SwingConstants.HORIZONTAL);
separator.setBackground(Color.RED);

When creating a 
JSeparator
, we must specify its orientation using the 
SwingConstants
enum. The following code example shows how to create a vertical separator: 

          
           
JSeparator separator = new JSeparator(SwingConstants.VERTICAL);
