org.objectweb.asm.Label

It is used to represent program locations in a class file or any other location within a program's bytecode. It can also be used to create branch and jump instructions, which allow the programmer to change the flow of execution of a program. The methods setTarget() and getTarget() can be used to set and retrieve a target label for branching and jump instructions, to redirect the flow of execution in a method. The setAttribute() and getAttribute() methods can be used to store and retrieve an arbitrary attribute that is associated with the label.
