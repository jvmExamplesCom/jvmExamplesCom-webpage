Introduction to javax.mail.Multipart

The javax.mail.Multipart package provides classes that are useful in creating and managing MIME multipart messages. It is derived from the javax.mail package, which contains core classes for handling email messages. The packages provide methods for constructing mail messages, specifying header information, setting message options, and managing attachments.

Creating a Message with an Attachment

To create a message with an attachment using the javax.mail.Multipart classes, you must first create an instance of the javax.mail.internet.MimeMultipart class. This will represent the message body and all its components. For example: 

          
           
MimeMultipart multipart = new MimeMultipart();
Next, create an instance of the javax.mail.internet.MimeBodyPart class which represents the attachment. This is done by specifying the file to be attached and a content disposition. For example: 

          
           
File file = new File("example.pdf");
MimeBodyPart attachment = new MimeBodyPart();
attachment.attachFile(file);
attachment.setDisposition(Part.ATTACHMENT);
Next, add the attachment to the multipart instance. For example: 

          
           
multipart.addBodyPart(attachment);

Sending the Mail Message

Now that the message has been created, it needs to be sent. This is achieved with the javax.mail.Transport class. All that is needed is the message with the appropriate recipient and protocol details, such as username and password. For example:
