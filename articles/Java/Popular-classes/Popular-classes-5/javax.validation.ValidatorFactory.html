The 

          
           
javax.validation.ValidatorFactory
is a class for configuring and creating instances of the 

          
           
javax.validation.Validator
interfaces. These interfaces allow programs to verify and validate object models according to specification within Java.

Getting a ValidatorFactory Instance

Getting a 
ValidatorFactory
instance can be done with the following method: 

          
           
ValidatorFactory factory = Validation.byDefaultProvider().configure().buildValidatorFactory();

Creating a Validator Instance

A 
Validator

ValidatorFactory
by calling the 
getValidator()
method on it. 

          
           
Validator validator = factory.getValidator();

Configuring the Validator

The 
ValidatorFactory
may optionally be configured when it is built. Currently, only one configuration setting is available: 
traverseProperties
. When this setting is enabled, the 
Validator
will also validate objects that are referenced by the root object. 

          
           
ValidatorFactory factory = Validation.byDefaultProvider().configure()
    .traverseProperties(true)
    .buildValidatorFactory();
