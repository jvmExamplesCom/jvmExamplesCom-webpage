Android Widget CheckedTextView

Android's 
CheckedTextView
is a type of view used for displaying checkable interface elements. It is an extension of the 
TextView
class and can be used to create lists with a single or multiple choice selection.

Using CheckedTextView in Android

The following example demonstrates how to use a basic CheckedTextView in an Android project. We start by creating a layout XML file that contains the CheckedTextView.


          
           
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
              android:orientation="vertical"
              android:layout_width="match_parent"
              android:layout_height="match_parent">
 
   <CheckedTextView
       android:id="@+id/checked_text_view"
       android:layout_width="match_parent"
       android:layout_height="wrap_content"
       android:text="Checked Text View!" />

</LinearLayout>

Next, we need to add the CheckedTextView to our Activity and configure it using Java code.


          
           
public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        CheckedTextView checkedTextView = findViewById(R.id.checked_text_view);
        checkedTextView.setChecked(true);
    }
}

Finally, we need to set an OnClickListener on the CheckedTextView so that it can respond to clicks.


          
           
checkedTextView.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View view) {
        CheckedTextView checkedTextView = (CheckedTextView) view;
        checkedTextView.toggle();
    }
});

This will toggle the CheckedTextView when it is clicked by the user.

Conclusion

The Android CheckedTextView can be used to create lists with a single or multiple choice selection. We have seen how to use this view in our projects, and how to respond to user clicks. This is a powerful and convenient way to create checkable interface elements in Android.
