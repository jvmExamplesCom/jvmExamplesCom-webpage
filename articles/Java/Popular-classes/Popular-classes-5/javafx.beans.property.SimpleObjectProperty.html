What is a Property?

A property is a special type of variable that has two components: a getter and a setter. The getter is used to retrieve the value of the property and the setter is used to set the value of the property. Both of these methods are usually implemented as public methods so they can be accessed from outside the class. Properties are often used to access the internal state of objects from outside the object without breaking encapsulation.
