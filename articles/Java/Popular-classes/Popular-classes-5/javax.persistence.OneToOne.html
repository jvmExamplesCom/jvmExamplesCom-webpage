javax.persistence.OneToOne

javax.persistence.OneToOne is an annotation used in object-relational mapping to denote a one-to-one relationship between two entities. This annotation is used to define a field in one entity that has a reference to a object of another entity. It can also be used with OneToMany and ManyToOne annotations to create unidirectional or bidirectional relationships.

Examples

Here are some examples of using the javax.persistence.OneToOne annotation.


          
           
@Entity
public class Person {
    @Id
    private Long id;

    @OneToOne
    private Address address;
}

@Entity
public class Address {
    @Id
    private Long id;

    private String street;
    private String city;
}

In this example, we are defining an entity called Person that has an association of type OneToOne with the entity Address. This means that each instance of Person can have one associated instance of Address.


          
           
@Entity
public class Person {
    @Id
    private Long id;
    
    @OneToOne(mappedBy="person")
    private Address address;
}

@Entity
public class Address {
    @Id
    private Long id;
    
    @OneToOne
    private Person person;

    private String street;
    private String city;
}

In this example, we are defining an entity called Person that has a one-to-one relationship with the entity Address. We are also specifying the mappedBy attribute, which tells the JPA provider which field of the second entity holds the reference back to the first entity. This creates a bidirectional relationship between the two entities.
