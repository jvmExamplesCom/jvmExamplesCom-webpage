org.reactivestreams.Subscriber

Reactive Streams is a standard for asynchronous stream processing with non-blocking back-pressure. The standard enables the development of efficient and robust libraries for asynchronous streams. The org.reactivestreams package defines the interfaces that are used by publishers and subscribers to communicate with each other.

The Subscriber interface defines the main methods that are used by publishers to communicate with subscribers. 


          
           
public interface Subscriber<T> {
    void onSubscribe(Subscription subscription);
    void onNext(T element);
    void onError(Throwable t);
    void onComplete();
}

onSubscribe

The onSubscribe() method is invoked when a Subscriber subscribes to a Publisher. This is where the Subscriber can request items from the Publisher, using the Subscription object that is passed as a parameter.

The most common way to request items is to call the Subscription's request() method. The Subscriber can specify the exact number of items it wishes to receive.


          
           
public void onSubscribe(Subscription subscription) {
    // Request 10 items
    subscription.request(10);
}

onNext

The onNext() method is invoked by the Publisher when a new item has been emitted. This is where the item should be consumed. 


          
           
public void onNext(T element) {
    // Consume the element
    System.out.println(element);
}

onError

The onError() method is invoked by the Publisher when something goes wrong. This could be due to an exception or a cancelled Subscription. The Throwable parameter contains the exception that caused the error. 


          
           
public void onError(Throwable t) {
    // Log the error
    System.err.println("Error: " + t.getMessage());
}

onComplete

The onComplete() method is invoked by the Publisher when all items have been emitted. This can be used by the Subscriber to clean up any resources that were in use. 


          
           
public void onComplete() {
    // Clean up resources
    System.out.println("Done");
}
