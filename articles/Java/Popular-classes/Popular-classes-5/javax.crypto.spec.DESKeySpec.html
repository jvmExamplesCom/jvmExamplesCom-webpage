javax.crypto.spec.DESKeySpec

javax.crypto.spec.DESKeySpec is a class within the Java Cryptography Architecture (JCA) which gives an application access to the DES algorithm or other DES-like algorithms. This class is the key specification for the DES algorithm and provides an immutable view of an DES (Data Encryption Standard) secret key.
