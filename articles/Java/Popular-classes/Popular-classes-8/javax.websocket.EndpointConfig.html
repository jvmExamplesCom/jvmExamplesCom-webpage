Java javax.websocket.EndpointConfig

The javax.websocket.EndpointConfig class is a configuration that can be used to configure a WebSocket endpoint in a Java application.

Creating an EndpointConfig

To create an EndpointConfig, you must first create a subclass of javax.websocket.EndpointConfig and provide it with a method to get the configuration:


          
           
public class MyEndpointConfig extends EndpointConfig {

    @Override
    public Map<String, Object> getUserProperties() {
        // Return your configuration here
    }

}

You can then configure the EndpointConfig by setting the user properties. This can be done either in the constructor of the EndpointConfig class or when the endpoint is deployed.

Configuring the EndpointConfig

The user properties of an EndpointConfig instance can be configured using the following methods:

  • 
            
             
setUserProperties(Map<String, Object>)

            
  • 
            
             
addUserProperty(String, Object)

            
  • 
            
             
removeUserProperty(String)

The user properties set on the EndpointConfig are then available via the method getUserProperties(), which returns a map of the user properties. In the above example, the user properties were added via the setUserProperties() method.

Example Usage

Here is an example of how to use the javax.websocket.EndpointConfig to configure a WebSocket endpoint:


          
           
// Create an instance of MyEndpointConfig
MyEndpointConfig config = new MyEndpointConfig();

// Set the user properties
config.setUserProperties(MyUserProperties);

// Deploy the endpoint
ServerEndpoint endpoint = new ServerEndpoint("/myendpoint", config);
ServerEndpointDeployment deployment = new ServerEndpointDeployment(endpoint);
ServerEndpointDeploymentManager.getInstance().deploy(deployment);

Once the endpoint is deployed, the user properties will be available via the method getUserProperties(). For example, to get the value of a user property called "myProperty":


          
           
Object myPropertyValue = config.getUserProperties().get("myProperty");
