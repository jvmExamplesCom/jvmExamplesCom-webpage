Constructors

The constructor for the javax.swing.border.BevelBorder class takes three parameters: the type of bevel, the highlight color, and the shadow color. The type of bevel can be one of the following values: BevelBorder.Raised, which creates an embossed appearance; BevelBorder.Lowered, which creates a depressed appearance; or BevelBorder.None, which does not paint any border.
