Run Class Overview

It then calls the build() method to execute the build, and finally the postBuild() method to report the results. The Run class also provides methods for getting various pieces of information about the build. For example, the getStatus() method can be used to retrieve the status of a build, and the getLog() method can be used to get the console log of the build.
