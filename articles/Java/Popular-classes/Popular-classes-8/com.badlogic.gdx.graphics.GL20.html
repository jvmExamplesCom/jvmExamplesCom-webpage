What is com.badlogic.gdx.graphics.GL20?

com.badlogic.gdx.graphics.GL20 is a Java library that provides an interface for OpenGL ES 2.0 applications. It provides access to all the features of the OpenGL ES 2.0 specification including: vertex and fragment shaders, vertex and index buffers, textures, renderbuffers, and framebuffers.
