Introduction to io.vertx.ext.unit.junit.VertxUnitRunner

io.vertx.ext.unit.junit.VertxUnitRunner is an annotation-based JUnit test runner for writing and running asynchronous tests with Vert.x. It allows programmers to easily write tests that can be run in JUnit.

Basic Setup

In order to use io.vertx.ext.unit.junit.VertxUnitRunner, the following setup is necessary.

  • The JUnit version should be 4.12 or higher
  • io.vertx:vertx-unit should be in the classpath.

Once the setup is done, a test class need to be annotated with @RunWith(VertxUnitRunner.class) and it should extends AbstractVerticle.


          
           
@RunWith(VertxUnitRunner.class)
public class MyTest extends AbstractVerticle {
    // test code
}

Writing Tests

Once the test class is setup, the test cases needs to be written. Writing test cases consists of writing the code that needs to be tested and the assertion part.

For setting the code to be tested, the @Before method should be used.


          
           
@Before
public void setUp() {
    // Set up code
}

Once the setup is done, the tests can be written by using the @Test annotation along with the assertion functions.


          
           
@Test
public void testSomething() {
    // Test code here
    assertEquals(expectedValue, actualValue);
}

Running Tests

Once all the tests are written, the test can be executed with the following command.


          
           
mvn test

This will execute all the tests written and output the results.

Conclusion

io.vertx.ext.unit.junit.VertxUnitRunner is an easy to use JUnit test runner for writing and running asynchronous tests with Vert.x. It provides a simple way to write tests and execute them with very little setup.
