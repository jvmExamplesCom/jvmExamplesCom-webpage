java.security.interfaces.ECPublicKey

The java.security.interfaces.ECPublicKey interface is a key interface for public keys used in Elliptic Curve cryptography. It extends the java.security.interfaces.ECKey interface and provides additional methods for generating the public keys.

Methods

The java.security.interfaces.ECPublicKey interface defines the following methods:

  • 
            
             
public ECPoint getW();
    – returns the public point W of this public key.
  • 
            
             
public String getAlgorithm();
    – returns the name of the algorithm associated with this public key.
  • 
            
             
public String getFormat();
    – returns the name of the format in which this public key is encoded.
  • 
            
             
public byte[] getEncoded();
    – returns the public key in its primary encoding format.
