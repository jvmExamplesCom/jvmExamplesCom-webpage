This makes it a perfect choice for when you want to get a result from a supplier but don't need to pass in any arguments. The LongSupplier interface is part of the Java 8 SDK and was released with the version 1.8.

Using LongSupplier

To make use of the LongSupplier interface, we have to create a class that implements this interface and then override its single method, getAsLong(). Inside this method, we can specify whatever logic is necessary to return a value. Here's an example of a simple implementation of the LongSupplier interface:


          
           
public class RandomLongSupplier implements LongSupplier {
    @Override
    public long getAsLong() {
        long min = -100;
        long max = 100;
        return min + (long)(Math.random() * (max - min));
    }
