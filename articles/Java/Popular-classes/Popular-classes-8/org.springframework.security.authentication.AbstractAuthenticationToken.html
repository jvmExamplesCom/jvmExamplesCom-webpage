org.springframework.security.authentication.AbstractAuthenticationToken

The 

          
           
org.springframework.security.authentication.AbstractAuthenticationToken
class is an abstract class used in Spring Security to represent an authenticated user. It provides methods to store the granted authorities, set authenticated boolean flag, set details and get principals. It also provides methods to facilitate the build up of such an authenticated user.

Code Example 1: Setting Authorities


          
           
Collection<GrantedAuthority> authorities = Arrays.asList(
        new SimpleGrantedAuthority("ROLE_USER"));
AbstractAuthenticationToken auth = 
        new AbstractAuthenticationToken(authorities) {};
auth.setAuthenticated(true);

Code Example 2: Get Principle


          
           
Object principal = auth.getPrincipal();

Code Example 3: Set Details


          
           
Map<String, String> details = new HashMap<>();
details.put("username", "john.doe");
auth.setDetails(details);

Code Example 4: Set AppDetails


          
           
AppUser appUser = new AppUser();
appUser.setUsername("john.doe");
auth.setAppDetails(appUser);
