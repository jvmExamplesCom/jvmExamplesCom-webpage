org.openqa.selenium.NoSuchElementException

org.openqa.selenium.NoSuchElementException is an exception that is thrown when a web element cannot be located. This often occurs when the web element in question is not present on the page, or when the web element is not configured properly.

Cause

The cause of org.openqa.selenium.NoSuchElementException can be due to a number of different factors. The most common cause of this exception is due to the web element not being defined properly. For example, if the web element is not specified correctly in the HTML source code, Selenium will not be able to find it and thus will throw an exception. Additionally, if the web element is not visible on the page, Selenium may also not be able to identify it and will also throw this exception.

Code Examples

Below are some code examples showing the use of org.openqa.selenium.NoSuchElementException:


          
           
try {
    driver.findElement(By.id("invalid_id"));
} catch (NoSuchElementException e) {
    System.out.println("Element not found!");
}
