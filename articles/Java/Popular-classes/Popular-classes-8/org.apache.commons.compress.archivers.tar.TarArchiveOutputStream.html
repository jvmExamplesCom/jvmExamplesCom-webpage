Introduction to the TarArchiveOutputStream Class

Tar files are a common archive format and there are many compression algorithms available for them. The TarArchiveOutputStream class provides access to these algorithms and makes it easy to write tar files from your Java code.

TarArchiveOutputStream out = new TarArchiveOutputStream(new FileOutputStream("example.tar"));
