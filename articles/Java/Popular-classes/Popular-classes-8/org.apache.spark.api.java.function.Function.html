Introduction to org.apache.spark.api.java.function.Function

Basic Syntax

Function<InputObject, OutputObject> myFunction = new Function<InputObject, OutputObject>() {
    @Override
    public OutputObject call(InputObject input) {
        // Function logic goes here
    }
};
