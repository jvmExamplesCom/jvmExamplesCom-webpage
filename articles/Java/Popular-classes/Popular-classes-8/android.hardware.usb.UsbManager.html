android.hardware.usb.UsbManager

The android.hardware.usb.UsbManager class provides access to USB devices connected to an Android device. It enables applications to discover available USB devices and request permission from the user to access them.

Getting Started

To get started with the UsbManager class, you will need to get a handle of the UsbManager object by getting a reference from the Android application context. This can be done by calling the getSystemService() method with the parameter “usb”.


          
           
UsbManager manager = (UsbManager) context.getSystemService(Context.USB_SERVICE);

HashMap<String, UsbDevice> deviceList = manager.getDeviceList();

Before your application is able to communicate with a USB device, it must first acquire permission from the user. The UsbManager class provides this functionality with the requestPermission() method.
