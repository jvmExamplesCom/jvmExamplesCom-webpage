javax.sound.sampled.UnsupportedAudioFileException

The 

          
           
javax.sound.sampled.UnsupportedAudioFileException
is a checked exception that is thrown when an audio file fails to be recognized by the Java Sound API, either due to an unrecognized format or corrupt audio data.

Example Usage

The following code snippet uses a try-catch block to catch any 


          
           
UnsupportedAudioFileException
s that may arise when attempting to play a sound file. 

          
           
try {
    AudioInputStream in = AudioSystem.getAudioInputStream(new File("sound.wav"));
    AudioFormat outFormat = new AudioFormat(Float.class, 44100, 16, 2, 4, 44100, false);
    AudioInputStream convertedIn = AudioSystem.getAudioInputStream(outFormat, in);
    Clip clip = AudioSystem.getClip();
    clip.open(convertedIn);
    clip.start();
} catch (UnsupportedAudioFileException e) {
    System.out.println("Error: unsupported audio file.");
}

If an unsupported audio format is encountered by this code, an 


          
           
UnsupportedAudioFileException
will be caught and the error message “Error: unsupported audio file.” will be printed.

UnsupportedAudioFileException
