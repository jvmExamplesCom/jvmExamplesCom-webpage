Introduction to JavaFX Stage DirectoryChooser

The JavaFX Stage DirectoryChooser class allows a user to open a directory, select a file, and choose various functions related to directories and files. It is a platform-independent library for creating graphical user interfaces (GUIs) in Java applications.

Using the DirectoryChooser Class

Using the DirectoryChooser class requires the following steps:

  • Create a new DirectoryChooser instance.
  • Set any optional properties.
  • Show the DirectoryChooser.
  • Retrieve the selected directory or file.

Creating a DirectoryChooser Instance

Creating a DirectoryChooser instance is simple: 

          
           
DirectoryChooser chooser = new DirectoryChooser();

Setting Optional Properties

The DirectoryChooser class has several optional properties that can be set:


          
           
title: The title of the dialog box.

          

          
           
initialDirectory: The directory that is initially displayed in the dialog box.

          

          
           
defaultDirectory: The directory that is used if none is specified.

For example, to set the title and initial directory of the chooser:


          
           
chooser.setTitle("Select a folder");
chooser.setInitialDirectory(new File("C:/"));
