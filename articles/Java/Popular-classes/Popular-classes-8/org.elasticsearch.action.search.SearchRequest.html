SearchRequest API for Elasticsearch in Java

Elasticsearch is a distributed search and analytics engine used to search and store data. The org.elasticsearch.action.search.SearchRequest API is a Java API for performing search requests against the Elasticsearch cluster. It provides a way of querying and retrieving information from the cluster.

Instantiating a SearchRequest

A SearchRequest object can be instantiated using two different ways: 

          
           
SearchRequest request = new SearchRequest();
or
SearchRequest request = new SearchRequest("my_index");
The first one creates an empty request object, while the second one sets the index used in the search process.

Setting Search Parameters

Search parameters can be set with the SearchSourceBuilder class. The SearchSourceBuilder takes a QueryBuilder instance as a parameter. 

          
           
SearchSourceBuilder sourceBuilder = new SearchSourceBuilder()
                .query(QueryBuilders.matchQuery("user", "kimchy"));
request.source(sourceBuilder);
This code matches the document field user to kimchy.

Executing the Request

Once the search request has been set up, it can be executed using the search method of the SearchRequest class. This method returns an ActionFuture, which is a type that implements java.util.concurrent.Future. 

          
           
ActionFuture<SearchResponse> future = client.search(request);
The ActionFuture can be used to retrieve the results of the search query.

Retrieving Results

To retrieve the results of the search query, the get method of the ActionFuture interface is used. The get method returns a SearchResponse object containing the search results. 

          
           
SearchResponse response = future.get();
The SearchResponse contains information about the search query, such as the number of documents found and the time taken to run the query. It also contains the actual search results, which can be accessed through the hits method. 

          
           
SearchHit[] hits = response.getHits().getHits();
The hits method returns an array of SearchHit objects, each containing the data for a single document returned by the search query.

Conclusion

The org.elasticsearch.action.search.SearchRequest API provides a simple way to perform search queries against an Elasticsearch cluster using Java. This API allows developers to set up search requests and execute them against the cluster, as well as retrieve the results of the search queries.
