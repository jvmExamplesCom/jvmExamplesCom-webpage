android.content.pm.ServiceInfo

android.content.pm.ServiceInfo is a component of Android that provides access to service-related information and meta-data associated with an application package. It stores information such as service name, enabled state, exported flags, Intent filters, and permissions.

Code Examples

This example shows how to create a ServiceInfo object using the package manager:


          
           
PackageManager pkgManager = getPackageManager();
ServiceInfo serviceInfo = pkgManager.getServiceInfo(
        new ComponentName("com.example.app", "com.example.app.MyService"),
        PackageManager.GET_SERVICES);

    // Service is enabled
}
To set the ServiceInfo's enabled state: 

          
           
serviceInfo.enabled = true;
