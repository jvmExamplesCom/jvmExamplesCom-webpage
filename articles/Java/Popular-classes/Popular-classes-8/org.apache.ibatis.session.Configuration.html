Configuration Basics

Configuration
class provides several methods for configuring a MyBatis application. The most basic configuration consists of specifying a database connection and some SQL maps: 

          
           
Configuration config = new Configuration();
config.addDataSource(dataSource);
config.addMapping("my-sql-map.xml");
The 

          
           
Configuration
class can also be used to specify type handlers, interceptors, and other options. For example, to register a type handler with the Configuration: 

          
           
TypeHandler typeHandler = new MyTypeHandler();
config.getTypeHandlerRegistry().register(typeHandler);
To register an interceptor: 

          
           
Interceptor interceptor = new MyInterceptor();
config.addInterceptor(interceptor);

SqlSessionFactory
