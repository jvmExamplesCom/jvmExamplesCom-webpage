Working with the BasicParser Class

To use this class, you must first create an instance of it. The constructor takes three parameters: 


          
           
BasicParser parser = new BasicParser(options, stopAtNonOption);

    options
    parameter is a 
    Options
    object which holds the information about the command line options that the user can use.
  • The 
    stopAtNonOption
    parameter indicates whether the parser should stop processing options when it encounters an unknown option.

Once the parser is initialized, you can then parse an array of 

String
objects containing the command line arguments that the user has provided. This is done by using the 
parse
method as follows: 

          
           
CommandLine commandLine = parser.parse(args);
The 
parse
method returns a 
CommandLine
object that contains all the command line options and values that the user has provided. It also provides methods for retrieving specific options and values. For example, you can use the 
hasOption

    //...
}

You can also retrieve the value of an option with the 

getOptionValue

String optionName = commandLine.getOptionValue("optionName");

Once you have finished parsing the command line arguments, you can then perform any additional actions necessary based on the values provided by the user.
