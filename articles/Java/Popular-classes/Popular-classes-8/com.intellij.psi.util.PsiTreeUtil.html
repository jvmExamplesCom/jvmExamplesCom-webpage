com.intellij.psi.util.PsiTreeUtil

com.intellij.psi.util.PsiTreeUtil is a utility class within IntelliJ's PSI API that provides a variety of useful methods related to manipulating PSI (Program Structure Interface) trees in Java. It is designed to provide basic features such as finding elements by type, and traversing parent, child, and sibling nodes within a tree.

Code Examples

Let's look at how we can use this class in some practical examples. First, let's define a PSI tree structure:

Element element1 = ...;
Element element2 = ...;
Element element3 = ...;

root.addChild(element1);
root.addChild(element2);
element1.addChild(element3);

Now we can use various PsiTreeUtil methods to manipulate the tree. We can find all children of a certain type with findChildrenOfType():


          
           
List<Element> elements = PsiTreeUtil.findChildrenOfType(root, Element.class);
