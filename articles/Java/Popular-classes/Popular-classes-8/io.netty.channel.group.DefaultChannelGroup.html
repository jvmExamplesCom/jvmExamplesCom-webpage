Netty is an asynchronous event-driven networking library which enables quick and easy development of network applications such as protocol servers and clients. One of the core components of Netty is DefaultChannelGroup which provides an efficient way to manage a collection of Channel instances.

DefaultChannelGroup Overview

DefaultChannelGroup is a specialized ChannelGroup implementation that stores the members of the group in a thread-safe Set. In addition to managing the collection, it also provides several utility methods for broadcast operations. The following example shows how to create and use a DefaultChannelGroup: 

          
           
// Create a new DefaultChannelGroup
DefaultChannelGroup channelGroup = new DefaultChannelGroup();

// Add Channels to the group
Channel ch1 = ...;
channelGroup.add(ch1);

// Broadcast a message
channelGroup.writeAndFlush("Hello World!");

// Close the Group when done
channelGroup.close();
The DefaultChannelGroup also provides methods for querying the size of the group, checking for membership, and iterating over each member of the group.

API Reference

The DefaultChannelGroup class extends the abstract ChannelGroup class and provides the following commonly used methods: 

          
           
size():
Returns the number of Channel instances in the group. 

          
           
add():
Adds a Channel to the group. 

          
           
remove():
Removes a Channel from the group. 

          
           
contains():
Checks if a Channel is a member of the group. 

          
           
write():
Broadcasts a message to all Channels in the group. 

          
           
flush():
Flushes all pending writes on all Channels in the group. 

          
           
close():
Closes the group and all its members.
