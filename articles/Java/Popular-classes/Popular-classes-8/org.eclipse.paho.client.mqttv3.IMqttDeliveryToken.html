org.eclipse.paho.client.mqttv3.IMqttDeliveryToken

The IMqttDeliveryToken interface represents a token used to track the delivery of a message published by an IMqttAsyncClient. It is returned by the IMqttAsyncClient when a message is published asynchronously, and can be used to track success or failure of the delivery.

Methods

void waitForCompletion(long timeout)
Waits for up to the specified timeout for the message delivery to complete.


          
           
//Code example
MqttMessage message = new MqttMessage();
message.setPayload("Hello World".getBytes());

IMqttDeliveryToken token = client.publish("topic/helloWorld", message);
token.waitForCompletion(5000);

boolean isComplete()
Checks if the message delivery has completed.


          
           
//Code example
boolean isComplete = token.isComplete();

String getTopics()
Returns an array of topics associated with this delivery token.


          
           
//Code example
String topics[] = token.getTopics();

MqttMessage getMessage()
Returns the MqttMessage associated with this delivery token.


          
           
//Code example
MqttMessage message = token.getMessage();

Throwable getException()
Returns any exception associated with the operation. If no exception has been thrown then returns null.


          
           
//Code example
Throwable exception = token.getException();

Additional interfaces

The IMqttDeliveryToken interface extends the IMqttToken interface which contains additional methods that can be used to track the progress of the delivery.


          
           
//Code example
client.setCallback(new MqttCallback() {
    public void deliveryComplete(IMqttDeliveryToken token) {
        try {
            String messageId = token.getMessageId();
            String topic = token.getTopics()[0];
            Logging.println("Message delivered. MessageId: " + messageId + ", Topic: " + topic);
        } catch (MqttException e) {
            e.printStackTrace();
        }
    }
});
