Java com.google.api.client.http.GenericUrl

The Java com.google.api.client.http.GenericUrl class is useful for constructing a URL in a convenient way. It provides a simple and powerful API for creating and manipulating URLs. This class allows developers to construct fully qualified URLs without having to manually construct the query strings and parameters. With the GenericUrl class, developers can easily specify the protocol, host, port, path, query string, and parameters.

A GenericUrl object can be created by passing a String representation of the URL to the constructor. Alternatively, developers can use the set methods to manually set the protocol, host, port, path, query string, and parameters.

Example


          
           
GenericUrl url = new GenericUrl("https://www.example.com/path?param1=value1&amp;param2=value2");

The following methods are available to manipulate the URL:

  • getProtocol(): Returns the protocol (e.g. http or https).
  • setProtocol(String): Sets the protocol.
  • getHost(): Returns the host.
  • setHost(String): Sets the host.
  • getPort(): Returns the port.
  • setPort(int): Sets the port.
  • getPath(): Returns the path.
  • setPath(String): Sets the path.
  • getQuery(): Returns the query string.
  • addParameter(String, String): Adds a new parameter to the query string.
  • removeParameter(String): Removes a parameter from the query string.

Once the URL is constructed, it can be accessed via the toString() method.


          
           
String urlString = url.toString();
