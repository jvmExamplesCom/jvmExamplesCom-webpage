Introduction to org.springframework.core.env.StandardEnvironment

org.springframework.core.env.StandardEnvironment is a core part of the Spring Framework that provides an environment-independent framework for managing and accessing environment information such as operating system settings and Java system properties.

StandardEnvironment env = new StandardEnvironment();
