com.fasterxml.jackson.core.JsonEncoding

JsonEncoding is an abstract class from the Jackson package which serves as a base class for all supported Json encoding formats. It contains several methods for writing and reading encoded data from a stream or reader.

Methods

The methods provided by JsonEncoding are get Encoding, get Java Name, get ClassLoaderName, writeBinary, readBinary and others. Let's have a look at the usage of some of these methods.

getEncoding()

The 

          
           
getEncoding()
method returns the type of encoding being used, be it UTF-8, UTF-16, ISO-8859-1 or one of the other supported formats.

getJavaName()

The 

          
           
getJavaName()
method returns the Java name of the encoding being used, such as “UTF-8”, “ISO-8859-1”, etc.

getClassLoaderName()

The 

          
           
getClassLoaderName()
method returns the name of the classloader associated with the encoding.

writeBinary()

The 

          
           
writeBinary(OutputStream out, byte[] data)
method writes encoded binary data to the given output stream.

readBinary()

The 

          
           
readBinary(InputStream in)
method reads encoded binary data from the given input stream.

Example Usage

Let's look at an example usage of JsonEncoding.

We'll create a simple program to read a file containing encoded data, decode it and print it out to the console.


           
            
// Create an instance of JsonEncoding
JsonEncoding jsonEncoding = new JsonEncoding();

// Read the encoded data from a file
InputStream inputStream = Files.newInputStream(Paths.get("data.json"));

// Decode the data
byte[] decodedData = jsonEncoding.readBinary(inputStream);

// Print the decoded data
System.out.println(new String(decodedData));

This is a very basic example of using the JsonEncoding class to read and decode encoded data.
