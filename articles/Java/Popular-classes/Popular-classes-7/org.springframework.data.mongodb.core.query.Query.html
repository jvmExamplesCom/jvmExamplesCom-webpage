Introduction to org.springframework.data.mongodb.core.query.Query

Creating Queries with the Query Class

createQuery()
methods. This method takes an array of criteria as arguments and returns a 

          
           
Query
object. For example, consider the following query that finds documents with a "name" field equal to "John": 

          
           
Query query = Query.createQuery(Criteria.where("name").is("John"));

Query query = Query.createQuery(
    Criteria.where("name").is("John"), 
    Criteria.where("age").gt(18)
