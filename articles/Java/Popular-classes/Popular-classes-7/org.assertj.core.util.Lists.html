Introduction to org.assertj.core.util.Lists

The org.assertj.core.util.Lists class is a collection of static factory methods used to create Lists in AssertJ. It provides various methods to create Lists from collections, arrays, Strings, and more.

Creating an Empty List

We can create an empty List using the following methods:


          
           
List<Object> emptyList = Lists.emptyList();
List<String> emptyStringList = Lists.emptyList();

Both of the above examples will create an empty List.

We can create a new List from an existing Collection using the following method:


          
           
List<String> listFromCollection = 
  Lists.newArrayList(Arrays.asList("foo", "bar"));

Creating a List from an Array

We can create a new List from an array using the following method:


          
           
String[] array = {"foo", "bar"};
List<String> listFromArray = Lists.newArrayList(array);

List<String> copiedList = Lists.newArrayList(list);
