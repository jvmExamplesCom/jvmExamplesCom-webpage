com.bumptech.glide.GlideBuilder

The GlideBuilder class is an entry point for configuring Glide to use custom settings and create a reusable Glide instance. It supports setting settings like the image loader, memory cache size, BitmapPool size, and more. Additionally, any registered RequestOptions can be set on all created requests.
