Introduction to com.google.common.collect.ImmutableBiMap

Java's com.google.common.collect.ImmutableBiMap package is a class that implements Map interface, allowing users to store key/value pairs that cannot be changed. This package can be used in any situation where you need an immutable collection of key/value pair associations. It is important to note that the same key cannot map to two different values and the order of insertion is not maintained.

Creating an ImmutableBiMap

An ImmutableBiMap can be created using various methods: 

          
           
ImmutableBiMap<Integer, String> numString = ImmutableBiMap.of(1, "one", 2, "two", 3,"three");
or 

          
           
ImmutableBiMap<Integer, String> numString= ImmutableBiMap.<Integer, String>builder().put(1, "one").put(2,"two").put(3, "three").build();

Map<Integer, String> numStringMap = new HashMap<>();
numStringMap.put(1, "one");

numStringMap.put(3, "three");
