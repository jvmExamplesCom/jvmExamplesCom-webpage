Retrofit Client Response Using Java

Retrofit is an open source networking library developed by Square that provides an easy and powerful way to make API calls in Java. Using Retrofit Client Response in Java

<dependency>

  <artifactId>retrofit</artifactId>
  <version>2.3.0</version>
</dependency>
