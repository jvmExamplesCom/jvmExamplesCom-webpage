What is a Boolean Query?

A Boolean query is a type of query that uses boolean logic to combine multiple query clauses together. These clauses are connected using logical operators such as AND and OR, which allow multiple search criteria to be evaluated at once. For example, if you wanted to search for all documents that contain either "cat" or "dog", you could use the following query:
