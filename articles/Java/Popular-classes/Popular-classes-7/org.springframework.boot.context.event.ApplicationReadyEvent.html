The 

          
           
ApplicationReadyEvent
is an event that is sent when a Spring Boot application has started and is ready to service requests. This event fires after the ApplicationContext is fully initialized and refreshed, and is useful for performing any final startup tasks in the application before it begins servicing requests.

Event listeners can be registered to listen for this event using the @EventListener annotation. In this example, we create a simple listener that prints out a message when the application is ready:


          
           
@Component
public class MyListener {
 
    @EventListener
	public void handleReadyEvent(ApplicationReadyEvent event) {
		System.out.println("My application is now ready to service requests!");
	}

}

When the application starts and is ready, the following will be printed out:


          
           
My application is now ready to service requests!
