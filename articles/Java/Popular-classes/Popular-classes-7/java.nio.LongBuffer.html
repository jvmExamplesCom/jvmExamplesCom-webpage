java.nio.LongBuffer

java.nio.LongBuffer is a class that allows for the handling of long data types in Java. It provides functions for manipulating, reading, writing, and converting between long values and other primitive data types. The LongBuffer class is part of the java.nio package. It is generally used when dealing with large volumes of data, such as when working with databases or file systems.

Methods

java.nio.LongBuffer provides many useful methods for dealing with longs. Some of the most common methods include:

  • get() – get a single long from the buffer.
  • put() – put a long into the buffer.
  • clear() – clear all longs from the buffer.
  • flip() – reset the buffer's position to the beginning.
  • mark() – remember the current position in the buffer.
  • reset() – reset the position to the remembered one.
  • rewind() – reset the position to the beginning.

Example

In this example, we will create a LongBuffer object with the capacity of 10 and fill it with random long values. We then use the methods described above to manipulate the LongBuffer.


          
           
//Create a LongBuffer object with a capacity of 10
LongBuffer buffer = LongBuffer.allocate(10); 

//Fill the buffer with random long values
Random rand = new Random();
for (int i = 0; i < buffer.capacity(); i++) {
    buffer.put(rand.nextLong());
}

//Print the contents of the buffer
System.out.println("Before manipulation:");
for (int i = 0; i < buffer.capacity(); i++) {
    System.out.print(buffer.get() + " ");
}

//Manipulate the buffer
buffer.clear();
buffer.mark();
buffer.put(12345L);
buffer.reset();
buffer.rewind();

//Print the contents of the buffer
System.out.println("\nAfter manipulation:");
for (int i = 0; i < buffer.capacity(); i++) {
    System.out.print(buffer.get() + " ");
}
