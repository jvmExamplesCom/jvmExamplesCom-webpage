Android.os.Messenger

Android.os.Messenger is an integral part of the Android operating system, providing reliable messaging services that allow applications to communicate with each other. The core features of this class include message delivery, message sending, and message handling. It also offers several convenience methods for creating messages, forwarding messages, and handling asynchronous messages.

Creating a Messenger

To create a Messenger, you must first instantiate a new instance of the class using the following code:


          
           
Messenger messenger = new Messenger();

Sending Messages

Once a Messenger has been created, you can send messages using the 


          
           
send
method. This method takes a 

          
           
Message
object as its only parameter, which contains the text and data to be sent. For example, the following code sends a hello world message: 

          
           
Message message = new Message("Hello World!");

messenger.send(message);

Handling Messages

When a message is received by a Messenger, it is delivered to a callback method specified when the Messenger was created. This callback method takes a 


          
           
Message
object as its only parameter, which contains the text and data of the message. The following example prints the body of the message: 

          
           
messenger.setMessageHandler(new MessageHandler() {
    public void handleMessage(Message message) {
        System.out.println(message.getBody());
    }
});

Forwarding Messages

In some cases, it may be necessary to forward messages between Messengers. This is done using the 


          
           
forward
method, which takes a source 

          
           
Messenger
, destination 

          
           
Messenger

Message

messenger.forward(sourceMessenger, destMessenger, message);
