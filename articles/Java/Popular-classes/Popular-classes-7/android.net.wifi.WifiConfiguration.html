android.net.wifi.WifiConfiguration

The android.net.wifi.WifiConfiguration class manages the configurations of Wi-Fi networks. It is mainly used by different applications to connect to a network and manage connection information.

Creating WifiConfigurations

You can use the WifiConfiguration class to create a new configuration for a Wi-Fi network or update existing configurations.

To create a new configuration, you will need to create a new instance of the WifiConfiguration class:


          
           
WifiConfiguration wifiConfig = new WifiConfiguration();

To set the network SSID, you can use the following code:


          
           
wifiConfig.SSID = "\"" + ssid + "\"";

Where ssid is the SSID of the network you want to connect to. You can also set the network security type using the following code:


          
           
wifiConfig.allowedKeyManagement.set(WifiConfiguration.KeyMgmt.NONE);

Where NONE is the type of network security (e.g. WPA, WPA2, etc.)

Once you have set the security type, you can set the pre-shared key using the following code:


          
           
wifiConfig.preSharedKey = "\"" + password + "\"";

Where password is the pre-shared key for the network.

Finally, you can set the network ID of the configuration using the following code:


          
           
wifiConfig.networkId = networkId;
Where networkId is the ID of the network.

Connecting to Networks

Once you have created a configuration, you can use it to connect to a network. The following code will connect to a network using the specified WifiConfiguration:


          
           
WifiManager wifiManager = (WifiManager) getSystemService(Context.WIFI_SERVICE);
int networkId = wifiManager.addNetwork(wifiConfig);
wifiManager.enableNetwork(networkId, true);

Where wifiConfig is a WifiConfiguration object created earlier.

Saving Configurations

Once you have created a configuration and connected to a network, you can save the configuration for later use. To do this, you will need to use the updateNetwork method from the WifiManager class:


          
           
int result = wifiManager.updateNetwork(wifiConfig);

Where wifiConfig is the configuration you want to save. This method will return 0 on success or -1 on failure.

Removing Configurations

If you want to remove a previously saved configuration, you can use the following code:


          
           
int result = wifiManager.removeNetwork(networkId);

Where networkId is the ID of the network you want to remove.

Conclusion

In this article, we saw how to use the WifiConfiguration class to manage Wi-Fi networks in Android. We saw how to create configurations, connect to networks, save configurations and remove configurations. We also saw how to use the WifiManager class to add, enable and update networks.
