JFormattedTextField field = new JFormattedTextField();

In order to specify a format, a constructor taking a Format parameter can be used:


          
           
DateFormat dateFormat = new SimpleDateFormat("dd/MM/yyyy");

JFormattedTextField maskedField = new JFormattedTextField(maskFormatter);
