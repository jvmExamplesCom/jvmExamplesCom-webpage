Creating a Connection

To create a connection, use the static factory method ConnectionFactory.newConnection(). This method accepts two arguments: a connection configuration object, and an IFrameHandler object.


          
           
Connection connection = ConnectionFactory.newConnection(
    new ConnectionConfig(), 
    new IFrameHandler()
);

ConnectionConfig config = new ConnectionConfig();
config.host("192.168.0.1");
