android.widget.FrameLayout.LayoutParams is a subclass of the ViewGroup.LayoutParams class and helps developers to set attributes for layout patterns on their Android applications. It provides an easy way to define the size, position, and margins for a given layout.

FrameLayout.LayoutParams lp = new FrameLayout.LayoutParams(300, 200, Gravity.TOP);

lp.setMargins(int left, int top, int right, int bottom);

lp.setMarginStart(int start);
lp.setMarginEnd(int end);

lp.setMarginBottom(int bottom);

lp.setLayoutWeight(float weight);
