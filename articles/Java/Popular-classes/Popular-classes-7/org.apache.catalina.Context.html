org.apache.catalina.Context

The org.apache.catalina.Context class is a fundamental component of the Java Servlet Container, Tomcat. It is responsible for providing the web application with an environment in which to run. The Context class provides a range of functions and methods that enable it to monitor and manage the web application.

Creating a Context Instance

Creating a Context instance is easy. It can be done with the following code snippet:


          
           
Context context = new StandardContext();

Adding Components to the Context

Adding components to the Context is simple. For example, to add an HTTP servlet, you can use the following code:


          
           
context.addServlet("myservlet", MyServlet.class);

You can also add filters and listeners to the Context instance. To add a filter, use the following code:


          
           
FilterDef filterDef = new FilterDef();
filterDef.setFilterName("myfilter");
filterDef.setFilterClass(MyFilter.class);
context.addFilterDef(filterDef);
To add a listener, use the following code: 

          
           
context.addApplicationListener(MyListener.class);

context.setResourceBase("/wwwroot");
context.setDefaultWebXml("/etc/web.xml");
context.addParameter("foo", "bar");

Starting and Stopping the Context

context.start(true);
context.stop();
