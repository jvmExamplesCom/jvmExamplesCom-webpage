org.junit.runner.JUnitCore

The 
JUnitCore
class is the main entry point for running JUnit tests programmatically. It provides methods to run a set of tests with or without filtering and reporting. It also gives access to the results of the tests that have been executed as well as any exceptions that might have occurred.

Example


          
           
import org.junit.runner.JUnitCore;
import org.junit.runner.Result;
import org.junit.runner.notification.Failure;
 
public class TestRunner {
  public static void main(String[] args) {
    Result result = JUnitCore.runClasses(TestJunit.class);
     
    for (Failure failure : result.getFailures()) {
      System.out.println(failure.toString());
    }
     
    System.out.println(result.wasSuccessful());
  }
}

Running a single test

In order to run just a single test method from a test class, you can use the 

runChild()
method. It takes two parameters, the first being the test class, the second the name of the test method to be executed: 

          
           
JUnitCore.runChild("TestJunit", "testAdd");

Filtering

You can also filter which tests are run according to certain criteria. To achieve this, create an instance of org.junit.runner.Request, e.g. Request.classes(), Request.method(), Request.aClass(), Request.aMethod() or Request.all():


          
           
Request request = Request.classes(TestJunit.class);
JUnitCore.run(request);

Results

The result of the tests are accessible through the 

Result
object returned by all of the 
JUnitCore
methods. It contains information about the number of successes and failures, as well as the total time spent executing the tests: 

          
           
result.getRunCount(); // number of tests run
result.getFailureCount(); // number of failures
result.getIgnoreCount(); // number of ignored tests
result.getRunTime(); // total time spent running tests in milliseconds
The 
Result
object also gives access to a list of failures encountered during the execution of the tests: 

          
           
List<Failure> failures = result.getFailures();

Summary

The 

          
           
org.junit.runner.JUnitCore
class provides an easy way to run JUnit tests programmatically. It allows tests to be run with or without filtering as well as examined for results and failures. The filtered tests can be specified by creating instances of 

          
           
org.junit.runner.Request
.
