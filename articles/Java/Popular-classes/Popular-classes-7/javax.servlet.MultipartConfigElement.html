javax.servlet.MultipartConfigElement

The javax.servlet.MultipartConfigElement class is a Java servlet API used to provide support for multipart/form-data requests, which allow clients (typically browser-based web clients) to send files of any type along with the normal form data. Multipart requests can be used to implement file upload functionality in your servlets and web applications.

Creating a MultipartConfigElement Object

A MultipartConfigElement object can be created using its constructor with several parameters. The two most important parameters are the location and the maxFileSize. The location specifies the directory where uploaded files will be temporarily stored and maxFileSize specifies the maximum size of an uploaded file, in bytes. For example:


          
           
MultipartConfigElement multipartConfig = 
    new MultipartConfigElement("/tmp", 102400, 204800, 0);

This creates a MultipartConfigElement object that points to the directory /tmp, with a maximum file size of 102,400 bytes and a maximum total request size of 204,800 bytes.

Adding a MultipartConfigElement to a Servlet

Once you’ve created your MultipartConfigElement object, you need to add it to the servlet in which you want to handle multipart requests. This can be done by using the 

@MultipartConfig
annotation on the servlet class. For example: 

          
           
@MultipartConfig(location="/tmp", maxFileSize=102400, 
maxRequestSize=204800)
public class MyServlet extends HttpServlet {
    // servlet logic
}

Once the annotation is added, the servlet is able to process incoming multipart requests.

Example

@MultipartConfig(location="/tmp", maxFileSize=102400, 
maxRequestSize=204800)
public class MultipartServlet extends HttpServlet {
    protected void doPost(HttpServletRequest request, 
        HttpServletResponse response) throws ServletException, IOException {

        // Check that we have a file upload request
        boolean isMultipart = ServletFileUpload.isMultipartContent(request);

        if (isMultipart) {
            // Create a factory for disk-based file items
            DiskFileItemFactory factory = new DiskFileItemFactory();

            // Configure a repository (to ensure a secure temp location is used)
            ServletContext servletContext = this.getServletConfig().getServletContext();
            File repository = (File) servletContext.getAttribute("javax.servlet.context.tempdir");
            factory.setRepository(repository);

            // Create a new file upload handler
            ServletFileUpload upload = new ServletFileUpload(factory);

            // Parse the request
            try {
                List<fileitem> items = upload.parseRequest(request);
                for (FileItem item : items) {
                    if (item.isFormField()) {
                        // Process regular form field (input type="text|radio|checkbox|etc", select, etc).
                        String fieldname = item.getFieldName();
                        String fieldvalue = item.getString();

                        System.out.println(fieldname + ":" + fieldvalue);
                    } else {
                        // Process form file field (input type="file").
                        String fieldname = item.getFieldName();
                        String filename = FilenameUtils.getName(item.getName());
                        InputStream stream = item.getInputStream();
                        System.out.println(fieldname + ":" + filename);

                    }
                }
            } catch (FileUploadException e) {
               throw new ServletException("Cannot parse multipart request.", e);
            }
        }
    }
}
