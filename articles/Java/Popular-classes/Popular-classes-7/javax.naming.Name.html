javax.naming.Name

The 

          
           
javax.naming.Name
interface represents a generic name -- an ordered sequence of components. For example, one might represent the name of a person in terms of the attributes given name and family name: 

          
           
GivenName = John
FamilyName = Smith
FullName = John Smith
In this case, the Name would have two components, the string "John" and the string "Smith", in that order.

Creating a Name Object

Context.createName()
method. This method creates an instance of a class that implements the Name interface. For example: 

          
           
Context ctx = new InitialContext();
Name n = ctx.createName( "John", "Smith" );
The components of the name can also be passed as an array of strings: 

          
           
String[] components = { "John", "Smith" };
Name n = ctx.createName( components );
