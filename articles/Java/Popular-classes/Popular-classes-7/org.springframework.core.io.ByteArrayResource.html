org.springframework.core.io.ByteArrayResource

The 

          
           
org.springframework.core.io.ByteArrayResource
is a specialized 

          
           
Resource
implementation for a given byte array. It creates a ByteArrayInputStream on the given byte array, and is seen as Resource interface in the Spring framework.

Create a ByteArrayResource

Creating a 

          
           
ByteArrayResource
requires a byte array which can be provided as an argument to the constructor. 

          
           
byte[] bytes = { 0x48, 0x65, 0x6c, 0x6c, 0x6f };

ByteArrayResource resource = new ByteArrayResource(bytes);

ByteArrayResource

getInputStream()

getDescription()
, and 

          
           
getFilename()

ByteArrayResource

A 

          
           
ByteArrayResource
