android.text.format.Time

android.text.format.Time is a Java class used to handle dates and times in Android applications. It provides various utility methods for manipulating date and time values, such as the ability to set time fields, retrieve time values, and compare dates. Construct Time Objects

Time curTime = new Time();

GregorianCalendar gc = new GregorianCalendar();

Time myTime = new Time();
myTime.set(12, 15, 2020); // set hour, minute and year
