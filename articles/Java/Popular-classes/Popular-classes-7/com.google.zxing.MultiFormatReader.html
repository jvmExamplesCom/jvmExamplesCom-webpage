Overview of com.google.zxing.MultiFormatReader

The 

          
           
com.google.zxing.MultiFormatReader
class is used to decode barcodes in multiple different formats. It supports most of the popular barcode formats, including QR Code, Aztec, Code 128, and others. The main purpose of the class is to read barcodes from image or camera sources and convert them into meaningful data, such as strings and numbers.

Example Usage

This code example shows how to use the 
MultiFormatReader
class to read a QR code from an image: 

          
           
BufferedImage image = ImageIO.read(new File("qrcode.png"));
LuminanceSource source = new BufferedImageLuminanceSource(image);
BinaryBitmap bitmap = new BinaryBitmap(new HybridBinarizer(source));
MultiFormatReader reader = new MultiFormatReader();
Result result = reader.decode(bitmap);
String text = result.getText();
System.out.println(text); // prints the contents of the QR code

String
.
