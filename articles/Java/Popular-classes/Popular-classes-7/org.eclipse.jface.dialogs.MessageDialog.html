org.eclipse.jface.dialogs.MessageDialog

The MessageDialog class provides a dialog for users to interact with messages. It is part of the Eclipse Platform’s JFace project, and allows developers to create dialogs that present messages to the user, with options for users to select from. The class can be used to display important messages, errors, warnings, questions, and information.

Example Dialog

For example, the following code creates a simple message dialog.


          
           
MessageDialog dialog = new MessageDialog(parentShell, "Dialog Title", null, "Dialog Message", 
            MessageDialog.INFORMATION, new String[] { "Button1", "Button2" }, 0);
dialog.open();

This will display a dialog window with the title “Dialog Title”, the message “Dialog Message”, and two buttons labeled “Button1” and “Button2”. The numbers and labels of the buttons are customizable, and different types of message dialogs can show different types of messages.

Message Types

The MessageDialog class provides several types of message dialogs. Each type displays a different icon in the dialog window and has different default buttons.

  • MessageDialog.ERROR
    – Displays an error message.
  • 
            
             
MessageDialog.INFORMATION
    – Displays an information message.
  • 
            
             
MessageDialog.WARNING
    – Displays a warning message.
  • 
            
             
MessageDialog.QUESTION
    – Displays a question message.

Buttons

The MessageDialog class also provides several methods to customize the buttons displayed in the dialog.

  • 
            
             
setButtonLabels(String[] labels)
    – Sets the labels of the buttons. The number of labels must be equal to the number of buttons.
  • getButtonLabels()
    – Returns the labels of the buttons.
  • getButtonCount()
    – Returns the number of buttons.
  • 
            
             
setDefaultButtonIndex(int index)
    – Sets the index of the default button. This is the button that is selected when the user presses Enter.

Return Value

When the user closes the dialog, the MessageDialog class returns an integer representing the index of the button that the user clicked. This value can then be used to determine which button the user chose.


          
           
int returnCode = dialog.open();

The default return value is -1, which indicates that the user closed the dialog without clicking any button.

MessageDialog dialog = new MessageDialog(parentShell, "Confirmation", null,
            "Are you sure?", MessageDialog.QUESTION, new String[] { "Yes", "No" }, 0);

    // User chose "Yes"
} else if (returnCode == 1) {
    // User chose "No"
