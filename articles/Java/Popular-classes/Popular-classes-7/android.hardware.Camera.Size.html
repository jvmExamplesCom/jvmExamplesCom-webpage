The android.hardware.Camera.Size class contains information about the size of a camera image. It can be used to determine the correct size for a picture when taking a photo or video.

public static Camera.Size findBestSizeMatch(List<Camera.Size> sizes, int targetWidth, int targetHeight)

This static method finds the best matching Camera.Size object from a list of Camera.Sizes that has the closest dimensions to the specified target width and height.

Example

The following example demonstrates how to find the best Camera.Size match from a list of Camera.Sizes:
