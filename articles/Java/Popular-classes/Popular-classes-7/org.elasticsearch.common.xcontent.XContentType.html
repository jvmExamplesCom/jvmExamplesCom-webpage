Introduction to org.elasticsearch.common.xcontent.XContentType

The org.elasticsearch.common.xcontent.XContentType class is an enum type used in Elasticsearch to define the types of content when serializing or deserializing data. This class defines two XContent types – JSON (application/json) and YAML (text/yaml). Each type is given a short name, used when constructing XContent objects.
