Introduction to javax.ws.rs.core.Response.ResponseBuilder

javax.ws.rs.core.Response.ResponseBuilder is a Java class used for constructing response objects in Java Enterprise Edition (Java EE) applications. It is a part of the Java API for RESTful Web Services (JAX-RS). The ResponseBuilder class provides methods for creating response objects, including specific status codes, entity headers and entity bodies.

Overview

The ResponseBuilder class is a child class of the abstract Response class, which contains the properties and methods from which responses are constructed. Each response is composed of three elements:

  • status code
  • entity headers
  • entity body

The ResponseBuilder class is used to construct all parts of a response, including the status code, entity header and entity body.

How To Use ResponseBuilder?

ResponseBuilder can be used to construct responses by following these steps:

  1. Create an instance of the ResponseBuilder.
  2. Set the desired status code.
  3. Construct an entity header if required.
  4. Construct the entity body if required.
  5. Call the build() method to create the response.

Examples of Usage

Let's look at some examples of how to use ResponseBuilder to construct responses.

Example 1 - Basic Response with No Body

The following example shows how to create a basic response with no body and a 200 OK status code.


          
           
ResponseBuilder rb = Response.ok();
Response response = rb.build();

Example 2 - Response with Entity Header

The following example shows how to create a response with an entity header and a 200 OK status code.


          
           
ResponseBuilder rb = Response.ok();
rb.header("Content-Type", "application/json");
Response response = rb.build();

Example 3 - Response with Entity Body

The following example shows how to create a response with an entity body and a 200 OK status code.


          
           
String body = "{\"name\":\"John Doe\"}";
ResponseBuilder rb = Response.ok().entity(body);
