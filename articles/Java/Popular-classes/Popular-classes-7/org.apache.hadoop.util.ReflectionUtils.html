Introduction to Apache Hadoop ReflectionUtils

Apache Hadoop ReflectionUtils is a set of utility methods for reflection in Java. It is a part of Apache Hadoop project, which is an open-source framework for distributed storage and processing of large data sets on clusters of commodity hardware. ReflectionUtils provides utility methods such as a generic constructor lookup (creating instances of classes with arbitrary constructors), invoking methods with arbitrary arguments, dealing with the unchecked exceptions thrown in reflection calls, etc.

Using the org.apache.hadoop.util.ReflectionUtils Class

The package consists of ReflectionUtils class and its nested classes. The ReflectionUtils class provides several static methods that take Object, Class and Method objects to perform various tasks related to reflection. This guide will show you how to:
  • Create an instance of a class with arbitrary constructors using newInstance() method.
  • Invoke a method with arbitrary arguments using invoke() method.
  • Catch and handle unchecked exceptions thrown in reflection calls using handleException() method.

Creating Instances of Classes Using newInstance() Method

The newInstance() method takes three arguments - a Class object representing the type of the class to create instance of, an Object array containing the arguments to the constructor and an Class array specifying the types of the arguments of the constructor. It then creates an instance of the given class using the provided constructor arguments and returns the newly created instance. For example, if we have a class ‘Foo’ with two constructors - one taking a String argument and another taking an int and a String argument, we can create an instance of ‘Foo’ as below: 

          
           
Class<Foo> clazz = Foo.class;
Object[] ctorArgs1 = new Object[]{“bar”};
Object[] ctorArgs2 = new Object[]{100, “bar”};
Class[] ctorArgTypes1 = new Class[]{String.class};
Class[] ctorArgTypes2 = new Class[]{int.class, String.class};

Foo foo1 = ReflectionUtils.newInstance(clazz, ctorArgs1, ctorArgTypes1);
Foo foo2 = ReflectionUtils.newInstance(clazz, ctorArgs2, ctorArgTypes2);

Invoking Methods Using invoke() Method

The invoke() method takes four arguments - an Object instance, a Method object, an Object array representing the arguments to the method and a Class array specifying the types of the arguments. It then invokes the method on the given instance using the provided arguments and returns the result. For example, if we have a class ‘Foo’ with a method ‘bar’ taking an int, a String and a boolean as arguments, we can invoke the method as below: 

          
           
Foo foo = new Foo();
Method method = Foo.class.getDeclaredMethod("bar", int.class, String.class, boolean.class);
Object[] methodArgs = new Object[]{100, “bar”, true};
Class[] methodArgTypes = new Class[]{int.class, String.class, boolean.class};

Object result = ReflectionUtils.invoke(foo, method, methodArgs, methodArgTypes);

Handling Unchecked Exceptions using handleException() Method

The handleException() method takes a Throwable object as a parameter. It then checks if the thrown exception is an unchecked exception and rethrows it as a RuntimeException, without changing any of its properties. For example, if we were to call the ‘bar’ method defined above and an exception was thrown, we can handle it as follows:

    Object result = ReflectionUtils.invoke(foo, method, methodArgs, methodArgTypes);
