What is org.apache.catalina.connector.Connector?

org.apache.catalina.connector.Connector is a Java class used to configure a Connector component that communicates with a web server. The connector can be configured by setting properties, attributes and listeners. This class supports HTTP/1.1, AJP/1.3, and other protocols.
