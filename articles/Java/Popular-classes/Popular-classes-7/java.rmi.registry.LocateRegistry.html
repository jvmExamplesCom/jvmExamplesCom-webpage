Introduction to java.rmi.registry.LocateRegistry

java.rmi.registry.LocateRegistry is a class that provides a static method used to obtain the Registry instance on a remote host. It is used to look up a name in the RMI registry and establish a connection with the remote object named by the name. The object passed as an argument in the LocateRegistry can be remote or local.

Features of java.rmi.registry.LocateRegistry

The main features of java.rmi.registry.LocateRegistry are:
  • Static method getRegistry() is used to retrieve an existing registry from the specified host, or create and export a registry on the local host if none already exists.
  • A static method, createRegistry() can be used to create a registry on the specified port.
  • The registry can be given a name when it is created, which is useful for clients to locate the registry.
  • The registry contains a list of names that clients can use to locate remote objects.
