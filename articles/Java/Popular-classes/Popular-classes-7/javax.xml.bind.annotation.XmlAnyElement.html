javax.xml.bind.annotation.XmlAnyElement

javax.xml.bind.annotation.XmlAnyElement is a JAXB annotation used to indicate that any XML element can be present in a given XML document. It can be used in combination with other JAXB annotations, such as XmlRootElement and XmlElementRef. This annotation allows for flexibility in an XML document by allowing for the unmarshalling of elements not specified by the application code.

Examples


          
           
// class definition
@XmlRootElement
public class Customer {
    @XmlElement
    private String name;

    @XmlAnyElement
    private Object data;
}

// xml document
<customer>
  <name>John Doe</name>
  <address>123 Main Street</address>
</customer>

In the example above, the address element is unmarshalled into the data field, even though it was not specified in the application code.
