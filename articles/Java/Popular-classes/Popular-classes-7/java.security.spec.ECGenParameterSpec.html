The ECGenParameterSpec class provides a constructor that takes a String as an argument. This String is the standard name of an EC parameter set as defined in RFC 5480, "A Generalized Framework for Elliptic Curve Cryptography". 

          
           
ECGenParameterSpec ecGenSpec =
  new ECGenParameterSpec("secp256r1");

Using ECGenParameterSpec
