javax.ws.rs.core.MultivaluedHashMap

The javax.ws.rs.core.MultivaluedHashMap class is an implementation of the MultivaluedMap interface which provides access to a set of key-value pairs. It stores elements in an unordered collection and allows read and write access. The keys are unique but values can be duplicated. This class provides a general-purpose associative array that can be used to store and retrieve arbitrary data.

Creating a MultivaluedHashMap

Use the constructor to create a new instance of the MultivaluedHashMap class. A parameterless constructor creates an empty map, or provide a Map object as an argument to populate the new instance with the provided elements.

source.put("key1", "value1");
source.put("key2", "value2");
MultivaluedHashMap<String, Object> map = new MultivaluedHashMap<>(source);

map.put("key2", "value2");
