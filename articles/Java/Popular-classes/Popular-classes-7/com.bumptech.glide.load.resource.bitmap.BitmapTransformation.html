com.bumptech.glide.load.resource.bitmap.BitmapTransformation

The 

          
           
BitmapTransformation
class is a utility from the Glide library for transforming bitmaps when loading them from a source. The transformation can also be applied when decoding a resource into a Bitmap. This class is used to apply both simple or complex operations to a bitmap, such as cropping or color filtering. It provides an easy way for developers to manipulate images, including those loaded from external sources.

BitmapTransformation
class to crop a bitmap by specifying the desired width and height: 

          
           
BitmapTransformation cropTransformation = new BitmapTransformation(width, height);
Bitmap croppedBitmap = cropTransformation.transform(originalBitmap);
