Exploring org.opencv.core.Mat in Java

Java is a powerful, object-oriented language that has found its way into many applications and frameworks. The opencv library, which is widely used for computer vision and image processing, provides a Mat class within org.opencv.core package. It offers an efficient way to manipulate single-channel or multi-channel 2D images. In this article, we will explore Mat and its uses through examples.

What is Mat?

Mat, short for Matrix, is an object of the class org.opencv.core.Mat. It is used to store and manipulate two-dimensional arrays of pixels called matrices. Each matrix is composed of multiple channels associated with one pixel.

Constructors

Mat is an abstract class, meaning that it cannot be directly instantiated. Instead, the constructors from its direct subclasses such as MatOfByte, MatOfFloat, MatOfDouble, etc. are used to create Mat objects. The constructors take either an array of bytes/floats/doubles/etc. or a size of the desired matrix as parameters. For example, the following code creates a Mat object with 3 rows and 2 columns: 

          
           
Mat m = new Mat(3, 2);
In addition, a Mat object can also be created from another Mat object. For example: 

          
           
Mat m2 = new Mat(m);

Methods

Mat offers several methods to manipulate the stored data. These methods include: 
  • 
            
             
put(int rowIndex, int colIndex, byte[] value)
    - inserts a byte array at the given row and column.
  • 
            
             
get(int rowIndex, int colIndex, byte[] dst)
    - retrieves a byte array from the given row and column.
  • setTo(Scalar value)
    - sets each pixel in the matrix to a given Scalar.
  • 
            
             
convertTo(Mat dstMat, int type)
    - converts the matrix to a specified type.
  • 
            
             
reshape(int channels)
    - reshapes the matrix to specified number of channels.

Usage Example

Letâ€™s consider an example where we want to create a 4x4 matrix filled with random integers between 0 and 10. The following code snippet should accomplish this task: 

          
           
// Create an empty 4x4 matrix
Mat m = new Mat(4, 4);

// Set random values between 0 and 10 
Random rand = new Random();
for (int rowIndex=0; rowIndex<m.rows(); rowIndex++) {
    for (int colIndex=0; colIndex<m.cols(); colIndex++) {
        int randomInt = rand.nextInt(11); // 0..10
        m.put(rowIndex, colIndex, new byte[]{(byte)randomInt});
    }
}

// Print contents of matrix
for (int rowIndex=0; rowIndex<m.rows(); rowIndex++) {
    for (int colIndex=0; colIndex<m.cols(); colIndex++) {
        byte[] value = new byte[1];
        m.get(rowIndex, colIndex, value);
        System.out.print(value[0]);
        System.out.print(" \t");
    }
    System.out.println();
}
