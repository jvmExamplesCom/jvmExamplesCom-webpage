Android Webkit JsResult

The Android Webkit JsResult class is an interface used by the WebView component to notify that an asynchronous JavaScript action has been completed. It contains three methods, confirm(), cancel() and setResult(). This gives the developer a way to return values from a WebView back to Java code.

confirm() Method

The confirm() method is used to trigger a confirmation dialog for a user. The prompt should contain a message and two buttons. One for OK or accept and one for Cancel or no. When the user selects an option either true or false will be returned. 

          
           
JsResult result = webView.confirm("Are you sure you want to delete this file?");
if (result.confirm()) {
    // File is deleted
} else {
    // File is not deleted
}

cancel() Method

The cancel() method is used to cancel an action in progress. It can also be used to explicitly close a dialog box. 

          
           
JsResult result = webView.cancel();
if (result.cancel()) {
    // Action canceled
}

setResult() Method

The setResult() method is used to set the result of the asynchronous action. It takes a boolean value as parameter (true or false). 

          
           
JsResult result = webView.setResult(true);
if (result.getResult()) {
    // Action succeeded
} else {
    // Action failed
}
