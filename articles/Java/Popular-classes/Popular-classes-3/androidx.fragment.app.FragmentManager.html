androidx.fragment.app.FragmentManager

The androidx.fragment.app.FragmentManager is a class which provides key functions related to working with Fragments. It is the primary class used for interacting with and managing Fragments within an Activity, and as such it serves as the central point of interaction between the Activity, Fragments, and other objects related to Fragment management. This article will cover the various aspects of the FragmentManager, including its methods, properties, and how it can be used to create practical applications.

Methods

The androidx.fragment.app.FragmentManager class has several methods which can be used to manage Fragments. The most commonly used methods are:

  • beginTransaction(): Used to begin a Fragment transaction. A Fragment transaction is a set of changes which will be applied to an Activity's Fragment layout.
  • add(): Used to add a Fragment to the Fragment layout.
  • replace(): Used to replace a Fragment in the Fragment layout.
  • remove(): Used to remove a Fragment from the Fragment layout.

Properties

The androidx.fragment.app.FragmentManager class has several properties which provide information about the current state of the Fragment layout. These properties include:

  • getFragments(): Used to retrieve a list of all Fragments in the Fragment layout.
  • findFragmentById(): Used to find a Fragment by its ID.
  • isDestroyed(): Used to check if the Fragment layout is destroyed.

Code Examples

The following code examples demonstrate some practical uses of the androidx.fragment.app.FragmentManager class.


          
           
// Begin a Fragment transaction
FragmentManager fragmentManager = getSupportFragmentManager();
FragmentTransaction fragmentTransaction = fragmentManager.beginTransaction();

// Add a Fragment to the Fragment layout
Fragment fragment = new MyFragment();
fragmentTransaction.add(R.id.my_fragment_container, fragment);

// Replace a Fragment in the Fragment layout
Fragment newFragment = new MyNewFragment();
fragmentTransaction.replace(R.id.my_fragment_container, newFragment);

// Remove a Fragment from the Fragment layout
Fragment oldFragment = fragmentManager.findFragmentById(R.id.my_old_fragment_id);
fragmentTransaction.remove(oldFragment);

// Commit the Fragment transaction
fragmentTransaction.commit();

          

          
           
// Get a list of all Fragments
List<Fragment> fragments = fragmentManager.getFragments();

          

          
           
// Check if the Fragment layout is destroyed
boolean isDestroyed = fragmentManager.isDestroyed();
