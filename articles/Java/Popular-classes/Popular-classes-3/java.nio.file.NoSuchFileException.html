Java nio File NoSuchFileException

The Java nio File NoSuchFileException is thrown when a file could not be found by the file system. This exception is typically encountered when attempting to access a file that does not exist in the specified directory, or when the file is not readable by the application.

Example 1


          
           
try {
    Path path = Paths.get("/data/file.txt");
    Files.readAllBytes(path);
} catch (NoSuchFileException e) {
    System.out.println("File does not exist: " + e.getMessage());
}
In this example, a 
NoSuchFileException
is thrown if the file at the given path does not exist. The exception is caught, and the error message is printed out.

}

byte[] contents = Files.readAllBytes(path);
