org.springframework.cache.annotation.Cacheable

org.springframework.cache.annotation.Cacheable is an annotation that is used to indicate a method invocation result should be cached. This annotation can be used for methods of any visibility: public, private, protected or package-private. The annotation can be applied at class level to cache all public methods in the class or at method level to cache the particular method.

Example 1: Caching All Methods in a Class

In this example, we are caching all methods of a class that is annotated with @Cacheable.


          
           
@Cacheable
public class MyService {
    
    public void doSomething() {
        // do something
    }
    
    public void doAnotherThing() {
        // do another thing
    }
}

Example 2: Caching A Single Method

In this example, we are only caching a single method within the class.


          
           
public class MyService {
    
    @Cacheable
    public void doSomething() {
        // do something
    }
    
    public void doAnotherThing() {
        // do another thing
    }
}

Example 3: Using Parameters

You can also use parameters in your caching annotations.


          
           
@Cacheable(key = "#param")
public List<Object> getObjects(String param) {
    // ...
}
You can also use multiple parameters. 

          
           
@Cacheable(key = {"#param1","#param2"})
public List<Object> getObjects(String param1, String param2) {
    // ...
}

Example 4: Maximum Age

You can also specify a maximum age for an entry in the cache.


          
           
@Cacheable(maxAge = 60)
public List<Object> getObjects(String param1, String param2) {
    // ...
}

Example 5: Setting a Time Unit

When specifying the maximum age, you can also specify a time unit.


          
           
@Cacheable(maxAge = 60, timeUnit = TimeUnit.SECONDS)
public List<Object> getObjects(String param1, String param2) {
    // ...
}

You can also specify a cache manager for a given method invocation.

    // ...
}

You can also specify a cache resolver for a given method invocation.
