android.graphics.drawable.LayerDrawable

The 

          
           
android.graphics.drawable.LayerDrawable
class is an object used to create complex drawable objects from an array of drawables. A LayerDrawable can be defined in an XML file with the 

          
           
<layer-list></layer-list>
element. Each Drawable in the layer is defined in a nested 

          
           
<item></item>
element.

Creating a LayerDrawable

To create a LayerDrawable, the constructor requires an array of Drawables and an optional set of 

          
           
StateListDrawables
. The following code example creates a LayerDrawable with two resources. 

          
           
Drawable[] layers = new Drawable[2];
layers[0] = getResources().getDrawable(R.drawable.image_a);
layers[1] = getResources().getDrawable(R.drawable.image_b);

LayerDrawable layerDrawable = new LayerDrawable(layers);
In this example, two Drawable resources are loaded from the app's resource folder. The 

          
           
image_a
and 

          
           
image_b
images are then added to the 

          
           
layers
array. Finally, the 

          
           
LayerDrawable
constructor is called using the array.

Setting Layer Attributes

Once the 

          
           
LayerDrawable
is created, you can then set some properties on each layer such as its size and position within the drawable. 

          
           
// Set the size of the drawable layers
layerDrawable.setLayerSize(0, 200, 200);
layerDrawable.setLayerSize(1, 200, 200);

// Set the position of the drawable layers
layerDrawable.setLayerPosition(0, 10, 10);
layerDrawable.setLayerPosition(1, 100, 100);
In this example, we use the 

          
           
setLayerSize
and 

          
           
setLayerPosition
methods to set the size and position of each drawable layer.

Adding StateListDrawable

You can also add a 

          
           
StateListDrawable
to a LayerDrawable. This allows you to provide different drawables based on the current state of the device. 

          
           
// Get the drawable from resources
StateListDrawable stateListDrawable = (StateListDrawable) getResources().getDrawable(R.drawable.statelist_drawable);

// Add the state list drawable to the layer drawable
layerDrawable.addState(new int[] {R.attr.state_pressed}, stateListDrawable);
In this example, the 

          
           
statelist_drawable
is added to the 

          
           
layerDrawable
. The 

          
           
addState
method is used to specify the attribute and Drawable to be used when the device is in the given state.

Using a LayerDrawable in Layout

Once the LayerDrawable is created, it can then be set on any 

          
           
View
or 

          
           
Activity
layout that supports a drawable background. The following example shows how to set the LayerDrawable as the background of a 

          
           
TextView
: 

          
           
TextView tv = findViewById(R.id.textView);
tv.setBackground(layerDrawable);
In this example, the 

          
           
layerDrawable
is assigned to the 

          
           
TextView

Conclusion

The 

          
           
LayerDrawable
class is a powerful way to create complex Drawable objects from multiple layers of Drawables. It can be used to quickly create sophisticated UI elements that respond to device states and changes.
