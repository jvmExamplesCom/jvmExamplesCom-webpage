com.fasterxml.jackson.annotation.JsonValue

This annotation is used to define how a Java property value is serialized as JSON. It is an annotation that can be used in one of two forms: as a method-level annotation or as a class-level annotation.

Method-Level Annotation Usage

When this annotation is applied to a getter method (or "accessor") on a class, the value returned from that method is used to generate the JSON representation of the value for the given property. 


          
           
@JsonValue  
public String getValue() {
    return "myValue";
}
In the above example, the return value of 
getValue()
will be used as the JSON representation of 
myValue
.

Class-Level Annotation Usage

When this annotation is applied to a class, the 
toString()
method of that class is used to generate the JSON representation of the value for the given property. 

          
           
@JsonValue  
public class MyClass {
    public String getValue() {
        return "myValue";
    }

    @Override  
    public String toString() {
        return "myStringValue";
    }
}
In the above example, the return value of the 
toString()
method will be used as the JSON representation of 
myStringValue
, not 
myValue
.

Conclusion

The 
@JsonValue
annotation is a powerful tool for customizing how Java objects are serialized as JSON. It provides flexibility in controlling the JSON representation of a property by either defining the serialization at the class level or at the method level.
