java.awt.Cursor

The Java AWT Cursor class provides the ability to modify certain aspects of the mouse cursor when the user interacts with your application. This includes the type of cursor you want to display, the cursor's background color, and the shape of the cursor. Using this class, you can create custom cursors with distinct shapes or colors to give your application a unique look and feel. Additionally, you can also make use of existing system cursors which are already included in the Java Runtime Environment.

Examples

In the following examples, we will demonstrate how to create and use a custom cursor. First, we must import the necessary packages: 

          
           
import java.awt.*;
import java.awt.event.*;
Next, we will create our custom cursor object. We will define its shape, size, and color: 

          
           
Cursor customCursor = new Cursor(Cursor.CUSTOM_CURSOR);
customCursor.setShape(new Rectangle(10, 10)); 
customCursor.setColor(Color.BLUE);
We can now set the new custom cursor as the default cursor of our application: 

          
           
Frame frame = new Frame();
frame.setCursor(customCursor);
Finally, we can add an action listener to the frame in order to respond to mouse interactions: 

          
           
MyListener myListener = new MyListener();
frame.addMouseListener(myListener);

The output of this program will be the shape, size, and color of the custom cursor as specified in the code. This can be seen by running the application and moving the mouse over the main window.
