org.apache.maven.plugin.MojoFailureException

org.apache.maven.plugin.MojoFailureException is an exception thrown when a plugin execution has failed due to an unexpected circumstance. The exception includes information about the plugin that has failed, additional information about the cause of the failure, and the messages from the plugin itself.

Example 1

In this example, the mojo fails with a MojoFailureException and the plugin provides specific details about the cause of the failure:


          
           
@Mojo(name = "failure-example")
public class FailureExampleMojo extends AbstractMojo {

    public void execute() throws MojoFailureException {
        throw new MojoFailureException("Something went wrong!");
    }
}

In this example, the mojo fails with a more detailed MojoFailureException and a message about the plugin:


          
           
@Mojo(name = "failure-example2")
public class FailureExample2Mojo extends AbstractMojo {

    public void execute() throws MojoFailureException {
        throw new MojoFailureException("Something went wrong!", 
                                       "MyPlugin", 
                                       "An error occurred while executing MyPlugin");
    }
}
