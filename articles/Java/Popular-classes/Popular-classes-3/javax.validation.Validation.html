javax.validation.Validation Overview

The javax.validation.Validation API is a powerful tool for validation and constraint declaration in a Java application. It was created to provide an extensible framework which allows custom components to be used for separating concerns of validating objects and declarations in a loosely coupled fashion. Declaring Constraints with Annotations

This approach is the preferred way of declaring constraints because it enables developers to specify validation constraints directly on their domain objects. It also enables them to group related constraints together and apply them to all of the properties of a class or a subset of properties. @NotNull
@Size(min = 5, max = 20)
