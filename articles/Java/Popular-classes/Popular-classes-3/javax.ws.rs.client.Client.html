javax.ws.rs.client.Client

The 

          
           
javax.ws.rs.client.Client
class provides a Java-based mechanism for processing RESTful (REpresentational State Transfer) web services. It provides methods for making requests to endpoints and for handling responses. It also provides methods for configuring request and response details. The Client class is a sub-class of the 
Ambit
class, which is part of the Jersey library. The Ambit class provides a framework for writing code to call web services and process their responses.

Creating a Client

Creating a Client is as simple as instantiating an instance of the Client class:


          
           
Client client = new Client();

The client instance created can then be used to access web resources. To do this, we need to configure the request and response details. We can do this by adding a 

WebTarget
object to the 
client
instance: 

          
           
WebTarget target = client.target("https://example.com/resources");

WebTarget

WebTarget
, we can then make requests using the 
request()

Invocation.Builder request = target.request();

addParameter()

request.addParameter("foo", "bar")

Response response = request.get();

String body = response.readEntity(String.class);
