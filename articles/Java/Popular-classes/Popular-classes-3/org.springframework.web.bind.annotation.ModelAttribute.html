org.springframework.web.bind.annotation.ModelAttribute

The @ModelAttribute annotation is an annotation that can be used on a method argument to access the current model data in a web request. It binds the model object to the method parameter and stores it in the request. This annotation has become popular in recent years due to its convenience in writing and reading code.

Using the Annotation

To use the annotation, a method parameter must be annotated with the @ModelAttribute annotation and the name of the model object that needs to be retrieved from the request. The following example shows how a model object named “person” can be accessed using the annotation: 

          
           
@ModelAttribute("person")
public Person getPerson() {
  return new Person();
}
The method returns the instance of the Person class that corresponds to the “person” model attribute.

Binding Request Parameters

In addition to retrieving model objects, the @ModelAttribute annotation can also be used to bind request parameters to method arguments. This allows developers to automatically populate method parameters with data from the incoming request. The following example shows how a request parameter named “name” can be bound to a method argument using the annotation: 

          
           
@ModelAttribute("name")
public String getName(@RequestParam("name") String name) {
  return name;
}
The method returns the value of the “name” request parameter.

Creating Model Attributes

The @ModelAttribute annotation can also be used to create model attributes. This allows developers to create and add model attributes to the model map. The following example shows how a model attribute named “address” can be created using the annotation: 

          
           
@ModelAttribute("address")
public Address getAddress() {
  return new Address();
}
The method returns an instance of the Address class which is then added to the model map.

Accessing the Model Map

The @ModelAttribute annotation allows developers to access the model map within the handler method. This allows the developer to modify the model map or retrieve model data from the request. The following example shows how the model map can be accessed within a handler method: 

          
           
@ModelAttribute("modifyMap")
public void modifyMap(@ModelAttribute("map") Map<String,Object> map) {
  map.put("key", "value");
}
The method adds a key/value pair to the model map.

Conclusion

The @ModelAttribute annotation is a powerful tool for accessing and manipulating model data in web requests. It can be used to retrieve model objects, bind request parameters to method arguments, create model attributes, and access the model map.
