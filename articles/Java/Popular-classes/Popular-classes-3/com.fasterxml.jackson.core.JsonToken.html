com.fasterxml.jackson.core.JsonToken

com.fasterxml.jackson.core.JsonToken is a class that contains an enumeration of potential JSON tokens that can be encountered while parsing a JSON string. They are widely used in Java applications to parse JSON strings and take action upon encountering each token. The following provides an overview of the available and common token types.

Common Token Types

  • START_OBJECT: Denotes the start of an object. It is the token associated with the opening brace ({) in a JSON string. This token is typically followed by a KEY_NAME token.
  • END_OBJECT: Denotes the end of an object. It is the token associated with the closing brace (}) in a JSON string.
  • START_ARRAY: Denotes the start of an array. It is the token associated with the opening bracket ([) in a JSON string. This token is typically followed by a VALUE token.
  • END_ARRAY: Denotes the end of an array. It is the token associated with the closing bracket (]) in a JSON string.
  • KEY_NAME: Denotes a key in a JSON object. It is the token associated with a key name, which is always a string.
  • VALUE: Denotes a value in a JSON string. This token may be any of the primitive or compound types (numbers, strings, booleans, arrays, and objects).

Example Usage

The following example illustrates how the JsonToken class can be used to parse a simple JSON string.


          
           
import com.fasterxml.jackson.core.JsonToken;
import com.fasterxml.jackson.core.JsonParser;

String jsonString = "{\"name\":\"John Doe\"}";

JsonParser parser = new JsonParser();
JsonToken token = parser.nextToken();
while (token != null) {
    // process the token
    switch (token) {
        case START_OBJECT:
            System.out.println("start object");
            break;
        case END_OBJECT:
            System.out.println("end object");
            break;
        case KEY_NAME:
            System.out.println("key name: " + parser.getCurrentName());
            break;
        case VALUE:
            System.out.println("value: " + parser.getText());
            break;
        default:
            break;
    }
    // read next token
    token = parser.nextToken();
}
The output of the above code should be: 

          
           
start object
key name: name
value: John Doe
end object
