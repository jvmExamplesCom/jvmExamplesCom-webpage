Introduction to org.json.simple.JSONObject

Using org.json.simple.JSONObject

To get started working with org.json.simple.JSONObject, you'll need to first create a new instance. You can do this by providing your JSON data as a string or as a File object:


          
           
String jsonData = "{\"name\":\"Bob\",\"age\":21}";
JSONObject obj = new JSONObject(jsonData);
