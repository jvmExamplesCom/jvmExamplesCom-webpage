javax.naming.InitialContext

The javax.naming.InitialContext class provides an object that can be used to look up names in the naming system. It is a subclass of the javax.naming.Context class.

Creating an InitialContext Object

The following code demonstrates how to create an instance of the InitialContext class:


          
           
InitialContext ctx = new InitialContext();

Looking Up a Name

The following code shows how to use the InitialContext object to look up a name from the underlying naming service, in this case the file system:


          
           
Object obj = ctx.lookup("/path/to/file.txt");

Binding a Name to an Object

ctx.bind("/path/to/file.txt", myObject);
