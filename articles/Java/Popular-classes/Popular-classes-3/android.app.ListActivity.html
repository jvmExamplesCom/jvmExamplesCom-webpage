Android List Activity

Android’s ListActivity is a class designed to facilitate the easy creation of lists in an Android app. It allows the programmer to easily create lists, and make use of other Activity classes to manage user interaction with the list items. In this article we will discuss the use of the ListActivity class to create a list in an Android application.

Using ListActivity

The ListActivity class is a pre-defined Activity class that allows the programmer to create a list quickly and easily. It gives the programmer the ability to create a list from an array, from a Cursor containing the result of a query against a content provider, or from any other object that implements the ListAdapter interface. The ListActivity class takes care of managing user interaction with the list items, and provides methods for adding listeners and capturing events related to the list items.

Example

Below is an example of how to use the ListActivity class in an Android application. In this example we will create a list from an array of strings. We will also add a click listener to the items in the list. 

          
           
public class MyListActivity extends ListActivity {

    private String[] listItems = {"Item 1", "Item 2", "Item 3"};
    
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);

        setListAdapter(new ArrayAdapter<string>(this, android.R.layout.simple_list_item_1, listItems));
        getListView().setOnItemClickListener(listListener);
    }

    private AdapterView.OnItemClickListener listListener = new AdapterView.OnItemClickListener() {
        @Override
        public void onItemClick(AdapterView<!--?--> parent, View view, int position, long id) {
            //Handle list item click event
        }
    };
}

In the above example we first define our array of strings which will be used to populate the list. We then override the onCreate() method, where we set the list adapter for our ListActivity, passing in an instance of an ArrayAdapter initialized with our string array and the resource layout, android.R.layout.simple_list_item_1. Finally, we add an OnItemClickListener to the list via the getListView() method.

Conclusion

The ListActivity class makes it easy to create lists in Android applications, and quickly add handlers to capture user input on the list items. With the ListActivity class it is possible to create lists from arrays, Cursors, or any object that implements the ListAdapter interface.
