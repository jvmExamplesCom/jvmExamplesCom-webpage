Using ListActivity

The ListActivity class is a pre-defined Activity class that allows the programmer to create a list quickly and easily. It gives the programmer the ability to create a list from an array, from a Cursor containing the result of a query against a content provider, or from any other object that implements the ListAdapter interface. The ListActivity class takes care of managing user interaction with the list items, and provides methods for adding listeners and capturing events related to the list items.

Example
