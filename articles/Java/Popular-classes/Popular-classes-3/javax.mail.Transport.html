javax.mail.Transport

javax.mail.Transport is a JavaMail class that provides functionality to send messages via SMTP. It enables applications to connect to an SMTP server and submit messages with commands defined in the SMTP protocol. The javax.mail.Transport class also provides methods to close the connection, check whether the connection is secure, flush the connection, and obtain the address of the server used.

Basic Usage

In order to use javax.mail.Transport, you'll need to create an instance of the class, which can be done using the static method:


          
           
Transport transport = Transport.getInstance(session);

This will return an instance of Transport for the given Session. After that, you can use various methods such as 

connect()
to establish a connection with an SMTP server and 
sendMessage()
to send a message.

Example Code

The following example shows how to use javax.mail.Transport to establish a connection to an SMTP server, authenticate with a username and password, and send an email:


          
           
Transport transport = Transport.getInstance(session);
transport.connect("smtp.example.com", username, password);
transport.sendMessage(message, message.getAllRecipients());
transport.close();
In this example, 
session
is an instance of Session, 
message
is an instance of Message, and 
username
and 
password
are strings containing the username and password for authenticating with the SMTP server.

Secure Connections

In some cases, you may need to connect to an SMTP server using a secure connection (SSL/TLS). This can be accomplished by using the following code:


          
           
Transport transport = Transport.getInstance(session);
transport.connect("smtp.example.com", username, password, "587", "TLS");
transport.sendMessage(message, message.getAllRecipients());
transport.close();

In this example, we're connecting to the server on port 587 (the standard port for TLS connections) and specifying the protocol as "TLS".

Conclusion

javax.mail.Transport is a powerful class that can be used to easily send messages via SMTP. With its support for secure connections, it's a great choice for sending emails securely and reliably.
