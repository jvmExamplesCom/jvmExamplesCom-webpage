What is com.google.common.util.concurrent.ListenableFuture?

com.google.common.util.concurrent.ListenableFuture is an interface in the Google Guava library for Java that provides an easy way to write asynchronous code. It provides a means of scheduling and executing tasks, as well as coordinating multiple tasks and handling their outcomes. ListenableFuture is the building block for writing applications that need to run tasks asynchronously, without having to worry about manually managing threads.

Code Examples

Creating a Basic ListenableFuture


          
           
ListenableFuture<String> future = 
    Executors.newSingleThreadExecutor().submit(() -> "Hello World!");

Handling Successful Results


          
           
Futures.addCallback(future, new FutureCallback<String>() {
    public void onSuccess(String result) {
        System.out.println("Result: " + result);
    }
});

Handling Exceptions


          
           
Futures.addCallback(future, new FutureCallback<String>() {
    public void onFailure(Throwable thrown) {
        System.out.println("Error: " + thrown.getMessage());
    }
});

Chaining ListenableFutures

You can chain ListenableFutures, allowing you to create complex workflows easily:


          
           
ListenableFuture<String> future1 = 
    Executors.newSingleThreadExecutor().submit(() -> "Hello World!");

ListenableFuture<String> future2 = 
    Futures.transform(future1, s -> s + " Goodbye!");

Futures.addCallback(future2, new FutureCallback<String>() {
    public void onSuccess(String result) {
        System.out.println("Result: " + result);
    }
});
