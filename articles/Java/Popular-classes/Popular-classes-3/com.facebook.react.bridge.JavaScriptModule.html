com.facebook.react.bridge.JavaScriptModule
is a JavaScript interface used in React Native to enable communication between native code and JavaScript components. It facilitates communication between the native environment and the JavaScript environment by allowing JavaScript objects to call native methods, and receive results from those calls, through a bridge that bridges the gap between the two environments.

@ReactMethod
  public void exampleMethod(String value) {

  }

Then, in the JavaScript code, it needs to be imported:


          
           
const { NativeModules } = require('react-native');

const ExampleModule = NativeModules.Example;

The module can then be used by calling its methods:


          
           
ExampleModule.exampleMethod("example value");
