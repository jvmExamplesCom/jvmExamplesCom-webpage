Java.util.AbstractCollection

An AbstractCollection is a Java utility class that simplifies a developer's task of creating and managing collections. This class provides basic implementations of most of the core collection interfaces. In this article, we will discuss various aspects of AbstractCollection in detail.

What is AbstractCollection?

An AbstractCollection is a utility class that is present in the java.util package. It provides basic implementations of the core Collection interfaces like List, Set, Map, and so on. It allows us to quickly build custom collections that are based on the existing collection interfaces.

How Does AbstractCollection Work?

An AbstractCollection provides several methods for adding, updating, removing and iterating over elements. These methods are abstract and must be implemented by concrete subclasses. Some of the most commonly used methods are as follows:
  • add(): adds an element to the collection.
  • remove(): removes an element from the collection.
  • contains(): checks if the collection contains the specified element.
  • iterator(): returns an iterator that can be used to iterate over the elements of the collection.
  • size(): returns the number of elements in the collection.

Example

Let us take an example of how we can use AbstractCollection to create a List. 

          
           
import java.util.AbstractCollection;
import java.util.ArrayList;
import java.util.Iterator;

public class MyList extends AbstractCollection {
    // the underlying data structure
    private ArrayList data = new ArrayList();

    // iterator method - required by AbstractCollection
    @Override
    public Iterator<Object> iterator() {
        return data.iterator();
    }

    // size method - required by AbstractCollection
    @Override
    public int size() {
        return data.size();
    }

    // add method
    public boolean add(Object e) {
        return data.add(e);
    }
}
In the above example, we defined a subclass of AbstractCollection called MyList. Note that we had to implement two methods (iterator() and size()) which are required by AbstractCollection. We also defined an add() method which allows us to add elements to the list. We can then create an instance of MyList and use it just like any other List. 

          
           
MyList myList = new MyList();
myList.add("foo");
myList.add("bar");
System.out.println(myList.size());
// Output: 2
