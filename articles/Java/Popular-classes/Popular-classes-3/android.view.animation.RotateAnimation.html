Android's RotateAnimation class enables developers to easily create rotation effects for their applications. The animation can be defined in XML or programmatically. It is a subclass of Animation and the parent class for any other rotational animation, such as Rotate3dAnimation and ScaleAnimation.

The "fromDegrees" and "toDegrees" properties set the starting and ending angle of the rotation. The "pivotX" and "pivotY" properties define the point at which the rotation will take place.     android:pivotY="50%" />

Creating a RotateAnimation Programmatically

You can also create a RotateAnimation programmatically. First you'll need to create an instance of the RotateAnimation class.
