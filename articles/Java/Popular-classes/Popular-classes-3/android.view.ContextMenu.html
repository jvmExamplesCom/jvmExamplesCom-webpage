Introduction to android.view.ContextMenu

android.view.ContextMenu is a class that provides a context menu for user interaction in Android applications. It allows the user to quickly access items in the application directly from the context menu, such as sending emails, making calls, opening websites, and more. The ContextMenu class provides a way to interact with data in the Android application by displaying a menu of available actions when the user performs certain actions on the data. The ContextMenu can also be used to display additional information related to the data in the form of contextual menus.

How to Create an Android ContextMenu

In order to create an Android ContextMenu, you will need to define a sub-class of the ContextMenu class. For example, the following code example shows how to create a new subclass of the ContextMenu class and add two menu items to it: 

          
           
public class MyContextMenu extends ContextMenu {

    public MyContextMenu() {
        super();
        addItem("Send Email");
        addItem("Open Website");
    }

}
You can then create an instance of your new ContextMenu class and attach it to a View object. To do this, you will need to call the View’s setOnLongClickListener method with a parameter of your newly created ContextMenu instance. For example, the following code snippet shows how to attach the ContextMenu instance to a button: 

          
           
Button myButton = findViewById(R.id.my_button); 
myButton.setOnLongClickListener(new MyContextMenu());
When a user presses and holds the button, the ContextMenu will appear on the activity’s screen.

How to Handle Menu Item Selection

The ContextMenu class provides a method called onMenuItemSelected that can be used to handle menu item selection. This method takes one parameter, the menu item that was selected. The following code example shows how to use the onMenuItemSelected method to handle the two menu items that were added to the ContextMenu instance: 

          
           
@Override
public boolean onMenuItemSelected(ContextMenuItem item) {
    
    if (item.getTitle().equals("Send Email")) {
        // Code to send email
    } else if (item.getTitle().equals("Open Website")) {
        // Code to open website
    }
    
    return true;
}
In this example, when the user selects an item from the menu, the onMenuItemSelected method is called and the code to perform the selected action is executed.
