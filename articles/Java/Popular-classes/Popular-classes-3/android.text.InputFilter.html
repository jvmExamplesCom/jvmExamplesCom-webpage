android.text.InputFilter

The android.text.InputFilter class is a powerful tool for filtering input into a TextView or EditText. It enables developers to control the input length, available character set, and text formatting.

What Does it Do?

InputFilter's main purpose is to filter the text that can be entered into a TextView or EditText. It can filter out invalid characters, limit the number of characters that can be entered, and ensure that only valid text is accepted.

How to Use an InputFilter

It is easy to use an InputFilter. All you need to do is create a new InputFilter and attach it to the TextView or EditText. Here is an example of creating an InputFilter that only allows letters and numbers.


          
           
InputFilter filter = new InputFilter() { 
    @Override 
    public CharSequence filter(CharSequence source, int start, 
      int end, Spanned dest, int dstart, int dend) { 
        if (source.equals("")) { 
            return source; 
        } 
        if (source.toString().matches("[a-zA-Z0-9 ]*")) { 
            return source; 
        }
        return ""; 
    }
};

editText.setFilters(new InputFilter[] {filter});

This example creates an InputFilter and attaches it to an EditText. Now, the EditText will only accept letters, numbers, and spaces. Any other character that is entered will be blocked.

Conclusion

The android.text.InputFilter class offers a powerful way to filter input into a TextView or EditText. It makes it easy to control the available character set, the input length, and the text format of an EditText. All you have to do is create a new InputFilter and attach it to the TextView or EditText.
