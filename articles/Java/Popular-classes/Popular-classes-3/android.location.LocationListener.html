Overview of Android.location.LocationListener

Android.location.LocationListener is an interface in the global namespace of the Android SDK that's part of the Location package. The interface is used to receive updates on the user's location based on the provider. It can provide information on the user's speed, altitude, and bearing as well as their longitude and latitude.
