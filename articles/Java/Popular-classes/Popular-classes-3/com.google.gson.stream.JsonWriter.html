What is com.google.gson.stream.JsonWriter

com.google.gson.stream.JsonWriter is a public API from Google's GSON library that enables writing JSON data to an output source in a streaming fashion. It provides an easy way of writing JSON data to any stream such as a file, network socket, or any other output source. In addition, it also supports features such as pretty printing, enabling the user to generate human-readable JSON output.

How to Use com.google.gson.stream.JsonWriter

The com.google.gson.stream.JsonWriter API is simple and easy to use. To use it, we need to first create an instance of the class: 

          
           
JsonWriter writer = new JsonWriter(outputStream);
Where 'outputStream' is the output stream to which the JSON data should be written. Now, to write the data, we use the following methods: 

          
           
writer.beginObject();
This method starts an object in the JSON output. Similarly, there is a method for beginning an array too: 

          
           
writer.beginArray();
We can then proceed to write the key-value pairs or element values for the JSON object/array: 

          
           
writer.name("key"); // Writes the key  
writer.value("value"); // Writes value corresponding to the key
Once all the necessary data has been written, the object/array must be closed using the 'endObject()' and 'endArray()' methods respectively. 

          
           
writer.endObject(); // Closes the object  
writer.endArray(); // Closes the array
And finally, close the writer when done: 

          
           
writer.close();

Example Usage

Let's consider an example where we are writing a JSON file containing details about a student. 

          
           
FileOutputStream fos = new FileOutputStream("student.json");  
JsonWriter writer = new JsonWriter(fos);  
writer.setIndent("    "); // Set indentation for better readability  
writer.beginObject();  
writer.name("name");  
writer.value("John Doe");  
writer.name("age");  
writer.value(22);  
writer.name("courses");  
writer.beginArray();  
writer.value("English");  
writer.value("Maths");  
writer.value("Science");  
writer.endArray();  
writer.endObject();  
writer.close();
This will generate the following output in the file 'student.json': 

          
           
{  
    "name": "John Doe",  
    "age": 22,  
    "courses": [  
        "English",  
        "Maths",  
        "Science"  
    ]  
}
