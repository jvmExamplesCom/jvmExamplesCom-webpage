Java.awt.geom.Point2D

Java.awt.geom.Point2D is a two-dimensional coordinate system used in Java programs. The Point2D class provides methods for setting, retrieving and manipulating coordinates located at a specified point. Through the use of this class, users can create shapes, angles and other graphics that can be manipulated with ease.

Constructors

The Point2D class contains multiple constructors that can be used to help create an instance of a Point2D object. These constructors include: 


          
           
Point2D() //creates a default Point2D object
Point2D(double x, double y) //creates a Point2D object with the specified coordinates
Point2D(Point2D p) //creates a Point2D object with the coordinates of the given Point2D object

Methods

The Point2D class contains several methods for accessing and manipulating the coordinates of a Point2D object. These methods include: 


          
           
double distance(double x, double y) //calculates the distance between this Point2D and the specified coordinates
double distance(Point2D pt) //calculates the distance between this Point2D and the specified Point2D
double getX() //returns the x coordinate of this Point2D
double getY() //returns the y coordinate of this Point2D
void setLocation(double x, double y) //sets the x and y coordinates of this Point2D

Examples

The following example demonstrates how to create and manipulate a Point2D object: 


          
           
// create a Point2D instance
Point2D p = new Point2D(1.0, 2.0);

// get the x and y coordinates of the Point2D
double x = p.getX();
double y = p.getY();
System.out.println("x: " + x + "; y: " + y); // prints: x: 1.0; y: 2.0

// calculate the distance between the Point2D and (3.0, 4.0)
double dist = p.distance(3.0, 4.0);
System.out.println("Distance: " + dist); // prints: Distance: 2.8284271247461903

p.setLocation(4.0, 5.0);
x = p.getX();
y = p.getY();
