Java org.osgi.framework.BundleContext

The Java org.osgi.framework.BundleContext is an interface that provides access to the current OSGi framework instance. It is available to all activated bundles in an OSGi container, and provides access to any necessary framework resources.

A BundleContext object can be obtained from the framework by implementing a BundleActivator class and overriding its start() method. The following example illustrates how to acquire a BundleContext object:


          
           
public class MyActivator implements BundleActivator {

    private BundleContext context;

    public void start(BundleContext context) throws Exception {
        this.context = context;
        // use context here
    }

}

Bundle bundle = context.installBundle("http://example.com/mybundle.jar");
