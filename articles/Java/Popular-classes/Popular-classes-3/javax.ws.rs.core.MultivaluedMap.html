javax.ws.rs.core.MultivaluedMap

The javax.ws.rs.core.MultivaluedMap is part of the Java API for RESTful web services (JAX-RS). It is a data structure that uses a string key to map to multiple values. It is similar to a Map interface, except that its methods return a List of values rather than a single value.

Creating a MultivaluedMap

The MultivaluedMap interface does not have a public constructor, so it must be created using one of its implementing classes. The most commonly used implementation is the LinkedHashMultiValuedMap:


          
           
LinkedHashMultiValuedMap<String, String> myMap = new LinkedHashMultiValuedMap<>();

This will create an empty MultivaluedMap ready to use.

Putting Values into a MultivaluedMap

Values can be added to the MultivaluedMap using the put() method. This takes two arguments - a string key and a value:


          
           
myMap.put("color", "blue");

If the same key is used more than once, the values will be stored in a list:


          
           
myMap.put("color", "red");
myMap.put("color", "green");

Now the key “color” will map to a list containing the values “blue”, “red”, and “green”.

Getting Values From a MultivaluedMap

Values can be retrieved from the MultivaluedMap using the get() method. This takes a single argument - a string key - and returns a list of values associated with that key:


          
           
List<String> colors = myMap.get("color");

In this example, the list “colors” will contain the values “blue”, “red”, and “green”.

Removing Values From a MultivaluedMap

Values can be removed from the MultivaluedMap using the remove() method. This takes two arguments - a string key and a value to remove:


          
           
myMap.remove("color", "green");

This will remove the value “green” from the list associated with the key “color”.
