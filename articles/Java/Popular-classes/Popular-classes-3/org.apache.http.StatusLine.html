org.apache.http.StatusLine

The org.apache.http.StatusLine class is a component of the HttpClient API that provides basic information about a HTTP response from the server, such as status code, protocol version and reason phrase.

Constructor

The StatusLine class has one constructor which takes three parameters:


          
           
public StatusLine(ProtocolVersion ver, int code, String reason)
Where :
  • ver is the protocol version used in the response.
  • code is the 3 digit status code of the response.
  • reason is the optional reason phrase provided in the response.

Methods

The StatusLine class has several methods for retrieving response information:

  • getProtocolVersion() returns the ProtocolVersion of the response.
  • getStatusCode() returns the 3 digit status code of the response.
  • getReasonPhrase() returns the reason phrase of the response.
  • toString() returns the String representation of the response.

Example

Let's take a look at an example using the StatusLine class. We'll start by creating a new HttpClient and sending a GET request to a server.


          
           
HttpClient httpclient = new DefaultHttpClient();
HttpGet httpget = new HttpGet("http://www.example.com/");
HttpResponse response = httpclient.execute(httpget);

We can then use the StatusLine to retrieve some basic information about the response:


          
           
StatusLine status = response.getStatusLine();
int code = status.getStatusCode();
String phrase = status.getReasonPhrase();
System.out.println(code + " " + phrase);
