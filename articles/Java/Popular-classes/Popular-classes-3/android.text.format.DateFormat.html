android.text.format.DateFormat

Android's DateFormat is a class for formatting and parsing dates in a locale-sensitive manner. DateFormat helps you to format and parse dates for any locale. DateFormat is an abstract class for date/time formatting subclasses which allow for formatting (i.e., date -> text), parsing (text -> date), and normalization. It provides many class methods for obtaining default date/time formatters based on the default or a given locale and a number of formatting styles.

Creating a DateFormat Instance

To create a DateFormat instance, you can use the static factory methods on the DateFormat class such as: 

          
           
DateFormat df = DateFormat.getDateInstance();
The getDateInstance() method returns a DateFormat object that can be used to format and parse a date in a locale-sensitive manner. The other factory methods available on DateFormat are getTimeInstance(), getDateTimeInstance(), and getInstance().

Formatting Dates

To format a Date object using a DateFormat instance, you can use the format() method like this: 

          
           
String str = df.format(date);
The format() method will return a String representation of the given Date object in the specified format.

Parsing Dates

To parse a date string using a DateFormat instance, you can use the parse() method like this: 

          
           
Date date = df.parse(str);
The parse() method will return a Date object from the given date string in the specified format.
