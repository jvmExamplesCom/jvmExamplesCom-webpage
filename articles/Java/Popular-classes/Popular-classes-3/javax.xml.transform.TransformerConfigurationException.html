javax.xml.transform.TransformerConfigurationException

javax.xml.transform.TransformerConfigurationException is an unchecked exception that goes hand in hand with a failure of initialization for a TransformerFactory, Transformer, or Templates object. This exception is thrown by methods that process XSLT documents when a problem is encountered with the configuration of the transformer factory.
