javax.swing.tree.DefaultMutableTreeNode

The 

          
           
javax.swing.tree.DefaultMutableTreeNode
class is a basic building block in the JTree component. It represents a single node in a tree data structure and can be used to construct a tree structure. This class provides methods for adding, inserting, deleting and getting child nodes, along with various getters and setters for accessing various properties of the node.

Creating a Simple Tree Node

Creating a new 

          
           
DefaultMutableTreeNode
is a simple matter of passing a string to its constructor: 

          
           
DefaultMutableTreeNode rootNode = new DefaultMutableTreeNode("Root");
The string passed in is used as the node's display name.

Adding and Accessing Child Nodes

To add a child node to an existing node, call the 

          
           
add()
method: 

          
           
DefaultMutableTreeNode child1 = new DefaultMutableTreeNode("Child 1");
rootNode.add(child1);
The 

          
           
add()
method takes a 

          
           
DefaultMutableTreeNode
as its only parameter, so it's easy to add any type of node. To access the child node, there are several different methods, such as 

          
           
getChildAt()
, which takes an index argument and returns the node at that index in the list of children: 

          
           
DefaultMutableTreeNode childAtIndexOne = rootNode.getChildAt(1);

getChildByName()

DefaultMutableTreeNode childWithNameChild1 = rootNode.getChildByName("Child 1");

Inserting and Deleting Nodes

In addition to adding nodes, you can also insert nodes at a specific index in the node's list of children, using the 

          
           
insert()
method: 

          
           
DefaultMutableTreeNode child2 = new DefaultMutableTreeNode("Child 2");
rootNode.insert(child2, 1);
This will insert the node at the specified index. If you want to remove a node, use the
