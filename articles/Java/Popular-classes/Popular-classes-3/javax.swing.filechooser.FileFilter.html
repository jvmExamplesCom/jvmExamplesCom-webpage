javax.swing.filechooser.FileFilter

The javax.swing.filechooser.FileFilter class is a useful utility class which provides support for file filtering in the use of file choosers. With this class, developers can easily filter the types of files presented in file choosers.

Let’s take a look at how this class works with a simple example. Here we have a basic Swing application with a file chooser:


          
           
import javax.swing.*;

public class FileFilterExample extends JFrame {

    public FileFilterExample() {
        JFileChooser chooser = new JFileChooser();
        chooser.setCurrentDirectory(new File("."));
        int retval = chooser.showOpenDialog(this);        
    }

    public static void main(String[] args) {
        SwingUtilities.invokeLater(new Runnable() {
            public void run() {
                FileFilterExample frame = new FileFilterExample();
            }
        });
    }
}

This will open a dialog that shows all the files and directories in the current directory. We want to add a filter so that only certain file types are shown in the dialog. To do this, we need to create an instance of the FileFilter class and define its accept() method. This method will be called by the chooser when it is determining whether to display a particular file.


          
           
FileFilter filter = new FileFilter() {
    @Override
    public boolean accept(File pathname) {
        String name = pathname.getName();
        return name.endsWith(".txt");
    }
};

This filter will only display files that end with “.txt”. Now we need to set the filter on the chooser:


          
           
chooser.setFileFilter(filter);

Now, when the chooser is displayed, only files with the “.txt” extension will be shown. This is just a simple example of what the FileFilter class can do. It can also be used to provide additional information about the files being shown (for example, their size or the last time they were modified).

Conclusion

This article has demonstrated how the javax.swing.filechooser.FileFilter class can be used to filter the types of files that are shown in a file chooser. This can be useful for limiting the user’s choices and improving the usability of your application.
