Java.security.InvalidParameterException

The 

          
           
java.security.InvalidParameterException
is a type of security exception that is thrown when an invalid parameter is passed to a method or constructor. It is a subclass of the 

          
           
java.security.GeneralSecurityException
.

Example

Consider the following example, where an invalid key is passed to the 

          
           
AESCipher.init()
method. 

          
           
try {
    SecretKeySpec key = new SecretKeySpec("invalid key".getBytes(), "AES");
    AESCipher cipher = new AESCipher();
    cipher.init(key);
} catch(InvalidParameterException e) {
    System.out.println("Invalid key: " + e.getMessage());
}
In this example, the 

          
           
AESCipher.init()
method throws an 

          
           
InvalidParameterException
when it sees that the key passed to it is invalid. The exception is caught, and the error message is printed out.

Usage

java.security.InvalidParameterException
class is often used to catch exceptions thrown by methods or constructors that take invalid parameters.
