Exploring Google’s com.google.common.collect.Collections2 Class

Collections2.transform()

Function

F

Collections2.transform()

List<String> myList = Arrays.asList("foo", "bar", "baz");

// Create a function to uppercase each element in the list
Function<String, String> toUpperCase = new Function<String, String>() {
    public String apply(String input) {

Collections2.filter()
