com.sun.jna.Platform Overview

The com.sun.jna.Platform class provides features for identifying the underlying operating system and architecture of the platform it is currently running on. It can be used to provide information about current system status, such as whether a 64-bit JVM is available, as well as to provide access to native system libraries.

In addition, it is also possible to detect the given platform's current language, timezone, and environment settings. This can be useful when developing applications that need to respect user preferences and regional nuances.

Platform.getPlatform()
as follows: 

          
           
Platform platform = Platform.getPlatform();

The Platform class can be used to detect the underlying operating system, as well as the processor architecture the JVM is running on. The 

os

arch
