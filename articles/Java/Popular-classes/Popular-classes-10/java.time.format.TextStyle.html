Java.time.format.TextStyle

Java.time.format.TextStyle is a part of the Java 8 java.time package, which provides access to powerful formatting capabilities for dates, times, and intervals. The Java.time.format.TextStyle class is a simple class that provides some methods for formatting text based on the current locale of the application.
