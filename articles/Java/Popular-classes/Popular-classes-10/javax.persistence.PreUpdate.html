javax.persistence.PreUpdate

The javax.persistence.PreUpdate annotation is used for defining callback method for the EntityListener class which is invoked before an entity is updated. It is used to determine what action should be taken before an update operation is carried out.

Usage Example

The following example illustrates a callback method that is executed before an entity is updated:


          
           
@Entity
public class Task {

    ...

    @PreUpdate
    public void preUpdate() {
        // Do some processing before an entity is updated.
    }
}

The callback method is always called before the update operation is performed. In this example, the code inside the method will be executed before an entity of type Task is updated.

The preUpdate callback method is invoked after the entity is retrieved from the database and changes are made to it. The output of the method invocation is returned as follows:
