javax.mail.util.ByteArrayDataSource

The 

          
           
javax.mail.util.ByteArrayDataSource
is a DataSource implementation that wraps an existing byte array and provides InputStreams for reading the bytes. It can also be used as an OutputStream for writing the data.

Creating a ByteArrayDataSource

To create a ByteArrayDataSource, you need to provide both a byte array and the content type. The content type may be specified as a parameter to the constructor.


          
           
byte[] data = ...;
String contentType = ...;
ByteArrayDataSource bads = new ByteArrayDataSource(data, contentType);

Reading from a ByteArrayDataSource is straightforward. You simply call the 

getInputStream()
method to obtain an InputStream to read the data.

// read from the InputStream here

Writing To a ByteArrayDataSource

Writing to a ByteArrayDataSource is even easier. You simply call the 

getOutputStream()
method to obtain an OutputStream to write the data.
