org.antlr.runtime.RecognitionException

org.antlr.runtime.RecognitionException is an exception that is thrown when a parser-recognized rule fails. This means that the input cannot be parsed using the grammar given and it's not possible to continue parsing. It is a checked exception and must be handled in the code.

In this example, we will use the ANTLR parser to process the following grammar rule:


          
           
A: 'Hello' B;
B: 'World';

MyGrammarParser parser = new MyGrammarParser(tokens);
try {
   parser.a();
} catch (RecognitionException e) {
   // handle exceptions
}
