Introduction to io.opentracing.propagation.Format

io.opentracing.propagation.Format is a class in the OpenTracing Java API that provides a set of classes and interfaces for propagating and extracting trace context across process boundaries.

It is used to provide a uniform way to encode/decode remote calls in distributed tracing across different services and different protocols.
