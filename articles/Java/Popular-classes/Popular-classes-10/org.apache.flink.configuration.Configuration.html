What is org.apache.flink.configuration.Configuration?

The org.apache.flink.configuration.Configuration class is part of the org.apache.flink package. It provides access to Flink’s configurations and allows users to set parameters for their applications. In other words, it allows developers to define static configurations that can be used to control how their Flink application runs.

import org.apache.flink.configuration.Configuration;

Configuration conf = new Configuration();
