java.lang.management.ClassLoadingMXBean

Java.lang.management.ClassLoadingMXBean is an interface in the Java Platform Management Extensions (JMX) API. It provides the properties and operations of a class loading system that is used by the Java virtual machine (JVM). It gives the information about the current state of the class loading system.

It is used to monitor and manage the loading of Java classes into the JVM. This includes the number of classes loaded, unloaded, and the total bytes loaded. This MXBean also provides operations for garbage collection of classes.

Properties

The following properties are available for the ClassLoadingMXBean:

  • Verbose: This is a boolean property that specifies whether the JVM will output additional debugging information as classes are loaded and unloaded. The default value is false.
  • TotalLoadedClassCount: This is an integer property that specifies the total number of classes that have been loaded since the JVM started.
  • UnloadedClassCount: This is an integer property that specifies the total number of classes that have been unloaded since the JVM started.
  • LoadedClassSize: This is a long property that specifies the total size in bytes of all the classes that have been loaded since the JVM started.

Operations

The following operations are available for the ClassLoadingMXBean:

  • gc(): This operation performs a garbage collection of all classes that are no longer being used. It returns a boolean value which is true if the garbage collection was successful, or false otherwise.
  • findClass(String name): This operation attempts to locate a class with the specified fully qualified class name. It returns the Class object for the specified class, or null if it could not be located.
