java.lang.reflect.GenericDeclaration

The java.lang.reflect.GenericDeclaration is an interface that represents the declaration of a generic type, as found in classes and interfaces. It provides methods for getting information about the declaration such as its modifiers, name, and type parameters.

The GenericDeclaration interface has several methods which can be used to get information about the generic type declaration.
