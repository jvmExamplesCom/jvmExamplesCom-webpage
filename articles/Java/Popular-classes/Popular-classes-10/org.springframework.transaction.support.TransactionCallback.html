The 

          
           
org.springframework.transaction.support.TransactionCallback

This callback object is usually used when the methods or code blocks need to participate in a transaction, for example for database operations. The TransactionCallback object provides a callback method that can be called from within a transactional context. The callback can receive a parameter from the transaction manager and then perform any actions required before returning a result.
