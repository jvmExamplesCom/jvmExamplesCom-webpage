Android.view.ViewGroup.MarginLayoutParams

Android.view.ViewGroup.MarginLayoutParams is an abstract class of LayoutParams used by ViewGroups to define the margins of their children. It is responsible for controlling the layout parameters of its associated View. This class provides a variety of ways to describe the width and height of the view, as well as the margins involved in the relationship between the View and its parent.

Constructors

Android.view.ViewGroup.MarginLayoutParams provides two different constructors with varying parameters: 

          
           
// Constructor with a width and height 
public MarginLayoutParams(int width, int height)

// Constructor with a specified set of layout parameters
public MarginLayoutParams(MarginLayoutParams source)

Methods

In addition to the constructors, Android.view.ViewGroup.MarginLayoutParams also provides several methods for setting and retrieving the width and height, as well as the margins between the parent ViewGroup and the child View: 

          
           
leftMargin()
Retrieves the left margin of the View. 

          
           
rightMargin()
Retrieves the right margin of the View. 

          
           
topMargin()
Retrieves the top margin of the View. 

          
           
bottomMargin()
Retrieves the bottom margin of the View. 

          
           
setMargins(int left, int top, int right, int bottom)
Sets the margins of the View. 

          
           
getLayoutDirection()
Retrieves the layout direction associated with this View. 

          
           
isLayoutRtl()
Indicates whether or not the layout is right-to-left.

Example Code

Below is an example of how a ViewGroup can use the MarginLayoutParams to define the layout parameters of a child View. 

          
           
final MarginLayoutParams params = new MarginLayoutParams(200, 100);
params.setMargins(20, 10, 30, 40);
viewGroup.addView(childView, params);
In this example, a ViewGroup called "viewGroup" is being initialized with a MarginLayoutParams object. The parameters for the object specify a width of 200px, a height of 100px, and margins of 20px on the left, 10px on the top, 30px on the right, and 40px on the bottom. The childView is then added to the ViewGroup using the initialized MarginLayoutParams object.
