com.facebook.FacebookException

The com.facebook.FacebookException is an exception class used for Facebook authentication errors. This class provides properties and static methods for dealing with errors encountered in the authentication process of Facebook. The exception class also provides base methods for formatting printed output and for returning readable messages for certain types of error.

Example 1: Basic Use of FacebookException

This example shows basic use of the FacebookException class and how it is used to catch and handle errors in the authentication process.


          
           
try {
	// Initialize Facebook instance
	Facebook facebook = new Facebook();
	
	// Authenticate user with Facebook
	facebook.authenticate();

} catch (FacebookException fe) {
	// Print error message
	System.err.print("Error: " + fe.getMessage());
}

Example 2: Checking Error Codes with FacebookException Method getErrorCode()

This example shows how to check error codes when dealing with a FacebookException object.


          
           
try {
	// Initialize Facebook instance
	Facebook facebook = new Facebook();

	// Authenticate user with Facebook
	facebook.authenticate();

} catch (FacebookException fe) {
	// Check error code
	if (fe.getErrorCode() == 1337) {
		// Print error message
		System.err.print("Error: Invalid login credentials");
	}
}

This example shows how to use multiple catch clauses in a single try block to catch multiple exceptions.


          
           
try {
	// Initialize Facebook instance
	Facebook facebook = new Facebook();

	// Authenticate user with Facebook
	facebook.authenticate();

} catch (IOException ioe) {
	// Handle IOException
	System.err.print("Error: " + ioe.getMessage());

} catch (FacebookException fe) {
	// Handle FacebookException
