org.apache.commons.net.ftp.FTPFile

The org.apache.commons.net.ftp.FTPFile class provides access to the data stored in an FTP file. It is used to read and parse the file system stored on an FTP server as well as to modify the server structure.

FTPFile ftpFile = new FTPFile();
Once you have an instance, you can call the connect() method to establish a connection with the FTP server: 

          
           
ftpFile.connect(hostname, port);
You can also specify the username and password of the FTP account using the setUsername() and setPassword() methods.

Reading data from an FTP file

Once you have established a connection, you can use the list() method to get a list of all the files in the current directory. The method returns an array of FTPFile objects, one for each file. 

          
           
FTPFile[] files = ftpFile.list();
