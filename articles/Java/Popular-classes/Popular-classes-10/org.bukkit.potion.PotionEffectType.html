org.bukkit.potion.PotionEffectType

The org.bukkit.potion.PotionEffectType class is a type of effect that can be applied to an entity through the use of potions. It defines all the possible effects that can be applied, and provides methods for manipulating them. This class is part of the Bukkit API.

Constructors


          
           
public PotionEffectType(String name)
