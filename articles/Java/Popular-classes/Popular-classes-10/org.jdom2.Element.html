org.jdom2.Element

The org.jdom2.Element class is the base class for all XML elements in the JDOM2 library. It provides a powerful interface to create and manipulate elements, as well as various methods to access and modify attributes of the element.

Creating Elements

Elements can be created by invoking the Element constructor. For example:


          
           
Element element = new Element("name");

This will create an element named "name". The constructor also accepts an additional argument of type String that specifies the element's namespace.

To set an attribute of the element, use the setAttribute() method. For example:


          
           
element.setAttribute("attr", "val");

will set the "attr" attribute to the value of "val". Note that the attribute must be a valid XML attribute name.

Adding Content

Content can be added to an element using the addContent() method. This accepts an array of content types, including strings, elements, comments and more.

For example, to add a new element as a child of the current element:


          
           
Element childElement = new Element("childElement");
element.addContent(childElement);

Modifying Content

Existing content in an element can be modified using the setText() and setAttribute() methods, or removed using the removeContent() method. For example:


          
           
element.setText("text");
element.setAttribute("attr2", "val2");
element.removeContent();

Traversing Elements

Elements can be traversed using the getChildren() and getParent() methods. These methods return a list of Elements or a single Element respectively, which can then be used to navigate the tree structure.

For example, to get a list of all child elements of the current element:


          
           
List<Element> childrenElements = element.getChildren();

One can also traverse up the hierarchy by calling the getParent() method:


          
           
Element parentElement = element.getParent();

Conclusion

The org.jdom2.Element class is a powerful and easy-to-use interface for manipulating XML elements. With it, one can create and modify elements, as well as traverse their tree structure.
