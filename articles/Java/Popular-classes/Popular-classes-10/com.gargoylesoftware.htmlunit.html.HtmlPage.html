Com.gargoylesoftware.htmlunit.html.HtmlPage

Com.gargoylesoftware.htmlunit.html.HtmlPage is an object representing the current page of a website parsed from HTML code. It extends from the abstract class HtmlPage, which provides methods that allow for manipulating the page and its contents.

Creating an Instance of HtmlPage

An instance of HtmlPage can be created by calling the static method 

parse
with the HTML code as argument: 

          
           
HtmlPage page = HtmlPage.parse(htmlCode);

Methods of HtmlPage

The most important method of this class is the 

          
           
getElementsByTagName
method, which allows for retrieving all elements in the page with a given tag name. For example: 

          
           
List<HtmlElement> allDivs = page.getElementsByTagName("div");
The returned list contains all
elements in the page. The individual elements in the list are represented as objects of type 
HtmlElement
. Other useful methods include 
getElementById
, which returns the element with a certain ID attribute; 
getForms
, which returns all form elements in the page; and 
getTextContent
, which returns the text content of the page.

Example Usage of HtmlPage

Here is an example of how to use 
HtmlPage
in practice: 

          
           
String htmlCode = "<html><body><div>This is a div</div></body></html>";
HtmlPage page = HtmlPage.parse(htmlCode);
HtmlElement divElement = page.getElementsByTagName("div").get(0);
System.out.println(divElement.getTextContent()); // prints "This is a div"
