Java.awt.geom.NoninvertibleTransformException

The java.awt.geom.NoninvertibleTransformException is a checked exception that is thrown when an operation is applied to a non-invertible affine transformation. It is a subclass of the java.awt.geom.TransformException. Here's an example of code that creates a non-invertible Affine Transformation:


          
           
AffineTransform at = new AffineTransform(0, 0, 0, 0, 0, 0);
try {
    at.createInverse();
} catch (NoninvertibleTransformException e) {
    System.out.println("Cannot create an inverse transform!");
}
