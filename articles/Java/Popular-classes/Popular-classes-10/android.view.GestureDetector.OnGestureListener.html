android.view.GestureDetector.OnGestureListener

android.view.GestureDetector.OnGestureListener is an interface that allows you to detect gestures in an Android application. It is part of the Android SDK and provides an easy way to detect common user interface gestures, such as double taps and swipes. This interface contains four methods that are called when a particular gesture is detected. The four methods are:
  • onDown(): This is called whenever the user touches down on the screen.
  • onLongPress(): This is called whenever the user presses down and holds the screen for a period of time.
  • onScroll(): This is called whenever the user scrolls across the screen.
  • onFling(): This is called whenever the user quickly flicks their finger across the screen.
