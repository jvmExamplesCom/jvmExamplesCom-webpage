org.apache.hadoop.mapred.FileSplit

The org.apache.hadoop.mapred.FileSplit class is a utility that helps read files stored on Hadoop’s HDFS in an efficient manner. It breaks files into large ‘splits’ and determines which parts of the files are handled by particular map tasks. It also provides methods to retrieve the file’s path, start and length positions, and its content.

Constructors

FileSplit offers two constructors for creating objects.


          
           
FileSplit()

This constructor creates an empty FileSplit object.


          
           
FileSplit(Path file, long start, long length, String[] hosts)

This constructor accepts four arguments and uses them to populate the FileSplit object.

  • file: The HDFS file path of the split.
  • start: The starting byte position of the split.
  • length: The length of the split.
  • hosts: An array containing the hostnames of the nodes storing portions of the split.

Methods

FileSplit offers several methods for retrieving and manipulating data.


          
           
public Path getPath()

This method returns the HDFS path of the given split.


          
           
public long getStart()

This method returns the starting byte position of the split.


          
           
public long getLength()
This method returns the length of the split. 

          
           
public String[] getLocations()

This method returns an array containing the hostnames of the nodes storing portions of the split.


          
           
public InputStream createInputStream(Configuration conf) throws IOException

This method creates an InputStream from the file split. It takes a Configuration object as an argument and returns the stream with the contents of the file split.

Example Usage

Let's take a look at a code sample to see how we might use FileSplit in practice.


          
           
Path inputPath = new Path("/input/myfile.txt");
FileSplit split = new FileSplit(inputPath, 0, 1024, null);
InputStream in = split.createInputStream(conf);
int ch = in.read();
System.out.println(ch); //prints the first byte of the given split

In the example above, we created a new FileSplit that reads the first 1024 bytes of the ‘myfile.txt’ file stored in the ‘/input’ folder of HDFS. We then created an InputStream from the FileSplit and used it to read the first byte of the file.
