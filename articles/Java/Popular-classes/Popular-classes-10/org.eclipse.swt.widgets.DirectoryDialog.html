org.eclipse.swt.widgets.DirectoryDialog

It is a Java SWT API file class used for displaying a dialog that allows the user to select a directory. This class lets the user navigate through the file system and select a directory, which will then be opened. It also provides access to the currently selected directory. This class is part of the org.eclipse.swt.widgets package.

DirectoryDialog
object by calling its constructor and passing it the Shell parent it should use. For example:

DirectoryDialog dirDialog = new DirectoryDialog(parentShell);

Once you have created the 
DirectoryDialog
object, you can set the path to the folder that should be initially selected when the dialog is opened. You can also set a dialog title to be displayed at the top of the dialog window. For example: 

          
           
String initialPath = "C:/Users/username/Documents";

dirDialog.setText(dialogTitle);

String selectedDir = dirDialog.open();

Getting the Selected Directory
