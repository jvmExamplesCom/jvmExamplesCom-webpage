The XYLineAndShapeRenderer class is a specialized renderer used to draw lines and shapes on a XYPlot. In addition, it supports customization of line and shape styles, as well as support for multiple data series.

Creating an XYLineAndShapeRenderer

Creating an XYLineAndShapeRenderer is straightforward. The following code creates a new instance of the renderer:


          
           
XYLineAndShapeRenderer renderer = new XYLineAndShapeRenderer();

The XYLineAndShapeRenderer class provides a range of methods for setting line and shape styles. For example, the following code sets the line and shape styles for a series index 0:
