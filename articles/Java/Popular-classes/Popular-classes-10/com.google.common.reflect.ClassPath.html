Overview of the ClassPath Class

ClassPath cp = ClassPath.from(Thread.currentThread().getContextClassLoader());

Finding All Classes in an Application

getAllClasses()

ClassPath cp = ClassPath.from(Thread.currentThread().getContextClassLoader());
