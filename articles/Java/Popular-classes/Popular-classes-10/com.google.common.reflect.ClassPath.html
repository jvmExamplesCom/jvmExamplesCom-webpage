Overview of the ClassPath Class

com.google.common.reflect.ClassPath

To use the ClassPath class, you must first create an instance of it with the following code: 

          
           
ClassPath cp = ClassPath.from(Thread.currentThread().getContextClassLoader());

Examples of Using the ClassPath Class

ClassPath

Finding All Classes in an Application

The 

          
           
getAllClasses()
method can be used to retrieve a 

          
           
Set<ClassInfo>
of all the classes in an application. This method is useful when you need to know all the classes that are available. For example: 

          
           
ClassPath cp = ClassPath.from(Thread.currentThread().getContextClassLoader());
Set<ClassInfo> allClasses = cp.getAllClasses();
for (ClassInfo ci : allClasses) {

}
