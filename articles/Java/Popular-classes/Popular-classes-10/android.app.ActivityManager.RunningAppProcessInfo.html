android.app.ActivityManager.RunningAppProcessInfo

Android's ActivityManager is a tool for managing the processes running on an Android device. It performs various tasks such as monitoring the running processes and managing their lifecycle. The android.app.ActivityManager.RunningAppProcessInfo class provides information about a running application process.

Constructors

public RunningAppProcessInfo (String processName, int pid, String[] pkgList)

Methods

public String getProcessName ()

public String [] getPkgList ()

public void setPid (int pid)

public void setProcessName (String processName)

public void setPkgList (String [] pkgList)

Examples

Here is an example of creating a new RunningAppProcessInfo instance and printing its properties:


          
           
ActivityManager.RunningAppProcessInfo runningAppProcessInfo = new ActivityManager.RunningAppProcessInfo();
runningAppProcessInfo.setProcessName("ExampleProcess");
runningAppProcessInfo.setPid(42);
runningAppProcessInfo.setPkgList(new String[]{"com.example.app1", "com.example.app2"});
