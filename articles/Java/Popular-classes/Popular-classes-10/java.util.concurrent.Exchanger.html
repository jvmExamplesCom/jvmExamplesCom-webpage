java.util.concurrent.Exchanger

The java.util.concurrent.Exchanger class provides an interface for exchanging objects between two threads in a thread-safe manner. It is used to create a communication channel between two threads and exchange the data between them. It enables two threads that synchronize their actions based on the exchanged data. The Exchanger class is part of the Java Concurrency package and is available since Java 5.

  • exchange(V x)
