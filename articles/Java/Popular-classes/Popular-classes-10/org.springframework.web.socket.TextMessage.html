Introduction to org.springframework.web.socket.TextMessage

How Does It Work?

We need a server that supports the Web Socket Protocol and a client who will connect to the server. In order for messages to be exchanged between the two parties, a suitable protocol must be agreed upon. Once a connection between the server and the client is established, the client can send messages to the server in whatever format they choose.
