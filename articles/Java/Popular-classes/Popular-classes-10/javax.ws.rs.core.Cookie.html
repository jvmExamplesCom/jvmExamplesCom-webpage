The Cookie class is part of the Java API for RESTful Web Services (JAX-RS), which provides support for the development of web services using the Representational State Transfer (REST) architectural pattern. With a Cookie object, you can access information stored within it, such as its name and value. You can also set or retrieve additional settings such as the domain, secure flag, maximum age, version, and so on.

Cookie cookie = new Cookie();
