The LookupElementBuilder class is a useful utility class that facilitates creating and configuring LookupElements, which are used with IntelliJ's code completion feature. LookupElements are the primary way of defining the list of options that appear when the user starts typing something (e.g. variable or method name).

Creating a LookupElement

LookupElements are created by calling one of the static factories methods provided by LookupElementBuilder. The simplest such method is 

create()
, which takes a 
String
argument and returns a LookupElement for it. 

          
           
LookupElement element = LookupElementBuilder.create("myString");

Configuring a LookupElement

with*

withIcon()

A wide range of configuration options are available to tweak the LookupElement's display and behavior. These include setting the type of the LookupElement, adding type information, setting an icon, specifying the sort order, setting a tooltip, and much more.

Reusing Configuration
