The 

          
           
com.intellij.psi.PsiDocumentManager

commitDocument(Document)

  • 
            
             
getCachedPsiFiles(VirtualFile)

  • getCachedPsiFiles()
    - returns all cached PsiFile instances.

    - updates the contents of the given document from the corresponding PsiFile instance.
