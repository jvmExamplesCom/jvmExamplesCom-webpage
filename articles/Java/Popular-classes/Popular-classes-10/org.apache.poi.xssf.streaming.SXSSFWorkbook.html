org.apache.poi.xssf.streaming.SXSSFWorkbook

The SXSSF workbook class is a streaming version of the XSSF workbook that allows large numbers of rows to be written without consuming huge amounts of memory. SXSSF works by pushing out window-sized row groups to temp files and then loading them back into memory at the same time. This allows very large spreadsheets to be created and manipulated without using massive amounts of memory.

Creating a SXSSF Workbook

The SXSSF workbook class can be instantiated like any other workbook:


          
           
SXSSFWorkbook wb = new SXSSFWorkbook();

Writing Data to the Workbook

Once a Workbook has been instantiated, it can be used to create Sheets and write data like a standard Workbook. For example, the following code creates a Sheet and writes some data to it:


          
           
Sheet sheet = wb.createSheet();
Row row = sheet.createRow(0);
Cell cell = row.createCell(0);
cell.setCellValue("Hello World!");

Writing Large Data Sets

When you have a large data set, you should use the SXSSFWorkbook's writeRows() method to quickly write the data to the sheet. This method takes an iterator containing the data and will write it to the sheet as quickly as possible. For example:


          
           
List<String> data = Arrays.asList("a", "b", "c");
sheet.writeRows(data.iterator());

Saving the Workbook

Once all the data has been added to the Workbook, it can be saved like any other Workbook:


          
           
FileOutputStream fos = new FileOutputStream("example.xlsx");
wb.write(fos);
fos.close();

Flushing Unused Data

Since the SXSSFWorkbook uses temporary files to store data, you should call the flushUnusedData() method when you are done using the Workbook. This will delete the temporary files and free up any used disk space. For example:


          
           
wb.flushUnusedData();
