javax.mail.internet.MimeUtility class provides methods for encoding and decoding MIME (Multipurpose Internet Mail Extensions) headers. It parses the input string into its constituent tokens and encodes or decodes them as needed. MIME header strings are used in email messages to identify their content type, encoding, etc.

public static String encodeWord(String s) throws UnsupportedEncodingException
