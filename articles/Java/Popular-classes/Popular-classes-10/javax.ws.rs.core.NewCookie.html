javax.ws.rs.core.NewCookie

The javax.ws.rs.core.NewCookie class is one of the important classes under Java API for RESTful Web Services (JAX-RS) 2.0 to create new cookie object. JAX-RS is a set of Java APIs that simplifies the development of web services using the Java programming language.

Methods

The javax.ws.rs.core.NewCookie class provides several methods to work with the newly created cookie object:

  • NewCookie(String name, String value) – creates a NewCookie object with given name and value.
  • NewCookie(String name, String value, String path, String domain, String comment, int maxAge, boolean secure) – Creates a NewCookie object with given name, value, path, domain, comment, maxAge and a boolean secure flag.
  • getComment() – Returns the comment described in the cookie.
  • getDomain() – Returns the domain described in the cookie.
  • getMaxAge() – Returns the maxAge described in the cookie.
  • getName() – Returns the name described in the cookie.
  • getPath() – Returns the path described in the cookie.
  • getSecure() – Returns a boolean flag indicating whether the cookie should be sent over secure connections only.
  • getValue() – Returns the value described in the cookie.
  • toString() – Returns the string representation of the cookie.

Example

The following example shows how to use the javax.ws.rs.core.NewCookie class to create a new cookie object:


          
           
String name = "sessionID";
String value = "12345";
String path = "/";
String domain = "example.com";
String comment = "session cookie";
int maxAge = 60*60;
boolean secure = true;

NewCookie myCookie = new NewCookie(name, value, path, domain, comment, maxAge, secure);
