Overview of org.apache.hadoop.hbase.client.Table

The org.apache.hadoop.hbase.client.Table is part of the HBase Java API, which allows users to perform various table-level operations such as creating tables, dropping tables, and performing scans and puts. This class provides methods for getting an instance of a Table, setting the read/writeTimeout, and enabling/disabling table operations. This article will provide an overview of the org.apache.hadoop.hbase.client.Table class, as well as code examples demonstrating how to use its various methods.

Methods

The following methods are available in org.apache.hadoop.hbase.client.Table:

  • getTableName(): Returns the name of this Table instance.
  • getConfiguration(): Returns the HBase Configuration object associated with this Table instance.
  • get(Get get): Performs a get operation on a single row.
  • put(Put put): Performs a put operation on a single row.
  • scan(Scan scan): Performs a scan operation over a set of rows.
  • close(): Closes the Table instance.
  • setReadTimeout(int timeout): Sets the read timeout.
  • setWriteTimeout(int timeout): Sets the write timeout.
  • isAutoFlush(): Returns true if auto flush is enabled.
  • setAutoFlush(boolean autoFlush): Enables/disables auto flush.

Code Examples

The following code examples demonstrate how to use the methods from org.apache.hadoop.hbase.client.Table.

Getting an Instance of a Table


          
           
Configuration conf = HBaseConfiguration.create();
Table table = new HTable(conf, "myTable");

Performing a Get Operation


          
           
Get get = new Get(Bytes.toBytes("rowKey"));
Result result = table.get(get);

Performing a Put Operation


          
           
Put put = new Put(Bytes.toBytes("rowKey"));
put.addColumn(Bytes.toBytes("family"), Bytes.toBytes("column"), Bytes.toBytes("value"));
table.put(put);

Performing a Scan Operation


          
           
Scan scan = new Scan();
scan.addFamily(Bytes.toBytes("family"));
ResultScanner scanner = table.getScanner(scan);
try {
  for (Result row : scanner) {
    System.out.println(row);
  }
} finally {
  scanner.close();
}

Setting the Read/Write Timeouts


          
           
table.setReadTimeout(10000);
table.setWriteTimeout(20000);

Enabling/Disabling Auto Flush


          
           
table.setAutoFlush(true);
