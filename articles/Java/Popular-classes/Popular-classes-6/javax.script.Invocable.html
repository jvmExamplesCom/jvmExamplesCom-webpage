javax.script.Invocable

javax.script.Invocable is a Java interface that allows scripts to be implemented and executed. It provides methods to invoke scripts, retrieve and set the values of variables, evaluate code and compile script files. This article will provide a brief overview of javax.script.Invocable and a few examples of how it can be used.

Overview

The javax.script.Invocable interface is part of the Java scripting API, which provides a framework for the integration of scripting languages into Java applications. The key feature of the scripting API is its ability to allow scripts written in different languages to interact with each other and with Java components. This enables developers to use a variety of scripting languages within their applications and to combine the strengths of each language. A wide range of scripting languages are supported, including JavaScript, Groovy, ECMAScript, Python and Ruby.

Examples

The following example shows how to use the javax.script.Invocable interface to compile and execute a script written in the Groovy language:


          
           
import java.util.*;
import javax.script.*;

public class Test {
    public static void main(String[] args) throws Exception {
        ScriptEngine engine = new ScriptEngineManager().getEngineByName("groovy");
        Invocable invocable = (Invocable) engine;

        String scriptText = "def hello() { return 'hello world'; }";

        engine.eval(scriptText);
        Object result = invocable.invokeFunction("hello");

        System.out.println(result);
    }
}
The above code will print out “hello world”.

import java.util.*;
