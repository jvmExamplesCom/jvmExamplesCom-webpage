Com.google.gson.stream.JsonToken

The com.google.gson.stream.JsonToken class is a class from the GSON library, specifically related to JSON parsing. It provides an enumeration of all the possible tokens that a stream can contain. It can be used when creating applications that need to parse JSON.

Here is a simple example of how to use the JsonToken class:


          
           
String json = "{\"name\":\"John\"}";
JsonReader reader = new JsonReader(new StringReader(json));

JsonToken token = reader.peek();
System.out.println(token); // prints BEGIN_OBJECT
This code reads in a JSON string and uses the 
JsonReader.peek()
method to check the first token in the stream. This returns a 
JsonToken
enum value that we can then use. We can also use the 
JsonToken
enum to control the parsing of a stream. For example, we could use it to advance through a JSON object until we find the token we want: 

          
           
String json = "{\"name\":\"John\", \"age\":25, \"gender\":\"male\"}";
JsonReader reader = new JsonReader(new StringReader(json));

while(reader.hasNext()) {
    JsonToken token = reader.peek();

    if(token == JsonToken.NAME) {
        String name = reader.nextName();
        System.out.println("Name: " + name);
    } else if (token == JsonToken.NUMBER) {
        int age = reader.nextInt();
        System.out.println("Age: " + age);
    } else if (token == JsonToken.STRING) {
        String gender = reader.nextString();
        System.out.println("Gender: " + gender);
    } else {
        reader.skipValue();
    }
}

In this example, we are reading in a JSON string and looping through it, printing out the name, age and gender when we encounter those values. The 


          
           
JsonReader.hasNext()
method checks if there are still more tokens to read while the 
JsonReader.peek()
method allows us to check the type of the upcoming token so we can decide which action to take. We can also use the 

          
           
JsonReader.nextName()
, 

          
           
JsonReader.nextInt()
and 

          
           
JsonReader.nextString()
methods to read the actual values from the stream. Finally, we can use the 

          
           
JsonReader.skipValue()
method to skip over any tokens that we don't need.

Conclusion

The com.google.gson.stream.JsonToken class is an important part of the GSON library and allows us to control how a JSON stream is parsed. With it, we can check the types of tokens in the stream and act accordingly, allowing us to parse even complex streams of JSON data.
