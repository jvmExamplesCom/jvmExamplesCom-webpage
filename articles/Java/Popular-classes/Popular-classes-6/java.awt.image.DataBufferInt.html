java.awt.image.DataBufferInt

java.awt.image.DataBufferInt
class is a subclass of the abstract DataBuffer class, and is used to store a data array of primitive type 
int

getElements()

getElem(int bank, int i)
, 

          
           
setElem(int bank, int i, int val)
