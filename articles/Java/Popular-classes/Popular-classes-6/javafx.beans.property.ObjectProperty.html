javafx.beans.property.ObjectProperty

The javafx.beans.property.ObjectProperty class is a class that allows for the binding of a JavaFX property to an object value. It provides methods for setting, getting, and listening for changes in the value of the object property. This makes it possible for the value of the object property to be updated automatically when changes occur in the associated object.

Simplyfying Property Access

This class simplifies the process of obtaining a value from a JavaFX property by providing access methods to the property values using a type-safe approach. The ObjectProperty class is generic and can be declared with any type of value.

Constructors

A new instance of the ObjectProperty class can be created with the following constructors: 

          
           
ObjectProperty<T>()
ObjectProperty<T>(T initialValue)
ObjectProperty<T>(Object bean, String name)
ObjectProperty<T>(Object bean, String name, T initialValue)
Where T is the type of value stored in the object property.

Methods

The ObjectProperty class contains the following public methods: 

          
           
getBean(), isBound(), getName(), get(), set(T value), unbind(), bind(ObservableValue<? extends T> observable), addListener(ChangeListener<? super T> listener), removeListener(ChangeListener<? super T> listener).

Example

An example of how to use the ObjectProperty class is shown below. In this example, an instance of the ObjectProperty class is initialized with a String value. The set() method is then used to update the value of the object property, which will fire the change event when the value is updated. 

          
           
ObjectProperty<String> objectProperty = new ObjectProperty<>("Initial Value");
objectProperty.addListener((observable, oldValue, newValue) -> {
    System.out.println("Value changed from " + oldValue + " to " + newValue);
});

objectProperty.set("New Value");
// Output: Value changed from Initial Value to New Value
The output of this code shows that the change event was fired when the value of the object property was changed.
