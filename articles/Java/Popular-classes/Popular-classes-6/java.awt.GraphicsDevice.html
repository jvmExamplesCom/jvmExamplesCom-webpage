java.awt.GraphicsDevice

The java.awt.GraphicsDevice class provides access to information about and control of the graphics devices associated with a local graphics environment. This class can be used to create a GraphicsConfiguration object, which describes the characteristics of a graphics device.

Constructor

The following constructor is available in this class: 

          
           
public GraphicsDevice()
This constructor creates a new GraphicsDevice object that is not associated with any physical device.

Methods

Some of the important methods available in this class are as follows: 

          
           
public String	getIDstring()
public GraphicsConfiguration[]	getConfigurations()
public GraphicsConfiguration getDefaultConfiguration()
public GraphicsConfiguration	getBestConfiguration(GraphicsConfigTemplate gct)
public int	getType()
public void	setFullScreenWindow(Window w)
The getIDstring() method returns a string representing the unique identifier of the graphics device. The getConfigurations() method is used to retrieve an array of all GraphicsConfiguration objects associated with the GraphicsDevice. The getDefaultConfiguration() method is used to return the default GraphicsConfiguration for this device. The getBestConfiguration(GraphicsConfigTemplate gct) method is used to find a suitable GraphicsConfiguration matching the specified template. The getType() method returns the type of this graphics device. And finally, the setFullScreenWindow(Window w) method is used to enter full-screen mode if possible.
