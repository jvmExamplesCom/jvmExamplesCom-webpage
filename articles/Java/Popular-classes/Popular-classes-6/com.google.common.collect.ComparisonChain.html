What is com.google.common.collect.ComparisonChain?

com.google.common.collect.ComparisonChain is a class from the Google Guava Library that provides an efficient way to compare multiple values. It was designed to improve the readability and conciseness of code when multiple comparisons are required. The Comparator chain allows for multiple comparisons of objects to be made without having to nest if-else statements.

How to Use ComparisonChain?

The ComparisonChain API provides two methods, one is compare(Object a, Object b) and the other is start(). The compare method takes two objects and compares them according to their respective compareTo or equals methods. If any of the arguments are null, then it will throw a NullPointerException. The start method is used to start building a comparison chain, which can later be executed by invoking the result() method.


          
           
ComparisonChain chain = ComparisonChain.start();
int result = chain
    .compare(a.getName(), b.getName())
    .compare(a.getId(), b.getId())
    .compare(a.getAge(), b.getAge()).result();

The result of the comparison can be either -1, 0, or 1. -1 indicates that the first argument is smaller than the second argument, 0 indicates equal values, and 1 indicates that the first argument is greater than the second argument.

Example Code

Here's an example of the ComparisonChain API in action.


          
           
public class Person {
  private String name;
  private int id;
  private int age;

  public Person(String name, int id, int age) {
    this.name = name;
    this.id = id;
    this.age = age;
  }

  public int compareTo(Person other) {
    return ComparisonChain.start()
        .compare(this.name, other.name)
        .compare(this.id, other.id)
        .compare(this.age, other.age)
        .result();
  }
}

In this example, we have a Person class that implements the Comparable interface. We use the ComparisonChain API to compare the three fields of the Person object – name, id, and age – in order to determine their relative order.

Conclusion

The ComparisonChain class is a useful and convenient way to compare multiple objects in a concise and readable way. Although it is not as powerful as other alternatives, such as Java 8’s Comparator.comparing() method, it is still worth knowing if you find yourself frequently needing to compare multiple objects.
