XmlSchemaType

The XmlSchemaType annotation allows the user to associate a Java type with a schema type. It maps a Java datatype to an XML Schema type.

Example


          
           
@XmlSchemaType(name="byte", type=Byte.class)
public class MyClass {
    private Byte byteValue;
    // getter and setter methods
}

Output

The above code will result in the following XML: 

          
           
<xsd:element name="byteValue" type="xsd:byte"/>

The XmlSchemaType annotation specifies that the byteValue element should be rendered as an XML Schema byte type.

Attributes

The XmlSchemaType annotation has the following attributes:

  • name - The local name of the type.
  • type - The class object of the type.

Usage

The XmlSchemaType annotation can be used with the following annotations:

  • XmlElement
  • XmlAttribute
  • XmlIDREF
For example: 

          
           
@XmlElement
@XmlSchemaType(name="date", type=Date.class)
public Date getDate(){
    return this.date;
}
The above code will result in the following XML: 

          
           
<xsd:element name="date" type="xsd:date"/>

The XmlSchemaType annotation specifies that the date element should be rendered as an XML Schema date type.
