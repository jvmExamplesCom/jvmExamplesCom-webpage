org.springframework.aop.support.AopUtils

The AopUtils is a class in Spring Framework that provides support for AOP (Aspect-Oriented Programming) programmatic configuration and proxy creation. The AopUtils class supports multiple AOP paradigms, such as dynamic proxies, static interfaces, and even aspects. The AopUtils class provides several methods to create proxies, register aspects and so on.

convertClassToProxy()

convertClassToProxy()
method is used to convert a given class into an AOP proxy that implements the passed in interfaces. The returned proxy instance will be a subclass of the provided class.

createProxy()

createProxy()
method is used to create an AOP proxy based on the provided configuration and interface contracts. This method returns an object of type ProxyFactoryBean which is used to create the actual proxy instance.
