javax.jws.WebService

Java Web Service (JWS) is an open source standard for developing web services and web applications in Java. It was initially released in 2007 and is part of the J2EE platform, which also includes Java Server Pages (JSP) and Java Servlet technology. JWS supports both SOAP (Simple Object Access Protocol) and REST (Representational State Transfer) web services.

Creating a Web Service

Creating a web service using JWS is easy. All it takes is a few steps:

  1. Write the class that will be used as the web service.
  2. Add the @WebService annotation to the class.
  3. Write the web service methods in the class.
  4. Package the web service into a .war file.
  5. Deploy the .war file to a web server.
  6. Test the web service.

Example

Let's look at a simple example of a web service created using JWS. Here is the source code for the class that will be used as the web service:


          
           
@WebService
public class HelloWorld {
    public String sayHello(String name) {
        return "Hello " + name;
    }
}

This class is simple enough and only has one method, 

sayHello()
, which takes a String as an argument and returns a String as a result. The 
@WebService
annotation tells JWS that this class should be used as the web service.

Next, we need to package the web service into a .war file. This can be done using the Java Archive tool, or simply by zipping up the class files into a .war file. Once the .war file has been created, it can be deployed to a web server.

Finally, we can test the web service by sending a request to the web service URL (the location where the web service has been deployed). If successful, the web service should respond with the expected output. In the case of our example, we should get the response “Hello [name]” if the request is successful.

Conclusion

This is a simple example of how to use Java Web Service to create a web service. There are many more features available with JWS that allow you to create more complex web services. With its support for both SOAP and REST web services, JWS is an excellent choice for creating web services in Java.
