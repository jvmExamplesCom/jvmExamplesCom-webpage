org.apache.maven.model.Model

The org.apache.maven.model.Model class is an important part of the Apache Maven build system. It is a data structure that represents the POM or Project Object Model, which is a model of a project or module. The Model class is the representation of this POM, and contains elements such as the name of the project, its description, its version, where to find dependencies, and so on.

The following code example shows how to create a Model object in Java:


          
           
Model model = new Model();
 model.setGroupId("org.apache");
 model.setArtifactId("maven.model");
 model.setVersion("1.0");

This creates a Model object that represents a project with group id 'org.apache', artifact id 'maven.model' and version '1.0'.

model.setName("Apache Maven Model Project");
