org.eclipse.core.runtime.CoreException

org.eclipse.core.runtime.CoreException is a class in the Eclipse Platform which provides an exception to indicate a fundamental error due to a problem that has occurred within Eclipse. This class can be extended by other classes to provide more specific exceptions.

In this example, we will create a method that throws a CoreException if a given value is null.


          
           
public void checkValue(Object value) throws CoreException {

    }
}
