Overview of io.netty.handler.codec.http.HttpResponse

The io.netty.handler.codec.http.HttpResponse class is an implementation of the HttpResponse interface, which can be used to send a response to an HTTP request. In addition, convenience methods are provided for encoding access control responses, modifiable cookie values and responses containing file or byte data.
