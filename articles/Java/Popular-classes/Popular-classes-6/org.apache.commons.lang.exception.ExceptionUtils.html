Getting an Exception's Stack Trace

The most basic feature of 
ExceptionUtils
is its ability to get a stack trace from an exception. This is incredibly useful as stack traces are often long and difficult to read, especially when they contain hundreds of lines of code. To get an exception's stack trace, use the 
getStackTrace()
method. This method takes an 
Throwable
as an argument, and returns a 
String[]
containing the stack trace. Here's an example: 

          
           
try {
    // some code
} catch (Exception e) {
    String[] stackTrace = ExceptionUtils.getStackTrace(e);
    for (String line : stackTrace) {
        System.out.println(line);
    }
}
