org.springframework.cloud.context.config.annotation.RefreshScope

RefreshScope is a Spring annotation and an implementation of the ApplicationContext interface. This annotation allows for the dynamic and immediate updating of application configuration, making it easier to deploy and update applications in cloud environments.

}
