org.apache.log4j.BasicConfigurator

Log4j BasicConfigurator provides basic and minimalistic logging capabilities for logging to console, file and socket output streams. It can be used to quickly configure the root logger with the default console appender while still allowing fine-grained control over all aspects of logging configuration. It also allows applications to quickly implement custom logging configurations.

Example 1: Basic Configuration

The following is a basic example of how to configure log4j using BasicConfigurator:


          
           
BasicConfigurator.configure();
Logger logger = Logger.getLogger("MyLogger");
logger.debug("This is a debug message");

In this example, the BasicConfigurator has been used to configure the root logger with the default console appender. This will cause all debug messages to be logged to standard out.

Example 2: Adding an Appender

The following example shows how to add an appender to the root logger using BasicConfigurator:


          
           
Appender someAppender = new SomeAppender();
BasicConfigurator.configure(someAppender);
Logger logger = Logger.getLogger("MyLogger");
logger.debug("This is a debug message");

In this example, the BasicConfigurator has been used to configure the root logger with the specified appender. This will cause all debug messages to be logged to this appender.

Example 3: Custom Configuration

The following example shows how to configure a custom logging configuration using BasicConfigurator:


          
           
Logger rootLogger = Logger.getRootLogger();

Appender appender1 = new SomeAppender();
Appender appender2 = new SomeOtherAppender();

rootLogger.addAppender(appender1);
