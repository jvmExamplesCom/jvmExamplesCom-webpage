Introduction to android.view.inputmethod.InputConnection

The android.view.inputmethod.InputConnection interface provides a connection from the application to an active Input Method that can receive text input from the user. It is used to communicate with the active Input Method about text compositions, such as querying the current cursor position and editing the text around it.

Overview

InputConnection provides ways to retrieve text entered from an active Input Method, to get and set the current edit cursor position and to commit or delete text around the current cursor position. It also allows applications to provide completion candidates for input, and to request the Input Method to switch its state. InputConnection is provided as part of the Activity, Fragment or Dialog's onCreateInputView callback. This callback happens when the Input Method has been initialized, usually when the user is focused on an editable view in the application. The InputConnection instance is finalized when the Activity, Fragment or Dialog's onDestroyInputView callback is received.

Key Classes and Interfaces

The following classes and interfaces are important for working with InputConnection:
  • android.view.inputmethod.InputConnection: This interface provides the primary API for getting and setting text and the current cursor position.
  • android.view.inputmethod.CompletionInfo: This class represents a single completion candidate.
  • android.view.inputmethod.ExtractedText: This class contains the extracted text from an InputConnection.
  • android.view.inputmethod.ExtractedTextRequest: This class encapsulates a request for extracted text from an InputConnection.

InputConnection inputConnection = getCurrentInputConnection();
if (inputConnection != null) {
    CharSequence text = inputConnection.getTextBeforeCursor(128, 0);
}

getTextBeforeCursor()

Setting Text of an InputConnection


          
           
String textToInsert = "Hello";
InputConnection inputConnection = getCurrentInputConnection();
if (inputConnection != null) {
    boolean success = inputConnection.commitText(textToInsert, 1);
}
This example sets the given text using the 

          
           
commitText()
method of the InputConnection. The second argument is an integer representing flags, which in this case is set to 1 to indicate that the text should be inserted at the current cursor position.

Completing Text of an InputConnection


          
           
InputConnection inputConnection = getCurrentInputConnection();
if (inputConnection != null) {
    List<CompletionInfo> completions = new ArrayList<>();
    completions.add(new CompletionInfo(1, "completion1"));
    completions.add(new CompletionInfo(2, "completion2"));
    inputConnection.setComposingText("completion", 1);
    inputConnection.setCompletionHints(completions);
}
This example provides two CompletionInfo objects to an InputConnection using the 

          
           
setCompletionHints()
method. The first argument is an array of CompletionInfo objects, each containing a description and an identifier used to distinguish between different completion candidates. The second argument is an integer representing flags, which in this case is set to 1 to indicate that the given text should be set as the composing text.
