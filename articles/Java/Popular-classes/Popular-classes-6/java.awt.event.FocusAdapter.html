java.awt.event.FocusAdapter

Java.awt.event.FocusAdapter is a built-in Java class that provides methods for responding to focus events in Java. Focus events occur when a component gains or loses the keyboard focus, and are commonly used to enable components to respond to user input. This class can be used to create custom focus-handling components and can be subclassed to create custom focus listeners for any component.

Creating a FocusAdapter

To create a FocusAdapter, simply create a new class that extends the FocusAdapter class. This class must then override the focusGained() and/or focusLost() methods as needed. The code below shows an example of a FocusAdapter that prints a message to the console when its component gains or loses the keyboard focus: 

          
           
public class MyFocusAdapter extends FocusAdapter {
    @Override
    public void focusGained(FocusEvent e) {
        System.out.println("Component gained focus");
    }

    @Override
    public void focusLost(FocusEvent e) {
        System.out.println("Component lost focus");
    }
}

Registering the FocusAdapter

Once the FocusAdapter is created, it must be registered with a component in order to respond to focus events. To register the FocusAdapter, use the addFocusListener() method of the component in question, passing in an instance of the FocusAdapter. The code below shows an example of registering the MyFocusAdapter created in the previous section with a JTextField component: 

          
           
JTextField myComponent = new JTextField();
MyFocusAdapter adapter = new MyFocusAdapter();
myComponent.addFocusListener(adapter);
Now whenever the component gains or loses the keyboard focus, the corresponding methods of MyFocusAdapter will be called.

Events and Properties

Focus events originate from the component that gains or loses the focus, and the FocusEvent class provides information about the source component as well as other properties. The code below shows an example of using properties of the FocusEvent class to print out some extra information: 

          
           
@Override public void focusGained(FocusEvent e) {
    System.out.println("Component gained focus");
    Component src = e.getComponent();
    System.out.println("Source component is " + src.getClass().getName());
    boolean isTemporary = e.isTemporary();
    if (isTemporary) {
        System.out.println("Focus is temporary");
    }
    else {
        System.out.println("Focus is permanent");
    }
}
When the component gains the focus, this code prints out both the type of the source component and whether the focus is temporary or permanent.

Conclusion

The FocusAdapter class allows code to respond to focus events in Java Swing applications. By extending the FocusAdapter class and overriding the focusGained() and focusLost() methods, it is possible to create custom focus listeners for any component. Once created, the FocusAdapter should be registered with the component in question using the addFocusListener() method. The FocusEvent class provides information about the source component as well as properties such as whether the focus is temporary or permanent.
