Introduction to org.eclipse.swt.graphics.Point

org.eclipse.swt.graphics.Point is a class from the SWT package of the Eclipse platform. It represents a point in two-dimensional space, which has an x and y coordinate. It allows a user to create, modify and manipulate points used in graphical applications. This article explains the usage of the org.eclipse.swt.graphics.Point class and provides code examples to illustrate its functionality.

Creating a Point

A point can be created with either the default constructor or with the constructors that takes x and y coordinates as arguments: 

          
           
Point p1 = new Point();
Point p2 = new Point(10, 10);
The default constructor creates a point with (0,0) coordinates: 

          
           
Point p3 = new Point();
System.out.println("X: " + p3.x + ", Y: " + p3.y);
// Outputs X: 0, Y: 0

Accessing and Modifying Point Values

The x and y values of a point can be accessed and modified using methods such as getX(), setX(), getY(), and setY(): 

          
           
Point p4 = new Point(5, 5);
int xValue = p4.getX();
System.out.println("X value of p4: " + xValue);
// Outputs X value of p4: 5

p4.setX(10);
xValue = p4.getX();
System.out.println("Updated X value of p4: " + xValue);
// Outputs Updated X value of p4: 10

Comparing Points

Points can be compared using the equals() method to determine if two points are equal: 

          
           
Point p5 = new Point(10, 10);
Point p6 = new Point(10, 10);

boolean areEqual = p5.equals(p6);
System.out.println("Are p5 and p6 equal? " + areEqual);
// Outputs Are p5 and p6 equal? true

p6.setX(11);
areEqual = p5.equals(p6);
System.out.println("Are p5 and p6 equal? " + areEqual);
// Outputs Are p5 and p6 equal? false

Translating Points

A point can be translated using the translate() method which moves the point by the given x and y values: 

          
           
Point p7 = new Point(10, 10);
Point translatedP7 = p7.translate(10, 10);
System.out.println("Translated point: (" + translatedP7.x + ", " + translatedP7.y + ")");
// Outputs Translated point: (20, 20)

Converting Points to Strings

The toString() method can be used to convert a point to a String object: 

          
           
Point p8 = new Point(10, 10);
String strP8 = p8.toString();
System.out.println("Point 8 as a String: " + strP8);
// Outputs Point 8 as a String: org.eclipse.swt.graphics.Point@70dea4e
