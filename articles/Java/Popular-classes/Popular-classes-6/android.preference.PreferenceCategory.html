android.preference.PreferenceCategory

The android.preference.PreferenceCategory class is a specialized type of Preference which serves as the basis for creating preference categories. A PreferenceCategory serves as a way to group related Preference items and provide a disabled title above the group of preferences.

Creating an Android Preference Category

Creating a PreferenceCategory requires adding it to a Preference hierarchy in your application as shown below: 

          
           
PreferenceScreen prefSet = getPreferenceManager().createPreferenceScreen(this);
// Create a new PreferenceCategory
PreferenceCategory prefCat = new PreferenceCategory(this);
prefCat.setTitle("My Preference Category"); 
// Add the newly created category to the PreferenceScreen
prefSet.addPreference(prefCat);

Adding Preference Items to a Category

Once you have created a PreferenceCategory, you can add other Preference objects to it by calling the PreferenceCategory's addPreference() method. For example: 

          
           
CheckBoxPreference checkBoxPref = new CheckBoxPreference(this);
checkBoxPref.setKey("checkbox_preference");
checkBoxPref.setTitle("Checkbox Preference");
checkBoxPref.setSummary("This is a checkbox preference");
// Add the checkbox preference to the preference category
prefCat.addPreference(checkBoxPref);

Enabling and Disabling a Preference Category

A PreferenceCategory can be enabled or disabled using its setEnabled() method. For example to disable a PreferenceCategory with an id my_pref_cat: 

          
           
prefCat.setEnabled(false);
Disabling a PreferenceCategory means that none of the preferences inside it will be visible. It can be re-enabled later by calling the setEnabled() method again with true.

Conclusion

The android.preference.PreferenceCategory class provides a way to group Preference items together and provide a disabled title for them. It can be added to a Preference hierarchy and other Preference objects can be added to it. The PreferenceCategory can also be enabled or disabled.
