com.github.benmanes.caffeine.cache.Caffeine

Caffeine is a high performance Java caching library providing thread-safe, composable and highly concurrent HashMap and caching support on top of Java's ConcurrentMap. It allows developers to quickly and easily build efficient, reliable and scalable caches. Caffeine provides access to advanced features such as eviction and variable expiration times, as well as cache loading and stats recording.

Using Caffeine, developers can quickly and easily create a variety of caches with different features and performance characteristics. Caffeine also supports various strategies for garbage collection and eviction which provide developers with the flexibility to custom tailor the cache to their needs. In addition, Caffeine provides extensive metrics and profiling capabilities, allowing developers to monitor cache performance and make adjustments accordingly.
