Introduction: org.apache.hadoop.hbase.HBaseConfiguration

This class provides the settings and configurations used by HBase applications. It is part of the Apache Hadoop project which provides support for a distributed file system and other associated projects such as HBase, Apache ZooKeeper, Apache Pig, and Apache Hive. By default, this class is used in most HBase request/response scenarios.

Creating a Configuration Object using HBaseConfiguration
