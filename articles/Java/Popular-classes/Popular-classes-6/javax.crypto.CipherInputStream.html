javax.crypto.CipherInputStream

The javax.crypto.CipherInputStream class is used to read data from an underlying input stream, applying a cipher transformation on the fly. It is used when reading data from an input stream that has been encrypted using a Cipher object.

To create a CipherInputStream, the application first creates a Cipher object. Then it instantiates a CipherInputStream object with a reference to the input stream and passes the same Cipher object to the constructor of the CipherInputStream.


          
           
Cipher cipher = Cipher.getInstance("AES");
cipher.init(Cipher.DECRYPT_MODE, key);
CipherInputStream cis = new CipherInputStream(in, cipher);

Once the CipherInputStream is created, the application can read bytes from it as if it were a normal input stream (for example, by calling read()). However, the bytes read have been cryptographically transformed as they come off the wire.
