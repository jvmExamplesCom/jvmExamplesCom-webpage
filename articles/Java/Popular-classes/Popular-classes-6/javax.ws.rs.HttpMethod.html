javax.ws.rs.HttpMethod

javax.ws.rs.HttpMethod is an enum of HTTP methods used in JAX-RS annotations. It provides the set of HTTP verbs to be used in web application interactions. It consists of two methods: GET and POST, which are used to create the @GET and @POST annotations. Each annotation specifies a method type which will be used in the application's request process.
