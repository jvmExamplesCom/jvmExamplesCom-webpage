JavaFX Scene Control ListView

The 

          
           
javafx.scene.control.ListView
class provides a view of a list of items from which one or more may be selected. It supports the ability to add, remove, and modify its items. The items are organized into a vertical list. A list view is scrollable, and supports selection of single or multiple items.

Example 1 (Creating a Simple List View)


          
           
//Creating a ListView 
ListView<String> listView = new ListView<>();

//Adding items to the ListView 
listView.getItems().addAll("Item1", "Item2", "Item3");

Example 2 (Displaying the Selected Item)


          
           
//Creating a ListView 
ListView<String> listView = new ListView<>();

//Adding items to the ListView 
listView.getItems().addAll("Item1", "Item2", "Item3");

//Displaying the selected item 
String selectedItem = listView.getSelectionModel().getSelectedItem();
System.out.println(selectedItem);

Example 3 (Listening for Selection Changes)


          
           
//Creating a ListView 
ListView<String> listView = new ListView<>();

//Adding items to the ListView 
listView.getItems().addAll("Item1", "Item2", "Item3");

//Listening for selection changes 
listView.getSelectionModel().selectedItemProperty().addListener((observable, oldValue, newValue) -> {
    System.out.println("Selected item: " + newValue);
});

Example 4 (Enable Multiple Selection)


          
           
//Creating a ListView 
ListView<String> listView = new ListView<>();

//Adding items to the ListView 
listView.getItems().addAll("Item1", "Item2", "Item3");

//Enabling multiple selection 
listView.getSelectionModel().setSelectionMode(SelectionMode.MULTIPLE);
