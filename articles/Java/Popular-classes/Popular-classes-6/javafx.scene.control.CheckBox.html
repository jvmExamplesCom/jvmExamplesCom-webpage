javafx.scene.control.CheckBox

javafx.scene.control.CheckBox is a class that is part of the JavaFX API which provides a graphical check box control to the JavaFX API. This control can be used to model two states such as true/false, yes/no, on/off, etc. In its most basic form, the javafx.scene.control.CheckBox simply provides an input field with a check box that can be checked or unchecked by the user.

// Constructor with label text parameter

The label text parameter allows the user to set the text that will appear next to the check box, indicating what it is used for. The initial state parameter allows the user to specify if the check box should initially be checked or not. The default constructor will create a check box with no initial text and in a unchecked state.

// Set the label text
public void setText(String text);

public String getText();

The setOnChange() method is used to specify an EventHandler that is called when the selected state of the CheckBox changes. This could be when the user checks or unchecks the CheckBox.

To demonstrate these features, we can look at the following example that shows how to use the CheckBox class.


          
           
// Create a CheckBox labelled 'Check Me!'
CheckBox cb = new CheckBox("Check Me!");

// Handle an action on the CheckBox

    // Print out the selected state
    System.out.println("CheckBox is checked? " + cb.isSelected());
