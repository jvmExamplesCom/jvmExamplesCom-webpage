Java.nio.file.attribute.PosixFilePermissions

This example shows how to use the PosixFilePermissions class to get, set, and modify the permissions of a file using the PosixFileAttributes class.

Attributes

The PosixFilePermissions class also has three useful attributes; the owner, group and others permissions.
