What is Com.github.tomakehurst.wiremock.junit.WireMockRule

Using WireMockRule with JUnit

The following code example illustrates how to use WireMockRule with JUnit:


          
           
@Rule
public WireMockRule wireMockRule = new WireMockRule(options().dynamicPort());

@Test
public void exampleTest() {

    verify(getRequestedFor(urlEqualTo("/some/thing")));
}

verify
