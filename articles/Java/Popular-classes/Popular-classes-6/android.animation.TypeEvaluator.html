android.animation.TypeEvaluator

The 

          
           
android.animation.TypeEvaluator
class is a utility class used in the animation framework of Android. It enables developers to create animations that transition between two values of a given type. It is an abstract class, so it cannot be directly instantiated. Instead, it must be extended and its evaluate() method overriden.

class MyEvaluator extends android.animation.TypeEvaluator<PointF> {

    @Override
    public PointF evaluate(float fraction, PointF startValue, PointF endValue) {

        ...
    }
}
The 

          
           
evaluate()
method takes three parameters: 
  • fraction

  • startValue

  • endValue

The 

          
           
evaluate()
method then takes these values and calculates a new value that is somewhere between the start and end values. This new value is returned as the result of the method. For example, if the start and end values were both points on a path and the fraction was 0.5, then the returned value would be the point exactly halfway along the path.
