org.json.JSONArray

The org.json.JSONArray class is part of the Java API for JSON Processing and provides an ordered collection of values (either as strings, numbers, booleans, or JSON objects). It also provides a range of methods for manipulating and navigating data stored within JSON arrays.

Creating a JSONArray

You can create an empty JSONArray by calling the constructor with no arguments, like this:


          
           
JSONArray jsonArray = new JSONArray();

It's also possible to create a JSONArray from a string, like this:


          
           
String jsonString = "[\"value1\", \"value2\", \"value3\"]";
JSONArray jsonArray = new JSONArray(jsonString);

Adding Values to a JSONArray

You can add values to a JSONArray using the add() method. This takes an Object parameter, so you can pass in a string, number, boolean, or other JSON object, like this:


          
           
String value1 = "Hello";
int value2 = 42;
boolean value3 = true;
JSONObject jsonObject = new JSONObject();
jsonObject.put("key", "value");

jsonArray.add(value1);
jsonArray.add(value2);
jsonArray.add(value3);
jsonArray.add(jsonObject);

Iterating Over a JSONArray

You can iterate over a JSONArray using the for-each loop syntax in Java:


          
           
for (Object value : jsonArray) {
    System.out.println(value);
}

This will print out each value in the JSONArray to the console.

Accessing Values in a JSONArray

You can access the values in a JSONArray using the get() method, which takes an integer parameter representing the index of the value you wish to access. For example, if you wanted to access the first value in the JSONArray, you would use the following code:


          
           
Object value = jsonArray.get(0);
System.out.println(value);

Removing Values from a JSONArray

You can remove values from a JSONArray using the remove() method, which takes an integer parameter representing the index of the value you wish to remove. For example, if you wanted to remove the first value from the JSONArray, you would use the following code:


          
           
Object removedValue = jsonArray.remove(0);
System.out.println(removedValue);

Conclusion

In this article, we've seen how to use the org.json.JSONArray class to create, manipulate, and access data stored within a JSON array. This class is an invaluable tool for working with JSON data in Java.
