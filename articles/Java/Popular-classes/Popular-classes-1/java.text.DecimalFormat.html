java.text.DecimalFormat

Java.text.DecimalFormat provides a class for formatting and parsing numbers and currency values according to user specified patterns. It is widely used in applications where the display of large numbers or currency values is desired. It supports string-based formatting and parsing, with a syntax similar to the type of formatting used in the Java programming language.

Creating a DecimalFormat Object

To create a DecimalFormat object, you need to specify a pattern. This pattern controls how numbers and currency values will be formatted and parsed. For example, the following code creates a DecimalFormat object with a pattern of "#,##0.00".


          
           
DecimalFormat df = new DecimalFormat("#,##0.00");

The pattern specifies that numbers should be formatted with a thousands separator (comma), followed by two decimal places. If a currency value is provided, it will be preceded by the currency sign. For example, with this pattern, the number 10000.45 would be formatted as "10,000.45" and the currency value $100.55 would be formatted as "$100.55".

Formatting Numbers and Currency Values

Once a DecimalFormat object has been created, it can be used to format numbers and currency values. This is done by calling the format() method and passing in the number or currency value to be formatted. For example, the following code will format the number 1000.25 with the pattern specified above.


          
           
String formattedNumber = df.format(1000.25);
// formattedNumber is "1,000.25"

Similarly, the following code will format a currency value of $100.55.


          
           
String formattedCurrency = df.format(100.55);
// formattedCurrency is "$100.55"

Parsing Formatted Strings

DecimalFormat can also be used to parse strings that are formatted according to the pattern. This is done by calling the parse() method and passing in the string to be parsed. For example, the following code parses a string formatted with the pattern specified above and returns a double value.


          
           
String s = "1,000.25";
double d = df.parse(s);
// d is 1000.25
