java.util.Random

java.util.Random is a utility class in java that can be used to generate random numbers. It can also be used for other purposes such as generating random passwords or selecting an item from a list. The Random class is thread-safe and all of its methods are synchronized.

Methods

Random has a number of different methods available for generating random values. These include:

  • nextInt(int n): Returns an integer between 0 (inclusive) and the specified value (exclusive).
  • nextDouble(): Returns a pseudorandom double between 0.0 and 1.0.
  • nextLong(): Returns a pseudorandom long value.
  • nextBoolean(): Returns a pseudorandom boolean value.
  • nextFloat(): Returns a pseudorandom float value.
  • nextBytes(byte[] bytes): Fills the specified array with random bytes.

Examples

Below are some examples of using the Random class.


          
           
Random random = new Random();
// Generates a random int between 0 and 10
int randomInt = random.nextInt(10);
System.out.println(randomInt);

// Generates a random double between 0.0 and 1.0
double randomDouble = random.nextDouble();
System.out.println(randomDouble);

// Generates a random boolean value
boolean randomBoolean = random.nextBoolean();
System.out.println(randomBoolean);

// Generates a random byte array
byte[] randomBytes = new byte[20];
random.nextBytes(randomBytes);
System.out.println(Arrays.toString(randomBytes));
