Introduction to Network Interface in Java

Network interface is a Java class which provides an application programming interface (API) to access the underlying networking configuration. It is present in the 

          
           
java.net
package and is part of the Java platform's core library. NetworkInterface provides network information about the system, such as details about all the installed network interfaces, IP addresses, broadcast addresses, and multicast addresses. This article will discuss some of the details related to the NetworkInterface class and give examples of how it can be used.

Creating an Instance

A NetworkInterface object can be created from either a String or an Integer. The String argument should provide either a name, a hardware address, or a display name of the network interface. The Integer should represent the index of the network interface. 

          
           
//Create an instance from a String
NetworkInterface nf = NetworkInterface.getByName("en0");

//Create an instance from an Integer
NetworkInterface nfi = NetworkInterface.getByIndex(2);

getByName

Getting Information About a Network Interface

Once an instance of NetworkInterface is created, the various methods defined in the NetworkInterface class can be used to get the details of the network interface. 

          
           
//Get the name of the network interface
String name = nf.getName();
System.out.println("Network Interface Name : " + name);

//Get the hardware address associated with the network interface
byte[] mac = nf.getHardwareAddress();
System.out.println("Hardware Address: " + Arrays.toString(mac));

//Get all IP addresses associated with the network interface
Enumeration<InetAddress> inetAddresses = nf.getInetAddresses();
while(inetAddresses.hasMoreElements()) {

Setting Network Interface Options

The NetworkInterface class also allows setting network interface options such as the broadcast address and the maximum transmission unit (MTU). 

          
           
//Set the broadcast address associated with the network interface
boolean isSet = nf.setBroadcast(InetAddress.getByName("10.0.0.255"));
if(isSet) {
   System.out.println("Broadcast address set successfully");
} else {
   System.out.println("Unable to set broadcast address");
}

//Set the maximum transmission unit for the network interface
isSet = nf.setMTU(1500);
if(isSet) {
   System.out.println("Maximum transmission unit set successfully");
} else {
   System.out.println("Unable to set maximum transmission unit");
}

Conclusion

The NetworkInterface class provides an easy way to programmatically access and configure the underlying network configuration of the system. Its methods allow the programmer to get the network interfaces installed on the system, get the IP addresses associated with each network interface, set the options for each network interface, etc. This article discussed some of the features offered by the NetworkInterface class and provided examples of how it can be used.
