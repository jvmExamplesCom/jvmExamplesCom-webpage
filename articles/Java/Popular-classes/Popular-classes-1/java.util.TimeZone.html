java.util.TimeZone

The java.util.TimeZone class is a subclass of Object and implements Serializable, Cloneable interfaces. It provides time zone information. It represents a system time zone or an offset from the UTC/Greenwich time zone.

Creating a TimeZone Object

The TimeZone class has many constructors that can be used to create a TimeZone object. Here are three of them: 

          
           
TimeZone tz = TimeZone.getDefault(); // get the default timezone
TimeZone tz = TimeZone.getTimeZone(String ID); // get the specified timezone
TimeZone tz = TimeZone.getTimeZone(String ID, Locale locale); // get the specified timezone in the specified country

Using the TimeZone Object

Once a TimeZone object is obtained, there are numerous methods available to get different types of info about the timezone. 

          
           
String getID() - returns the ID string of this time zone.
For example, the following code prints out the ID string of the timezone: 

          
           
TimeZone tz = TimeZone.getDefault();
System.out.println(tz.getID()); // Prints out the ID string of the time zone

          

          
           
String getDisplayName() - returns the display name of this time zone.
For example, the following code prints out the display name of the timezone: 

          
           
TimeZone tz = TimeZone.getDefault();
System.out.println(tz.getDisplayName()); // Prints out the display name of the time zone

          

          
           
int getOffset(long date) - returns the offset from UTC of this time zone at the specified date.
For example, the following code prints out the offset from UTC of the timezone at the current date: 

          
           
TimeZone tz = TimeZone.getDefault();
long date = System.currentTimeMillis();
System.out.println(tz.getOffset(date)); // Prints out the offset from UTC of the time zone

          

          
           
boolean useDaylightTime() - returns true if this time zone uses daylight savings time.
For example, the following code prints out whether or not the timezone uses daylight savings time: 

          
           
TimeZone tz = TimeZone.getDefault();
System.out.println(tz.useDaylightTime()); // Prints out whether or not the timezone uses daylight savings time

Conclusion

The java.util.TimeZone class provides a great interface for obtaining and working with timezone information. With its various constructors, it can be used to obtain various types of info about the specified timezones such as their ID, display names and offset from UTC.
