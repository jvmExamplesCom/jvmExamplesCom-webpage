Java and android.provider.Settings

One of the most powerful and versatile tools that developers have to work with when creating Android applications is the android.provider.Settings class. This powerful class provides access to a variety of system settings, allowing developers to make their applications fully interactive with the users' devices. This article will give an overview of what is possible with the android.provider.Settings class, as well as provide examples of how to use it to make your applications more flexible.

Basic Usage

In order to begin using the android.provider.Settings class, you must first import it into your application: 

          
           
import android.provider.Settings;
Once you have the class imported, you can begin calling methods from the class to access different device settings. The following example shows how to access the user's language selection: 

          
           
String language = Settings.System.getString(getContentResolver(), Settings.Global.DEVICE_LANGUAGE);
This code retrieves a String object containing the currently selected language. Another useful method available in android.provider.Settings has to do with screen brightness: 

          
           
int brightnessValue = Settings.System.getInt(getContentResolver(), Settings.System.SCREEN_BRIGHTNESS);
This code retrieves a value between 0 and 255 which represents the current screen brightness setting.

In order to create a menu, you must first create an XML file that contains the layout for the menu. This XML file should include all of the options you wish to present to the user.         </menu>
    </item>
</menu>
Once you have your menu layout XML file created, you can now begin using android.provider.Settings to display it. To do this, you will need to call the createSettingMenu() method which will create the menu and return a Menu object that can be used to interact with the user's selections. The following code shows an example of how this might look: 

          
           
Menu menu = Settings.createSettingMenu(this, R.menu.settings_menu);
String language = 
    Settings.getString(getContentResolver(), Settings.Global.DEVICE_LANGUAGE, menu);
int brightnessValue = 
    Settings.getInt(getContentResolver(), Settings.System.SCREEN_BRIGHTNESS, menu);

Conclusion
