org.w3c.dom.Document

The org.w3c.dom.Document interface provides the primary access to the document's data, as well as providing methods to traverse the document and modify its contents and structure. It is also the root of the document tree and is used to create new elements and set the document's properties. This article explores the org.w3c.dom.Document interface and provides code examples.

Creating a DOM Document

A DOM Document can be created with the 

          
           
DocumentBuilderFactory
: 

          
           
DocumentBuilderFactory factory = DocumentBuilderFactory.newInstance();
DocumentBuilder builder = factory.newDocumentBuilder();
Document document = builder.newDocument();

Once the document is created, it can be populated with data, or have other DOM operations performed on it:


          
           
Element root = document.createElement("root");
document.appendChild(root);
Attr attr = document.createAttribute("attribute");
attr.setValue("value");
root.setAttributeNode(attr);
Text text = document.createTextNode("Text");
root.appendChild(text);
The 
Document
object contains a number of methods to traverse, edit and query the document, such as 

          
           
getDocumentElement()
and 

          
           
getElementsByTagName()
: 

          
           
Element elem = document.getDocumentElement();
NodeList nodes = document.getElementsByTagName("root");

Writing a DOM Document

When a DOM Document has been created, it can be written to an output stream using a 

Transformer
: 

          
           
TransformerFactory tf = TransformerFactory.newInstance();
Transformer transformer = tf.newTransformer();
transformer.setOutputProperty(OutputKeys.OMIT_XML_DECLARATION, "yes");
StringWriter writer = new StringWriter();
transformer.transform(new DOMSource(document), new StreamResult(writer));
String output = writer.getBuffer().toString().replaceAll("\n|\r", "");
And the output can be written as HTML: 

          
           
<html>
  <head></head>
  <body>
    <root attribute="value">Text</root>
  </body>
</html>
