Java.net.URLDecoder

The Java.net.URLDecoder class is a part of the java.net package and provides a convenient, static utility method for decoding URLs encoded with the "application/x-www-form-urlencoded" MIME type. This class contains static methods for decoding the various components of a URL, including the query string and path information.

Code Examples

The following code example demonstrates using Java.net.URLDecoder for decoding an encoded URL:


          
           
String url = "http://example.com/hello%20world";
String decodedUrl = java.net.URLDecoder.decode(url);
System.out.println(decodedUrl); // prints http://example.com/hello world

This example decodes the "hello world" portion of the URL and prints out the decoded version.

The Java.net.URLDecoder also supports encoding parameters in the query string of a URL. The following example demonstrates how to decode a parameter encoded in the query string:


          
           
String url = "http://example.com?key=value%20with%20spaces";
Map<String, String> params = new HashMap<>();
String query = java.net.URLDecoder.decode(url.split("[?]")[1]);
for (String param : query.split("[&amp;]")) {
    String[] keyValue = param.split("[=]");
    params.put(keyValue[0], keyValue[1]);
}
System.out.println(params.get("key")); // prints value with spaces

This example decodes the "key=value%20with%20spaces" portion of the URL and prints out the decoded version of the "key".
