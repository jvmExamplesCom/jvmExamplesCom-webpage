Java.io.OutputStream

The 

          
           
java.io.OutputStream
class is a fundamental component of the Java Input/Output system, and is used for writing raw data bytes to a target stream. Output streams are capable of writing data to any type of output device, such as a file, connection, or another stream.

An OutputStream is connected to a specific target, such as a FileOutputStream connected to a particular file, or an OutputStream connected to a TCP socket.

The OutputStream class has several useful methods for writing data, such as write(), flush(), and close(). The write() method is used to write bytes to the stream, while the flush() method forces any buffered output bytes to be written immediately. The close() method closes and releases any system resources associated with the stream.

Example 1 : Writing Byte Data

The following example shows how to use the write() method to write a single byte to an OutputStream.


          
           
OutputStream os = new FileOutputStream("data.txt");
os.write(0xFF);  // write a single byte
os.close();

Example 2 : Writing Array of Bytes

The following example shows how to use the write() method to write an array of bytes to an OutputStream.


          
           
OutputStream os = new FileOutputStream("data.txt");

byte[] buffer = { 0x41, 0x42, 0x43 };
os.write(buffer);  // write array of bytes

os.close();

Example 3 : Writing Int Data

The following example shows how to use the write() method to write an int data to an OutputStream.


          
           
OutputStream os = new FileOutputStream("data.txt");
int value = 0xFF;
os.write(value);  // write int data
os.close();

Example 4 : Flushing Output Stream

The following example shows how to use the flush() method to force any buffered output bytes to be written immediately to the underlying stream.


          
           
OutputStream os = new FileOutputStream("data.txt");
os.write(0xFF);
os.flush();  // flush output stream
os.close();

Example 5 : Closing Output Stream

The following example shows how to use the close() method to close and release any system resources associated with the stream.


          
           
OutputStream os = new FileOutputStream("data.txt");
os.write(0xFF);
os.close();  // close output stream
