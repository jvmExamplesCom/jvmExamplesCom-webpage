android.view.animation.Interpolator

An android.view.animation.Interpolator is a class of the Android framework API that allows developers to create smooth and consistent animation over the course of a single action. A common example of how this works is when using multiple interpolators together to create a continuous animation, like in games or interactive applications. Interpolators are mostly used to provide smoother and more natural animations. They are also used to produce animations with a specific duration, speed, and looping capabilities. The following sections will discuss the class and its use in greater detail.

Doing Animation with Interpolators

Interpolators are used for the purposes of doing animations on Android devices. To do this, a developer must create an instance of the android.view.animation.Interpolator class, set it's fields to the desired values, and call the startAnimation() method. This will start the animation. To end the animation, the developer must call the stopAnimation() method. This can be done from any thread as long as the thread holds the appropriate lock. The android.view.animation.Interpolator class also supports other features such as setRepeatMode(), resetCycle(), and setSpeed(). These can be used to modify the behavior of the animation.
