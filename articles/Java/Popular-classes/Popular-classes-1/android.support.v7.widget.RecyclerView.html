Android Support V7 RecyclerView

The android.support.v7.widget.RecyclerView is a powerful user interface component used for displaying collections of items in a list or for navigating through pages or categories. It is a modernized and flexible alternative to the traditional ListView widget, allowing for much greater flexibility and customization. This article will provide an overview of the RecyclerView and its features, including a detailed look at how to use it in an Android app.

Overview of the RecyclerView

The RecyclerView is designed to be a more versatile and customizable version of the ListView widget. It provides a specialized layout manager that can be used to display a collection of items in a vertical or horizontal scrolling list. It supports item animations, item decoration, item selection, and item click/long press events. It also offers specific features such as ViewHolder reuse, DividerItemDecoration, ItemTouchHelper, and ItemAnimators. The RecyclerView is fully customizable and can be used with any layout manager.

Using the RecyclerView in an Android App

To use the RecyclerView in an Android app, the first step is to add the RecyclerView dependency to the app's build.gradle file: 

          
           
compile ‘com.android.support:recyclerview-v7:25.3.1’

Now create a class for the RecyclerView's adapter. This class will extend from the RecyclerView.Adapter class and will be responsible for binding the data from the model to the views in the list. 

          
           
public class MoviesAdapter extends RecyclerView.Adapter<MoviesAdapter.ViewHolder> {

  private List<Movie> movies;

  public MoviesAdapter(List<Movie> movies) {
    this.movies = movies;
  }

  @Override
  public ViewHolder onCreateViewHolder(ViewGroup parent, int viewType) {
    // Inflate the view and return the new ViewHolder
    return new ViewHolder(LayoutInflater.from(parent.getContext())
           .inflate(R.layout.list_item_movie, parent, false));
  }

  @Override
  public void onBindViewHolder(ViewHolder holder, int position) {
    // Set the data for the list item
    Movie movie = movies.get(position);
    holder.titleTextView.setText(movie.getTitle());
    holder.ratingTextView.setText(String.valueOf(movie.getRating()));
    holder.releaseDateTextView.setText(movie.getReleaseDate().toString());
  }

  @Override
  public int getItemCount() {
    return movies.size();
  }

  class ViewHolder extends RecyclerView.ViewHolder {

    TextView titleTextView;
    TextView ratingTextView;
    TextView releaseDateTextView;

    ViewHolder(View itemView) {
      super(itemView);
      titleTextView = (TextView) itemView.findViewById(R.id.titleTextView);
      ratingTextView = (TextView) itemView.findViewById(R.id.ratingTextView);
      releaseDateTextView = (TextView) itemView.findViewById(R.id.releaseDateTextView);
    }
  }
}

  android:layout_height="wrap_content"
