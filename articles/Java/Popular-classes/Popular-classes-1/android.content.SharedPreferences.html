Android SharedPreferences

Android SharedPreferences provides an API to save data in key-value pairs locally and share it between activities with a common context. It provides a fast and secure way to store and retrieve data in an efficient manner and works as a bridge between applications and the Android operating system.

Overview of Android SharedPreferences

Android SharedPreferences is a framework to save and retrieve data from a local memory or remote server. It works as a back-up utility for saving small amounts of data such as Strings, boolean values, etc. It also helps store user settings that should be kept even after the app has been closed.

Syntax of Android SharedPreferences

To use Android SharedPreferences, you need to declare an instance of SharedPreferences. 

          
           
SharedPreferences prefs = PreferenceManager.getDefaultSharedPreferences(context);
This is the basic syntax for using the Android SharedPreferences.

Working with Android SharedPreferences

To work with Android SharedPreferences, we have to first create an instance of SharedPreferences. After that we can either store the value, retrieve the value or remove it from memory.

Storing Values

We can store values in Android SharedPreferences by using the 

          
           
edit()
method of the SharedPreferences instance. We can store any type of data in the key-value pair format. 

          
           
SharedPreferences prefs = PreferenceManager.getDefaultSharedPreferences(context);

SharedPreferences.Editor editor = prefs.edit();
editor.putString("key", "value");

editor.apply();

Retrieving Values

We can retrieve the stored values from the SharedPreferences instance by using the 

          
           
get()
method. This method returns an Object. We need to cast this object to the required data type. 

          
           
SharedPreferences prefs = PreferenceManager.getDefaultSharedPreferences(context);
String value = prefs.getString("key",null);

Removing Values

We can remove values from the SharedPreferences instance with the 

          
           
remove()
method. 

          
           
SharedPreferences prefs = PreferenceManager.getDefaultSharedPreferences(context);

SharedPreferences.Editor editor = prefs.edit();
editor.remove("key");

editor.apply();

Conclusion

Android SharedPreferences is a fast and secure way of storing and retrieving small amounts of data. It helps us store the user settings and preferences so that they are not lost when the app closes. It provides an easy way for developers to save and retrieve data effectively.
