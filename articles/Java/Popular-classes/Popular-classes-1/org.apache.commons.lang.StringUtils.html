org.apache.commons.lang.StringUtils

org.apache.commons.lang.StringUtils is a utility class for manipulating strings in Java. It provides methods for creating, manipulating and transforming strings, as well as for performing various operations on a string, such as splitting, comparing, formatting, concatenating and replacing. It is a part of the Apache Commons Lang library and is an important tool for developers working with Java.

Methods of StringUtils

StringUtils provides a wide range of methods, some of the more common ones are discussed below.

Contains

The contains method is used to determine if a string contains a given substring. This method returns true if the substring is found and false if not.


          
           
StringUtils.contains("Hello World", "World")
// Output: true

SubstringBefore

The substringBefore method takes two strings as parameters and returns the substring from the beginning of the first parameter up to but not including the second parameter. If the second parameter is not found in the first, the entire string is returned.


          
           
StringUtils.substringBefore("Hello World", "World")
// Output: "Hello "

Replace

The replace method takes three parameters, a string to be searched for, a string to be replaced, and a string to replace it with. This method is useful for when a certain part of a string needs to be replaced with something else.


          
           
StringUtils.replace("Hello World", "World", "Java")
// Output: "Hello Java"

StartsWith

The startsWith method can be used to check if a string starts with a given substring. This method returns true if the search string is at the beginning of the target string and false if not.


          
           
StringUtils.startsWith("Hello World", "Hello")
// Output: true

EndsWith

The endsWith method is similar to the startsWith method, but it checks if a string ends with a given substring. This method returns true if the search string is at the end of the target string and false if not.


          
           
StringUtils.endsWith("Hello World", "World")
// Output: true

Trim

The trim method is used to remove leading and trailing white space from a string. This is useful for cleaning up user input before further processing.


          
           
StringUtils.trim("   Hello World   ")
// Output: "Hello World"

Conclusion

org.apache.commons.lang.StringUtils is a powerful utility class that provides a wide range of methods for manipulating strings in Java. It is used for tasks such as searching for a substring, replacing part of a string, and trimming white space from the beginning and end. These methods can be very useful for working with strings in Java.
