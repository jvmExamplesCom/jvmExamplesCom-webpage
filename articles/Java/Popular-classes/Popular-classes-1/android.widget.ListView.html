Android Widget ListView

The 

          
           
android.widget.ListView
is an important component in Android user Interface. It is used to display a list of items in a scrollable view. The ListView widget provides several features like adding dividers, custom views, selection and click listeners, item selection and much more. In this article, we will be looking at how you can use the ListView widget in your Android applications.

Creating a ListView

To create a ListView, you will need to add the ListView widget to your layout file. You can do so by adding the following code to your layout XML file: 

          
           
<ListView
    android:id="@+id/myListView"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"/>
Now that you have added the ListView to your layout, you can get a reference to the ListView in your activity using the ID you set in the layout file. For example, if you set the ID to “myListView”, then you could get a reference to the ListView in your activity as follows: 

          
           
ListView myListView = (ListView) findViewById(R.id.myListView);

Filling the ListView

Once you have a reference to the ListView, the next step is to populate it with data. This can be done in two ways:

Using an ArrayAdapter

The simplest way to fill the ListView is to use an Adapter. An Adapter is an object that provides the ListView with data. One type of Adapter is the ArrayAdapter, which takes an array of objects and displays each one in the ListView. For example, say you have an array of Strings that you want to display in the ListView: 

          
           
String[] strings = {"String 1", "String 2", "String 3"};
You can create an ArrayAdapter like so: 

          
           
ArrayAdapter arrayAdapter = new ArrayAdapter(this, android.R.layout.simple_list_item_1, strings);
Finally, you set the ArrayAdapter on the ListView using the 

          
           
setAdapter
method: 

          
           
myListView.setAdapter(arrayAdapter);

Using a Custom Adapter

Another option is to create a custom Adapter. A custom Adapter gives you more control over the appearance of the ListView, as well as allowing you to use other types of data sources. To create a custom Adapter, you must extend the 

          
           
BaseAdapter
class and implement the required methods. Here is a simple implementation of a custom Adapter: 

          
           
public class MyAdapter extends BaseAdapter {

    private Context context;
    private List<String> strings;

    public MyAdapter(Context context, List<String> strings) {
        this.context = context;
        this.strings = strings;
    }

    @Override
    public int getCount() {
        return strings.size();
    }

    @Override
    public Object getItem(int position) {
        return strings.get(position);
    }

    @Override
    public long getItemId(int position) {
        return position;
    }

    @Override
    public View getView(int position, View convertView, ViewGroup parent) {
        // inflate the layout for each list row
        if (convertView == null) {
            convertView = LayoutInflater.from(context).
                    inflate(R.layout.list_row, parent, false);
        }

        // get current item to be displayed
        String currentString = (String) getItem(position);

        // get the TextView for item name and item description
        TextView textViewName = (TextView) 
                convertView.findViewById(R.id.text_view_name);

        //sets the text for item name and item description
        textViewName.setText(currentString);

        // returns the view for the current row
        return convertView;
    }
}
Then, you create an instance of your custom Adapter and set it on the ListView: 

          
           
MyAdapter adapter = new MyAdapter(this, strings);
myListView.setAdapter(adapter);

Adding Listeners

The ListView also supports various click and selection listeners that can be used to respond to user interaction. To add a click listener, you can use the 

          
           
setOnItemClickListener
method: 

          
           
myListView.setOnItemClickListener(new OnItemClickListener() {
    @Override
    public void onItemClick(AdapterView<?> parent, View view, int position, long id) {
        // Do something when a listitem is clicked
    }
});
And to add a selection listener, you can use the 

          
           
setOnItemSelectedListener
method: 

          
           
myListView.setOnItemSelectedListener(new OnItemSelectedListener() {
    @Override
    public void onItemSelected(AdapterView<?> parent, View view, int position, long id) {
        // Do something when a listitem is selected
    }
});

Conclusion

The ListView widget is a powerful tool for displaying lists of data in Android. In this article, we have looked at how to create a ListView, populate it with data, and add listeners for user interaction. With these skills, you should be able to create user interfaces with ListViews that look great and work well.
