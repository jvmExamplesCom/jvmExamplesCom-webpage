java.util.Collections

Java.util.Collections is a utility class in the Java language that consists of static methods that operate on instances of Collection classes such as List, Set, and Map. It provides a set of algorithms to manipulate collections, such as sorting, searching, reversing, shuffling, etc.

Sorting Collections with sort()

The java.util.Collections.sort() method can be used to sort a collection in ascending order. The sort() method takes a List as an argument and rearranges the elements using either their natural order or a user-defined Comparator.


          
           
List<String> list = new ArrayList<>();
list.add("David");
list.add("Alice");
list.add("John");

Collections.sort(list);

System.out.println(list);
// Output: [Alice, David, John]

Reversing a Collection

The java.util.Collections.reverse() method can be used to reverse the order of the elements in a collection. The reverse() method takes a List as an argument and rearranges the elements in reverse order.


          
           
List<String> list = new ArrayList<>();
list.add("David");
list.add("Alice");
list.add("John");

Collections.reverse(list);

System.out.println(list);
// Output: [John, Alice, David]

Shuffling a Collection

The java.util.Collections.shuffle() method can be used to randomly shuffle the elements in a collection. The shuffle() method takes a List as an argument and rearranges the elements in a random order.


          
           
List<String> list = new ArrayList<>();
list.add("David");
list.add("Alice");
list.add("John");

Collections.shuffle(list);

System.out.println(list);
// Output: [Alice, David, John] (may be different)

Searching a Collection

The java.util.Collections.binarySearch() method can be used to search for an element in a collection. The binarySearch() method takes a List and a key as arguments and returns the index of the element if found or -1 if not found.


          
           
List<Integer> list = new ArrayList<>();
list.add(10);
list.add(20);
list.add(30);

int key = 20;
int index = Collections.binarySearch(list, key);

System.out.println(index);
// Output: 1

Filling a Collection with a Value

The java.util.Collections.fill() method can be used to fill a collection with a specific value. The fill() method takes a List as an argument and a value and fills the entire List with the given value.


          
           
List<String> list = new ArrayList<>();
list.add("David");
list.add("Alice");
list.add("John");

String value = "Bob";
Collections.fill(list, value);

System.out.println(list);
// Output: [Bob, Bob, Bob]
