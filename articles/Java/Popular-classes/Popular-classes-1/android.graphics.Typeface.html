android.graphics.Typeface

android.graphics.Typeface is a class in Android that allows developers to easily apply custom fonts to their application. It provides methods for creating typefaces from files and resources, and for querying and applying the current typeface and font family. Creating and Retrieving Typefaces

The simplest way to create a Typeface is to use one of the static methods provided by android.graphics.Typeface. These methods allow you to create a Typeface from either a file or a resource. For example, to create a Typeface from a file: 

          
           
Typeface myTypeface = Typeface.createFromFile("myfont.ttf");
Alternatively, you can create a Typeface from a resource: 

          
           
Typeface myTypeface = Typeface.createFromAsset(getAssets(), "myfont.ttf");

String fontFamily = myTypeface.getFontFamily();

myTextView.setTypeface(myTypeface);
