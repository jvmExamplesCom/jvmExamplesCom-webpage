java.lang.reflect.Modifier

The java.lang.reflect.Modifier class provides static methods and constants to decode class and member access modifiers. This class is primarily used in the reflection package to set up the class and instance fields that are needed to specify runtime access permission to classes and members.

Examples of Usage

The following code snippets show how the Modifier class can be used to find and interpret access modifiers attached to class items.

Retrieve Class Modifiers


          
           
int modifiers = MyClass.class.getModifiers();
System.out.println(Modifier.isPublic(modifiers));
System.out.println(Modifier.isAbstract(modifiers));

The above code retrieves the modifiers associated with the class MyClass and prints out if it is public or abstract.

Retrieve Field Modifiers


          
           
Field field = MyClass.class.getDeclaredField("fieldName");
int modifiers = field.getModifiers();
System.out.println(Modifier.isFinal(modifiers));
System.out.println(Modifier.isStatic(modifiers));

The above code retrieves the modifiers associated with the field fieldName of the class MyClass, and prints out if it is final or static.

Retrieve Method Modifiers


          
           
Method method = MyClass.class.getDeclaredMethod("methodName");
int modifiers = method.getModifiers();
System.out.println(Modifier.isProtected(modifiers));
System.out.println(Modifier.isSynchronized(modifiers));

The above code retrieves the modifiers associated with the method methodName of the class MyClass, and prints out if it is protected or synchronized.

int modifiers = constructor.getModifiers();
