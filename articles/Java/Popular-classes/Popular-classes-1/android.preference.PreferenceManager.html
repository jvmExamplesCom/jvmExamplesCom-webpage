android.preference.PreferenceManager is a class from the Android framework used to help manage application preferences. This class provides convenience methods for accessing and modifying preferences that are stored in SharedPreferences instances: retrieving, setting, and listening for preference updates.

Then the getString() method is used to retrieve a preference value associated with the given key.

 PreferenceManager.getDefaultSharedPreferences(context);

 PreferenceManager.getDefaultSharedPreferences(context);

editor.apply();
