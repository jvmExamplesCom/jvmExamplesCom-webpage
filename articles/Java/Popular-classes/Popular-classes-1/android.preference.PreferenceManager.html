android.preference.PreferenceManager

android.preference.PreferenceManager is a class from the Android framework used to help manage application preferences. This class provides convenience methods for accessing and modifying preferences that are stored in SharedPreferences instances: retrieving, setting, and listening for preference updates.

The following code example shows how to retrieve a preference value using the getDefaultSharedPreferences() method to get a SharedPreferences instance associated with the given context. Then the getString() method is used to retrieve a preference value associated with the given key. 

          
           
SharedPreferences sharedPreferences = 
 PreferenceManager.getDefaultSharedPreferences(context);
String prefValue = sharedPreferences.getString("my_key", "");

Setting Preferences

The following code example shows how to set a preference value using the getDefaultSharedPreferences() method to get a SharedPreferences instance associated with the given context. Then the edit() method is used to begin editing the values in the SharedPreferences instance. The putString() method is then used to set a preference value associated with the given key. Finally, the apply() method is used to commit the edited values to the SharedPreferences instance. 

          
           
SharedPreferences sharedPreferences = 
 PreferenceManager.getDefaultSharedPreferences(context);
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.putString("my_key", "my_value");
editor.apply();
