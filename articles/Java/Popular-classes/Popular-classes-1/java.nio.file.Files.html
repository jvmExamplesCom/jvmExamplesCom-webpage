java.nio.file.Files

The java.nio.file.Files class provides a convenient way of manipulating files and file paths in Java. It provides utility methods for working with files, directories and other file-related objects such as symlinks. In this article, we will explore some of the methods provided by this class.

Creating Files and Directories

The java.nio.file.Files class provides convenient methods for creating files and directories. The following example shows how to create a file called 'hello.txt' in the current working directory:


          
           
Path path = Paths.get("hello.txt");
try {
    Files.createFile(path);
} catch (IOException e) {
    e.printStackTrace();
}
Similarly, we can create a directory using the 
createDirectory()
method as shown below: 

          
           
Path path = Paths.get("mydir");
try {
    Files.createDirectory(path);
} catch (IOException e) {
    e.printStackTrace();
}

Writing and Reading Files

The java.nio.file.Files class also provides convenient methods for writing to and reading from files. The following example shows how to write a simple string to a file:


          
           
String data = "This is some sample data";
Path path = Paths.get("data.txt");
try {
    Files.write(path, data.getBytes());
} catch (IOException e) {
    e.printStackTrace();
}
We can also read the contents of a file using the 
readAllBytes()
method as shown in the following example: 

          
           
Path path = Paths.get("data.txt");
try {
    byte[] data = Files.readAllBytes(path);
} catch (IOException e) {
    e.printStackTrace();
}

Copying, Moving and Deleting Files

The java.nio.file.Files class provides convenient methods for copying, moving and deleting files. To copy a file, we can use the 

copy()
method as follows: 

          
           
Path source = Paths.get("source.txt");
Path destination = Paths.get("destination.txt");
try {
    Files.copy(source, destination);
} catch (IOException e) {
    e.printStackTrace();
}
Similarly, we can move a file using the 
move()
method and delete a file using the 
delete()
method as shown below: 

          
           
Path source = Paths.get("source.txt");
Path destination = Paths.get("destination.txt");
try {
    Files.move(source, destination);
} catch (IOException e) {
    e.printStackTrace();
}

try {
    Files.delete(source);
} catch (IOException e) {
    e.printStackTrace();
}
