java.net.MalformedURLException

A 

          
           
java.net.MalformedURLException
is an exception which indicates that a malformed URL has been specified. This could be due to missing or incorrect parameters, invalid syntax, or a URL which is otherwise not correct. The exception is thrown when attempting to create an instance of a 
java.net.URL
object.

Example

To demonstrate, consider an example where we are attempting to connect to a resource via a malformed URL:


          
           
String urlStr = "https://www.mywebsite.com";
URL url = new URL(urlStr);

In this example, the URL string is missing the port parameter and as such, attempting to create an instance of 

java.net.URL
will result in a 

          
           
java.net.MalformedURLException
being thrown.

Catching the Exception

In order to handle the exception, it must be caught within a 

try
/ 
catch
block: 

          
           
try {
    String urlStr = "https://www.mywebsite.com";
    URL url = new URL(urlStr);
} catch (MalformedURLException e) {
    System.out.println("The specified URL is malformed: " + e.getMessage());
    e.printStackTrace();
}

This will print out a descriptive error message and the stack trace, allowing us to easily debug any issues with our URLs.

Summary

In summary, a 

          
           
java.net.MalformedURLException
is thrown when an invalid URL is specified. It can be caught within a 
try
/ 
catch
block to allow for debugging and further investigation.
