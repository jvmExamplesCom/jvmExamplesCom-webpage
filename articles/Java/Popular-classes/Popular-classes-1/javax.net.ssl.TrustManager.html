javax.net.ssl.TrustManager

The javax.net.ssl.TrustManager class is a core part of the Java Secure Socket Extension (JSSE). It provides a set of classes and methods for verifying the authenticity of an incoming communication. The TrustManager allows Java components to accept or reject particular incoming connections based on their trustworthiness.

Overview

A TrustManager is an entity that can validate an encrypted connection. It makes sure that data travelling over the connection is both secure and authentic. It is responsible for handling the certificate exchange process between two parties, also known as the handshake process.
