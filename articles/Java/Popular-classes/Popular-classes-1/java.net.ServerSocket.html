The ServerSocket class allows applications to listen for incoming requests from clients, providing a server-side view of the connection. The ServerSocket class provides two constructors. The ServerSocket class also provides methods for managing server sockets, such as setSoTimeout() and setReuseAddress().

Example

In the following example, a simple web server is created that listens for connections on port 8080 and sends an HTML page in response. The example also demonstrates how to set a timeout for the server socket, allowing it to automatically close after 5 seconds of inactivity.


          
           
ServerSocket server = new ServerSocket(8080);
server.setSoTimeout(5000);
while (true) {
    System.out.println("Waiting for connection...");
    Socket connection = server.accept();
    System.out.println("Connection established.");

    // Send HTML page
    PrintWriter out = new PrintWriter(connection.getOutputStream());
    out.println("<html>");
    out.println("<head><title>My Web Server</title></head>");
    out.println("<body>Hello World!</body>");
    out.println("</html>");
    out.close();

    connection.close();
}

In this example, the server is set up to automatically close after 5 seconds of inactivity. This ensures that the server is not left running unnecessarily and wasting resources.
