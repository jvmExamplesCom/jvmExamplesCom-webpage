Introduction to Java.time.Duration

Java.time.Duration is a class in the Java Standard Edition 8 date-time package (java.time). Duration is designed to store time as a number of seconds and nanoseconds. Durations are commonly used to measure durations between two points in time, such as the time taken for an operation to complete.

Creating a Duration

// Create a Duration of one minute
