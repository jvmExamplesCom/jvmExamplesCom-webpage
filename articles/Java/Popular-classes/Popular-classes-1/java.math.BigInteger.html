java.math.BigInteger

java.math.BigInteger is a class that provides methods for operations on immutable arbitrary precision signed integers (from the Java API documentation). It is used to represent integers with arbitrarily large magnitudes, which is useful for calculations that require numbers that are too large for longs.

Constructors

java.math.BigInteger offers constructors that take strings, arrays, and other BigIntegers as input. Here are some examples:


          
           
BigInteger b1 = new BigInteger("123456789"); //from a string

byte[] b2Array = {1, 2, 3, 4, 5}; //from an array
BigInteger b2 = new BigInteger(b2Array);

BigInteger b3 = new BigInteger("12345678901234567890"); //from another BigInteger
BigInteger b4 = new BigInteger(b3);

Methods

java.math.BigInteger contains many methods for various arithmetic and comparison operations. Here are some examples of common operations:


          
           
BigInteger b1 = new BigInteger("123456789");
BigInteger b2 = new BigInteger("987654321");

b1.add(b2); //sum of b1 and b2
b1.subtract(b2); //difference of b1 and b2
b1.multiply(b2); //product of b1 and b2
b1.divide(b2); //quotient of b1 and b2
b1.mod(b2); //b1 mod b2

It is also possible to do comparison operations on BigIntegers. Here are some examples:


          
           
BigInteger b1 = new BigInteger("123456789");
BigInteger b2 = new BigInteger("987654321");

b1.compareTo(b2); //compares magnitude of b1 and b2
b1.equals(b2); //returns true if b1 and b2 are equal
b1.max(b2); //returns the largest of b1 and b2
b1.min(b2); //returns the smallest of b1 and b2

Conclusion

java.math.BigInteger provides powerful methods for working with integers of arbitrary precision. Its flexibility and wide range of supported methods make it a great choice for any calculations requiring numbers larger than those handled by longs.
