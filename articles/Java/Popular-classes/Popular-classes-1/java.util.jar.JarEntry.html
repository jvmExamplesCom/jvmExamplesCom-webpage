java.util.jar.JarEntry

The 

          
           
java.util.jar.JarEntry
class is a subclass of 

          
           
java.util.zip.ZipEntry
. It represents an entry in a JAR file (Java archive) which is a collection of files, directories, or both that have been compressed into a single file. The 
JarEntry
class provides methods for obtaining compressed length, original length, name, data, etc., of entries within a JAR file.

Constructors

The 
JarEntry
class has two constructors: 

          
           
JarEntry(String name) 
// Constructs a new JarEntry for the specified JAR file entry name.

JarEntry(ZipEntry ze) 
// Constructs a new JarEntry from the specified ZipEntry object.

Methods

The following methods are contained in the 
JarEntry
class: 

          
           
getAttributes() 
// Returns an attribute view of this entry's attributes.

getCertificates() 
// Returns an array of certificates for the entry, or null if none.

getCodeSigners() 
// Returns an array of CodeSigner objects for the entry, or null if none.

getCompressedSize() 
// Returns the compressed size of the entry data, or -1 if not known.

getCreationTime() 
// Returns the ZipEntry associated with this JAR file entry.

getExtra() 
// Returns the extra information for the entry, or null if none.

getName() 
// Returns the name of the JAR file entry.

getOriginalSize() 
// Returns the original size of the entry data, or -1 if not known.

getTime() 
// Returns the time at which the file represented by this JAR entry was last modified.

isDirectory() 
// Returns true if this JAR file entry is a directory, and false otherwise.

isJAR() 
// Returns true if this JAR file entry is a JAR entry, and false otherwise.

isNative() 
// Returns true if this JAR file entry is a native library, and false otherwise.

Example

The following example demonstrates the use of the 
JarEntry
class: 

          
           
import java.util.jar.*;
import java.io.*;

public class JarExample {

    public static void main(String[] args) {
        try {
            JarFile jarFile = new JarFile("myJarFile.jar");

            // Get a list of all JAR entries in the jar file
            Enumeration<JarEntry> entries = jarFile.entries();

            // Iterate over the entries
            while (entries.hasMoreElements()) {
                JarEntry entry = entries.nextElement();
                System.out.println("Name: " + entry.getName());
                System.out.println("Type: " + entry.isDirectory());
                System.out.println("Size: " + entry.getSize());
            }
        } catch (IOException ex) {
            ex.printStackTrace();
        }
    }
}
