Introduction to java.lang.reflect.InvocationTargetException

The java.lang.reflect.InvocationTargetException is an unchecked exception thrown by the Java virtual machine during the invocation of a method or constructor in order to indicate that the method or constructor has thrown an exception or an error during its execution. By using reflection, it is possible to invoke methods or constructors that are not accessible by any other means, such as private methods or constructors. This can lead to errors which can be harder to track down than other kinds of exceptions.

An InvocationTargetException can occur when the target of an invocation throws an exception. The class also contains a getCause() method, which allows access to the underlying exception or error that caused the problem.

Example 1: Constructor Throwing an Exception

In this example, we create a class called MyClass. This class has a private constructor that always throws an exception. We then use the Java Reflection API to invoke this constructor and catch the resulting InvocationTargetException.


          
           
public class MyClass {

    // Private constructor which always throws an exception
    private MyClass() throws Exception {
        throw new Exception("This constructor always fails!");
    }

    public static void main(String[] args) {
        try {
            // Use the Java Reflection API to invoke the private constructor
            Constructor ctor = MyClass.class.getDeclaredConstructor();
            ctor.setAccessible(true);
            Object obj = ctor.newInstance();
        } catch (NoSuchMethodException | 
                IllegalAccessException | 
                InvocationTargetException | 
                InstantiationException e) {
            // Catch the InvocationTargetException
            if (e instanceof InvocationTargetException) {
                InvocationTargetException ite = (InvocationTargetException) e;
                // Print out the cause
                System.out.println(ite.getCause().getMessage());
            }
        }
    }

}
The output of this example is: 

          
           
This constructor always fails!

public class MyClass {
