Introduction to Java.util.Map.Entry

Java.util.Map.Entry is an interface which is a member of the Java Collections Framework. It provides a simple way to access key-value pairs in a map. This interface enables a map to be iterated over its entries, whether they are held in the map or stored elsewhere, and to perform basic operations like add, remove, modify, clear and search for items. The Map.Entry interface is a generic type that has two type parameters; one for the key and one for the value.

Example Code and Output showing the use of Java.util.Map.Entry

We will start by constructing a map using the Map.Entry interface. The following code example shows how to create and populate a Map.Entry instance with sample data:


          
           
Map<String, Integer> m = new HashMap<>();
m.put("one", 1);
m.put("two", 2);
m.put("three", 3);

//Iterating over entries
for (Map.Entry<String, Integer> entry : m.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}
The output of this code snippet is as follows: 

          
           
one = 1
two = 2
three = 3

The above code example shows how to iterate over all of the entries in a map, printing out each key and its associated value to the console.

The Map.Entry interface also provides a convenient way to modify the values in a map. The following example demonstrates how to use the setValue() method to modify an existing entry in the map: 


          
           
Map<String, Integer> m = new HashMap<>();
m.put("one", 1);
m.put("two", 2);
m.put("three", 3);

//Iterating over entries
for (Map.Entry<String, Integer> entry : m.entrySet()) {
    if (entry.getKey().equals("two")) {
        entry.setValue(20);
    }
    System.out.println(entry.getKey() + " = " + entry.getValue());
}
The output of this code is as follows: 

          
           
one = 1
two = 20
three = 3

As can be seen from the output above, the value associated with the key "two" has been changed to 20.

The Map.Entry interface also provides a method for removing specific entries from a map. The following example shows how to use the remove() method to remove an entry from a map: 


          
           
Map<String, Integer> m = new HashMap<>();
m.put("one", 1);
m.put("two", 2);
m.put("three", 3);

//Iterating over entries
Iterator<Map.Entry<String, Integer>> entries = m.entrySet().iterator();
while (entries.hasNext()) {
    Map.Entry<String, Integer> entry = entries.next();
    if (entry.getKey().equals("two")) {
        entries.remove();
    }
    System.out.println(entry.getKey() + " = " + entry.getValue());
}
The output of this code snippet is as follows: 

          
           
one = 1
three = 3

As can be seen from the output, the entry with the key "two" has been removed from the map.

Finally, the Map.Entry interface also provides a convenient way to search for entries in a map. The following example shows how to use the containsValue() method to search for an entry in a map:


          
           
Map<String, Integer> m = new HashMap<>();
m.put("one", 1);
m.put("two", 2);
m.put("three", 3);

boolean found = m.entrySet().containsValue(2);

Found? true

As can be seen from the output, the containsValue() method returns true if it finds a matching value in the map.
