java.util.LinkedHashSet

The java.util.LinkedHashSet class is a HashSet implementation that keeps track of the order in which its elements were inserted. This implementation is based on a linked list, so all of the standard Collection operations will work properly. LinkedHashSet also provides an iterator over its elements that returns them in their insertion order. The LinkedHashSet class is a member of the Java Collection Framework.

Create LinkedHashSet

To create a LinkedHashSet, you can use the following syntax:


          
           
LinkedHashSet<String> set = new LinkedHashSet<>();

Add Elements to LinkedHashSet

You can add elements to a LinkedHashSet using the following methods:


          
           
set.add("foo");
set.add("bar");
set.add("baz");

These methods will add the elements to the set in the same order they were specified. If an element already exists in the set, it won't be added again.

Iterate Over LinkedHashSet Elements

You can iterate over the elements of a LinkedHashSet in their insertion order using the following syntax:


          
           
for(String s : set) {
    System.out.println(s);
}
This will print out: 

          
           
foo
bar
baz

Search for Elements in LinkedHashSet

You can search for elements in a LinkedHashSet using the contains() method. This method will return true if the element exists in the set, and false otherwise. For example:


          
           
boolean containsFoo = set.contains("foo"); // true
boolean containsQux = set.contains("qux"); // false

Remove Elements from LinkedHashSet

You can remove elements from a LinkedHashSet using the remove() method. This method takes the element to be removed as its argument and will return true if the element was found and removed, or false if the element was not found. For example:


          
           
boolean removedFoo = set.remove("foo"); // true
boolean removedQux = set.remove("qux"); // false

Clear a LinkedHashSet

You can clear all elements from a LinkedHashSet using the clear() method. This method will return all elements of the set, removing them from the LinkedHashSet. For example:


          
           
set.clear();
