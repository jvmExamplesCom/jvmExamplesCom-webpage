android.support.test.runner.AndroidJUnit4

AndroidJUnit4 is a class in the Android Support Test runner, part of the Android Support Library. It provides a JUnit4-based test runner for both local unit tests as well as instrumented tests. This class is an extension of the standard JUnit4 test runner, which provides support for running Android tests on the Android framework.

Example Usage

To use the AndroidJUnit4 test runner, you must first create a 


          
           
@RunWith(AndroidJUnit4.class)
annotation within your test class. This annotation tells the test runner to use AndroidJUnit4 when running your test. 

          
           
@RunWith(AndroidJUnit4.class)
public class ExampleTest {
    // Test methods go here
}

Once you have declared the runWith annotation, you can now annotate methods with @Test to indicate that they are test methods. You may also specify other options such as timeout and expected exception.


          
           
@Test
public void exampleTest() {
    // Test code goes here
}

Running Tests

To manually run tests with the AndroidJUnit4 test runner, use the gradle command 


          
           
./gradlew connectedCheck
. This will run all instrumented tests, including those annotated with AndroidJUnit4.

Alternatively, you can also launch Android Studio, select 'Run' from the menu, and select either a single method or an entire test class to be executed.

Assertions

assertEquals(expected, actual);
