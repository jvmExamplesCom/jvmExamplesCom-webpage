org.springframework.util.Assert

org.springframework.util.Assert is a utility class that provides convenient methods for asserting conditions in code such as preconditions, postconditions, and invariants. The methods provided by org.springframework.util.Assert provide extra functionality to the standard Java assertions, allowing for more powerful expression of program logic. The following code examples demonstrate the various ways in which the methods of org.springframework.util.Assert can be used.

assertTrue()

The 
assertTrue()
method allows for a boolean expression to be tested. If the expression evaluates to false, an exception will be thrown. 

          
           
assertTrue("Expected true but was false", someBooleanExpression);

assertFalse()

assertFalse()
