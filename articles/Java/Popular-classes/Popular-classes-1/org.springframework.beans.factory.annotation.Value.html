Introduction to org.springframework.beans.factory.annotation.Value

org.springframework.beans.factory.annotation.Value is a Spring annotation used to inject a value property into a bean. It provides a convenient way to access and assign specific properties of values to bean fields or methods without the need to explicitly specify the property name. This annotation is often used when working with configuration files and when dealing with multiple environments or configurations.

Example Usage

In this example, we will create a simple bean that uses the Value annotation to inject a property from an application.properties file. For the sake of this example, we will assume the application.properties file looks like the code snippet below:


          
           
my.property.name=myvalue

We then create a bean with a property annotated with @Value and the location of the property in the file:


          
           
@Component
public class MyBean {
 
    @Value("${my.property.name}")
    private String value;
    ...
}

When the context is created, the Value annotation is used to inject the value from the application.properties file. In our example, the value of the 

value
field in our bean would be set to 
myvalue
.

Using Default Values

The @Value annotation also supports the use of default values. If no value is found in the application.properties file, a value can be specified as a parameter to the @Value annotation:


          
           
@Value("${my.property.name:defaultValue}")
private String value;
In this example, if the 
my.property.name
is not found in the application.properties file, the value of the 
value
field in our bean will be set to the default value of 
defaultValue
.

Conclusion

The @Value annotation provides a convenient way to inject values from configurations and/or property files into beans. With the ability to provide default values, it is possible to have multiple environments or configurations that can be used without the need to explicitly specify the property names for each environment.
