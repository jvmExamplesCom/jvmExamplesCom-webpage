Java.time.LocalDateTime

The java.time.LocalDateTime class is a combination of two classes, the java.time.LocalDate and the java.time.LocalTime classes. It provides a single API for dealing with dates, times, and date-times. This class is immutable and thread-safe. We can create instances of this class by using various factory methods or the parse() method.

Example 1: Creating an instance of LocalDateTime using now() Method


          
           
LocalDateTime time = LocalDateTime.now();
System.out.println("Current DateTime: " + time);

Output:
Current DateTime: 2019-12-25T10:11:45.154

Example 2: Creating an instance of LocalDateTime using of() Method


          
           
LocalDateTime time = LocalDateTime.of(2019, Month.DECEMBER, 25, 10, 10);
System.out.println("date-time: " + time);

Output:
date-time: 2019-12-25T10:10

Example 3: Creating an instance of LocalDateTime using parse() Method


          
           
String date = "2019-12-25T10:11:45";
LocalDateTime ldt = LocalDateTime.parse(date);
System.out.println("date-time: " + ldt);

Output:
date-time: 2019-12-25T10:11:45

Example 4: Creating an instance of LocalDateTime using ofInstant() Method


          
           
Instant instant = Instant.now();
LocalDateTime ldt = LocalDateTime.ofInstant(instant, ZoneId.systemDefault());
System.out.println("date-time: " + ldt);

Output:
date-time: 2019-12-25T10:14:23.805

Example 5: Retrieving Date and Time components from an Instance of LocalDateTime


          
           
LocalDateTime dt = LocalDateTime.now();
int year = dt.getYear();
Month month = dt.getMonth();
int day = dt.getDayOfMonth();
int hour = dt.getHour();
int min = dt.getMinute();
int sec = dt.getSecond();

System.out.println("year = "+year);
System.out.println("month = "+month);
System.out.println("day = "+day);
System.out.println("hour = "+hour);
System.out.println("min = "+min);
System.out.println("sec = "+sec);

Output:
year = 2019
month = DECEMBER
day = 25
hour = 10
min = 17
sec = 13

Example 6: Converting an Instance of LocalDateTime to String


          
           
LocalDateTime time = LocalDateTime.now();
DateTimeFormatter fmt = DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm:ss");
String strTime = time.format(fmt);
System.out.println("Time in String : " + strTime);

Output:
Time in String : 25-12-2019 10:19:22

Example 7: Modifying a LocalDateTime Instance


          
           
LocalDateTime time = LocalDateTime.now();
LocalDateTime futureTime = time.plusYears(2).plusMonths(3).plusDays(4).plusHours(5).minusSeconds(20);
System.out.println("Future DateTime: " + futureTime);

Output:
