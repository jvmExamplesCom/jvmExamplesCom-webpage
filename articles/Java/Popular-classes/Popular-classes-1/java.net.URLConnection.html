Java.net.URLConnection

Java.net.URLConnection is a class in the Java Networking API which enables applications to use the protocol of their choice when accessing web-based resources, such as web pages, audio files, and videos. It also allows for the configuration of timeouts, as well as other parameters to be set for each request.

Methods

The URLConnection class contains many methods which enable developers to customize and configure requests. These methods are listed below:

  • getContentType(): Returns the content type for the connection.
  • setRequestProperty(): Sets the value of the request header field.
  • getInputStream(): Returns an InputStream to read from the connection.
  • getOutputStream(): Returns an OutputStream to write to the connection.
  • connect(): Opens a connection to the remote resource.
  • setAllowUserInteraction(): Allows user interaction with the connection.
  • setConnectTimeout(): Sets the timeout value for connecting.
  • setReadTimeout(): Sets the timeout value for reading from the connection.

Example

Below is a simple example of how to use URLConnection to open a connection to a web page and read its contents:


          
           
// Create URL object
URL url = new URL("http://www.example.com");

// Open connection
URLConnection conn = url.openConnection();

// Read response
BufferedReader in = new BufferedReader(new InputStreamReader(conn.getInputStream()));

String line;
while ((line = in.readLine()) != null) {
    System.out.println(line);
}
in.close();

This example will open a connection to the URL specified and read all of the contents of the page. It can then be processed however the developer desires.

Conclusion

The URLConnection class provides a powerful way to access web resources. By leveraging the various methods it provides, developers can create robust applications that perform custom HTTP requests and retrieve data from remote sites.
