Java.awt.event.ActionEvent

The 

          
           
java.awt.event.ActionEvent
class is a type of event that occurs when an action is performed on a specific component of the Graphical User Interface (GUI). It's a subclass of the superclass

addActionListener(ActionListener l)

Example

Here is a simple example of how to use 

          
           
ActionEvent

JButton okButton = new JButton("OK");
