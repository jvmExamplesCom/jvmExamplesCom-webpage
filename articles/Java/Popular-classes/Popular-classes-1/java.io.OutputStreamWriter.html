java.io.OutputStreamWriter

A Java.io.OutputStreamWriter is a subclass of the abstract class java.io.Writer, which is used for writing stream of characters. It uses an OutputStream as its output source. The OutputStreamWriter wraps a character-based output stream and provides a convenient way to write a stream of characters. It also offers the flexibility of choosing the charset that the output is written in.

Constructors

The OutputStreamWriter class has two constructors:


          
           
OutputStreamWriter(OutputStream out)
OutputStreamWriter(OutputStream out, Charset cs)

The first constructor creates an OutputStreamWriter with the provided OutputStream and the default charset. The second constructor creates an OutputStreamWriter with the provided OutputStream and charset.

Methods

Below are a few of the methods available for OutputStreamWriter class:


          
           
void flush()
void close()
String getEncoding()
void write(int c)
The 

          
           
flush()
method flushes the output stream. The 

          
           
close()
method closes the stream and releases any resources associated with it. The 

          
           
getEncoding()
method returns the name of the charset being used. The 

          
           
write(int c)
method writes the specified character to the OutputStreamWriter.

Example

Below is an example illustrating how to use OutputStreamWriter:


          
           
import java.io.*;
  
public class Main {
    public static void main(String[] args) throws IOException {
        File file = new File("sample.txt");
        OutputStream os = new FileOutputStream(file);
        Writer writer = new OutputStreamWriter(os,"UTF-8");
        writer.write("Hello World!");
        writer.close();
        os.close();
    }
}

In this example, we create a File instance and a FileOutputStream object. Then, we create an OutputStreamWriter object with the OutputStream and charset provided. Finally, we write the string “Hello World!” to the file and close both the Writer and OutputStream objects.
