android.support.v4.view.ViewCompat

Android's ViewCompat class provides static methods to make working with views in the Android framework easier. This library is located at android.support.v4.view, and contains a number of classes such as ViewPropertyAnimatorCompat, AccessibilityDelegateCompat, NestedScrollingParent, and others. The ViewCompat class itself provides a broad range of methods to help developers work with Android views, including methods for animating views, setting or getting clip bounds, checking whether a view has focus or hover, setting or getting visibility states, and so much more.

isAttachedToWindow()
This method returns true if the view is currently attached to a window, and false otherwise. 

          
           
boolean isAttached = ViewCompat.isAttachedToWindow(view);
getAlpha()
This method returns the current alpha value of the view, which is a float value between 0 (completely transparent) and 1 (completely opaque).
