Introduction to org.springframework.beans.factory.annotation.Autowired

org.springframework.beans.factory.annotation.Autowired is an annotation used in the Java programming language to allow for the dependency injection of components and beans. This annotation is used to indicate a constructor, field or a setter method as a method for autowiring the beans properties.

Example of Autowired Annotation

The following example demonstrates the usage of the @Autowired annotation:


          
           
@Autowired
private SomeBean someBean;
Or 

          
           
@Autowired
public void setSomeBean(SomeBean someBean) {
    this.someBean = someBean;
}

Attributes of Autowired Annotation

The @Autowired annotation has two optional attributes: required and autowireCandidates.

required Attribute

The “required” attribute is used to indicate that the target bean of the @Autowired annotation must be instantiated or else an exception will be thrown.


          
           
@Autowired(required=true)
private SomeBean someBean;

If the required attribute is set to “false” then the property can remain uninitialized if no matching bean is found.


          
           
@Autowired(required=false)
private SomeBean someBean;

autowireCandidate Attribute

The “autowireCandidate” attribute is used to indicate that the target bean of the @Autowired annotation should be considered as a candidate for autowiring. This attribute takes a boolean value, with “true” indicating that the target bean should be considered as a candidate for autowiring and “false” indicating that it should not be considered as a candidate.


          
           
@Autowired(autowireCandidate=true)
private SomeBean someBean;

Autowiring using Qualifiers

A qualifier can be used to further refine the autowiring process. When a qualifier is used with the @Autowired annotation, the autowiring process will only match beans which have a matching qualifier value.


          
           
@Autowired
@Qualifier("someBean")
private SomeBean someBean;

The qualifiers can also be used to determine which implementation of an interface should be used for autowiring.


          
           
@Autowired
@Qualifier("someImpl")
private SomeInterface someInterface;

Conclusion

The @Autowired annotation is an important tool for implementing dependency injection.
