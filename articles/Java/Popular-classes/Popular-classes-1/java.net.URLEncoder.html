Java.net.URLEncoder

Java.net.URLEncoder is a class in Java used for encoding URLs. It is used for encoding parameters and sending them over the web. This class encodes an input string using the “application/x-www-form-urlencoded” MIME format and returns a String in which all non-ASCII characters are replaced with their hexadecimal equivalents.

Example 1

Below is an example of how to use the encode() method of the URLEncoder class.


          
           
String encodedString = URLEncoder.encode("MyParams");
System.out.println(encodedString);
The output would be: 

          
           
MyParams%3D

Example 2

Below is another example of how to use the encode() method of the URLEncoder class.


          
           
String myParams = "name=John&amp;age=40"; 
String encodedString = URLEncoder.encode(myParams);
System.out.println(encodedString);
The output would be: 

          
           
name%3DJohn%26age%3D40
