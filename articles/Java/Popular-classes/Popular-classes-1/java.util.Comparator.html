Introduction to Java.util.Comparator

Java.util.Comparator is an interface in the Java API that provides a natural ordering for a collection of objects. It is used to sort a collection of objects based on their properties. This interface defines two methods for comparing two objects: compare() and equals(). The compare() method returns an integer, which indicates whether one object is greater than, equal to, or less than the other. The equals() method is used to determine if two objects are equal.

Implementing Comparator Interface

The Comparator interface must be implemented in order to use its functions. To use the Comparator interface, create a class that implements it, and then override the compare() method. For example, consider a class called Person which has fields for first name, last name, and age, and define a PersonComparator class that implements Comparator:


          
           
public class Person {
  private String firstName;
  private String lastName;
  private int age;
  
  // Getters and Setters here
}

public class PersonComparator implements Comparator<Person> {
  @Override
  public int compare(Person p1, Person p2) {
    return p1.getAge() - p2.getAge();
  }
}

In this example, the compare() method compares the ages of two Person objects. Where the difference between the two ages is positive, p1 is greater than p2, and where the difference is negative, p2 is greater than p1. The equals() method is never overridden in this example, so the default implementation from the Object class is used.

Using Comparator with Collections

Once a Comparator has been created and the compare() method is overridden, it can be used with the collections framework. It can be used to sort a List using the 

Collections.sort()
method, passing a List and a Comparator as arguments. This will sort the List by the criteria specified in the Comparator's compare() method. Consider the following code: 

          
           
List<Person> people = Arrays.asList(
  new Person("John", "Doe", 30),
  new Person("Jane", "Doe", 20),
  new Person("Jim", "Smith", 40)
);

Collections.sort(people, new PersonComparator());

In this example, the List of Person objects is sorted by age, as specified in the compare() method of the PersonComparator class. The result is a sorted List of Person objects by age, with John Doe first (age 30), Jane Doe second (age 20), and Jim Smith third (age 40).

Using Comparator with Lambdas

Since Java 8, Comparator can also be used with lambda expressions. Instead of creating a separate class to implement the interface, the Comparator can be passed as a lambda expression. Consider the following example:


          
           
List<Person> people = Arrays.asList(
  new Person("John", "Doe", 30),
  new Person("Jane", "Doe", 20),
  new Person("Jim", "Smith", 40)
);

Collections.sort(people, (p1, p2) -> p1.getAge() - p2.getAge());

This code has the same outcome as the code above, with the Person objects sorted by age. In this example, the Comparator is passed as a lambda expression to the Collections.sort() method. This allows for more concise and readable code, as well as better performance, since less code needs to be written and compiled for the Comparator.
