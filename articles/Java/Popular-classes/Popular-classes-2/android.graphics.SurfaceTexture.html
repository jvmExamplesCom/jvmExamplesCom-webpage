android.graphics.SurfaceTexture

SurfaceTexture is an Android API class in the package 

android.graphics
which allows applications to produce data from one component of the Android system to another component. It is part of the Android graphics framework and provides an easy way for applications to produce frames of data without having to worry about managing thread synchronization and buffering. It offers a simple API to set up the buffer queue, attach it to media sources or graphics processors and to receive the frames of data produced.

Creating SurfaceTexture

Creating a SurfaceTexture instance is done by calling the constructor with a desired width and height.


          
           
// Create SurfaceTexture with desired width and height
SurfaceTexture surfaceTexture = new SurfaceTexture(width, height);

Updating SurfaceTexture

The ‘updateTexImage’ method can be used to update a SurfaceTexture object with the contents of a new image buffer. This should be called each time the buffer being used to produce the image has been filled with new image data.


          
           
// Update the SurfaceTexture using the buffer 
surfaceTexture.updateTexImage();

Getting Transform Matrix

The 'getTransformMatrix' method can be used to retrieve a 4x4 transform matrix which is equivalent to the current texture matrix that was set by calling 'setTransformMatrix'. The result can be used to apply transformations to the image before display.


          
           
// Get transform matrix 
float[] mtx = new float[16]; 
surfaceTexture.getTransformMatrix(mtx);

Releasing SurfaceTexture
