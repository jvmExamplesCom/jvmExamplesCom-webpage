android.widget.HorizontalScrollView

android.widget.HorizontalScrollView is a ViewGroup object that allows the user to scroll horizontally within the view. It can be used in conjunction with a ListView or any other layout. The widget provides the ability to scroll along the x axis by swiping side to side. This is useful when displaying a large dataset that needs to be horizontally scrolled.

Example

The following example code creates a HorizontalScrollView with a ListView inside:


          
           
HorizontalScrollView hsv = new HorizontalScrollView(context);
ListView lv = new ListView(context);
// add items to ListView here
hsv.addView(lv);

To set up the scroll listener, we use the setOnScrollChangeListener() method:


          
           
hsv.setOnScrollChangeListener(new OnScrollChangeListener() {
    @Override
    public void onScrollChange(View v, int scrollX, int scrollY, int oldScrollX, int oldScrollY) {
        // do something here
    }
});

To set the scrolling speed, we use the setOverScrollMode() method:


          
           
hsv.setOverScrollMode(View.OVER_SCROLL_NEVER);

Finally, to enable horizontal scrolling we call the requestDisallowInterceptTouchEvent() method on the ListView:


          
           
lv.requestDisallowInterceptTouchEvent(true);
