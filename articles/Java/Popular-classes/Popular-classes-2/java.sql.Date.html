Java.sql.Date

Java.sql.Date is a subclass of java.util.Date, and it represents a date in the format yyyy-mm-dd. It is used mainly for storing data in databases like SQLite. This class provides various constructors and methods which can be used to perform various operations on dates.

Constructors

There are two types of constructors provided by Java.sql.Date class - one with no parameters and other with three parameters.

  • No parameter constructor - This constructor creates a Date object corresponding to the current date and time.
  • Three parameter constructor - This constructor creates a Date object corresponding to the specified parameters - year, month and day.

Methods

Following are some of the methods provided by Java.sql.Date class - 

  • 
            
             
void setTime(long time)
    - Sets the Date object corresponding to the specified long value.
  • long getTime()
    - Returns the long value representing the date.
  • String toString()
    - Returns the string representation of the date in the format "yyyy-mm-dd". 
  • int getYear()
    - Returns the year part of the date.
  • int getMonth()
    - Returns the month part of the date.
  • int getDay()
    - Returns the day part of the date.
  • 
            
             
boolean equals(Object obj)
    - Compares the specified Object with the Date object and returns true if they are equal, false otherwise.
  • 
            
             
int compareTo(Date date)
    - Compares the specified Date object with the current Date object and returns a negative integer, zero or a positive integer if the current Date object is less than, equal to or greater than the specified Date object.

Example

This example shows how to use the methods provided by Java.sql.Date class. 


          
           
import java.sql.Date;

public class Test {
    public static void main(String[] args) {
        // create a Date object with no parameters
        Date date = new Date();
        System.out.println(date); // prints the current date and time

        // create a Date object with three parameters
        Date date1 = new Date(2020, 03, 20);
        System.out.println(date1); // prints 2020-03-20

        // using setTime(long time) method
        long millis = date1.getTime();
        date.setTime(millis);
        System.out.println(date); // prints 2020-03-20

        // using getTime() method
        long time = date1.getTime();
        System.out.println(time); // prints the long value of the date

        // using toString() method
        String str = date.toString();
        System.out.println(str); // prints 2020-03-20

        // using getYear(), getMonth(), and getDay() methods
        int year = date1.getYear();
        int month = date1.getMonth();
        int day = date1.getDay();
        System.out.println("year: "+year+", month: "+month+", day: "+day); // prints year: 2020, month: 3, day: 20

        // using equals() method
        Date date2 = new Date(2020, 03, 20);
        boolean isEqual = date1.equals(date2);
        System.out.println(isEqual); // prints true

        // using compareTo() method
        Date date3 = new Date(2020, 02, 10);
        int result = date1.compareTo(date3);
        System.out.println(result); // prints 1
    }
}
