Introduction to com.fasterxml.jackson.core.JsonGenerator

The 

          
           
com.fasterxml.jackson.core.JsonGenerator
class is an important part of the Jackson Core package which provides programmatic methods for writing JSON content from Java. JsonFactory factory = new JsonFactory(); 
JsonGenerator jsonGenerator = factory.createJsonGenerator(System.out);
