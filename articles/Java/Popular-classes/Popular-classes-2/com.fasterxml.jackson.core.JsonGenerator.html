Introduction to com.fasterxml.jackson.core.JsonGenerator

The 

          
           
com.fasterxml.jackson.core.JsonGenerator
class is an important part of the Jackson Core package which provides programmatic methods for writing JSON content from Java. The JsonGenerator also has the ability to write JSON as a stream, making it easier to work with large amounts of data. JsonFactory factory = new JsonFactory(); 
JsonGenerator jsonGenerator = factory.createJsonGenerator(System.out);
