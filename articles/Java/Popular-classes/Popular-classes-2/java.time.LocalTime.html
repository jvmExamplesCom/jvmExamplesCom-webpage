Java.time.LocalTime

Java.time.LocalTime is a part of the JavaTime library which enables developers to work with dates and times using a concise, readable format. It is one of the most commonly used classes in the library and contains a variety of methods for instantiating, manipulating, and formatting times.

Instantiation

The LocalTime class provides a few different ways to instantiate it. The most convenient way is to use the static method now(). 


          
           
LocalTime now = LocalTime.now();
System.out.println(now); //20:09:41.071

Another way of creating a LocalTime instance is when you have a complete time, in the format of hour, minute, second, and nanoseconds. 


          
           
LocalTime time = LocalTime.of(16, 20, 15);
System.out.println(time); //16:20:15

You can also instantiate it with a string that represents a valid ISO-8601 format. 


          
           
LocalTime anotherTime = LocalTime.parse("14:40:45.487");
System.out.println(anotherTime); //14:40:45.487

Manipulation

Once a LocalTime instance is created, one of the most useful methods is withHour, which takes an int parameter and sets the hour portion of the Time. 


          
           
LocalTime localTime = LocalTime.now();
localTime = localTime.withHour(22);
System.out.println(localTime); //22:09:41.071

Another useful manipulation is plusHours, which adds the given number of hours to the LocalTime instance. 


          
           
LocalTime localTime = LocalTime.now();
localTime = localTime.plusHours(2);
System.out.println(localTime); //22:09:41.071

LocalTime also supports other methods such as minusHours, withMinute, plusMinutes, minusMinutes, withSecond, plusSeconds, minusSeconds, withNano, plusNanos, and minusNanos for further manipulation.

Formatting

One of the great features of LocalTime is that it provides formatting functions that enable developers to display times exactly as they'd like. For example, one can use the DateTimeFormatter class to construct a pattern in which the time should be displayed. 


          
           
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("hh:mm a");
LocalTime localTime = LocalTime.now();
System.out.println(localTime.format(formatter)); //08:09 PM

In addition, methods such as toLocalDate, toLocalDateTime, and toString also provide different formats for displaying times. With these methods, it's easy to display times in different formats.

System.out.println(time.toString()); //20:09:41.071
System.out.println(time.toLocalDateTime()); //2020-05-29T20:09:41.071
