android.os.Vibrator
class is used to control the vibrator on a device. It enables applications to cause the device to vibrate for a specified length of time or to do so periodically.

Vibrator

Vibrator vibrator = (Vibrator) getSystemService(Context.VIBRATOR_SERVICE);

Once an object of the 

          
           
Vibrator
class has been created, it can be used to vibrate the device. There are two methods to do this: 

          
           
vibrate()
and 

          
           
vibrate()
.

vibrate()

vibrate()
method takes an array of long values as its parameter. This array specifies the sequence of on and off periods with which the device should vibrate. For example: 

          
           
long[] pattern = {0, 100, 1000, 300, 200, 100, 500, 200, 100};
vibrator.vibrate(pattern, 0);
In the example above, the device will vibrate for 0ms, then pause for 100ms, then vibrate for 1000ms and so on.

vibrate()

vibrate()
method takes two long values as parameters. The first value defines how long the device should vibrate in milliseconds and the second value defines how long the device should pause once it has finished vibrating. For example: 

          
           
vibrator.vibrate(1000, 500);
In the example above, the device will vibrate for 1000ms and then pause for 500ms before repeating.

The

cancel()
