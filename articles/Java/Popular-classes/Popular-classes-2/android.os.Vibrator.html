android.os.Vibrator

Vibrator
class has been created, it can be used to vibrate the device. There are two methods to do this: 

          
           
vibrate()
and 

          
           
vibrate()

vibrate()

vibrate()

long[] pattern = {0, 100, 1000, 300, 200, 100, 500, 200, 100};
vibrator.vibrate(pattern, 0);
