android.view.Surface

Android.view.Surface is an interface used to communicate between the application and the display subsystem. This allows applications to interact with display hardware, enabling them to render graphics directly to the screen.

Surface is part of the Android framework, and can be used to render graphical objects on the display. It provides APIs for creating and using surfaces, manipulating their contents and properties, and updating the underlying view structure. The Surface class encapsulates surface properties such as size and format and provides interfaces for drawing and animating 2D and 3D objects on the surface.

create()
static method: 

          
           
Surface surface = Surface.create();

method takes an optional argument that allows the application to specify a specific format for the surface: 

          
           
Surface surface = Surface.create(PixelFormat.OPAQUE);
To create a surface with a particular size, the 
create()
method takes two additional parameters that specify the width and height of the surface: 

          
           
Surface surface = Surface.create(100, 200);
