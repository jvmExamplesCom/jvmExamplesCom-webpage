Java.security.Key

Java.security.Key provides a set of classes to manage cryptographic keys in the Java platform. The keys are used for encryption and decryption, digital signatures, and other encryption operations.

Java Key Classes

The following classes are available for use with Java security keys:
  • SecretKey – used for encryption and decryption operations.
  • PublicKey and PrivateKey – used for digital signatures.
  • KeyFactory – required to convert keys between different formats.
  • KeyPairGenerator – used to generate key pairs.
  • KeyStore – used to store keys securely.
  • KeyAgreement – used for key agreement protocols.

Using Secret Keys

Secret keys can be used for symmetric encryption and decryption operations. The following code example demonstrates how to use the SecretKey class to generate and encrypt data using the AES encryption algorithm. 

          
           
//Generate a secret key
Cipher cipher = Cipher.getInstance("AES"); 
KeyGenerator keyGenerator = KeyGenerator.getInstance("AES"); 
SecretKey secretKey = keyGenerator.generateKey(); 

//Encrypt data 
cipher.init(Cipher.ENCRYPT_MODE, secretKey); 
byte[] encryptedData = cipher.doFinal("Hello".getBytes());

Using Public and Private Keys

Public and private keys can be used for digital signatures and hashing operations. The following code example demonstrates how to use public and private keys to sign data using the RSA algorithm. 

          
           
//Generate key pair
KeyPairGenerator keyGen = KeyPairGenerator.getInstance("RSA"); 
keyGen.initialize(1024); 
KeyPair keyPair = keyGen.generateKeyPair(); 
PublicKey publicKey = keyPair.getPublic();
PrivateKey privateKey = keyPair.getPrivate();

//Sign data
Signature signature = Signature.getInstance("SHA1withRSA"); 
signature.initSign(privateKey); 
signature.update("Hello".getBytes());
byte[] signedData = signature.sign();

Using Key Factories

Key factories are used to convert keys between different formats. The following code example demonstrates how to use a KeyFactory to convert a PublicKey to an encoded X.509 format. 

          
           
//Convert PublicKey to encoded X.509 format 
KeyFactory keyFactory = KeyFactory.getInstance("RSA"); 
X509EncodedKeySpec x509spec = keyFactory.getKeySpec(publicKey, X509EncodedKeySpec.class);
byte[] encodedKey = x509spec.getEncoded();

Using Key Stores

Key stores are used to store keys securely. The following code example demonstrates how to use a KeyStore to store a SecretKey. 

          
           
//Store SecretKey in KeyStore 
KeyStore keyStore = KeyStore.getInstance("JKS"); 
keyStore.load(null); 
keyStore.setEntry("secretkey", new KeyStore.SecretKeyEntry(secretKey), new KeyStore.PasswordProtection("password".toCharArray()));
