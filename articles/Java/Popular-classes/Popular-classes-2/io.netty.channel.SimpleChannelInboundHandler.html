The SimpleChannelInboundHandler class is a generic message handler which allows you to convert your incoming message into the type of your choice. Basics of SimpleChannelInboundHandler

The SimpleChannelInboundHandler class implements both ChannelInboundHandler and ChannelInboundHandlerAdapter.
