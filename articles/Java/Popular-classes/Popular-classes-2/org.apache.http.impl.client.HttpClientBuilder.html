Introduction to org.apache.http.impl.client.HttpClientBuilder

The following example shows how to create a new instance of an Apache HttpClient using the HttpClientBuilder class:


          
           
HttpClient client = HttpClientBuilder.create().build();

The above code creates a new instance of an Apache HttpClient with the default configuration settings. To customize the configuration settings, you can use the setter methods on the HttpClientBuilder class.
