Introduction to io.reactivex.Observable

io.reactivex.Observable is a powerful tool for creating and managing streams of data in a reactive programming language like Java. It can help developers create applications that are more efficient and reactive, as well as providing a simplified and understandable way of managing and manipulating data. In this article, we will discuss the features and capabilities of io.reactivex.Observable and provide code examples for creating and manipulating streams.

Creating Observables

The simplest way to create an Observable is to call the static create method on the Observable class. This method takes a single argument, which is a function or lambda expression that will generate the stream of items that will be emitted by the Observable. Here’s an example of creating an Observable that emits the numbers from 0 to 10: 

          
           
Observable.create(subscriber -> {
    // Generate stream of numbers 0..10
    for (int i = 0; i <= 10; i++) {
        subscriber.onNext(i);
    }
});
It’s also possible to create Observables from other sources, such as collections or files. Here’s an example of creating an Observable from a list of strings: 

          
           
List<String> list = Arrays.asList("One", "Two", "Three");
Observable.fromIterable(list);

Subscribing to Observables

Once you’ve created an Observable, you can subscribe to it using a Subscriber. The Subscriber provides a set of callbacks that will be invoked when Observable emits an item. The onNext() callback will be called when the Observable emits a new item, while the onError() callback will be called if the Observable encounters an unrecoverable error. Here’s an example of subscribing to an Observable that emits the numbers from 0 to 10: 

          
           
Observable.create(subscriber -> {
    // Generate stream of numbers 0..10
    for (int i = 0; i <= 10; i++) {
        subscriber.onNext(i);
    }
}).subscribe(
    item -> System.out.println(item),
    error -> System.err.println(error),
    () -> System.out.println("Done!")
);
This will print out each number emitted by the Observable on a separate line, and then print “Done!” when the stream is complete.

Operators

Another powerful feature of io.reactivex.Observable are operators, which are functions that can transform the items emitted by an Observable. Observable.create(subscriber -> {
    // Generate stream of numbers 0..10
