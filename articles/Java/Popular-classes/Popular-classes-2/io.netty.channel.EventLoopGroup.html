EventLoopGroup in Java

In Java, EventLoopGroup is a type of class from 
io.netty.channel
package which is used to handle and process I/O operations. It manages a pool of Threads with different event loops for handling various channels operations such as connection making, data transferring, etc. In general, the Networking application is built upon multiple EventLoopGroups; one for the main thread and the other for worker threads.

newEventLoopGroup()
. This takes two parameters: the number of threads in the group, and the thread factory. 

          
           
// construct EventLoopGroup object
