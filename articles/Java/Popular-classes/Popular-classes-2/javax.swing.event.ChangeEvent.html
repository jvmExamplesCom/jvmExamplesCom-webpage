javax.swing.event.ChangeEvent

The 

          
           
ChangeEvent
class is a part of the Java Swing package ( 

          
           
javax.swing
) that is used to indicate that something has been changed in a Swing component. This event is typically fired as a result of an action by a user or an application. The 

          
           
ChangeEvent
class provides methods for getting the source of the change, an object that identifies the type of change, and a time stamp indicating when the change occurred.

public ChangeEvent(Object source, Object type, long timestamp)

Methods


          
           
public Object getSource()
public Object getType()
public long getTimestamp()

getType()

getTimestamp()

ChangeEvent
