javax.swing.event.ChangeEvent

The 

          
           
ChangeEvent
class is a part of the Java Swing package ( 

          
           
javax.swing

ChangeEvent
class provides methods for getting the source of the change, an object that identifies the type of change, and a time stamp indicating when the change occurred.

getTimestamp()

ChangeEvent
