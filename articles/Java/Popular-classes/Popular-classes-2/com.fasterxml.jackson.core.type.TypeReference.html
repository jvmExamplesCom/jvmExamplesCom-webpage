com.fasterxml.jackson.core.type.TypeReference

com.fasterxml.jackson.core.type.TypeReference is an abstract class that represents a generic type and serves as a reference to it. It can be used to access the actual type information via reflection. It is useful when you need to work with generic types such as collections where the generic type information is not available at runtime.

Example

Let's take a simple example. We have a List of Strings and want to convert the list to JSON and back. First, we create a TypeReference for the generic type:


          
           
TypeReference<List<String>> typeRef = new TypeReference<List<String>>() {};

Now we can use this typeRef to convert the list to JSON:


          
           
String json = new ObjectMapper().writeValueAsString(list, typeRef);
To convert it back we can use the same typeRef: 

          
           
List<String> list = new ObjectMapper().readValue(json, typeRef);

Advanced Example

TypeReference can also be used to convert generic types to/from JSON. Let's take a more advanced example. We have a Map of String to List of Integers:


          
           
Map<String, List<Integer>> map;

We can convert it to JSON using a typeRef for the generic type:


          
           
TypeReference<Map<String, List<Integer>>> typeRef = new TypeReference<Map<String, List<Integer>>>() {};

And to convert it back:

Conclusion

com.fasterxml.jackson.core.type.TypeReference is an abstract class that provides a way to access generic type information. It is useful when you need to convert generic types from/to JSON, and cannot access the generic type information at runtime.
