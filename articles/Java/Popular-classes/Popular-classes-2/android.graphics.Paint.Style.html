android.graphics.Paint.Style

The android.graphics.Paint.Style class is used to define the types of styles that can be used to draw objects on a canvas. This class contains three static fields which can be used to set different painting styles. These fields are FILL, STROKE and FILL_AND_STROKE.

FILL

The FILL field defines a painting style, in which the inside of the object is filled with the specified color. For example, consider a circle: 

          
           
Paint paint = new Paint();
paint.setStyle(Paint.Style.FILL);
canvas.drawCircle(x, y, radius, paint);
This will draw a solid circle with its interior colored according to the current paint's color.

STROKE

The STROKE field defines a painting style, in which the outline of the object is drawn with the specified color. For example, consider a circle: 

          
           
Paint paint = new Paint();
paint.setStyle(Paint.Style.STROKE);
canvas.drawCircle(x, y, radius, paint);
This will draw an outline of the circle with its border colored according to the current paint's color.

FILL_AND_STROKE

The FILL_AND_STROKE field defines a painting style, in which the inside of the object is filled with the specified color, while the outline of the object is drawn with a different color. For example, consider a circle: 

          
           
Paint paint = new Paint();
paint.setStyle(Paint.Style.FILL_AND_STROKE);
canvas.drawCircle(x, y, radius, paint);
This will draw a circle with its interior colored according to the current paint's fill color and its border colored according to the current paint's stroke color.
