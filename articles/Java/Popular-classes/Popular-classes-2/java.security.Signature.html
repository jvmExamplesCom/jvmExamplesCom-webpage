Java Security Signature

The 

          
           
java.security.Signature
class allows applications to generate and verify digital signatures. It provides a framework for generating, verifying and storing digital signatures based on cryptographic techniques. This class can be used to sign and verify data in digital applications. It is part of the Java Cryptography Architecture (JCA).

Creating A Signature Object

The 
getInstance()
method can be used to create a new Signature object with the specified algorithm. The following code creates a Signature object for the RSA algorithm: 

          
           
Signature signature = Signature.getInstance("RSA");

Initializing The Signature Object

Before using the Signature object, it must first be initialized. This is done using the 

initSign()
or 
initVerify()
methods. The 
initSign()
method initializes the Signature object for signing data, while the 
initVerify()
method initializes the object for verifying data.

For example, the following code initializes the Signature object for signing data:


          
           
signature.initSign(privateKey);

Updating The Signature Object

update()

signature.update(dataString.getBytes());

Signing The Data

The data that has been added to the Signature can then be signed using the 

sign()
method. This method returns an array of bytes containing the signed data. The following example signs the data: 

          
           
byte[] signedData = signature.sign();

Verifying The Data

The 
verify()
