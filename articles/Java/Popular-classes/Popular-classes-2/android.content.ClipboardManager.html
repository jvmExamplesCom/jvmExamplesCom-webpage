android.content.ClipboardManager

The android.content.ClipboardManager class provides an interface to the system clipboard on devices running Android API level 11 or higher. It is used to store and share data between applications, activities, or windows. The ClipboardManager can be used to copy text to the clipboard, check for existing text in the clipboard, paste text from the clipboard, or clear the clipboard.

Copy Text To Clipboard

To copy text to the clipboard, first create a new ClipboardManager instance: 

          
           
ClipboardManager clipMan = (ClipboardManager) getSystemService(Context.CLIPBOARD_SERVICE);
Then use the setText() method to add the text to the clipboard: 

          
           
clipMan.setText("Text to Copy");

Check For Existing Text In Clipboard

To check if there is existing text in the clipboard, use the hasText() method: 

          
           
boolean hasText = clipMan.hasText();
The hasText() will return true if there is existing text in the clipboard that can be copied.

Paste Text From Clipboard

To paste the text from the clipboard, use the getText() method: 

          
           
CharSequence text = clipMan.getText();
The getText() method will return a CharSequence object containing the text from the clipboard.

Clear Clipboard

To clear the clipboard, use the clearText() method: 

          
           
clipMan.clearText();
This will clear all text from the clipboard.
