javax.xml.transform.TransformerException is a checked exception that is thrown by an XML parser when it encounters an error while transforming an XML document. Causes of javax.xml.transform.TransformerException

There are many causes of javax.xml.transform.TransformerException, including:

  • When an unrecognised XSLT element is encountered.
  • When a XSLT statement is syntactically incorrect.
  • When a XSLT file uses unsupported features.
  • When a runtime error occurs during processing of XSLT statements.
  • When an invalid source or result of a transformation is provided.
