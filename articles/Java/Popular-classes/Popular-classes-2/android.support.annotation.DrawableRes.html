Introduction to android.support.annotation.DrawableRes

The 

          
           
android.support.annotation.DrawableRes
annotation is an Android Support Library annotation used to indicate that an integer parameter, field or method return value is expected to be a drawable resource reference (e.g. 
@DrawableRes
int). It offers compile-time type safety for Android APIs that require drawables.

Drawable resources are a collection of multiple images and audio files that can be used by an Android application. For instance, an application may want to display an image based on user input. The application will need to construct the correct drawable resource and specify it in a view.

Using the @DrawableRes Annotation

The @DrawableRes annotation requires that any variable, parameter or method return type passed to it is an integer corresponding to a drawable resource. If a non-integer is passed, the compiler will throw an error.

The following code example demonstrates how to use the 

@DrawableRes
annotation in an Android application: 

          
           
public class ExampleActivity extends Activity {

    private ImageView mImageView;
    
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_example);
        
        mImageView = (ImageView) findViewById(R.id.image_view);
    }
    
    public void setImage(@DrawableRes int drawableResId) {
        mImageView.setImageResource(drawableResId);
    }
}
In the above example, the 
setImage()
method accepts an 
int
parameter annotated with the 
@DrawableRes
annotation. This indicates that the parameter is expected to be an integer corresponding to a drawable resource. The example then uses the given 
int

ImageView

                showImage(R.drawable.cat);
                break;
            case R.id.button_rabbit:
                showImage(R.drawable.rabbit);
                break;
        }
    }
}

showImage()
method takes an integer parameter which has been annotated with the 
@DrawableRes

int
