Introduction to MockitoJUnitRunner

MockitoJUnitRunner is an extension of JUnit's BlockJUnit4ClassRunner which provides support for using Mockito in unit tests. It takes care of initializing mocks and validating the framework's state throughout the test execution. It is often used when the test subject depends on external collaborators, allowing the test author to isolate the test subject from its environment.

Creating a Unit Test with MockitoJUnitRunner

Creating a unit test with MockitoJUnitRunner is straightforward. The following code example demonstrates how to initialize a test class with the Runner: 

          
           
@RunWith(MockitoJUnitRunner.class)
public class MyTest {
    // test methods go here
}
The @RunWith annotation is used to specify which runner should be used for executing the unit test. In this case, MockitoJUnitRunner is specified as the Runner for executing the MyTest class.

Mocking Collaborators
