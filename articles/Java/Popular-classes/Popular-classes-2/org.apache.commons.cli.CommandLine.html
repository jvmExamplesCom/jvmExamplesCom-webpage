Introduction to org.apache.commons.cli.CommandLine

org.apache.commons.cli.Command Line
is a Java class that provides an easy way for developers to parse command line arguments from the user and store them in an organized way. This can be especially useful when creating applications that require the user to specify certain options or arguments before execution.

Basic Usage and Syntax

To use this class, the user must first create a 

          
           
Options
object that defines all of the possible options that the program can accept. This can be done by calling the 

          
           
new Option()
constructor for each option you wish to add, and adding it to the Options object with the 

          
           
addOption()
method. Once all the options have been added to the Options object, the user can call the 

          
           
CommandLineParser.parse()
method, passing in the Options and the 

          
           
args[]
array from the main method, to create an instance of the CommandLine class. Once the CommandLine instance has been created, the user can use its various methods to check for the presence and value of any given option specified by the user. The following example demonstrates how to check if the 

          
           
-v
option was included, and if so, what value it had. 

          
           
public static void main(String[] args) {
	Options options = new Options();
	options.addOption("v", false, "verbose");

	CommandLineParser parser = new DefaultParser();
	CommandLine cmd = parser.parse(options, args);

	if (cmd.hasOption("v")) {
		System.out.println("option v: " + cmd.getOptionValue("v"));
	}
}
