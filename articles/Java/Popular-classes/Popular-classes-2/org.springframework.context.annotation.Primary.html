org.springframework.context.annotation.Primary

The org.springframework.context.annotation.Primary annotation indicates that a bean should be given priority when multiple candidates are qualified to autowire a single-valued dependency. It is useful when multiple beans might qualify as primary autowire candidates and you need to decide which one to use by default.

The @Primary annotation can be used at the class level or at the method level. When used at the class level, all the methods of the annotated class are considered as primary autowire candidates.
