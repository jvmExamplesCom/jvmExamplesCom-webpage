org.apache.commons.codec.digest.DigestUtils

The org.apache.commons.codec.digest.DigestUtils class provides a convenient interface for generating various types of message digests, such as MD5 and SHA-1.

Example Code

The following code samples illustrate how to use the DigestUtils class to generate message digests. 


          
           
String data = "This is a test";
String md5Hex = DigestUtils.md5Hex(data);
System.out.println(md5Hex); // output: d1937aa8ab8bfe58e2bff7f14d29533b

          

          
           
String sha256Hex = DigestUtils.sha256Hex(data);
System.out.println(sha256Hex); // output: 5daf97f55cb69cb57cb50bb2b5f54e5ea94b5a52767d3ba0066384756f80c59a

          

          
           
String sha512Hex = DigestUtils.sha512Hex(data);
System.out.println(sha512Hex); // output: 6ed24710389f937e174cf575be6e41eb29862f86ef40cd81c59d672ef65fdde11df22fb10db43dcefdf63308090a32318f4b7fde3a4a4cdc0ad21e9bb739c2dd

The org.apache.commons.codec.digest.DigestUtils class provides a convenient interface for generating various types of message digests, such as MD5 and SHA-1. It can be used to generate secure and unique identifiers for resources or to verify the integrity of transmitted data.
