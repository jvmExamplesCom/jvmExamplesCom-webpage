Android Text Style ForegroundColorSpan

Android Text Style ForegroundColorSpan helps to apply a foreground color to the text of a widget. This style can be used on any type of text widget, such as an EditText, or a TextView. The ForegroundColorSpan class is a subclass of the CharacterStyle class and is a part of the Android Text package. When this style is applied to a widget, the color of the text within that widget is changed to the one specified.

Using ForegroundColorSpan

The way ForegroundColorSpan works is that it takes in a color as an argument when initiating the span. This color will then be applied to the selected text. To apply a ForegroundColorSpan to a TextView: 

          
           
TextView tv = (TextView) findViewById(R.id.textview); 
ForegroundColorSpan fgcs = new ForegroundColorSpan(Color.BLUE);
tv.setText(Spannable.Factory.getInstance().newSpannable(tv.getText()).setSpan(fgcs, 0, tv.getText().length(), Spannable.SPAN_EXCLUSIVE_EXCLUSIVE));
This code snippet will set the color of all the text in the TextView to be blue.

Additional Features

The ForegroundColorSpan class also provides several convenience methods which allow for finer control over how the color is applied. For example, you can use the updateDrawState() method to dynamically set the color based on the widget's state. 

          
           
public void updateDrawState(TextPaint tp) {
    if (widget.isEnabled()) {
        tp.setColor(Color.BLUE);
    } else {
        tp.setColor(Color.GRAY);
    }
}
In the above example, the color of the text is set to blue if the widget is enabled and gray if it is disabled.
