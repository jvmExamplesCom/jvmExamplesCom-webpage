android.os.PowerManager

The android.os.PowerManager class was introduced in Android 1.0 and provides access to the power management functions on the device. It allows applications to access and control the state of the power sources on the device. It also allows applications to request an operation to be performed when the power source changes state. This can be used to wake up the device when it is powered off or put it into sleep mode when the battery is low.

Overview

The PowerManager class provides methods to register and unregister power state change listeners. It also provides methods to acquire, release, and manage WakeLocks, which allow applications to keep the device from sleeping or to prevent it from dimming the display.
