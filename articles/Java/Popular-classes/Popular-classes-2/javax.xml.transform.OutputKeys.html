javax.xml.transform.OutputKeys

javax.xml.transform.OutputKeys is a class in the javax.xml.transform package that defines constants to be used as values of output properties for the Transformer and TransformerHandler classes.

Overview

The OutputKeys class defines constants for naming keys of output properties available to a Transformer or TransformerHandler. These properties control the serialization of the output.

Example Usage

The following example creates a data structure which contains output properties as needed by the class Transformer to serialize an XML document.


          
           
public static Properties createTransformerProperties() {
    Properties props = new Properties();
    props.put(OutputKeys.METHOD, "xml");
    props.put(OutputKeys.INDENT, "yes");
    props.put(OutputKeys.ENCODING, "UTF-8");
    props.put(OutputKeys.OMIT_XML_DECLARATION, "no");
    return props;
}

In this example, the METHOD value is set to “xml” so the output will be valid XML. The INDENT value is set to “yes” so the output will be indented for improved readability. The ENCODING value is set to “UTF-8” for encoding the output document, and the OMIT_XML_DECLARATION value is set to “no” so the document will include the XML declaration.

Constants

The OutputKeys class defines the following constants which can be used as keys for output properties:

  • METHOD
    - Indicates the output method to be used (e.g. “xml”, “html”, “text”, etc).
  • INDENT
    - Indicates whether or not the output should be indented.
  • ENCODING
    - Indicates the character encoding to be used in the output document.

OMIT_XML_DECLARATION
    - Indicates whether or not to include the XML declaration at the beginning of the output document.

  • MEDIA_TYPE
    - Indicates the media type (aka MIME type) of the output document.
  • STANDALONE
    - Indicates whether the output document should include a standalone document declaration.
  • DOCTYPE_PUBLIC
    - Indicates the public identifier to be used in the document type declaration.
  • DOCTYPE_SYSTEM
    - Indicates the system identifier to be used in the document type declaration.
  • VERSION

    - Indicates the preferred form of whitespace normalization to be performed.

Conclusion

The javax.xml.transform.OutputKeys class defines constants to be used as values of output properties for the Transformer and TransformerHandler classes. These constants provide a convenient way to set output properties needed to serialize an XML document.
