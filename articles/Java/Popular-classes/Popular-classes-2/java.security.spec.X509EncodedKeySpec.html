Java Security Spec - X509EncodedKeySpec

The Java security spec, X509 Encoded Key Spec, implements an interface that allows a public key to be retrieved from an encoded key. It is used to securely store the private key of an entity. The encoded key can be converted into a format that is readable by applications.

Example 1 - Generating an X509 Encoded Key


          
           
public static void main(String[] args) {
    try {
        KeyPairGenerator keyGen = KeyPairGenerator.getInstance("RSA");
        SecureRandom random = new SecureRandom();
        keyGen.initialize(1024, random);

        KeyPair pair = keyGen.generateKeyPair();
        PublicKey pubKey = pair.getPublic();
        PrivateKey privKey = pair.getPrivate();

        X509EncodedKeySpec x509EncodedPubKey = new X509EncodedKeySpec(pubKey.getEncoded());
    } catch (Exception e) {
        System.out.println("Error: " + e);
    }
}

Example 2 - Retrieving the Key from the Encoded Value


          
           
try {
    KeyFactory kf = KeyFactory.getInstance("RSA");
    PublicKey pubKey = kf.generatePublic(x509EncodedPubKey);
    System.out.println("Public key: " + pubKey);

} catch (NoSuchAlgorithmException | InvalidKeySpecException e) {
    System.out.println("Error: " + e);
}

Example 3 - Displaying the Key in Base64 Format


          
           
try {
    byte[] encodedKey = x509EncodedPubKey.getEncoded();
    String base64Key = Base64.getEncoder().encodeToString(encodedKey);
    System.out.println("Base64 encoded key: " + base64Key);

} catch (Exception e) {
    System.out.println("Error: " + e);
}

Example 4 - Writing the Key to a File


          
           
try {
    File f = new File("public_key.dat");
    if (!f.exists()) {
        f.createNewFile();
    }
    FileWriter writer = new FileWriter(f);
    writer.write(base64Key);
    writer.close();

} catch (Exception e) {
    System.out.println("Error: " + e);
}
