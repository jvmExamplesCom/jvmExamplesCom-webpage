The Java Security AccessController class is a part of the Java Security package which provides an API to provide a secure environment for executing Java code. The AccessController class is used to check whether a certain piece of code (which is called a “permission”) has access to perform a particular task or not. This is done by comparing the permission's attributes with the attributes of the current execution context.
