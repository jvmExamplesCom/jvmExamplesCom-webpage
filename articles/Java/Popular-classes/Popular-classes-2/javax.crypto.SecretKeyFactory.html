javax.crypto.SecretKeyFactory

The 

          
           
javax.crypto.SecretKeyFactory
is a class in the Java Cryptography Architecture which provides the functionality of converting secret keys of different formats into SecretKey objects. It is used to convert opaque cryptographic keys of type 

          
           
SecretKey
into secret keys for use by a 

          
           
SecretKeyFactory
object. 

          
           
SecretKeyFactory
objects are also capable of generating new keys.

Creating a SecretKeyFactory Object

The 

          
           
SecretKeyFactory
class can be instantiated using one of the following constructors: 

          
           
public SecretKeyFactory(String algorithm) throws NoSuchAlgorithmException
public SecretKeyFactory(String algorithm, Provider provider) throws NoSuchAlgorithmException
The parameters to be passed to the constructor are an algorithm name and a Java Security Provider.

Generating Keys


          
           
SecretKeyFactory
has two methods to generate keys: 

          
           
public SecretKey generateSecret(KeySpec keySpec) throws InvalidKeySpecException 
public KeySpec getKeySpec(SecretKey key, Class keySpec) throws InvalidKeySpecException
The first method generates a new key using the specified 

          
           
KeySpec
. The second method is used to generate a 

          
           
KeySpec
for the specified key.

Converting Keys


          
           
SecretKeyFactory
can be used to convert keys between different types. 

          
           
public SecretKey translateKey(SecretKey key) throws InvalidKeyException
This method converts a key from one type to another.
