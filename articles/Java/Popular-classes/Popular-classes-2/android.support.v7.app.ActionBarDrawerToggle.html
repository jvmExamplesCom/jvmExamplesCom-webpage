Introduction to Android Support V7 App ActionBarDrawerToggle

Using the ActionBarDrawerToggle

The ActionBarDrawerToggle can be used in any activity, as long as it is connected to a navigation drawer. To use the ActionBarDrawerToggle, you must first initialize it and set the appropriate callbacks. The following code snippet shows an example of such initialization: 

          
           
ActionBarDrawerToggle actionBarDrawerToggle = new ActionBarDrawerToggle(this, mDrawer, toolbar, R.string.open_drawer, R.string.close_drawer);

The above code initializes a new ActionBarDrawerToggle and adds it as a DrawerListener to the navigation drawer. Finally, it syncs the state of the ActionBarDrawerToggle so that it displays the correct icon on the toolbar.

ActionBarDrawerToggle actionBarDrawerToggle = new ActionBarDrawerToggle(this, mDrawer, toolbar, R.string.open_drawer, R.string.close_drawer);
