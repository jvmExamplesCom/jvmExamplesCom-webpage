Introduction to javax.xml.bind.Unmarshaller

javax.xml.bind.Unmarshaller is an important part of the Java API for XML Processing (JAXP). It provides an efficient way to convert a XML document into an object. With this class, you can unmarshal XML documents into Java objects, and marshal Java objects back into XML.

To create an Unmarshaller object, use the static method newInstance() of the JAXBContext class. This method takes two arguments. The first argument is the name of the package where the objects to be unmarshaled are located. The second argument is the ClassLoader to load classes.


          
           
JAXBContext jc = JAXBContext.newInstance("my.package", this.getClass().getClassLoader());
Unmarshaller u = jc.createUnmarshaller();

Unmarshaling a XML File

Once you have created an Unmarshaller object, you can use it to unmarshal a XML file. Use the unmarshal() method to do this. This method takes a DOMSource, SAXSource or StreamSource as argument. When unmarshaling, all the elements of the XML file are mapped to Java objects.


          
           
File f = new File("MyXMLFile.xml");
JAXBElement<?> element = (JAXBElement<?>)u.unmarshal(f);
MyObject myObject = (MyObject)element.getValue();
