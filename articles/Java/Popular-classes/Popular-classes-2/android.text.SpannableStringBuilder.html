Introduction to Android Text SpannableStringBuilder

The Android Text SpannableStringBuilder provides an efficient way of creating and manipulating styled strings. To create a new instance of the Android Text SpannableStringBuilder class, you must first get a CharSequence object. The CharSequence can be any string, including an empty one. CharSequence input = "Hello World!";
