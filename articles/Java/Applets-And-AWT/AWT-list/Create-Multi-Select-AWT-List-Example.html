Create Multi Select AWT List Example

The Java Abstract Window Toolkit (AWT) allows you to create a multi select list that can be used in GUI applications. This tutorial will provide an example of how to create a multi select list using the AWT class library.

Creating a Multi Select List

To create a multi select list, you first need to create a 

java.awt.List
object: 

          
           
List myList = new List();

You can then add items to your list with the following code:


          
           
myList.add("Item 1");
 myList.add("Item 2");
 myList.add("Item 3");

You can also set the list's selection mode to allow for multiple selections:


          
           
myList.setMultipleMode(true);

Adding Listener to the List

A listener must be added to the list in order to react to changes in the list. To do this, create an 

ItemListener
object and add it to the list: 

          
           
ItemListener myItemListener = new ItemListener(){ 
     public void itemStateChanged(ItemEvent e){ 
         //Handle Item Event 
     }
 };
 myList.addItemListener(myItemListener);

Getting Selected Items from the List

In order to get the selected items from the list, use the 

getSelectedItems()
method. This method returns an array of 
String
objects containing the selected items. For example: 

          
           
String[] selectedItems = myList.getSelectedItems();

The returned array can then be used to process the selected items as needed.

Conclusion

This tutorial provided an example of how to create a multi select list using the AWT class library in Java. The creation and usage of the list was demonstrated, as well as how to add a listener to the list and how to get the selected items from it.
