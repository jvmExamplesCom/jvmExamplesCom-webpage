Create AWT List Example in Java

AWT List is an example of AWT GUI component used to display list of items. The AWT List class is used to create a List object, which can then be used to show the list of items in AWT GUI window. In this tutorial, we will discuss how to create an AWT List object and use it in a Java program. First, we need to create a Frame object in which the AWT List will be placed. To do this, we can use the following code: 

          
           
Frame frame = new Frame("AWT List Demo");
frame.setSize(250, 150);
frame.setVisible(true);
This code creates a Frame object with its title set as "AWT List Demo". It also sets the size of the Frame at 250px by 150px, and makes it visible on the screen. Next, we need to create an AWT List object that will be placed within the Frame. To do this, we can use the following code: 

          
           
List list = new List();
list.add("Item 1");
list.add("Item 2");
list.add("Item 3");
list.setMultipleMode(true);
frame.add(list);
This code creates a List object, and adds three items to it. We also set the MultipleMode property to true, so that multiple items can be selected simultaneously. Finally, we add the list to the Frame. Now, we have created an AWT List object in Java, and added it to the Frame. To see the result, we can run the program, which will open a window containing the List like this: The List allows us to select multiple items, and we can also use different methods to get the selected items from the List. For example, we can use the following code to print all the selected items in the List: 

          
           
int[] indices = list.getSelectedIndexes();
for (int i = 0; i < indices.length; i++) {
    System.out.println(list.getItem(indices[i]));
}
The above code uses the getSelectedIndexes() method of the List object to get an array containing the indices of the selected items. It then iterates over the array and prints the item corresponding to each index. Finally, we can also add or remove items from the list at any time using the add() and remove() methods. For example, we can use the following code to add a new item to the List: 

          
           
list.add("Item 4");
And the following code to remove an item from the List:
