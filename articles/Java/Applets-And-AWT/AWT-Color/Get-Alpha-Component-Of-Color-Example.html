Java: Get Alpha Component Of Color Example

The alpha component of a color in Java is an integer between 0 and 255 inclusive, with 0 being opaque and 255 being fully transparent. It is possible to get, set and manipulate the alpha component of a color using the method 

Color.getAlpha()
. In this article we will explore how to use this method to get the alpha component of a color.

Example 1: Get Alpha Component From Color

In this example we will declare a 
Color
object and get its alpha component using the method 
Color.getAlpha()
. We will then print out the result. 

          
           
Color c = new Color(255, 0, 0, 127);
int alphaComponent = c.getAlpha();
System.out.println("Alpha component: " + alphaComponent);
In this example the output should be 

          
           
Alpha component: 127
.

Example 2: Change Alpha Component Of A Color

In this example we will demonstrate how to use the method 

Color.getAlpha()
to change the alpha component of a color. We will declare a 
Color
object and then set the alpha component using the method 
Color.setAlpha()
. We will then print out the result. 

          
           
Color c = new Color(255, 0, 0);
int oldAlphaComponent = c.getAlpha();
c.setAlpha(127);
int newAlphaComponent = c.getAlpha();
System.out.println("Old alpha component: " + oldAlphaComponent);
System.out.println("New alpha component: " + newAlphaComponent);
In this example the output should be 

          
           
Old alpha component: 255
, followed by 

          
           
New alpha component: 127
.
