How to Remove AWT Checkbox in Java Applet or Window Frame Example

In this article we will learn how to remove an AWT Checkbox from an Applet or Window Frame in Java. As AWT Checkboxes are used for user input, they can be easily removed if no longer needed. We'll cover two ways of removing an AWT Checkbox: by disabling the checkbox and by destroying the object. Let’s look at some code examples to help us understand how to remove AWT Checkboxes.

Disabling the AWT Checkbox

The first way of removing an AWT Checkbox is by disabling it. This means that the Checkbox is still present but is not interactive and has no effect on the application. To do this you can use the 

          
           
awtCheckbox.setEnabled(false);
command. This will disable the Checkbox and make it so that it cannot be interacted with any more.

Destroying the AWT Checkbox

The second way to remove an AWT Checkbox is by destroying the object. This means that the Checkbox is no longer part of the application and is no longer visible. To do this you can use the 

          
           
awtCheckbox.dispose();
command. This will destroy the Checkbox and make it so that it no longer exists in the application.

Example Code

Let’s look at a code example to illustrate how these commands are used. The following code creates a window frame with an AWT Checkbox: 

          
           
import java.awt.*;

public class Test {
    public static void main(String[] args) {
        Frame f = new Frame("Test");
        Checkbox awtCheckbox = new Checkbox("Test Checkbox");
        f.add(awtCheckbox);
        f.setSize(400, 300);
        f.setVisible(true);

    }
}
