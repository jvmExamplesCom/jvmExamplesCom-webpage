Create Checked Checkbox Example Using Java

In this article, we will learn how to create a checked checkbox example using Java. Checkboxes are commonly used in User Interface (UI) design, allowing the user to select one or more items from a list of options. In this example, we will create a checked checkbox where the user can select one or more items from a list that are already checked by default.

Java Code Example

Let's start by writing the Java code for creating the checked checkbox: 

          
           
public class CheckBoxExample {
    public static void main(String[] args) {  
        // Create a JCheckBox   
        JCheckBox checkBox = new JCheckBox("Apple", true);
        JCheckBox checkBox2 = new JCheckBox("Banana", true);
        JCheckBox checkBox3 = new JCheckBox("Orange", true);
       
        // Create a panel to add the checkbox  
        JPanel panel = new JPanel();
       
        // Add the checkbox to the panel  
        panel.add(checkBox);
        panel.add(checkBox2);
        panel.add(checkBox3);
       
        // Create a frame to add the panel 
        JFrame frame = new JFrame();
       
        // Add the panel to the frame 
        frame.add(panel);
       
        // Set the size of the frame 
        frame.setSize(400, 400);
       
        // Set the title of the frame  
        frame.setTitle("Check Box Example");
       
        // Make the frame visible  
        frame.setVisible(true);
    }  
}
In the above Java code example, we first created three JCheckBox objects with the label "Apple", "Banana" and "Orange". We then created a panel and added the checkboxes to it. Finally, we created a frame and added the panel to it. We also set the size of the frame and set its title to "Check Box Example" and made it visible.

Output HTML

The HTML code which is the output of the Java code example given above is as follows: 

          
           
<input type="checkbox" name="Apple" value="Apple" checked> Apple
<input type="checkbox" name="Banana" value="Banana" checked> Banana
<input type="checkbox" name="Orange" value="Orange" checked> Orange
This HTML code creates three checkboxes with the labels "Apple", "Banana" and "Orange". The checkboxes are already checked by default.

Conclusion

In this article, we learned how to create a checked checkbox example using Java. We wrote the Java code example, which we then converted into HTML. This code creates three checkboxes with the labels "Apple", "Banana" and "Orange" which are already checked by default.
