Get Applet Context Example

Applet context allows an applet to interact with the browser environment. It provides the applet class with a set of tools to access and modify the browser environment. This tutorial will show you how to get an Applet context example and the output in HTML.

Example 1


          
           
AppletContext ac = getAppletContext();
   URL url = new URL("file:///c:/foo.html"); 
   ac.showDocument(url);
This simple example shows how to get an Applet context and use it to show a document. The code creates a new URL and then uses the AppletContext to show this document in the browser.

Example 2


          
           
AppletContext ac = getAppletContext();
   String param = ac.getParameter("name"); 
   System.out.println("The parameter is: "+param);
This example shows how to get an Applet context and use it to retreive a parameter from the browser environment. The code retrieves the value of the "name" parameter from the browser and prints it out.

AppletContext ac = getAppletContext();
   URL url = new URL("http://www.example.com"); 
   ac.showDocument(url,"_top");
This example shows how to get an Applet context and use it to show a document in the same window. The code creates a new URL and then uses the AppletContext to show this document in the same window.

Example 5


          
           
AppletContext ac = getAppletContext();
   String soundFile = "http://www.example.com/sound.wav";
