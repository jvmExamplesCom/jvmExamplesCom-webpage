Display Image in an Applet Example

In this article, we will learn how to display an image in an applet. It is quite easy to do, but requires some understanding of the basic concepts of Java.

Example Code

The following code will display an image in an applet:


          
           
import java.applet.*;
import java.awt.*;

public class ImageApplet extends Applet {

    /*
    * The name of the image file. 
    */
   private String imgName = "myimage.jpg";

   public void init(){
      try{
          /*
          * Create an image object using the image file
          */
          Image img = getImage(getCodeBase(), imgName);
          /*
          * Draw the image on the applet using the Graphics drawImage() method
          */
          Graphics g = getGraphics();
          g.drawImage(img, 0, 0, this); 
      } catch(Exception e) {
          System.out.println("Error : "+e);
      }
   }
}

Save this simple program to a file and name it ImageApplet.java. Compile the program using the command line:


          
           
javac ImageApplet.java

After compilation is successful, run it using the following command:


          
           
appletviewer ImageApplet.class

You will see the image that you specified in the code displayed in the Applet viewer window.

Output

The output of this program should be the image specified in the code, which will be displayed in the applet viewer window.
