Get Foreground Color Of an Applet Window Example

The following example demonstrates how to get the foreground color of an applet window. It can be used to create a customized look and feel for an applet, making it stand out from a generic container.

Java Code


          
           
import java.awt.Color;
import java.applet.Applet;
import java.applet.AppletContext;

public class AppletBackground extends Applet {

    public void init() {

        AppletContext context = this.getAppletContext();
        Color foreground = context.getForeground();
        System.out.println("The foreground color is: " + foreground);
    }
}

Output

The output of the code is: 

          
           
The foreground color is: java.awt.Color[r=0,g=0,b=0]
