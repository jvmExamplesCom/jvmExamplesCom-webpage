Java Change AWT Button Font Example

Changing a font in a button can be a difficult task for novice Java programmers. In this tutorial we will show you a step by step guide of how to change the font in an AWT Button using Java.

Arguments of the setFont Method

In order to change the font in an AWTButton, we need to use the setFont() method. The setFont() method takes three arguments: 

          
           
setFont(Font fontName, int style, int size)
The fontName argument is a Font object that specifies the name of the font to be used on the AWT Button. It can be any of these fonts: SansSerif, Serif, Monospaced, Dialog, or DialogInput. The style argument is an integer value. It specifies the style of the font. It can be one of the following constants: 

          
           
PLAIN, BOLD, ITALIC, or BOLD + ITALIC.
The size argument is an integer value that specifies the size of the font.

Example Code

Now that we understand the arguments of the setFont() method, letâ€™s look at an example of how to actually use it. In this example, we will create an AWT Button with a font size of 20 and a style of BOLD. 

          
           
import java.awt.*; 
import java.awt.event.*; 
public class ChangeFontExample {  
   // Create a Button 
   private static Button btn; 

   public static void main(String[] args) {  
      // create a new frame 
      Frame frame = new Frame("AWT Button Font Example"); 
      // Create a Button 
      btn = new Button("My Button"); 
      // Setting font size 20 and style BOLD  
      btn.setFont(new Font("Arial", Font.BOLD, 20)); 
   
      // add button to frame 
      frame.add(btn); 
      frame.setSize(400, 400); 
      frame.setVisible(true); 
   } 
}
