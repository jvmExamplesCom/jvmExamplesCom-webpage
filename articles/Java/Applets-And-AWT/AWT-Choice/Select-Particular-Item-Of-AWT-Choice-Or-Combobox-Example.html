Java AWT Choice and Combobox Example

The AWT choice and combobox components are two of the user-interface controls found in the AWT package. These components can be used to select an item from a list of choices either through a drop-down list or a combo box. In this article, we'll look at an example of how to use AWT choice and combobox components.

Creating the UI

The first step is to create the necessary components for our example. We'll create a labeled field, an AWT choice component, and an AWT combobox component: 

          
           
Label lblField = new Label("Select your favorite color: ");
Choice chcChoice = new Choice();
ComboBox cmbComboBox = new ComboBox();
Next, we'll add the items to our choice and combobox components: 

          
           
chcChoice.add("Red");
chcChoice.add("Green");
chcChoice.add("Blue");

cmbComboBox.add("Red");
cmbComboBox.add("Green");
cmbComboBox.add("Blue");
Finally, we'll add a listener to both components so that we can handle the selection events: 

          
           
chcChoice.addItemListener(new ItemListener() {
   public void itemStateChanged(ItemEvent e) {
      // Handle the item selection event
   }
});

cmbComboBox.addActionListener(new ActionListener() {
   public void actionPerformed(ActionEvent e) {
      // Handle the item selection event
   }
});

Handling the Event

Now that we have our UI set up, we need to handle the item selection event. This will involve getting the selected item from either the choice or the combobox and then performing any necessary logic. For this example, we'll just print out the selected item to the console: 

          
           
Object selectedItem = null;
if (e.getSource() == chcChoice) {
   selectedItem = chcChoice.getSelectedItem();
} else if (e.getSource() == cmbComboBox) {
   selectedItem = cmbComboBox.getSelectedItem();
}
System.out.println("Selected item: " + selectedItem);

Rendering the UI

Finally, we'll render our UI by adding the components to a container and displaying the container in a frame: 

          
           
Frame frame = new Frame("AWT Choice and Combobox Example");
frame.setLayout(new FlowLayout());
frame.add(lblField);
frame.add(chcChoice);
frame.add(cmbComboBox);
frame.pack();
frame.setVisible(true);
And with that, we've created an example of how to use AWT choice and combobox components in Java.
