Disable AWT Choice Or Combobox Example

The following example illustrated how to disable an AWT choice or combo box. The example creates a frame with two buttons – one for disabling the combo box and another for enabling it. The combo box has several list items and a text field that displays the selected list item. 

          
           
import java.awt.*;
import java.awt.event.*;

public class DisableExample extends Frame {
  private Choice cb;

  public DisableExample() {
    cb = new Choice();

    cb.add("USA");
    cb.add("UK");
    cb.add("India");
    cb.add("Russia");
    cb.add("China");

    add(cb, BorderLayout.NORTH);

    Button btDisable = new Button("Disable");
    add(btDisable, BorderLayout.WEST);
    btDisable.addActionListener(new ActionListener() {
      public void actionPerformed(ActionEvent e) {
        cb.setEnabled(false);
      }
    });

    Button btEnable = new Button("Enable");
    add(btEnable, BorderLayout.EAST);
    btEnable.addActionListener(new ActionListener() {
      public void actionPerformed(ActionEvent e) {
        cb.setEnabled(true);
      }
    });

    setSize(400, 200);
    setVisible(true);
  }

  public static void main(String[] args) {
    new DisableExample();
  }
}
When the “Disable” button is clicked, it will disable the combo box and when the “Enable” button is clicked, it will enable the combo box again.

Output

When you run the above example, you can see that the combo box will be disabled after clicking the “Disable” button and it will be enabled after clicking the “Enable” button.
