Get Selected Item Of AWT Choice Or Combobox Example

AWT (Abstract Window Toolkit) is a powerful library for creating user interfaces in Java. A particularly popular element of AWT is the Choice or ComboBox, which displays a selection of items from which the user can choose. This article will explore how to retrieve the selected item of an AWT Choice or ComboBox programmatically.

Retrieving the Value of an AWT Choice or ComboBox

The first step in determining which item has been selected in an AWT Choice or ComboBox is to obtain the string value of the currently selected item. To do this, you must use the 

          
           
getSelectedItem()
method of the Choice or ComboBox instance. This method returns the string value of the currently selected item. For example, consider the following code snippet which creates an instance of a ComboBox with three different items added to it: 

          
           
ComboBox cb = new ComboBox();
cb.add("Apple");
cb.add("Banana");
cb.add("Carrot");
To retrieve the string value of the selected item in the ComboBox, you would use the following code: 

          
           
String selectedItem = cb.getSelectedItem();

Retrieving the Index of an AWT Choice or ComboBox

Often, it can be useful to determine the index of the selected item in an AWT Choice or ComboBox instance. To do this, you should use the 

          
           
getSelectedIndex()
the method of the Choice or ComboBox instance. This method returns the index of the currently selected item. For example, consider the same code snippet used in the previous section which creates an instance of a ComboBox with three different items added to it: 

          
           
ComboBox cb = new ComboBox();
cb.add("Apple");
cb.add("Banana");
cb.add("Carrot");
To retrieve the index of the selected item in the ComboBox, you would use the following code: 

          
           
int selectedItemIndex = cb.getSelectedIndex();
