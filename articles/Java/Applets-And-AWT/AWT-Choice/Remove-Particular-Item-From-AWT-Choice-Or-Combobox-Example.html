Remove Particular Item From AWT Choice Or Combobox Example

Introduction

Java AWT Choice and Combobox are two GUI components contain list of items. In this article, we will discuss how to remove a particular item from a Choice or ComboBox component in Java. We will provide code examples to help illustrate the concepts and show what the output looks like.

Example 1: Removing an Item from Choice

In this example, we will remove an item from a Choice component in Java. The code for this example is given below. 

          
           
Choice choice = new Choice();
choice.add("Item 1");
choice.add("Item 2");
choice.add("Item 3");
choice.remove("Item 2");
The output of this code will be a Choice component that contains two items: "Item 1" and "Item 3".

Example 2: Removing an Item from ComboBox

In this example, we will remove an item from a ComboBox component in Java. The code for this example is given below. 

          
           
ComboBox combobox = new ComboBox();
combobox.add("Item 1");
combobox.add("Item 2");
combobox.add("Item 3");
combobox.remove("Item 2");
The output of this code will be a ComboBox component that contains two items: "Item 1" and "Item 3".
