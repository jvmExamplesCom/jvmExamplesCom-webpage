Get Particular Item Of AWT Choice Or Combobox Example

AWT Choice or Combobox is a widget for Java that enables users to select an item from a list. It is widely used in GUI applications for displaying a list of options and allowing users to select one of them. The following example demonstrates how to get a particular item of an AWT Choice or Combobox.

Java Code Example

The following code snippet creates a Choice object and adds some items to it. Then, it gets the selected item and prints it out. 

          
           
Choice choice = new Choice();
choice.add("Item 1");
choice.add("Item 2");
choice.add("Item 3");

String selectedItem = choice.getSelectedItem();
System.out.println(selectedItem);
The output will be: 

          
           
Item 1

Getting a Particular Item from a Choice Object

It is possible to get a particular item from a Choice object by using the getItem(int index) method. The following code snippet demonstrates it. This code snippet creates a Choice object and adds some items to it. Then, it gets the item at index 1 and prints it out. 

          
           
Choice choice = new Choice();
choice.add("Item 1");
choice.add("Item 2");
choice.add("Item 3");

String itemAtIndexOne = choice.getItem(1);
System.out.println(itemAtIndexOne);
The output will be: 

          
           
Item 2

In this article, you have learned how to get a particular item of an AWT Choice or Combobox object in Java. You can use the getSelectedItem() and getItem(int index) methods to achieve this.
